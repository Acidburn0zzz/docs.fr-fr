# <a name="getting-started-with-f"></a>Bien démarrer avec F# #

Pour bien démarrer avec F#, trois méthodes principales s’offrent à vous :

Si vous êtes sur Windows, consultez [Bien démarrer avec F# dans Visual Studio](getting-started-visual-studio.md).

Si vous souhaitez écrire du code dans un IDE léger sur Windows, Linux ou macOS, consultez [Bien démarrer avec F# dans Visual Studio Code avec Ionide](getting-started-vscode.md).

Si vous préférez utiliser la ligne de commande sur un système d’exploitation, quel qu’il soit, consultez [Bien démarrer avec F# à l’aide des outils en ligne de commande](getting-started-command-line.md).
