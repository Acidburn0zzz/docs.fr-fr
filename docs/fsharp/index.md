# <a name="f-guide"></a>Guide F#

F# est un langage de programmation multiplateforme et open source pour .NET qui offre une prise en charge de première classe de la programmation fonctionnelle. F# prend également en charge la programmation orientée objet et impérative.  Grâce au compilateur et aux outils Visual F# implémentés par Microsoft, le langage de programmation F# fait partie intégrante de .NET.

## <a name="if-youre-new-to-f"></a>Si vous débutez avec F # #

Si vous utilisez F #, commencer par le [visite guidée de F #](tour.md) pour obtenir une vue d’ensemble du langage.

Il est également recommandé que vous parcourez les [fonctionne comme valeurs de première classe](introduction-to-functional-programming/functions-as-first-class-values.md) <!--[Introduction to Functional Progamming](introduction-to-functional-programming/index.md)--> pour apprendre les concepts de programmation fonctionnelle qui sont essentielles à l’utilisation de F #.

Les [didacticiels](tutorials/getting-started/index.md) comprennent également des guides pas à pas couvrant les fonctionnalités du langage pour différents niveaux de compétence.

## <a name="if-youre-experienced-with-f"></a>Si vous êtes familiarisé avec F # #

Si vous utilisez régulièrement F#, les [informations de référence sur le langage](language-reference/index.md) vous seront d’une grande utilité. Cette rubrique présente de manière approfondie chaque aspect du langage et propose de nombreux exemples de code.  Vous trouverez aussi beaucoup à découvrir dans les [informations de référence sur la bibliothèque principale F#](https://msdn.microsoft.com/visualfsharpdocs/conceptual/fsharp-core-library-reference).  La référence de bibliothèque F # Core déplacera finalement en s’éloignant de MSDN et dans ces documents en cours.

## <a name="the-f-software-foundation"></a>F# Software Foundation

Bien que Microsoft soit le principal développeur du langage F# et des outils Visual F#, F# bénéficie également de l’appui d’une organisation indépendante, la FSSF (F# Software Foundation).

La FSSF a pour but non seulement de promouvoir, protéger et développer le langage de programmation F#, mais aussi de soutenir et d’étendre la communauté internationale et diversifiée de programmeurs F#.

Pour en savoir plus et vous impliquer, consultez [fsharp.org](http://fsharp.org).
