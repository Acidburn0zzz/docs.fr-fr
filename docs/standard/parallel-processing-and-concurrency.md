# <a name="parallel-processing-concurrency-and-async-programming-in-net"></a>Traitement parallèle, accès concurrentiel et programmation asynchrone dans .NET
.NET offre plusieurs façons d’écrire du code asynchrone permettant de rendre votre application plus réactive pour l’utilisateur, et d’écrire du code parallèle qui utilise plusieurs threads d’exécution afin d’optimiser les performances de l’ordinateur de votre utilisateur.  
  
## <a name="in-this-section"></a>Dans cette section  
 [Programmation asynchrone](../../docs/standard/async.md)  
 Décrit les mécanismes de la programmation asynchrone fournis par .NET.  
  
 [Programmation parallèle](../../docs/standard/parallel-programming/index.md)  
 Décrit un modèle de programmation basée sur les tâches qui simplifie le développement parallèle, vous permettant d’écrire du code parallèle efficace, précis et scalable dans un idiome naturel sans devoir utiliser directement des threads ou le pool de threads.  

 [Thread](../../docs/standard/threading/index.md)  
 Décrit les mécanismes de base de l’accès concurrentiel et de la synchronisation fournis par .NET.
