# <a name="introduction-tothe-docker-application-life-cycle"></a>Introduction au cycle de vie d’application Docker
 
Le cycle de vie des applications en conteneur naît de la réflexion du développeur. Le développeur choisit d’implémenter des conteneurs et Docker pour éliminer les frictions dans les déploiements et les opérations informatiques, contribuant ainsi à accroître l’agilité, la productivité de bout en bout et la rapidité d’exécution de tous les intervenants.
