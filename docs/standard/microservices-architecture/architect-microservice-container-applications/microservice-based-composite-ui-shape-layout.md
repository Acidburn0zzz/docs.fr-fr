<span data-ttu-id="e2356-114">En revanche, une interface utilisateur composite est générée avec précision et composée par le microservices eux-mêmes.</span><span class="sxs-lookup"><span data-stu-id="e2356-114">In contrast, a composite UI is precisely generated and composed by the microservices themselves.</span></span> <span data-ttu-id="e2356-115">Parmi les microservices définissent la forme visual des zones spécifiques de l’interface utilisateur.</span><span class="sxs-lookup"><span data-stu-id="e2356-115">Some of the microservices drive the visual shape of specific areas of the UI.</span></span> <span data-ttu-id="e2356-116">La principale différence est que vous disposez des composants d’interface utilisateur client (classes TS, par exemple) basées sur des modèles et le ViewModel UI mise en forme des données de ces modèles proviennent de chaque microservice.</span><span class="sxs-lookup"><span data-stu-id="e2356-116">The key difference is that you have client UI components (TS classes, for example) based on templates, and the data-shaping-UI ViewModel for those templates comes from each microservice.</span></span>
