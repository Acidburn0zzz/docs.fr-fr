# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Choix entre .NET Core et .NET Framework pour les conteneurs Docker

Il existe deux frameworks pris en charge pour générer des applications Docker en conteneur côté serveur avec .NET : [.NET Framework et .NET Core](https://www.microsoft.com/net/download). Ils partagent de nombreux composants de plateforme .NET et vous permettent de partager du code entre les deux. Toutefois, il existe des différences fondamentales entre eux et votre choix dépend de ce que vous souhaitez accomplir. Cette section fournit des instructions pour éclairer le choix de chaque framework.
