# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Choix entre .NET Core et .NET Framework pour les conteneurs Docker

Il existe deux implémentations prises en charge pour générer des applications Docker en conteneur côté serveur avec .NET : [.NET Framework](https://www.microsoft.com/net/download/framework) et [.NET Core](https://www.microsoft.com/net/download/core). Elles partagent de nombreux composants .NET Standard et vous permettent de partager du code entre les deux. Toutefois, il existe des différences fondamentales entre elles et votre choix dépend de ce que vous souhaitez accomplir. Cette section fournit des instructions pour éclairer le choix de chaque implémentation.
