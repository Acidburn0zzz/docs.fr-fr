# <a name="removing-nodes-content-and-values-from-an-xml-document"></a>Suppression de nœuds, de contenu et de valeurs d'un document XML
Une fois qu’un DOM (Document Object Model) XML est en mémoire, vous pouvez supprimer des nœuds de l’arborescence ou supprimer du contenu et des valeurs de certains types de nœuds. Pour plus d’informations sur la façon de supprimer un nœud sans descendant ou une sous-arborescence entière d’un document, consultez [suppression de nœuds du DOM](../../../../docs/standard/data/xml/removing-nodes-from-the-dom.md). Pour plus d’informations sur la façon de supprimer des attributs sur un élément, consultez [suppression d’attributs d’un nœud d’élément dans le DOM](../../../../docs/standard/data/xml/removing-attributes-from-an-element-node-in-the-dom.md). Pour plus d’informations sur la suppression du contenu d’un nœud, mais en laissant le nœud dans l’arborescence, consultez [suppression du nœud contenu dans le DOM](../../../../docs/standard/data/xml/removing-node-content-in-the-dom.md).
