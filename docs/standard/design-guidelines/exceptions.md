# Instructions de conception pour les Exceptions
La gestion des exceptions présente de nombreux avantages sur le rapport d’erreurs basé sur la valeur de retour. Conception de bonne infrastructure permet de tirer parti des exceptions au développeur d’applications. Cette section présente les avantages des exceptions et présente des conseils pour les utiliser efficacement.
