# Using TPL with Other Asynchronous Patterns
La bibliothèque parallèle de tâches peut être utilisée avec les modèles traditionnels de programmation asynchrone .NET Framework de plusieurs façons.  
  
## Dans cette section  
 [TPL and Traditional .NET Framework Asynchronous Programming](../../../docs/standard/parallel-programming/tpl-and-traditional-async-programming.md)  
 Décrit comment les objets <xref:System.Threading.Tasks.Task> peuvent être utilisés conjointement aux modèles \(APM\) de programmation asynchrone et au modèle asynchrone basé sur les événements.  
  
 [How to: Wrap EAP Patterns in a Task](../../../docs/standard/parallel-programming/how-to-wrap-eap-patterns-in-a-task.md)  
 Indique comment utiliser des objets <xref:System.Threading.Tasks.Task> pour encapsuler des modèles asynchrones basés sur des événements.
