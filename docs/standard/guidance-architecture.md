# <a name="net-architecture-guidance"></a>Guide d’architecture .NET

## <a name="containerized-docker-application-lifecycle-with-the-microsoft-platform-and-toolscontainerized-lifecycle-architectureindexmd"></a>[Cycle de vie des applications en conteneur Docker avec la plate-forme Microsoft et les outils](./containerized-lifecycle-architecture/index.md)

Ce guide est une introduction aux processus de bout en bout du cycle de vie recommandée que vous utiliserez pour développer, valider et déployer des applications en conteneur Docker à l’aide de Visual Studio et Microsoft Azure.

## <a name="modernize-existing-net-applications-with-azure-cloud-and-windows-containersmodernize-with-azure-and-containersindexmd"></a>[Moderniser des Applications .NET existantes avec Azure cloud et les conteneurs Windows](./modernize-with-azure-and-containers/index.md)

Ce guide est une introduction aux stratégies que vous devez migrer les applications web existantes pour le cloud Azure et les conteneurs Windows. Vous allez en savoir plus sur les stratégies de code, la migration des données, orchestrators et traite de l’élément de configuration/CD.

## <a name="architect-modern-web-applications-with-aspnet-core-and-azuremodern-web-apps-azure-architectureindexmd"></a>[Architecture des applications web avec ASP.NET Core et Azure](modern-web-apps-azure-architecture/index.md)

Ce guide est une introduction à l’architecture recommandée, la conception et le processus de déploiement que vous utiliserez pour créer des applications ASP.NET et ASP.NET Core et héberger ces applications dans Azure.

## <a name="architecting-container-and-microservice-based-applicationsmicroservices-architectureindexmd"></a>[Architecture d’applications conteneur et basées sur des microservices](microservices-architecture/index.md)

Ce guide est une introduction au développement d’applications basées sur les microservices et à la gestion de celles-ci au moyen de conteneurs. Il traite de la conception architecturale et des approches d’implémentation utilisant .NET Core et les conteneurs Docker.
