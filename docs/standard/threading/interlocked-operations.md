# <a name="interlocked-operations"></a><span data-ttu-id="dbe37-102">Opérations verrouillées</span><span class="sxs-lookup"><span data-stu-id="dbe37-102">Interlocked Operations</span></span>

 <span data-ttu-id="dbe37-107">La classe <xref:System.Threading.Interlocked> fournit les opérations suivantes :</span><span class="sxs-lookup"><span data-stu-id="dbe37-107">The <xref:System.Threading.Interlocked> class provides the following operations:</span></span>  
  
-   <span data-ttu-id="dbe37-108">Dans .NET Framework version 2.0, la méthode <xref:System.Threading.Interlocked.Add%2A> ajoute une valeur entière à une variable et retourne la nouvelle valeur de la variable.</span><span class="sxs-lookup"><span data-stu-id="dbe37-108">In the .NET Framework version 2.0, the <xref:System.Threading.Interlocked.Add%2A> method adds an integer value to a variable and returns the new value of the variable.</span></span>  
  
-   <span data-ttu-id="dbe37-109">Dans .NET Framework version 2.0, la méthode <xref:System.Threading.Interlocked.Read%2A> lit une valeur entière 64 bits sous forme d’opération atomique.</span><span class="sxs-lookup"><span data-stu-id="dbe37-109">In the .NET Framework version 2.0, the <xref:System.Threading.Interlocked.Read%2A> method reads a 64-bit integer value as an atomic operation.</span></span> <span data-ttu-id="dbe37-110">Cela est utile sur les systèmes d’exploitation 32 bits, où la lecture d’un entier 64 bits n’est en général pas une opération atomique.</span><span class="sxs-lookup"><span data-stu-id="dbe37-110">This is useful on 32-bit operating systems, where reading a 64-bit integer is not ordinarily an atomic operation.</span></span>  
  
-   <span data-ttu-id="dbe37-111">Les méthodes <xref:System.Threading.Interlocked.Increment%2A> et <xref:System.Threading.Interlocked.Decrement%2A> incrémentent ou décrémentent une variable et retournent la valeur résultante.</span><span class="sxs-lookup"><span data-stu-id="dbe37-111">The <xref:System.Threading.Interlocked.Increment%2A> and <xref:System.Threading.Interlocked.Decrement%2A> methods increment or decrement a variable and return the resulting value.</span></span>  
  
-   <span data-ttu-id="dbe37-112">La méthode <xref:System.Threading.Interlocked.Exchange%2A> exécute un échange atomique de la valeur dans une variable spécifiée, en renvoyant la valeur et en la remplaçant par une nouvelle valeur.</span><span class="sxs-lookup"><span data-stu-id="dbe37-112">The <xref:System.Threading.Interlocked.Exchange%2A> method performs an atomic exchange of the value in a specified variable, returning that value and replacing it with a new value.</span></span> <span data-ttu-id="dbe37-113">Dans .NET Framework version 2.0, une surcharge générique de cette méthode peut être utilisée pour exécuter cet échange sur une variable de tout type de référence.</span><span class="sxs-lookup"><span data-stu-id="dbe37-113">In the .NET Framework version 2.0, a generic overload of this method can be used to perform this exchange on a variable of any reference type.</span></span> <span data-ttu-id="dbe37-114">Consultez <xref:System.Threading.Interlocked.Exchange%60%601%28%60%600%40%2C%60%600%29>.</span><span class="sxs-lookup"><span data-stu-id="dbe37-114">See <xref:System.Threading.Interlocked.Exchange%60%601%28%60%600%40%2C%60%600%29>.</span></span>  
  
-   <span data-ttu-id="dbe37-115">La méthode <xref:System.Threading.Interlocked.CompareExchange%2A> échange également deux valeurs, mais dépend du résultat d’une comparaison.</span><span class="sxs-lookup"><span data-stu-id="dbe37-115">The <xref:System.Threading.Interlocked.CompareExchange%2A> method also exchanges two values, but contingent on the result of a comparison.</span></span> <span data-ttu-id="dbe37-116">Dans .NET Framework version 2.0, une surcharge générique de cette méthode peut être utilisée pour exécuter cet échange sur une variable de tout type de référence.</span><span class="sxs-lookup"><span data-stu-id="dbe37-116">In the .NET Framework version 2.0, a generic overload of this method can be used to perform this exchange on a variable of any reference type.</span></span> <span data-ttu-id="dbe37-117">Consultez <xref:System.Threading.Interlocked.CompareExchange%60%601%28%60%600%40%2C%60%600%2C%60%600%29>.</span><span class="sxs-lookup"><span data-stu-id="dbe37-117">See <xref:System.Threading.Interlocked.CompareExchange%60%601%28%60%600%40%2C%60%600%2C%60%600%29>.</span></span>  
  
 <span data-ttu-id="dbe37-118">Sur les processeurs modernes, les méthodes de la classe <xref:System.Threading.Interlocked> peuvent souvent être implémentées par une seule instruction.</span><span class="sxs-lookup"><span data-stu-id="dbe37-118">On modern processors, the methods of the <xref:System.Threading.Interlocked> class can often be implemented by a single instruction.</span></span> <span data-ttu-id="dbe37-119">Par conséquent, elles fournissent d’excellentes performances de synchronisation et peuvent être utilisées pour générer des mécanismes de synchronisation de niveau supérieur, comme des verrous de rotation.</span><span class="sxs-lookup"><span data-stu-id="dbe37-119">Thus, they provide very high-performance synchronization and can be used to build higher-level synchronization mechanisms, like spin locks.</span></span>
