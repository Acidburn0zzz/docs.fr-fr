# Op&#233;rations de cha&#238;nes de base dans .NET Framework
Les applications répondent souvent aux utilisateurs en construisant des messages sur la base de l'information saisie par lui.  Par exemple, il n'est pas inhabituel que les sites Web répondent à un utilisateur récemment connecté par un message d'accueil spécialisé reprenant le nom de l'utilisateur.  Plusieurs méthodes dans les classes <xref:System.String?displayProperty=fullName> et <xref:System.Text.StringBuilder?displayProperty=fullName> vous permettent de construire de façon dynamique des chaînes personnalisées à afficher dans votre interface utilisateur.  Ces méthodes vous aident également à réaliser un certain nombre d'opérations de chaînes de base telles que la création de nouvelles chaînes à partir de tableaux d'octets, la comparaison des valeurs des chaînes et la modification de chaînes existantes.
