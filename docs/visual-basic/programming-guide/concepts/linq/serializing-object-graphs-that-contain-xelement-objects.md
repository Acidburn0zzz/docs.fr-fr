# <a name="serializing-object-graphs-that-contain-xelement-objects-visual-basic"></a>Sérialisation de graphiques d’objets qui contiennent des objets XElement (Visual Basic)
Cette rubrique présente la fonctionnalité de sérialisation de graphiques d’objets qui contiennent des références aux objets de type <xref:System.Xml.Linq.XElement>.</xref:System.Xml.Linq.XElement> Afin de faciliter ce type de sérialisation, <xref:System.Xml.Linq.XElement>implémente le <xref:System.Xml.Serialization.IXmlSerializable>interface.</xref:System.Xml.Serialization.IXmlSerializable> </xref:System.Xml.Linq.XElement>  
  
 Notez que seul le <xref:System.Xml.Linq.XElement>classe implémente la sérialisation.</xref:System.Xml.Linq.XElement>
