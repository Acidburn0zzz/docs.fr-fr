# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Axes intégrés au langage en Visual Basic (LINQ to XML)
Cette section décrit les fonctionnalités intégrées directement dans le [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] language pour en faciliter l’accès au code XML. La plupart des exemples de la documentation LINQ to XML utilisent ces axes [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] intégrés.  
  
## <a name="in-this-section"></a>Dans cette section  
  
|Rubrique|Description|  
|-----------|-----------------|  
|[Propriété d’axe enfant XML](../../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|Fournit un accès aux enfants de l'un des éléments suivants : un objet <xref:System.Xml.Linq.XElement>, un objet <xref:System.Xml.Linq.XDocument>, une collection d'objets <xref:System.Xml.Linq.XElement> ou une collection d'objets <xref:System.Xml.Linq.XDocument>. Cet axe est équivalent à l'axe <xref:System.Xml.Linq.XContainer.Elements%2A>.|  
|[Propriété d’axe descendant XML](../../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|Fournit un accès aux descendants de l'un des éléments suivants : un objet <xref:System.Xml.Linq.XElement>, un objet <xref:System.Xml.Linq.XDocument> ou une collection d'objets <xref:System.Xml.Linq.XElement>. Cet axe est équivalent à l'axe <xref:System.Xml.Linq.XContainer.Descendants%2A>.|  
|[Propriété d’axe d’attribut XML](../../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|Fournit un accès à un attribut d'un objet <xref:System.Xml.Linq.XElement>. Cet axe est à peu près équivalent à l'axe <xref:System.Xml.Linq.XElement.Attribute%2A>. Cet axe diffère de l'axe <xref:System.Xml.Linq.XElement.Attribute%2A>, dans le sens où il retourne la valeur de l'attribut, et non un objet <xref:System.Xml.Linq.XAttribute>.|  
|[Propriété d’indexeur d’extension](../../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|Fournit un accès aux différents éléments d’une collection.|
