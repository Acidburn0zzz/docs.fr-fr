# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML fournit une interface de programmation XML en mémoire qui exploite le cadre .NET Language-Integrated Query (LINQ). LINQ to XML utilise les dernières fonctionnalités du langage .NET Framework et est comparable à une interface de programmation XML du modèle DOM (Document objet Model) mis à jour et repensée.  
  
 La famille LINQ de technologies fournit une expérience de requête cohérente pour les objets (LINQ to Objects), les bases de données relationnelles (LINQ to SQL) et XML (LINQ to XML).
