# <a name="programming-guide-linq-to-xml-visual-basic"></a>Guide de programmation (LINQ to XML) (Visual Basic)
Cette section fournit des informations conceptuelles et de procédure sur la programmation avec [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)].  
  
## <a name="who-should-read-this-documentation"></a>À qui cette documentation est-elle destinée ?  
 Cette documentation est destinée aux développeurs qui connaissent déjà Visual Basic et certains aspects de base du [!INCLUDE[dnprdnshort](../../../../csharp/getting-started/includes/dnprdnshort_md.md)].  
  
 L’objectif de cette documentation est de rendre [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] facile à utiliser pour tous les types de développeurs. [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)]facilite la programmation XML. Son utilisation n'est pas restreinte aux développeurs expérimentés.  
  
 [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)]s’appuie fortement sur les classes génériques. Par conséquent, il est très important de comprendre l’utilisation des classes et des types génériques. Pour plus d’informations, consultez [des Types génériques en Visual Basic](../../../../visual-basic/programming-guide/language-features/data-types/generic-types.md).
