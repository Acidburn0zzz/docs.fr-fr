# <a name="linq-and-reflection-visual-basic"></a>LINQ et réflexion (Visual Basic)
Les API de réflexion bibliothèque .NET Framework classe peut être utilisé pour examiner les métadonnées dans un assembly .NET et créer des collections de types, les membres de type, les paramètres, etc. qui se trouvent dans cet assembly. Étant donné que ces collections prennent en charge le type générique `IEnumerable` interface, elles peuvent être interrogées à l’aide de LINQ.
