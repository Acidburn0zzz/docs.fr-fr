# <a name="program-structure-and-code-conventions-visual-basic"></a>Structure de programme et conventions de codage (Visual Basic)
Cette section présente le type [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] structure du programme, fournit un simple [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] du programme, « Hello, World » et présente [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] conventions de code. Conventions de code sont des suggestions qui reposent pas sur la logique d’un programme mais sur sa structure physique et son apparence. Suivant les rend plus faciles à lire, comprendre et gérer votre code. Conventions de code incluent, entre autres :  
  
-   Formats normalisés pour l’étiquetage et les commentaires de code.  
  
-   Instructions pour l’espacement, la mise en forme et la mise en retrait du code.  
  
-   Conventions d’affectation de noms pour les objets, des variables et des procédures.  
  
 Les rubriques suivantes présentent un ensemble de la programmation des recommandations pour [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] programmes, ainsi que des exemples d’utilisation appropriée.
