## <a name="arguments"></a><span data-ttu-id="c7f12-118">Arguments</span><span class="sxs-lookup"><span data-stu-id="c7f12-118">Arguments</span></span>  
 <span data-ttu-id="c7f12-119">Un *argument* représente la valeur que vous passez à un paramètre de procédure lorsque vous appelez la procédure.</span><span class="sxs-lookup"><span data-stu-id="c7f12-119">An *argument* represents the value that you pass to a procedure parameter when you call the procedure.</span></span> <span data-ttu-id="c7f12-120">Le code appelant fournit les arguments lorsqu’il appelle la procédure.</span><span class="sxs-lookup"><span data-stu-id="c7f12-120">The calling code supplies the arguments when it calls the procedure.</span></span>  
  
 <span data-ttu-id="c7f12-121">Lorsque vous appelez un `Function` ou `Sub` procédure, vous incluez un *liste d’arguments* entre parenthèses le nom de la procédure qui suit immédiatement.</span><span class="sxs-lookup"><span data-stu-id="c7f12-121">When you call a `Function` or `Sub` procedure, you include an *argument list* in parentheses immediately following the procedure name.</span></span> <span data-ttu-id="c7f12-122">Chaque argument correspond au paramètre dans la même position dans la liste.</span><span class="sxs-lookup"><span data-stu-id="c7f12-122">Each argument corresponds to the parameter in the same position in the list.</span></span>  
  
 <span data-ttu-id="c7f12-123">Contrairement à la définition des paramètres, arguments n’ont pas de noms.</span><span class="sxs-lookup"><span data-stu-id="c7f12-123">In contrast to parameter definition, arguments do not have names.</span></span> <span data-ttu-id="c7f12-124">Chaque argument est une expression qui peut contenir zéro ou plusieurs variables, de constantes et des littéraux.</span><span class="sxs-lookup"><span data-stu-id="c7f12-124">Each argument is an expression, which can contain zero or more variables, constants, and literals.</span></span> <span data-ttu-id="c7f12-125">Le type de données de l’expression évaluée doit correspondre en général, le type de données défini pour le paramètre correspondant, et dans tous les cas, il doit être convertible au type de paramètre.</span><span class="sxs-lookup"><span data-stu-id="c7f12-125">The data type of the evaluated expression should typically match the data type defined for the corresponding parameter, and in any case it must be convertible to the parameter type.</span></span>
