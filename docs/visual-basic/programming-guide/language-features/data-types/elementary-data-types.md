# <a name="elementary-data-types-visual-basic"></a>Types de données élémentaires (Visual Basic)
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]fournit un jeu de types de données prédéfinis, vous pouvez utiliser pour la plupart de vos éléments de programmation. Cette section décrit ces types et comment les utiliser.  
  
> [!NOTE]
>  Chaque type de données élémentaire en Visual Basic est prise en charge par une structure ou une classe qui se trouve dans le <xref:System> espace de noms. Le compilateur utilise chaque mot clé type de données en tant qu’alias pour la structure sous-jacente ou de la classe. Par exemple, déclarer une variable en utilisant le mot réservé `Byte` est identique à la déclarer en utilisant le nom de structure qualifié <xref:System.Byte?displayProperty=nameWithType>.
