# <a name="creating-xml-in-visual-basic"></a>Création de code XML dans Visual Basic
[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]Vous pouvez utiliser *littéraux XML* directement dans votre code. La syntaxe de littéral XML représente [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] objets, qui est similaire à la syntaxe XML 1.0. Cela rend plus facile de créer par programme des éléments, des documents et des fragments XML, car votre code possède la même structure que le XML final.
