# <a name="creating-xml-in-visual-basic"></a>Création de code XML dans Visual Basic
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]Vous pouvez ainsi utiliser *littéraux XML* directement dans votre code. La syntaxe de littéral XML représente [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] objets et il est similaire à la syntaxe XML 1.0. Cela rend plus facile de créer par programmation des éléments, des documents et des fragments XML, car votre code possède la même structure que le XML final.
