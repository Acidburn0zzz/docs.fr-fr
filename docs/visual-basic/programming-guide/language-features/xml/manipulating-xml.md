# <a name="manipulating-xml-in-visual-basic"></a>Manipulation de code XML dans Visual Basic
Vous pouvez utiliser *littéraux XML* charger XML à partir d’une source externe comme une chaîne, un fichier ou un flux. Vous pouvez ensuite utiliser [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] pour manipuler le fichier XML et utiliser [!INCLUDE[vbteclinqext](../../../../csharp/getting-started/includes/vbteclinqext_md.md)] pour interroger le document XML.
