# <a name="visual-basic-intellisense-code-snippets"></a>Extraits de code IntelliSense Visual Basic
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]inclut une bibliothèque de code composée de plusieurs centaines de segments de code, appelés extraits de code IntelliSense, qui sont prêts à être insérés dans votre application. Chaque extrait de code exécute une tâche de programmation complète comme la création d’une exception personnalisée, envoyer un message électronique ou dessiner un cercle. Vous pouvez insérer un extrait de code dans votre code source en quelques clics de souris. Vous pouvez également créer vos propres extraits de code adapté à vos besoins, les ajouter à la bibliothèque, puis les utiliser si nécessaire. Pour plus d’informations, consultez [Extraits de code](https://docs.microsoft.com/visualstudio/ide/code-snippets).  
  
## <a name="a-productivity-tool-for-visual-basic-users"></a>Un outil de productivité pour les utilisateurs de Visual Basic  
 Ces extraits de code augmentent votre productivité en réduisant la quantité de temps passé à rechercher des exemples de code, en réduisant le temps nécessaire pour apprendre à utiliser une fonctionnalité peu familière et en prenant en charge la réutilisation du code.  
  
 La bibliothèque de code prend en charge les fonctionnalités suivantes :  
  
-   Insérer l’un des extraits de code dans le **éditeur de Code**.  
  
-   Créer des tâches que vous pouvez réutiliser dans vos projets.  
  
-   Créer de nouvelles tâches à partager avec votre groupe de travail et de vos collègues.  
  
-   Modifier les tâches.  
  
-   Télécharger d’autres tâches auprès de fournisseurs tiers.  
  
 Ces blocs de code sont disponibles tout au long de [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]:  
  
-   Vous pouvez les ajouter via le menu contextuel de la **éditeur de Code**.  
  
-   Vous pouvez faire glisser les fichiers de code XML à partir de **Explorateur de fichiers** sur votre fichier de code source.
