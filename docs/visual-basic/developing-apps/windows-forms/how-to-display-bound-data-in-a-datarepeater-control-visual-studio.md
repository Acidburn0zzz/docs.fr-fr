### <a name="to-create-a-data-bound-datarepeater"></a><span data-ttu-id="800c2-109">Pour créer un contrôle DataRepeater lié aux données</span><span class="sxs-lookup"><span data-stu-id="800c2-109">To create a data-bound DataRepeater</span></span>  
  
1.  <span data-ttu-id="800c2-110">Faites glisser un <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> contrôle depuis la **Visual Basic PowerPacks** onglet dans le **boîte à outils** à un contrôle de formulaire ou un conteneur.</span><span class="sxs-lookup"><span data-stu-id="800c2-110">Drag a <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control from the **Visual Basic PowerPacks** tab in the **Toolbox** to a form or container control.</span></span>  
  
2.  <span data-ttu-id="800c2-111">Faites glisser les poignées de dimensionnement et de position pour dimensionner et positionner le contrôle.</span><span class="sxs-lookup"><span data-stu-id="800c2-111">Drag the sizing and position handles to size and position the control.</span></span>  
  
     <span data-ttu-id="800c2-112">Notez que le contrôle a les deux régions rectangulaires.</span><span class="sxs-lookup"><span data-stu-id="800c2-112">Note that the control has two rectangular regions.</span></span> <span data-ttu-id="800c2-113">Région du haut est la *modèle d’élément*; les contrôles ajoutés au modèle seront répétés dans chaque élément dans le <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> contrôle au moment de l’exécution.</span><span class="sxs-lookup"><span data-stu-id="800c2-113">The upper region is the *item template*; controls added to the template will be repeated in each item in the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control at run time.</span></span> <span data-ttu-id="800c2-114">La région inférieure est la *viewport*, où les éléments seront affichés.</span><span class="sxs-lookup"><span data-stu-id="800c2-114">The lower region is the *viewport*, where the items will be displayed.</span></span>  
  
     <span data-ttu-id="800c2-115">Vous pouvez également dimensionner et positionner le contrôle ou le modèle d’élément en modifiant le **taille** et **Position** propriétés dans la fenêtre Propriétés.</span><span class="sxs-lookup"><span data-stu-id="800c2-115">You can also size and position the control or the item template by changing the **Size** and **Position** properties in the Properties window.</span></span>  
  
3.  <span data-ttu-id="800c2-116">Dans le menu **Données** , cliquez sur **Afficher les sources de données**.</span><span class="sxs-lookup"><span data-stu-id="800c2-116">On the **Data** menu, click **Show Data Sources**.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="800c2-117">Si le **des Sources de données** fenêtre est vide, ajoutez une source de données à ce dernier.</span><span class="sxs-lookup"><span data-stu-id="800c2-117">If the **Data Sources** window is empty, add a data source to it.</span></span> <span data-ttu-id="800c2-118">Pour plus d’informations, consultez [Ajouter de nouvelles sources de données](/visualstudio/data-tools/add-new-data-sources).</span><span class="sxs-lookup"><span data-stu-id="800c2-118">For more information, see [Add new data sources](/visualstudio/data-tools/add-new-data-sources).</span></span>  
  
4.  <span data-ttu-id="800c2-119">Dans le **des Sources de données** fenêtre, sélectionnez le nœud de niveau supérieur pour la table qui contient les données que vous voulez lier.</span><span class="sxs-lookup"><span data-stu-id="800c2-119">In the **Data Sources** window, select the top-level node for the table that contains the data that you want to bind.</span></span>  
  
5.  <span data-ttu-id="800c2-120">Modifier le type de déplacement de la table à `Details` en cliquant sur `Details` dans la liste déroulante sur le nœud de la table.</span><span class="sxs-lookup"><span data-stu-id="800c2-120">Change the drop type of the table to `Details` by clicking `Details` in the drop-down list on the table node.</span></span>  
  
6.  <span data-ttu-id="800c2-121">Sélectionnez le nœud de la table et faites-le glisser vers la région du modèle d’élément de la <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> contrôle.</span><span class="sxs-lookup"><span data-stu-id="800c2-121">Select the table node and drag it onto the item template region of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span>  
  
     <span data-ttu-id="800c2-122">Vous pouvez spécifier des types de contrôles sont affichés pour chaque champ.</span><span class="sxs-lookup"><span data-stu-id="800c2-122">You can specify which types of controls are displayed for each field.</span></span> <span data-ttu-id="800c2-123">Pour plus d’informations, consultez [définir le contrôle à créer lors du déplacement de la fenêtre Sources de données](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span><span class="sxs-lookup"><span data-stu-id="800c2-123">For more information, see [Set the control to be created when dragging from the Data Sources window](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span></span>
