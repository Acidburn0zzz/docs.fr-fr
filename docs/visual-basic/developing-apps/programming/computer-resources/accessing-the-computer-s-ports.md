# <a name="accessing-the-computer39s-ports-visual-basic"></a>Accès aux ports de l’ordinateur (Visual Basic)
L’objet `My.Computer.Ports` fournit une propriété et une méthode pour accéder aux ports série de l’ordinateur.

 [Guide pratique : recevoir des chaînes des ports série](../../../../visual-basic/developing-apps/programming/computer-resources/how-to-receive-strings-from-serial-ports.md)
