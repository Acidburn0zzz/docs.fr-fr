# <a name="unable-to-apply-security-attributes-to-39ltnamegt39-lterrorgt"></a>Impossible d’appliquer des attributs de sécurité à '&lt;nom&gt;' : &lt;erreur&gt;
Une erreur s’est produite pendant l’application des attributs de sécurité. Cette erreur peut provenir de l’utilisation incorrecte d’un attribut de sécurité ou indiquer un problème avec le compilateur.  
  
 **ID d’erreur :** BC31520  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez la syntaxe et les arguments de l’attribut.  
  
2.  Recompilez le programme pour voir si l’erreur se produit à nouveau.  
  
3.  Si l'erreur se produit encore, enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
4.  Si l’erreur se reproduit, réinstallez [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
5.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
