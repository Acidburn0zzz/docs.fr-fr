# <a name="comma-or--expected-type-argument-list"></a>Virgule ou ')' attendue (liste d’arguments de type)
Une liste d’arguments de type contient un élément inattendu. Utilisez des virgules (`,`) pour séparer les arguments de type dans ce genre de liste, et utilisez une parenthèse fermante (`)`) à la fin de la liste.  
  
 **ID d’erreur :** BC32094  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une virgule et continuez la liste d’arguments de type, ou ajoutez un caractère de parenthèse fermante pour terminer la liste.
