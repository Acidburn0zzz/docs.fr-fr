# <a name="type-character-ltcharactergt-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Caractère de type '&lt;caractère&gt;' ne peut pas être utilisé dans une déclaration avec un type explicite
Vous avez utilisé un caractère de type, tel que `$`, en spécifiant explicitement un type.  
  
 **ID d’erreur :** BC30302  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le caractère de type ou supprimez la clause `As <Type>` de la déclaration.
