# <a name="cannot-convert-start-value-of-type-39lttype1gt39-limit-value-of-39lttype2gt39-and-step-value-of-39lttype3gt39-to-a-common-type"></a>Impossible de convertir le valeur de départ de type &#39; &lt;type1&gt;&#39;, valeur de limite de &#39; &lt;type2&gt; &#39; et la valeur de pas de &#39; &lt;type3&gt; &#39; à un type commun
Les valeurs de contrôle `For...Next` spécifiées ont des types différents et ne sont donc pas valides, car elles ne peuvent pas être converties en type commun.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Modifiez le type d’une ou de plusieurs valeurs pour que toutes les valeurs puissent être converties en type commun.
