# <a name="39ltmembergt39-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>«&lt;membre&gt;' est en conflit avec le membre réservé par ce nom qui est implicitement déclaré dans tous les enums
Le nom d’un membre de type est en conflit avec le nom d’un membre implicitement déclaré dans toutes les énumérations.  
  
 Le [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilateur crée des membres implicites correspondant à certains éléments de programmation que vous déclarez. Les énumérations déclarent implicitement le membre `value__member`.  
  
 **ID d’erreur :** BC31420  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez le nom du membre.
