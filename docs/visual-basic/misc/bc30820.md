# <a name="39ltlsetstmtgt39-is-not-declared"></a>«&lt;lsetstmt&gt;' n’est pas déclaré
«\<lsetstmt >' n’est pas déclaré. Les instructions LSet ne sont plus prises en charge ; utilisez Microsoft.VisualBasic.LSet à la place.  
  
 Plusieurs fonctions qui étaient intrinsèques à [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] dans les versions précédentes ont été déplacés vers le <xref:Microsoft.VisualBasic?displayProperty=fullName>espace de noms.</xref:Microsoft.VisualBasic?displayProperty=fullName> Leurs fonctionnalités sont ainsi plus généralement accessibles à tous les langages de programmation.  
  
 **ID d’erreur :** BC30820  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez plutôt la fonction `LSet` dans l’espace de noms `Microsoft.VisualBasic` .
