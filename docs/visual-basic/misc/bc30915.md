# <a name="39ltnamegt39-cannot-expose-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-39lttypegt39"></a>&#39; &lt;nom&gt;&#39; ne peut pas exposer le type délégué sous-jacent &#39;&lt; type_délégué&gt;&#39; de l’événement qu’il implémente en dehors du projet via &lt;spécificateur&gt; &#39;&lt; type de&gt;&#39;
Le code expose un type privé en dehors de la classe publique.  
  
 **ID d’erreur :** BC30915  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez le type `Public`.  
  
     - ou -  
  
-   Utilisez un autre type.
