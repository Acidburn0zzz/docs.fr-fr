# <a name="ltnamegt-cannot-expose-the-underlying-delegate-type-ltdelegatetypegt-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-lttypegt"></a>«&lt;nom&gt;'ne peut pas exposer le type délégué sous-jacent'&lt;type_délégué&gt;' de l’événement qu’il implémente en dehors du projet via &lt;spécificateur&gt; '&lt;type &gt;'
Le code expose un type privé en dehors de la classe publique.  
  
 **ID d’erreur :** BC30915  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez le type `Public`.  
  
     ou  
  
-   Utilisez un autre type.
