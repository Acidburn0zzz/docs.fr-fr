# <a name="method-39ltmethodnamegt39-cannot-handle-event-39lteventnamegt39-because-they-do-not-have-the-same-signature"></a>Méthode '&lt;methodname&gt;« ne peut pas gérer les événements »&lt;eventname&gt;» car ils n’ont pas la même signature
Les paramètres ou types de paramètre du gestionnaire d’événements spécifié ne correspondent pas à ceux définis pour l’événement spécifié.  
  
 **ID d’erreur :** BC31029  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Redéfinissez l’événement ou le gestionnaire d’événements pour que les listes de paramètres correspondent.
