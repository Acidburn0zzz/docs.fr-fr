# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-39optional39"></a>Valeurs par défaut ne peut pas être fournies pour les paramètres qui ne sont pas déclarés &#39;facultatif&#39;
Vous devez fournir des arguments obligatoires quand la procédure est appelée. Ils ne prennent pas de valeurs par défaut dans la déclaration.  
  
 **ID d’erreur :** BC32024  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez l’argument pour qu’il soit `Optional` ou supprimez la valeur par défaut.
