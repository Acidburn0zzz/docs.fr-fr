# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>La balise de commentaire XML &#39; &lt;balise&gt; &#39; n’est pas autorisée sur un &#39; &lt;élément&gt; &#39; élément de langage
La balise de commentaire XML \<balise > n’est pas autorisée sur un «\<élément >' élément. Le commentaire XML est ignoré.  
  
 Une balise de commentaire XML n’est pas prise en charge par l’élément de langage auquel elle est appliquée.  
  
 **ID d’erreur :** BC42306  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez la balise de commentaire XML non prise en charge.
