# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-ltmethodnamegt"></a>Argument non spécifié pour le paramètre '&lt;nom_paramètre&gt;'of'&lt;nom_méthode&gt;'
Vous n’avez pas fourni d’argument pour un paramètre obligatoire.  
  
 **ID d’erreur :** BC30455  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez un argument pour le paramètre spécifié.
