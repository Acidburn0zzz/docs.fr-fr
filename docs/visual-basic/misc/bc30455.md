# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-39ltmethodnamegt39"></a>Argument non spécifié pour le paramètre &#39; &lt;nom_paramètre&gt; &#39; de &#39; &lt;methodname&gt;&#39;
Vous n’avez pas fourni d’argument pour un paramètre obligatoire.  
  
 **ID d’erreur :** BC30455  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez un argument pour le paramètre spécifié.
