# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>Les instructions 'Wend' ne sont plus prises en charge ; utilisez des instructions 'End While' à la place
La construction `End While` remplace le mot clé `Wend` .  
  
 **ID d’erreur :** BC30809  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez `End While` pour terminer un bloc `While` .
