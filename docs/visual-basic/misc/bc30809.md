# <a name="39wend39-statements-are-no-longer-supported-use-39end-while39-statements-instead"></a>&#39;Wend&#39; instructions ne sont plus prises en charge ; Utilisez &#39;End While&#39; instructions à la place
La construction `End While` remplace le mot clé `Wend` .  
  
 **ID d’erreur :** BC30809  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez `End While` pour terminer un bloc `While` .
