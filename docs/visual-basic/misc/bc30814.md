# <a name="39gosub39-statements-are-no-longer-supported"></a>&#39; GoSub &#39; les instructions ne sont plus prises en charge.
`GoSub` ne peut pas être utilisé pour appeler une procédure.  
  
 **ID d’erreur :** BC30814  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Appelez directement les procédures sans utiliser `GoSub`. Par exemple :  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
