# Division par z&#233;ro (erreur d’ex&#233;cution Visual Basic)
Une expression utilisée en tant que diviseur possède une valeur égale à zéro.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez l’orthographe des variables de l’expression. Un nom de variable mal orthographié peut créer implicitement une variable numérique dont la valeur est zéro.  
  
2.  Vérifiez les opérations précédentes sur les variables de l’expression, particulièrement celles qui sont passées à la procédure en tant qu’arguments à partir d’autres procédures.
