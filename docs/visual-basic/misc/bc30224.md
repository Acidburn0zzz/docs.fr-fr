# &#39;Is&#39; attendu
Un opérateur `TypeOf` apparaît sans clause `Is`.  
  
 **ID d’erreur :** BC30224  
  
### Pour corriger cette erreur  
  
-   Ajoutez une clause `Is` après l’opérateur `TypeOf` ; par exemple, `TypeOf MyControl Is ComboBox`.
