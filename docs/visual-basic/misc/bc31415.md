# <a name="ltimplementsclausegt-cannot-implement-lttypenamegt-because-lttypenamegt-is-a-reserved-name"></a>«&lt;clause_implements&gt;'ne peut pas implémenter'&lt;typename&gt;'car'&lt;typename&gt;' est un nom réservé
Une instruction `Implements` a spécifié un nom réservé.  
  
 **ID d’erreur :** BC31415  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Renommez le membre d’interface qui n’a pas pu être implémenté.
