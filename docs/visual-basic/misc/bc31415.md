# <a name="39ltimplementsclausegt39-cannot-implement-39lttypenamegt39-because-39lttypenamegt39-is-a-reserved-name"></a>&#39;&lt;clause_implements&gt; &#39; ne peut pas implémenter &#39; &lt;typename&gt; &#39; car &#39; &lt;typename&gt; &#39; est un nom réservé
Une instruction `Implements` a spécifié un nom réservé.  
  
 **ID d’erreur :** BC31415  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Renommez le membre d’interface qui n’a pas pu être implémenté.
