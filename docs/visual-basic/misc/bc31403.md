# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>Les alias Imports &#39; &lt;nom&gt;&#39; est en conflit avec &#39;&lt; name2&gt;&#39; déclarés dans l’espace de noms racine
Le nom d’alias utilisé dans l’instruction `Imports` est en conflit avec un autre nom déclaré dans l’espace de noms racine.  
  
 **ID d’erreur :** BC31403  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Choisissez un nom unique pour l’alias utilisé par l’instruction `Imports` .
