# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>L’alias Imports&lt;nom&gt;'est en conflit avec'&lt;name2&gt;' déclaré dans l’espace de noms racine
Le nom d’alias utilisé dans l’instruction `Imports` est en conflit avec un autre nom déclaré dans l’espace de noms racine.  
  
 **ID d’erreur :** BC31403  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Choisissez un nom unique pour l’alias utilisé par l’instruction `Imports` .
