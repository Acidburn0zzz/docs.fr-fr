# <a name="39option-strict39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Option Strict&#39; peut être suivie que de &#39;sur&#39; ou &#39;hors tension&#39;
Les instructions`Option Strict` doivent spécifier `On` ou `Off`. Toutes les autres valeurs ne sont pas valides.  
  
 **ID d’erreur :** BC30620  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Spécifiez `On` ou `Off` pour l’instruction `Option Strict` .
