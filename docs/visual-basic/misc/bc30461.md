# <a name="classes-cannot-be-declared-ltspecifiergt"></a>Les classes ne peuvent pas être déclarées '&lt;spécificateur&gt;'
Vous avez tenté de déclarer `Class` ou `Module` avec un spécificateur non valide.  
  
 **ID d’erreur :** BC30461  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
