# L’interface &#39;&lt;nom_interface&gt;&#39; ne peut &#234;tre impl&#233;ment&#233;e qu’une seule fois par ce type
L’interface spécifiée est déjà implémentée par ce type. Elle ne peut pas être implémentée à nouveau.  
  
 **ID d’erreur :** BC31033  
  
### Pour corriger cette erreur  
  
-   Implémentez cette interface une seule fois à l’aide de ce type.
