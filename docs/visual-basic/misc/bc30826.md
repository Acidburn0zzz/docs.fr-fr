# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39;EndIf&#39; instructions ne sont plus prises en charge ; Utilisez &#39;End If&#39; à la place
L’instruction `EndIf` n’est plus prise en charge.  
  
 **ID d’erreur :** BC30826  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez `End If` pour terminer les instructions `If` .
