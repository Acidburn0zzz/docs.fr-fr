# <a name="addhandler-declaration-must-end-with-a-matching-end-addhandler"></a>La déclaration 'AddHandler' doit se terminer par un 'End AddHandler' correspondant
Une déclaration `AddHandler` doit se terminer par une instruction `End AddHandler` .  
  
 **ID d’erreur :** BC31115  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que la déclaration `AddHandler` se termine par une instruction `End AddHandler` .
