# <a name="removehandler-is-already-declared"></a>'RemoveHandler' est déjà déclaré
Plusieurs déclarations `RemoveHandler` sont apparues dans une déclaration d’événement personnalisée. Une déclaration `RemoveHandler` déclare la procédure utilisée pour supprimer un gestionnaire d’événements.  
  
 **ID d’erreur :** BC31128  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction `RemoveHandler` redondante.
