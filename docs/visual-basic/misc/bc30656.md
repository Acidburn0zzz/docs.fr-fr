# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Champ &#39; &lt;fieldname&gt; &#39; est d’un type non pris en charge
Un champ est déclaré comme un type qui ne peut pas être représenté par Visual Studio.  
  
 **ID d’erreur :** BC30656  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Attribuez au champ un type pris en charge.  
  
2.  Développez un nouveau type décrivant les données du champ.
