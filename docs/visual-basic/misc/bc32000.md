# <a name="local-variable-39ltvariablenamegt39-cannot-be-referred-to-before-it-is-declared"></a>Variable locale &#39; &lt;variablename&gt;&#39; ne peut pas être référencée avant qu’il est déclaré
Une variable à l’intérieur d’une procédure est utilisée avant sa déclaration.  
  
 **ID d’erreur :** BC32000  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déplacez la déclaration de sorte qu’elle précède la première référence à la variable.
