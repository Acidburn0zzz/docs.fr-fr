# <a name="39ltnamegt39-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-39lttypenamegt39-through-ltnamegt"></a>&#39; &lt;nom&gt;&#39; ne peut pas exposer le type &lt;type1&gt; dans &lt;spécificateur&gt; &lt;type2&gt; &#39;&lt; TypeName&gt;&#39; via &lt;nom&gt;
Vous avez tenté d’exposer un type privé en dehors de la classe publique `MyClass`.  
  
 **ID d’erreur :** BC30508  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez le type `Public`.
