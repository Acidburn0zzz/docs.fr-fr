# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>«&lt;nom&gt;' ne peut pas exposer le type &lt;type1&gt; dans &lt;spécificateur&gt; &lt;type2&gt; '&lt;typename&gt;' via &lt;nom&gt;
Vous avez tenté d’exposer un type privé en dehors de la classe publique `MyClass`.  
  
 **ID d’erreur :** BC30508  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez le type `Public`.
