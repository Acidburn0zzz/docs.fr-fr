# <a name="array-exceeds-the-limit-of-32-dimensions"></a>Le tableau dépasse le nombre de 32 dimensions autorisées
Les tableaux ne peuvent pas avoir plus de 32 dimensions.  
  
 **ID d’erreur :** BC30052  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Réduisez le nombre de dimensions dans la déclaration du tableau.
