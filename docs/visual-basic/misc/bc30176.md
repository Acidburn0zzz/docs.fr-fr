# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Seul un des &#39; Public &#39; &#39; Privé &#39; &#39; Protégé &#39; &#39; Friend &#39; ou &#39; Protected Friend &#39; peut être spécifié.
Des conflits d’accessibilité se produisent dans une déclaration d’élément. Une seule accessibilité peut être spécifiée.  
  
 **ID d’erreur :** BC30176  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Choisissez une accessibilité et supprimez les autres mots clés de la déclaration.
