# <a name="39redim39-statements-require-a-parenthesized-list-of-the-new-bounds-of-each-dimension-of-the-array"></a>&#39; ReDim &#39; instructions requièrent une liste entre parenthèses des nouvelles limites de chaque dimension du tableau
Vous devez spécifier la nouvelle taille d’un tableau dans le cadre d’une instruction `ReDim` .  
  
 **ID d’erreur :** BC30670  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Indiquez la taille de chaque rang du tableau. Par exemple :  
  
    ```  
    ReDim arr(5, 6)  
    ```
