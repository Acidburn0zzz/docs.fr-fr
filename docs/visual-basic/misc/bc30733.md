# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>«&lt;membername&gt;'est déjà déclaré par'&lt;nom_procédure&gt;», qui a été généré pour ce &lt;containername&gt;
Un membre qui porte le même nom qu’un membre déclaré par le Common Language Runtime a été déclaré.  
  
 **ID d’erreur :** BC30733  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez le nom du membre pour supprimer le conflit.
