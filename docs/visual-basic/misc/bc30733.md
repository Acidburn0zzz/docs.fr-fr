# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>&#39; &lt;membername&gt;&#39; est déjà déclaré par &#39;&lt; nom_procédure&gt;&#39; qui a été généré pour ce &lt;containername&gt;
Un membre qui porte le même nom qu’un membre déclaré par le Common Language Runtime a été déclaré.  
  
 **ID d’erreur :** BC30733  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez le nom du membre pour supprimer le conflit.
