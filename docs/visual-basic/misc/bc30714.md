# <a name="39resume39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Reprise &#39; les instructions ne sont pas valides dans la fenêtre exécution
Les instructions Resume ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30714  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Resume` du code de débogage.
