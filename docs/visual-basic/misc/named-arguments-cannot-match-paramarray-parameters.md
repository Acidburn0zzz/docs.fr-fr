# <a name="named-arguments-cannot-match-paramarray-parameters"></a>Les arguments nommés ne peuvent pas correspondre aux paramètres ParamArray
Les tableaux de paramètres doivent être passés par valeur.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Passez les arguments `ByVal`, plutôt qu’en les nommant.
