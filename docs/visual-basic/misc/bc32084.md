# <a name="type-parameter-39lttypeparametername1gt39-must-have-either-a-39new39-constraint-or-a-39structure39-constraint-to-satisfy-the-39new39-constraint-for-type-parameter-39lttypeparametername2gt39"></a><span data-ttu-id="1c460-102">Paramètre de type &#39; &lt;nom_paramètre_type1&gt;&#39; doit contenir soit un &#39; Nouveau &#39; contrainte ou un &#39; Structure &#39; contrainte pour satisfaire la &#39; Nouveau &#39; contrainte de paramètre de type &#39; &lt;nom_paramètre_type2&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="1c460-102">Type parameter &#39;&lt;typeparametername1&gt;&#39; must have either a &#39;New&#39; constraint or a &#39;Structure&#39; constraint to satisfy the &#39;New&#39; constraint for type parameter &#39;&lt;typeparametername2&gt;&#39;</span></span>
<span data-ttu-id="1c460-103">Une instruction construit un type générique en passant un paramètre de type qui n’est pas contraint pour satisfaire une contrainte `New` .</span><span class="sxs-lookup"><span data-stu-id="1c460-103">A statement constructs a generic type passing a type parameter that is not constrained to satisfy a `New` constraint.</span></span>  
  
 <span data-ttu-id="1c460-104">La contrainte `New` signifie que l’argument de type fourni à ce paramètre de type doit exposer un constructeur sans paramètre accessible au code qui crée des objets à partir de celui-ci.</span><span class="sxs-lookup"><span data-stu-id="1c460-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="1c460-105">Tous les types valeur ont des constructeurs sans paramètre, mais ce n’est pas le cas de tous les types référence.</span><span class="sxs-lookup"><span data-stu-id="1c460-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="1c460-106">La contrainte `Structure` satisfait donc la contrainte `New` , mais la contrainte `Class` , ou un nom de classe ou d’interface, ne la satisfont pas.</span><span class="sxs-lookup"><span data-stu-id="1c460-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>  
  
 <span data-ttu-id="1c460-107">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="1c460-107">The following statements can generate this error.</span></span>  
  
```  
Public Class c1(Of t As New)  
End Class  
Public Class c2(Of u)  
    Public testObject As New c1(Of u)  
End Class  
```  
  
 <span data-ttu-id="1c460-108">Quand la classe `c2` crée un objet à partir de `c1`, le paramètre de type `u` ne satisfait pas la contrainte `New` appliquée au paramètre de type `t`.</span><span class="sxs-lookup"><span data-stu-id="1c460-108">When class `c2` creates an object from `c1`, type parameter `u` does not satisfy the `New` constraint on type parameter `t`.</span></span>  
  
 <span data-ttu-id="1c460-109">**ID d’erreur :** BC32084</span><span class="sxs-lookup"><span data-stu-id="1c460-109">**Error ID:** BC32084</span></span>
