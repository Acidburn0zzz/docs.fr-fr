# <a name="39ltspecifiergt39-is-not-valid-on-an-enum-declaration"></a>&#39; &lt;spécificateur&gt;&#39; n’est pas valide dans une déclaration Enum
Vous avez utilisé un spécificateur non valide dans une déclaration `Enum` .  
  
 **ID d’erreur :** BC30396  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez le spécificateur.
