# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;Module&#39; instruction doit se terminer par une mise en correspondance &#39;End Module&#39;
Blocs de module doivent se terminer par un `End Module` instruction.  
  
 **ID d’erreur :** BC30625  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une instruction `End Module` à la fin du bloc `Module` .
