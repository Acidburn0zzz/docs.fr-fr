# <a name="import-of-type-lttypenamegt-from-assembly-or-module-ltnamegt-failed"></a>Importation du type '&lt;typename&gt;'de l’assembly ou du module'&lt;nom&gt;' a échoué
Vérifiez que toutes les bibliothèques référencées sont compatibles les unes avec les autres.  
  
 **ID d’erreur :** BC31091  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que les fichiers qui contiennent le type référencé ne sont pas endommagés.
