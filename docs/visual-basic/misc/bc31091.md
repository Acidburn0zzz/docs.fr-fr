# <a name="import-of-type-39lttypenamegt39-from-assembly-or-module-39ltnamegt39-failed"></a>Importation de type &#39; &lt;typename&gt;&#39; de l’assembly ou module &#39;&lt; nom&gt;&#39; a échoué
Vérifiez que toutes les bibliothèques référencées sont compatibles les unes avec les autres.  
  
 **ID d’erreur :** BC31091  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que les fichiers qui contiennent le type référencé ne sont pas endommagés.
