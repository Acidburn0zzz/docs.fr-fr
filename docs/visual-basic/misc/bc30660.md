# <a name="attributes-cannot-be-applied-to-local-variables"></a>Les attributs ne peuvent pas être appliqués aux variables locales
Les attributs ne peuvent pas être appliqués aux variables locales.  
  
 **ID d’erreur :** BC30660  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Déclarez cette variable au niveau de portée supérieur, par exemple au niveau du module ou de la classe.
