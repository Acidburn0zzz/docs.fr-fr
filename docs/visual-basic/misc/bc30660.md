# Les attributs ne peuvent pas &#234;tre appliqu&#233;s aux variables locales
Les attributs ne peuvent pas être appliqués aux variables locales.  
  
 **ID d’erreur :** BC30660  
  
### Pour corriger cette erreur  
  
1.  Déclarez cette variable au niveau de portée supérieur, par exemple au niveau du module ou de la classe.
