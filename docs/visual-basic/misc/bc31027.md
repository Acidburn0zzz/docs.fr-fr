# <a name="unable-to-open-file-ltfilenamegt-lterrorgt"></a>Impossible d’ouvrir le fichier '&lt;filename&gt;' : &lt;erreur&gt;
Une erreur s’est produite pendant l’ouverture d’un fichier dans ce projet.  
  
 **ID d’erreur :** BC31027  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que le fichier existe et qu’il n’est pas endommagé ou verrouillé par une autre application.
