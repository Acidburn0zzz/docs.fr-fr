# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39;MyBase&#39; ne peut pas être utilisé avec la méthode &#39; &lt;methodname&gt; &#39; , car il est déclaré &#39;MustOverride&#39;
Vous avez essayé d’utiliser `MyBase` avec une méthode qui a été déclarée `MustOverride`.  
  
 **ID d’erreur :** BC30399  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez la déclaration `MustOverride` .
