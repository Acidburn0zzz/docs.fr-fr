# <a name="property-ltpropertynamegt-cannot-be-declared-default-because-it-overrides-a-property-on-the-base-lttypegt-lttypenamegt-that-is-not-default"></a>Propriété '&lt;propertyname&gt;' ne peut pas être déclarée 'Default', car elle substitue une propriété sur la base de &lt;type&gt; '&lt;typename&gt;» qui est pas par défaut
Vous avez essayé de déclarer une propriété `Default`, ce qui n’est pas une déclaration valide, car elle se substitue à une propriété qui n’a pas été déclarée `Default` dans la classe de base.  
  
 **ID d’erreur :** BC30504  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez la déclaration `Default` .
