# <a name="property-39ltpropertynamegt39-cannot-be-declared-39default39-because-it-overrides-a-property-on-the-base-lttypegt-39lttypenamegt39-that-is-not-default"></a>Propriété &#39; &lt;propertyname&gt; &#39; ne peut pas être déclaré &#39;par défaut&#39; car elle substitue à une propriété sur la base de &lt;type&gt; &#39; &lt;typename&gt; &#39; c'est-à-dire pas par défaut
Vous avez essayé de déclarer une propriété `Default`, ce qui n’est pas une déclaration valide, car elle se substitue à une propriété qui n’a pas été déclarée `Default` dans la classe de base.  
  
 **ID d’erreur :** BC30504  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez la déclaration `Default` .
