# <a name="39ltqualifiedmethodgt39-is-not-accessible-in-this-context-because-it-is-39ltmodifiergt39"></a>&#39;&lt;méthode_qualifiée&gt; &#39; n’est pas accessible dans ce contexte, car il est &#39; &lt;modificateur&gt;&#39;
Vous avez tenté d’accéder à une méthode qui n’est pas accessible dans ce contexte, car un modificateur lui a été appliqué.  
  
 **ID d’erreur :** BC30390  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le modificateur.
