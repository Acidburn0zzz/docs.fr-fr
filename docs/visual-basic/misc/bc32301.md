# <a name="project-has-been-closed"></a>Le projet a été fermé
Une erreur d’accès au modèle de code s’est produite dans le compilateur [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] .  
  
 **ID d’erreur :** BC32301  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Si l'erreur se produit à nouveau, réinstallez le compilateur [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] .  
  
3.  Si l’erreur persiste après la réinstallation, rassemblez des informations sur le contexte dans lequel est survenue l’erreur, puis avertissez les Services de support technique Microsoft.
