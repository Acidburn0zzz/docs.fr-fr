# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39;Définissez&#39; instruction doit se terminer par une mise en correspondance &#39;End Set&#39;
Les procédures de propriété`Set` doivent se terminer par une instruction `End Set` .  
  
 **ID d’erreur :** BC30633  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que la procédure de propriété `Set` se termine par une construction `End Set` .
