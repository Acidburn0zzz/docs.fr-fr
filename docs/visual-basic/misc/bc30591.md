# <a name="39withevents39-variable-does-not-raise-any-events"></a>&#39;WithEvents&#39; variable ne déclenche pas d’événements
La variable objet déclarée avec le mot clé `WithEvents` ne fait pas référence à une instance de classe qui peut déclencher des événements.  
  
 **ID d’erreur :** BC30591  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Assurez-vous que la variable objet fait référence à une instance de classe qui peut déclencher des événements.
