# <a name="value-39ltvaluename1gt39-cannot-be-converted-to-39ltvaluename2gt39"></a>Valeur &#39; &lt;nom_valeur1&gt;&#39; ne peut pas être converti en &#39;&lt; nom_valeur2&gt;&#39;
Une conversion de type a été tentée entre deux types incompatibles.  
  
 **ID d’erreur :** BC30742  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez les types qui font l’objet d’un cast.
