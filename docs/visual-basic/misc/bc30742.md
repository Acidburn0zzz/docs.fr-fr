# <a name="value-ltvaluename1gt-cannot-be-converted-to-ltvaluename2gt"></a>Valeur '&lt;nom_valeur1&gt;« ne peut pas être converti en »&lt;nom_valeur2&gt;'
Une conversion de type a été tentée entre deux types incompatibles.  
  
 **ID d’erreur :** BC30742  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez les types qui font l’objet d’un cast.
