# Les param&#232;tres ParamArray doivent avoir un type tableau
Des paramètres `ParamArray` ont un type non valide à cause d’un problème qui s’est produit lors de l’accès à l’assembly MSCorlib.  
  
 **ID d’erreur :** BC31092  
  
### Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Le cas échéant, enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].  
  
3.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
4.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
