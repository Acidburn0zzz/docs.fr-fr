# <a name="argument-39ltargumentnamegt39-must-be-in-the-range-of--32768-to-65535"></a>Argument &#39; &lt;nom_argument&gt;&#39; doit être comprise entre-32 768 et 65 535.
La valeur d’un argument n’est pas comprise dans la plage autorisée de -32 768 - 65 535.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que la valeur de l’argument est comprise dans la plage valide.  
  
2.  Si l’argument est fourni par un calcul, vérifiez les valeurs traitées par le calcul.
