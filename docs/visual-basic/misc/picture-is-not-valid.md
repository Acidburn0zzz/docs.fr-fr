# Image non valide
Vous avez tenté d’assigner un format graphique autre qu’une bitmap, une icône ou un métafichier Windows à la propriété `Picture` d’un formulaire ou d’un contrôle.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que le fichier que vous essayez de charger dans la propriété `Picture` est un fichier graphique valide pris en charge par [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].
