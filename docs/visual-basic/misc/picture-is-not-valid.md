# <a name="picture-is-not-valid"></a>Image non valide
Vous avez tenté d’assigner un format graphique autre qu’une bitmap, une icône ou un métafichier Windows à la propriété `Picture` d’un formulaire ou d’un contrôle.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que le fichier que vous essayez de charger dans le `Picture` propriété est un fichier graphique valide pris en charge par [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].
