# <a name="field-or-property-ltmembernamegt-is-not-found"></a>Champ ou la propriété '&lt;membername&gt;' est introuvable
Une propriété ou un champ non défini est référencé pour l’attribut.  
  
 **ID d’erreur :** BC30661  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez l’orthographe du champ ou de la propriété.  
  
2.  Vérifiez que votre projet peut accéder à la classe d’attributs appropriée, puis ajoutez-le aux références de projet, si nécessaire.
