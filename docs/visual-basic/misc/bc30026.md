# <a name="end-sub-expected"></a>'End Sub' attendu
Une instruction `End` doit correspondre au bloc qu’elle termine. Une définition de procédure `Sub` n’a pas été correctement terminée.  
  
 **ID d’erreur :** BC30026  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une instruction `End Sub` à la fin de la procédure.
