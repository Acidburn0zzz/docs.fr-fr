# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-39ltdelegatenamegt39"></a><span data-ttu-id="27d69-102">Option Strict On n’autorise pas conversions restrictives lors des conversions de types implicites entre l’expression lambda et le délégué &#39; &lt;nom_délégué&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="27d69-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate &#39;&lt;delegatename&gt;&#39;</span></span>
<span data-ttu-id="27d69-103">Quand `Option Strict` est On, vous ne pouvez pas avoir une conversion restrictive entre le type de données d’un paramètre dans un délégué et le paramètre correspondant d’une expression lambda assignée à une variable de ce type de délégué.</span><span class="sxs-lookup"><span data-stu-id="27d69-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="27d69-104">Par exemple, dans le code suivant, le délégué `Del` a un paramètre de type `Integer`.</span><span class="sxs-lookup"><span data-stu-id="27d69-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="27d69-105">Ainsi, le paramètre correspondant d’une expression lambda assignée à une variable de type `Del` peut être un `Integer` ou n’importe quel type de données pour lequel il existe une conversion étendue à partir de `Integer`.</span><span class="sxs-lookup"><span data-stu-id="27d69-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="27d69-106">**ID d’erreur :** BC36662</span><span class="sxs-lookup"><span data-stu-id="27d69-106">**Error ID:** BC36662</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="27d69-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="27d69-107">To correct this error</span></span>  
  
-   <span data-ttu-id="27d69-108">Modifiez le type de données du paramètre dans le délégué ou l’expression lambda pour que la relation d’étendue nécessaire existe.</span><span class="sxs-lookup"><span data-stu-id="27d69-108">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>  
  
-   <span data-ttu-id="27d69-109">Ne spécifiez pas de types de données de paramètres dans l’expression lambda.</span><span class="sxs-lookup"><span data-stu-id="27d69-109">Do not specify parameter data types in the lambda expression.</span></span> <span data-ttu-id="27d69-110">Les types seront déduits des paramètres correspondants dans le délégué.</span><span class="sxs-lookup"><span data-stu-id="27d69-110">Types will be inferred from the corresponding parameters in the delegate.</span></span>  
  
```vb  
Dim example4 As Del = Function(n) "Valid"
```  
