# <a name="resource-name-39ltresourcenamegt39-cannot-be-used-more-than-once"></a>Nom de la ressource &#39; &lt;resourcename&gt;&#39; ne peut pas être utilisé plusieurs fois
Vous avez utilisé plusieurs fois le nom d’une ressource spécifiée dans une bibliothèque de liens dynamiques ou dans un fichier exécutable.  
  
 **ID d’erreur :** BC31502  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Nommez les ressources de manière unique.
