# Le nom de ressource &#39;&lt;nom_ressource&gt;&#39; ne peut pas &#234;tre utilis&#233; plusieurs fois
Vous avez utilisé plusieurs fois le nom d’une ressource spécifiée dans une bibliothèque de liens dynamiques ou dans un fichier exécutable.  
  
 **ID d’erreur :** BC31502  
  
### Pour corriger cette erreur  
  
-   Nommez les ressources de manière unique.
