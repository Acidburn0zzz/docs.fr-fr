# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>'Currency' n’est plus un type pris en charge ; utilisez le type 'Decimal' à la place
Le type de données `Currency` a été remplacé par le type de données `Decimal` .  
  
 **ID d’erreur :** BC30803  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez le type de données `Decimal` au lieu de `Currency`.
