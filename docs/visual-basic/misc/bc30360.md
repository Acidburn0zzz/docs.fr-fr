# <a name="property-39ltpropertynamegt39-must-be-declared-39default39-because-it-overrides-a-default-property-on-the-base-39ltnamegt39"></a>Propriété &#39; &lt;propertyname&gt;&#39; doit être déclaré &#39; par défaut &#39; car elle substitue à une propriété par défaut sur la base de &#39;&lt; nom&gt;&#39;
Une instruction `Property` dans votre code remplace une propriété `Default` sur la classe, la structure ou l’interface de base. Elle doit donc être déclarée `Default`.  
  
 **ID d’erreur :** BC30360  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez la propriété `Default`.
