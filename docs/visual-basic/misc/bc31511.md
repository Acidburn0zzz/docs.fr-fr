# <a name="attribute-member-ltmembernamegt-cannot-be-the-target-of-an-assignment-because-it-is-not-declared-public"></a>Membre d’attribut '&lt;membername&gt;' ne peut pas être la cible d’une assignation, car il n’est pas déclaré 'Public'
Vous avez essayé d’assigner une valeur à un membre privé d’un attribut.  
  
 **ID d’erreur :** BC31511  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez l’assignation.  
  
2.  Si vous utilisez des attributs personnalisés que vous avez développés, affectez `Public`comme modificateur d’accès du membre d’attribut.
