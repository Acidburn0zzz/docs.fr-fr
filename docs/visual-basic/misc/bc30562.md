# <a name="ltnamegt-is-ambiguous-between-declarations-in-modules-ltmodulename1gt-and-ltmodulename2gt"></a>«&lt;nom&gt;'est ambigu entre les déclarations dans les Modules'&lt;nom_module1&gt;'et'&lt;nom_module2&gt;»
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le compilateur Visual Basic n’a pas les règles de résolution de conflit ; Vous devez lever l’ambiguïté des noms vous-même.  
  
 **ID d’erreur :** BC30562  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez un nom qualifié complet.
