# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>Type '&lt;typename&gt;'est importé à partir de différentes versions d’assembly'&lt;assemblyname&gt;»
Type '\<typename >' est importé à partir de différentes versions d’assembly '\<assemblyname > ». Un même projet ne peut pas utiliser plusieurs versions du même assembly.  
  
 Un type a été importé à partir de plusieurs versions du même assembly, créant ainsi une ambiguïté.  
  
 **ID d’erreur :** BC31525  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les références aux différentes versions du même assembly.
