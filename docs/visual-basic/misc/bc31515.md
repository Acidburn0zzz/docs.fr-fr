# <a name="project-39ltprojectnamegt39-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39ltclassnamegt39"></a>Projet «&lt;nom_projet&gt;'établit une référence indirecte à l’assembly'&lt;assemblyname&gt;« qui contient »&lt;classname&gt;»
Projet «\<projectname >' établit une référence indirecte à l’assembly '\<assemblyname >' qui contient '\<classname > ». Ajoutez une référence à '\<assemblyname > » à votre projet.  
  
 Ce projet contient des membres qui dépendent d’un assembly qui n’est pas contenu dans les références de projet.  
  
 **ID d’erreur :** BC31515  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une référence à l’assembly en cliquant avec le bouton droit sur **Références** dans l’ **Explorateur de solutions** puis en cliquant sur **Ajouter une référence**.
