# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-with-these-argumentslterrorgt"></a>A échoué, car la résolution de surcharge n’accessible '&lt;méthode&gt;» peut être appelée avec ces arguments :&lt;erreur&gt;
Vous avez appelé une méthode surchargée, mais le compilateur ne peut pas trouver de méthode avec une liste d’arguments correspondante.  
  
 **ID d’erreur :** BC30518  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez la liste d’arguments que vous avez fournie.
