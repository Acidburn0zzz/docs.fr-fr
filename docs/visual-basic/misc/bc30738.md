# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39;Sub Main&#39; est déclarée plusieurs fois dans &#39; &lt;espace de noms&gt;&#39;: &lt;message&gt;
`Sub Main` ne peut être déclaré qu’une seule fois dans un espace de noms.  
  
 **ID d’erreur :** BC30738  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que votre projet ne contient qu’une seule procédure `Sub Main` .
