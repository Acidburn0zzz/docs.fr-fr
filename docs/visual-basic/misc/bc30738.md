# <a name="sub-main-is-declared-more-than-once-in-ltnamespacegt-ltmessagegt"></a>'Sub Main' est déclaré plusieurs fois dans '&lt;espace de noms&gt;' : &lt;message&gt;
`Sub Main` ne peut être déclaré qu’une seule fois dans un espace de noms.  
  
 **ID d’erreur :** BC30738  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que votre projet ne contient qu’une seule procédure `Sub Main` .
