## <a name="example"></a><span data-ttu-id="f0525-111">Exemple</span><span class="sxs-lookup"><span data-stu-id="f0525-111">Example</span></span>  
 <span data-ttu-id="f0525-112">L’exemple suivant utilise l’attribut `AttributeUsage` pour spécifier les éléments de programme auxquels la nouvelle version de `ExtensionAttribute` peut s’appliquer.</span><span class="sxs-lookup"><span data-stu-id="f0525-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="f0525-113">L’exemple spécifie trois membres de l’énumération `AttributeTargets` : `Assembly`, `Class`et `Method`.</span><span class="sxs-lookup"><span data-stu-id="f0525-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="f0525-114">L’omission de l’un de ces éléments provoque cette erreur.</span><span class="sxs-lookup"><span data-stu-id="f0525-114">The omission of any one of these elements will cause this error.</span></span>
