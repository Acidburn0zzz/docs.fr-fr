# <a name="type-39lttypenamegt39-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Type de &#39; &lt;typename&gt;&#39; n’est pas défini ou le module qui le contient n’est pas chargé dans la session de débogage
Un type a été référencé et n’est pas disponible ou il n’est pas défini.  
  
 **ID d’erreur :** BC30743  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que le nom du type est correctement orthographié.  
  
2.  Définissez le type dans la classe ou le module actuels.
