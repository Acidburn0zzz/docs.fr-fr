# <a name="arrays-cannot-be-declared-with-39new39"></a><span data-ttu-id="9e4f6-102">Les tableaux ne peuvent pas être déclarés avec &#39;nouveau&#39;</span><span class="sxs-lookup"><span data-stu-id="9e4f6-102">Arrays cannot be declared with &#39;New&#39;</span></span>
<span data-ttu-id="9e4f6-103">Le mot clé `New` peut figurer uniquement dans la partie initialisation d’une déclaration de tableau.</span><span class="sxs-lookup"><span data-stu-id="9e4f6-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="9e4f6-104">Cela signifie que `New` doit être situé à droite du signe égal (`=`) pour lui permettre de créer un type de tableau à assigner à la variable tableau.</span><span class="sxs-lookup"><span data-stu-id="9e4f6-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="9e4f6-105">Le raccourci d’initialisation de classe n’est pas disponible pour les tableaux.</span><span class="sxs-lookup"><span data-stu-id="9e4f6-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="9e4f6-106">Les deux lignes de code suivantes sont valides et équivalentes, car elles initialisent toutes les deux un objet de classe.</span><span class="sxs-lookup"><span data-stu-id="9e4f6-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="9e4f6-107">Toutefois, l’initialisation de tableau ne peut pas utiliser le même raccourci comme initialisation de classe.</span><span class="sxs-lookup"><span data-stu-id="9e4f6-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="9e4f6-108">Notez que pour un tableau, la clause `New` doit contenir des parenthèses ( `()`) et des accolades ( `{}`).</span><span class="sxs-lookup"><span data-stu-id="9e4f6-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="9e4f6-109">Les parenthèses indiquent que le nouveau type est un tableau, tandis que les accolades fournissent les valeurs d’initialisation.</span><span class="sxs-lookup"><span data-stu-id="9e4f6-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="9e4f6-110">Le compilateur exige les accolades même si elles sont vides, c’est-à-dire, même si vous n’initialisez aucune valeur du tableau.</span><span class="sxs-lookup"><span data-stu-id="9e4f6-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="9e4f6-111">**ID d’erreur :** BC30053</span><span class="sxs-lookup"><span data-stu-id="9e4f6-111">**Error ID:** BC30053</span></span>
