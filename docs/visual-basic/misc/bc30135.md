# <a name="39synclock39-statements-are-not-valid-in-the-immediate-window"></a>&#39; SyncLock &#39; les instructions ne sont pas valides dans la fenêtre exécution
L’instruction `SyncLock` synchronise les threads et n’est pas autorisée dans un contexte de débogage.  
  
 **ID d’erreur :** BC30135  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   N’émettez pas une instruction `SyncLock` dans la fenêtre **Exécution** .
