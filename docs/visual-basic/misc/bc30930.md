# <a name="ltnamegt-necessary-for-compiling-this-construct-cannot-be-found"></a>«&lt;nom&gt;», qui est nécessaire pour compiler cette construction, est introuvable
Un élément nécessaire pour la compilation est introuvable.  
  
 **ID d’erreur :** BC30930  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Fournissez l’élément d’assistance nécessaire.  
  
2.  Vérifiez que le code est managé et valide.
