# Impossible d’&#233;valuer &#39;Nothing&#39;
Le mot clé `Nothing` a été utilisé dans la fenêtre de débogage.  
  
 **ID d’erreur :** BC30701  
  
### Pour corriger cette erreur  
  
-   N’utilisez pas `Nothing` pendant le débogage.
