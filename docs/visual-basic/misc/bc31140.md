# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>&#39;Personnalisé&#39; modificateur ne peut être utilisé qu’immédiatement avant une &#39;événement&#39; déclaration
Le mot clé `Custom` doit précéder uniquement le mot clé `Event` dans une déclaration d’événement personnalisé.  
  
 **ID d’erreur :** BC31140  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Placez le mot clé `Custom` avant le mot clé `Event` .  
  
     - ou -  
  
-   Supprimez le mot clé `Custom` .
