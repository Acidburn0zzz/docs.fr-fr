# <a name="39default39-can-be-applied-to-only-one-property-name-in-a-lttypegt"></a>&#39;Par défaut&#39; peut être appliqué à un nom de propriété qu’une seule dans un &lt;type&gt;
Vous avez tenté d’appliquer `Default` à plusieurs noms de propriétés dans une classe, interface ou structure.  
  
 **ID d’erreur :** BC30359  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez toutes les déclarations `Default` sauf une.
