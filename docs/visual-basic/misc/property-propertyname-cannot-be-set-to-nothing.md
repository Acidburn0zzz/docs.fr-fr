# <a name="property-ltpropertynamegt-cannot-be-set-to-nothing"></a>Propriété &lt;propertyname&gt; ne peut pas être définie sur Nothing
La propriété exige une valeur autre que `Nothing`. Cette erreur peut être le résultat d’un calcul incorrect de la valeur.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez une valeur valide.
