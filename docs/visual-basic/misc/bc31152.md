# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Préfixe d’espace de noms XML &#39;préfixe&#39; est réservée pour une utilisation par XML et l’espace de noms URI ne peut pas être modifié.
Un préfixe d’espace de noms XML réservé, tel que `xmlns`, a été utilisé dans un littéral XML.  
  
 **ID d’erreur :** BC31152  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Remplacez le préfixe de l’espace de noms XML par une valeur qui n’est pas réservée.
