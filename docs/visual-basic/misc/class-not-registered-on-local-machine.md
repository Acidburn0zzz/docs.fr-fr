# <a name="class-not-registered-on-local-machine"></a>La classe n’est pas inscrite sur l’ordinateur local.
Cette erreur est généralement due à un fichier endommagé. Vous devrez peut-être réinstaller [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Réinstallez [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].
