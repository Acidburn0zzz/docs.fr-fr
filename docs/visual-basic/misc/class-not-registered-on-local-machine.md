# La classe n’est pas inscrite sur l’ordinateur local.
Cette erreur est généralement due à un fichier endommagé. Vous devrez peut\-être réinstaller [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].  
  
### Pour corriger cette erreur  
  
-   Réinstallez [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].
