# <a name="lterrorgt-39ltstructurename1gt39-contains-39ltstructurename2gt39"></a>&lt;erreur&gt;: &#39;&lt; nom_structure1&gt;&#39; contient &#39;&lt; nom_structure2&gt;&#39;
Une définition de structure circulaire se produit quand une structure se désigne comme l’un de ses propres membres.  
  
 **ID d’erreur :** BC30293  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez le nom du membre de la structure.
