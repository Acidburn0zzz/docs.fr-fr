# <a name="cannot-inherit-interface-39ltinterfacename1gt39-because-the-interface-39ltinterfacename2gt39-from-which-it-inherits-could-be-identical-to-interface-39ltinterfacename3gt39-for-some-type-arguments"></a><span data-ttu-id="23a1d-102">Impossible d’hériter d’interface &#39; &lt;nom_interface1&gt;&#39; car l’interface &#39;&lt; nom_interface2&gt;&#39; dont elle hérite peut être identique à l’interface &#39;&lt; nom_interface3&gt;&#39; pour certains arguments de type</span><span class="sxs-lookup"><span data-stu-id="23a1d-102">Cannot inherit interface &#39;&lt;interfacename1&gt;&#39; because the interface &#39;&lt;interfacename2&gt;&#39; from which it inherits could be identical to interface &#39;&lt;interfacename3&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="23a1d-103">Une interface générique hérite d’au moins deux interfaces génériques, et deux des héritages peuvent entrer en conflit pour certaines valeurs d’arguments de type.</span><span class="sxs-lookup"><span data-stu-id="23a1d-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="23a1d-104">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="23a1d-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="23a1d-105">Si `derivedInterface` est construit ou implémenté en fournissant le même type à `t1` et à `t2`, il doit hériter de deux versions de `interfaceX` avec des arguments de type identiques.</span><span class="sxs-lookup"><span data-stu-id="23a1d-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="23a1d-106">Il en résulte une ambiguïté quant à la version à laquelle accéder.</span><span class="sxs-lookup"><span data-stu-id="23a1d-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="23a1d-107">**ID d’erreur :** BC32121</span><span class="sxs-lookup"><span data-stu-id="23a1d-107">**Error ID:** BC32121</span></span>
