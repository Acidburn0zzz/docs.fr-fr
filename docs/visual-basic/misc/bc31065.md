# <a name="39set39-parameter-cannot-be-declared-39ltspecifiergt39"></a>&#39; Définir le &#39; le paramètre ne peut pas être déclaré &#39; &lt;spécificateur&gt;&#39;
Un modificateur non valide a été spécifié avec un paramètre dans une procédure de propriété `Set` .  
  
 **ID d’erreur :** BC31065  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les modificateurs non valides, comme `ByRef`, `ParamArray`et `Optional`.
