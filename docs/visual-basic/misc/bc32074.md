    -   <span data-ttu-id="2031e-110">Type construit à partir d’un type générique</span><span class="sxs-lookup"><span data-stu-id="2031e-110">A type constructed from a generic type,</span></span>  
  
    -   <span data-ttu-id="2031e-111">Paramètre de type d’un type conteneur</span><span class="sxs-lookup"><span data-stu-id="2031e-111">A type parameter of a containing type, or</span></span>  
  
    -   <span data-ttu-id="2031e-112">Type construit à partir d’un paramètre de type d’un type conteneur</span><span class="sxs-lookup"><span data-stu-id="2031e-112">A type constructed from a type parameter of a containing type.</span></span>
