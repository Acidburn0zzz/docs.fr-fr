# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>Longueur de l’argument '\<nom_argument >' doit être supérieure à zéro
Un argument a une valeur inférieure ou égale à zéro.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez un argument supérieur à zéro.
