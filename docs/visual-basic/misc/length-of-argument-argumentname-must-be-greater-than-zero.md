# La longueur de l’argument &#39;&lt;nom_argument&gt;&#39; doit &#234;tre sup&#233;rieure &#224; z&#233;ro
Un argument a une valeur inférieure ou égale à zéro.  
  
### Pour corriger cette erreur  
  
-   Fournissez un argument supérieur à zéro.
