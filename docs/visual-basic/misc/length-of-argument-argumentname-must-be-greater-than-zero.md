# <a name="length-of-argument-39ltargumentnamegt39-must-be-greater-than-zero"></a>Longueur d’argument &#39; &lt;nom_argument&gt;&#39; doit être supérieure à zéro
Un argument a une valeur inférieure ou égale à zéro.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez un argument supérieur à zéro.
