# <a name="length-of-argument-ltargumentnamegt-must-be-greater-than-zero"></a>Longueur de l’argument '&lt;nom_argument&gt;' doit être supérieure à zéro
Un argument a une valeur inférieure ou égale à zéro.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez un argument supérieur à zéro.
