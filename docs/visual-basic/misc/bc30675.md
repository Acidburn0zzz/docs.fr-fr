# <a name="39synclock39-statement-must-end-with-a-matching-39end-synclock39"></a>&#39;SyncLock&#39; instruction doit se terminer par une mise en correspondance &#39;End SyncLock&#39;
Les blocs`SyncLock` commencent par le mot clé `SyncLock` et se terminent par la construction `End SyncLock` .  
  
 **ID d’erreur :** BC30675  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que le bloc `SyncLock` se termine par une construction `End SyncLock` .
