# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>La version ciblée du .NET Compact Framework ne prend pas en charge la &#39;fin&#39; instruction
La version du .NET Compact Framework que vous utilisez ne prend pas en charge l’instruction `End` .  
  
 **ID d’erreur :** BC30769  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction.
