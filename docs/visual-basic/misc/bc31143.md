 <span data-ttu-id="0a42d-105">Le code suivant illustre des échecs de conversion.</span><span class="sxs-lookup"><span data-stu-id="0a42d-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="0a42d-106">Le délégué est `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="0a42d-106">The delegate is `FunDel`.</span></span>

Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date

 <span data-ttu-id="0a42d-108">Chacune des instructions d’assignation suivantes provoque l’erreur.</span><span class="sxs-lookup"><span data-stu-id="0a42d-108">Each of the following assignment statements causes the error.</span></span>

    ' are not compatible.
