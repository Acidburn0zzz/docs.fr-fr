# <a name="the-value-of-ltargumentnamegt-must-be-greater-than-or-equal-to-1000"></a>La valeur de &lt;nom_argument&gt; doit être supérieure ou égale à 1 000
La valeur de la propriété <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> doit être supérieure ou égale à 1 000.  
  
 La propriété <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> spécifie le nombre maximal d’octets autorisé dans un fichier journal avant la création d’un autre fichier journal.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Affectez à la propriété <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> un nombre supérieur ou égal à 1 000.
