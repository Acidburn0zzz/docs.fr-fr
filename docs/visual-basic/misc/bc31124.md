# <a name="end-addhandler-must-be-preceded-by-a-matching-addhandler-declaration"></a>'End AddHandler' doit être précédé d’une déclaration 'AddHandler' correspondante
Une instruction `End AddHandler` s’est produite sans instruction `AddHandler` correspondante. `End AddHandler` doit être précédé d’une instruction `AddHandler` correspondante.  
  
 **ID d’erreur :** BC31124  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que l’instruction `AddHandler` précédente est valide et correctement orthographiée.
