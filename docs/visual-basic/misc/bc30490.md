# <a name="default-cannot-be-combined-with-specifier"></a>'Default' ne peut pas être combiné avec '\<spécificateur >'
Vous avez tenté d’associer le mot clé `Default` avec un spécificateur, tel que `Shared` ou `Private`, qui n’est pas valide dans ce contexte.  
  
 **ID d’erreur :** BC30490  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez `Default`.  
  
     ou  
  
-   Supprimez le spécificateur en conflit.
