# Le chemin du Registre sp&#233;cifi&#233; ne commence pas par un nom de ruche valide
Le chemin du Registre spécifié ne commence pas par un nom de ruche valide. Les noms de ruche valides sont les suivants :  
  
-   `HKEY_CLASSES_ROOT`  
  
-   `HKEY_CURRENT_CONFIG`  
  
-   `HKEY_CURRENT_USER`  
  
-   `HKEY_DYNDATA`  
  
-   `HKEY_LOCAL_MACHINE`  
  
-   `HKEY_PERFORMANCE_DATA`  
  
-   `HKEY_USERS`  
  
### Pour corriger cette erreur  
  
-   Veillez à fournir un nom de ruche valide.
