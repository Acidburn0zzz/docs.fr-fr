# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Continue' ne sont pas valides dans la fenêtre exécution
`Continue`instructions ne peuvent pas être utilisées dans le code source.  
  
 **ID d’erreur :** BC30780  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction `Continue` du code de débogage.
