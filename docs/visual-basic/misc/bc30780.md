# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Continuer &#39; les instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Continue` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30780  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction `Continue` du code de débogage.
