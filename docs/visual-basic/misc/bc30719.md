# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Case' ne sont pas valides dans la fenêtre Exécution
Les instructions`Case` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30719  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Case` de la fenêtre **Exécution** .
