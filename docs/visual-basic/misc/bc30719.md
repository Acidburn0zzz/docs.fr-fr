# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Cas &#39; les instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Case` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30719  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Case` de la fenêtre **Exécution** .
