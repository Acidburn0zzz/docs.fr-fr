# <a name="file-io-with-type-39lttypenamegt39-is-not-valid"></a>Fichier e/s avec type &#39; &lt;typename&gt;&#39; n’est pas valide
Vous tentez d’écrire un type non pris en charge dans un fichier.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez la structure pour déterminer où apparaît le type non pris en charge, puis remplacez-le par un type valide.
