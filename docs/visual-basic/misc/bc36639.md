# <a name="byref-parameter-ltparameternamegt-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="c5260-102">Paramètre 'ByRef' '&lt;nom_paramètre&gt;' ne peut pas être utilisé dans une expression lambda</span><span class="sxs-lookup"><span data-stu-id="c5260-102">'ByRef' parameter '&lt;parametername&gt;' cannot be used in a lambda expression</span></span>
<span data-ttu-id="c5260-103">Une expression lambda déclarée dans un élément `Sub` ou une fonction ne peut pas utiliser les paramètres `ByRef` de cet élément `Sub` ou de cette fonction.</span><span class="sxs-lookup"><span data-stu-id="c5260-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="c5260-104">Par exemple, le code suivant provoquera cette erreur, car le paramètre `ByRef` `n` est utilisé dans l’expression lambda.</span><span class="sxs-lookup"><span data-stu-id="c5260-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="c5260-105">**ID d’erreur :** BC36639</span><span class="sxs-lookup"><span data-stu-id="c5260-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c5260-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="c5260-106">To correct this error</span></span>  
  
-   <span data-ttu-id="c5260-107">Assignez le paramètre `ByRef` à une variable locale et utilisez la variable locale dans l’expression lambda, comme illustré dans le code suivant :</span><span class="sxs-lookup"><span data-stu-id="c5260-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>

    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
