# <a name="39byref39-parameter-39ltparameternamegt39-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="b0c86-102">&#39; ByRef &#39; paramètre &#39; &lt;nom_paramètre&gt;&#39; ne peut pas être utilisé dans une expression lambda</span><span class="sxs-lookup"><span data-stu-id="b0c86-102">&#39;ByRef&#39; parameter &#39;&lt;parametername&gt;&#39; cannot be used in a lambda expression</span></span>
<span data-ttu-id="b0c86-103">Une expression lambda déclarée dans un élément `Sub` ou une fonction ne peut pas utiliser les paramètres `ByRef` de cet élément `Sub` ou de cette fonction.</span><span class="sxs-lookup"><span data-stu-id="b0c86-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="b0c86-104">Par exemple, le code suivant provoquera cette erreur, car le paramètre `ByRef` `n` est utilisé dans l’expression lambda.</span><span class="sxs-lookup"><span data-stu-id="b0c86-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="b0c86-105">**ID d’erreur :** BC36639</span><span class="sxs-lookup"><span data-stu-id="b0c86-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b0c86-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="b0c86-106">To correct this error</span></span>

    ```  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
