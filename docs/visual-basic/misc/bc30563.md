# <a name="ltnamegt-is-ambiguous-in-the-application-objects-ltlistgt"></a>«&lt;nom&gt;« est ambigu dans les objets d’application »&lt;liste&gt;»
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le compilateur Visual Basic n’a pas les règles de résolution de conflit ; Vous devez lever l’ambiguïté des noms vous-même. La liste fournie dans le message d’erreur décrit en détail les objets d’application dont le nom est ambigu.  
  
 **ID d’erreur :** BC30563  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez un nom qualifié complet.
