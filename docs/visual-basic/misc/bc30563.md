# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39;&lt;nom&gt; &#39; est ambigu dans les objets d’application &#39; &lt;liste&gt;&#39;
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le compilateur Visual Basic ne dispose pas de toutes les règles de résolution de conflit ; Vous devez lever toute ambiguïté des noms vous-même. La liste fournie dans le message d’erreur décrit en détail les objets d’application dont le nom est ambigu.  
  
 **ID d’erreur :** BC30563  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez un nom qualifié complet.
