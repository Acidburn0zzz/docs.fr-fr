# &#39;/&#39; attendu pour la balise de fin XML
La balise de fin XML pour un littéral XML ne possède pas le caractère '\/' obligatoire.  
  
 **ID d’erreur :** BC31167  
  
### Pour corriger cette erreur  
  
-   Ajoutez le caractère '\/' au début de la balise de fin XML pour le littéral XML.
