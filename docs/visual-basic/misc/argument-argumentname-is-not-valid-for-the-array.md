# <a name="argument-argumentname-is-not-valid-for-the-array"></a>Argument '\<nom_argument >' n’est pas valide pour le tableau
Un tableau contient un argument qui n’est pas valide.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez l’orthographe des arguments dans l’expression. Un nom de variable mal orthographié peut créer implicitement une variable numérique dont la valeur est zéro.
