# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>Opérateur &#39; &lt;nom_opérateur&gt; &#39; n’est pas défini pour le type &#39; &lt;typename&gt;&#39;
Vous avez tenté d’utiliser un opérateur unaire (`+`, `-`ou `Not`) sur un type tel qu’un `String`pour lequel il n’est pas défini.  
  
 **ID d’erreur :** BC30487  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Remplacez l’opérateur unaire par un opérateur adapté à l’expression.
