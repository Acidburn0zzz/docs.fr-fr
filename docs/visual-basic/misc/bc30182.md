# <a name="type-expected"></a>Type attendu
Des caractères de code source ne sont pas reconnus comme nom de type. Le contexte nécessite un type de données élémentaire ou un nom d’élément déclaré représentant un type.  
  
 **ID d’erreur :** BC30182  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que le type a été défini.  
  
-   Vérifiez que le nom de type est orthographié correctement.
