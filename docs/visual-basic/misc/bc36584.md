# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Paramètre &#39; &lt;nom_paramètre&gt; &#39; dans la méthode d’extension &#39; &lt;methodname&gt; &#39; définies dans &#39; &lt;typename&gt; &#39; déjà a un argument correspondant
Un argument est passé par nom plusieurs fois dans le même appel de procédure.  
  
 **ID d’erreur :** BC36584  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez si l’un des noms d’argument est mal orthographié.  
  
-   Supprimez l’argument dupliqué de la liste d’arguments.
