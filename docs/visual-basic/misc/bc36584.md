# <a name="parameter-ltparameternamegt-in-extension-method-ltmethodnamegt-defined-in-lttypenamegt-already-has-a-matching-argument"></a>Paramètre '&lt;nom_paramètre&gt;« dans la méthode d’extension'&lt;nom_méthode&gt;'définie dans'&lt;typename&gt;' a déjà un argument correspondant
Un argument est passé par nom plusieurs fois dans le même appel de procédure.  
  
 **ID d’erreur :** BC36584  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez si l’un des noms d’argument est mal orthographié.  
  
-   Supprimez l’argument dupliqué de la liste d’arguments.
