# <a name="39get39-is-already-declared"></a>&#39;Obtenir&#39; est déjà déclaré
Une instruction `Get` apparaît plusieurs fois dans un bloc de code. La déclaration `Get` déclare une procédure de propriété `Get` utilisée pour récupérer une valeur à une propriété.  
  
 **ID d’erreur :** BC30443  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction `Get` redondante.
