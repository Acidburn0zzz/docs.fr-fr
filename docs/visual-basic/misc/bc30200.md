# &#39;New&#39; n’est pas valide dans ce contexte
Le mot clé `New` apparaît alors qu’un nom de type est demandé. Le contexte nécessite un type valeur qui ne peut pas être combiné avec `New`.  
  
 **ID d’erreur :** BC30200  
  
### Pour corriger cette erreur  
  
-   Supprimez le mot clé `New`.
