# <a name="39new39-is-not-valid-in-this-context"></a>&#39; Nouveau &#39; n’est pas valide dans ce contexte
Le mot clé `New` apparaît alors qu’un nom de type est demandé. Le contexte nécessite un type valeur qui ne peut pas être combiné avec `New`.  
  
 **ID d’erreur :** BC30200  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le mot clé `New` .
