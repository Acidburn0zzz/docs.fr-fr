# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;méthode&gt; &#39; &lt;nom_méthode1&gt; &#39; doit être déclaré &#39;surcharges&#39; , car une autre &#39; &lt;nom_méthode2&gt; &#39; est déclaré &#39; Les surcharges&#39;
Une autre propriété ou méthode portant le même nom est déclarée avec le modificateur `Overloads` .  
  
 **ID d’erreur :** BC31409  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Ajoutez le modificateur `Overloads` à la propriété ou à la méthode.  
  
2.  Supprimez le modificateur `Overloads` de toutes les propriétés ou méthodes portant le même nom.
