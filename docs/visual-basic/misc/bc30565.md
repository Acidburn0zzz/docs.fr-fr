# <a name="array-initializer-has-too-few-dimensions"></a>L’initialiseur de tableau n’utilise pas suffisamment de dimensions
Vous n’avez pas fourni suffisamment de dimensions dans l’initialiseur de votre tableau.  
  
 **ID d’erreur :** BC30565  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez votre initialiseur de tableau pour déterminer le nombre de dimensions nécessaire.  
  
-   Fournissez les arguments manquants.
