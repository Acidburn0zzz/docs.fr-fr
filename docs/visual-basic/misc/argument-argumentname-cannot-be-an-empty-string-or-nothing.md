# <a name="argument-argumentname-cannot-be-an-empty-string-or-nothing"></a>Argument \<nom_argument > ne peut pas être une chaîne vide ou Nothing
L’argument exige une valeur autre qu’une chaîne vide ("") ou `Nothing`. Cette erreur peut être le résultat d’un calcul incorrect de la valeur.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez une valeur valide pour l’argument.
