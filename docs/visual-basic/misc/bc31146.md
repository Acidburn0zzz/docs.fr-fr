# <a name="xml-name-expected"></a>Nom XML attendu
Un littéral XML ou une propriété d’axe XML a été spécifié(e) sans nom.  
  
 **ID d’erreur :** BC31146  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez un nom au littéral XML ou à la propriété d’axe XML.
