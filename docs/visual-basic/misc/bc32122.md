# <a name="cannot-inherit-interface-ltinterfacename1gt-because-the-interface-ltinterfacename2gt-from-which-it-inherits-could-be-identical-to-interface-ltinterfacename3gt-from-which-the-interface-ltinterfacename4gt-inherits-for-some-type-arguments"></a><span data-ttu-id="77f62-102">Ne peut pas hériter de l’interface '&lt;nom_interface1&gt;', car l’interface '&lt;nom_interface2&gt;'dont elle hérite peut être identique à l’interface'&lt;nom_interface3&gt;' à partir de laquelle l’interface '&lt;nom_interface4&gt;' hérite pour certains arguments de type</span><span class="sxs-lookup"><span data-stu-id="77f62-102">Cannot inherit interface '&lt;interfacename1&gt;' because the interface '&lt;interfacename2&gt;' from which it inherits could be identical to interface '&lt;interfacename3&gt;' from which the interface '&lt;interfacename4&gt;' inherits for some type arguments</span></span>
<span data-ttu-id="77f62-103">Une interface générique hérite d’au moins deux interfaces génériques, et deux des héritages peuvent entrer en conflit pour certaines valeurs d’arguments de type.</span><span class="sxs-lookup"><span data-stu-id="77f62-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="77f62-104">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="77f62-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface interfaceY(Of w)  
    Inherits interfaceA(Of w)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceX(Of t1), interfaceY(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="77f62-105">Si `derivedInterface` est construit ou implémenté en fournissant le même type à `t1` et à `t2`, il doit hériter de deux versions de `interfaceA` avec des arguments de type identiques.</span><span class="sxs-lookup"><span data-stu-id="77f62-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="77f62-106">Il en résulte une ambiguïté quant à la version à laquelle accéder.</span><span class="sxs-lookup"><span data-stu-id="77f62-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="77f62-107">**ID d’erreur :** BC32122</span><span class="sxs-lookup"><span data-stu-id="77f62-107">**Error ID:** BC32122</span></span>
