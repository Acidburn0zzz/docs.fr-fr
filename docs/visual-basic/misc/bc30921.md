# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-to-ltregiongt-ltregionnamegt"></a><span data-ttu-id="2c6e7-102">«&lt;nom_type_dérivé&gt;' ne peut pas hériter &lt;type&gt; '&lt;nom_type_base_construit&gt;', car il étend l’accès de type »&lt;nom_type_interne&gt;' à &lt;région&gt; '&lt;regionname&gt;'</span><span class="sxs-lookup"><span data-stu-id="2c6e7-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' to &lt;region&gt; '&lt;regionname&gt;'</span></span>
<span data-ttu-id="2c6e7-103">Une classe ou une interface dérivée essaie d’étendre le niveau d’accès d’un type interne en l’utilisant comme argument de type dans une classe ou une interface de base.</span><span class="sxs-lookup"><span data-stu-id="2c6e7-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="2c6e7-104">Le code suivant peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="2c6e7-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="2c6e7-105">Le code à l’extérieur de `containingClass` n’est pas autorisé à accéder à `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="2c6e7-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="2c6e7-106">Toutefois, `derivedClass` est accessible à partir de n’importe quel code dans le même assembly.</span><span class="sxs-lookup"><span data-stu-id="2c6e7-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="2c6e7-107">Par conséquent, `derivedClass` ne peut pas hériter de `baseClass` s’il utilise `internalStructure` comme argument de type, car cela peut exposer `internalStructure` dans l’ensemble de la région de code de définition.</span><span class="sxs-lookup"><span data-stu-id="2c6e7-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="2c6e7-108">**ID d’erreur :** BC30921</span><span class="sxs-lookup"><span data-stu-id="2c6e7-108">**Error ID:** BC30921</span></span>
