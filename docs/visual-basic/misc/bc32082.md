# <a name="type-argument-typeargumentname-is-declared-mustinherit-and-does-not-satisfy-the-new-constraint-for-type-parameter-typeparametername"></a><span data-ttu-id="9d018-102">Argument de type '\<nom_argument_type >' est déclaré 'MustInherit' et ne satisfait pas la contrainte 'New' pour le paramètre de type '\<nom_paramètre_type >'</span><span class="sxs-lookup"><span data-stu-id="9d018-102">Type argument '\<typeargumentname>' is declared 'MustInherit' and does not satisfy the 'New' constraint for type parameter '\<typeparametername>'</span></span>
<span data-ttu-id="9d018-103">Un type générique est appelé avec une classe `MustInherit` comme argument de type, alors que le paramètre de type correspondant est déclaré avec la contrainte `New` .</span><span class="sxs-lookup"><span data-stu-id="9d018-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="9d018-104">La contrainte `New` requiert que le type passé dans l’argument de type correspondant prenne en charge la création d’objets.</span><span class="sxs-lookup"><span data-stu-id="9d018-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="9d018-105">Toutefois, une classe *abstraite* , c’est-à-dire une classe déclarée comme `MustInherit`, n’expose pas de constructeurs car vous ne pouvez pas créer d’objets à partir de celle-ci.</span><span class="sxs-lookup"><span data-stu-id="9d018-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="9d018-106">**ID d’erreur :** BC32082</span><span class="sxs-lookup"><span data-stu-id="9d018-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9d018-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="9d018-107">To correct this error</span></span>  
  
1.  <span data-ttu-id="9d018-108">Si la classe utilisée dans l’argument de type ne doit pas être abstraite, supprimez le mot clé `MustInherit` de sa déclaration.</span><span class="sxs-lookup"><span data-stu-id="9d018-108">If the class used in the type argument does not need to be abstract, then remove the `MustInherit` keyword from its declaration.</span></span>  
  
2.  <span data-ttu-id="9d018-109">Si la classe utilisée dans l’argument de type doit être abstraite, mais ne doit pas être utilisée pour construire le type générique, passez une autre classe dans l’argument de type.</span><span class="sxs-lookup"><span data-stu-id="9d018-109">If the class used in the type argument needs to be abstract but does not need to be used to construct the generic type, then pass a different class in the type argument.</span></span>  
  
3.  <span data-ttu-id="9d018-110">Si le paramètre de type correspondant ne doit pas créer d’objets du type qui lui est passé, supprimez la contrainte `New` de sa déclaration.</span><span class="sxs-lookup"><span data-stu-id="9d018-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>
