# <a name="member-39ltmembernamegt39-cannot-be-found-in-class-39ltclassnamegt39"></a>Membre &#39; &lt;membername&gt; &#39; ne figure pas dans la classe &#39; &lt;classname&gt;&#39;
Membre '\<nom_membre >' est introuvable dans la classe\<nom_classe >'. Cette condition est généralement le résultat d'un 'Microsoft.VisualBasic.dll' incompatible.  
  
 Un membre défini est introuvable.  
  
 **ID d’erreur :** BC31097  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Si l’erreur persiste, enregistrez votre travail et redémarrez Visual Studio.  
  
3.  Si l’erreur persiste, réinstallez Visual Basic.  
  
4.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
