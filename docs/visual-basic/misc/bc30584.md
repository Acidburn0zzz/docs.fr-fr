# &#39;&lt;type&gt;&#39; ne peut pas &#234;tre h&#233;rit&#233; plus d’une fois
Vous avez dupliqué une instruction `Inherits`.  
  
 **ID d’erreur :** BC30584  
  
### Pour corriger cette erreur  
  
-   Supprimez l’instruction `Inherits` redondante.
