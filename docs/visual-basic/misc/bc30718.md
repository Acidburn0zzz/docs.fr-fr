# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Sélectionnez&#39; instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Select` peuvent uniquement être utilisées dans le code source.  
  
 **ID d’erreur :** BC30718  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Select` du code de débogage.
