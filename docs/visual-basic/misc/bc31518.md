# <a name="assembly-39ltassemblynamegt39-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Assembly &#39; &lt;assemblyname&gt; &#39; ne peut pas être créé, car son chemin d’accès excède 259 caractères
Le chemin de cet assembly dépasse la longueur maximale de 259 caractères.  
  
 **ID d’erreur :** BC31518  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Créez un chemin plus court en utilisant des noms moins longs ou en sélectionnant une hiérarchie de fichiers moins profonde.
