# &#39;&lt;nom_espace_de_noms&gt;&#39; est ambigu
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le compilateur [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] n’a pas de règles de résolution de conflit. Vous devez lever vous\-même toute ambiguïté sur les noms.  
  
 **ID d’erreur :** BC30554  
  
### Pour corriger cette erreur  
  
-   Utilisez un nom qualifié complet.
