# <a name="39ltmathfunction1gt39-is-not-declared"></a>«&lt;mathfunction1&gt;' n’est pas déclaré
«\<mathfunction1 >' n’est pas déclaré. Fonction a été déplacé vers la classe System.Math et s’appelle désormais '\<mathfunction2 > ».  
  
 Plusieurs fonctions qui étaient intrinsèques à [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] dans les versions précédentes ont été déplacés vers le <xref:System.Math?displayProperty=fullName>espace de noms.</xref:System.Math?displayProperty=fullName> Leurs fonctionnalités sont ainsi plus généralement accessibles à tous les langages de programmation.  
  
 **ID d’erreur :** BC30819  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez les méthodes déclarées dans <xref:System.Math?displayProperty=fullName>.</xref:System.Math?displayProperty=fullName>
