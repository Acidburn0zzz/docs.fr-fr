# <a name="lambda-expression-cannot-be-converted-to-39lttypenamegt39-because-type-39lttypenamegt39-is-declared-39mustinherit39-and-cannot-be-created"></a>Expression lambda ne peut pas être convertie en &#39; &lt;typename&gt; &#39; car type &#39; &lt;typename&gt; &#39; est déclaré &#39;MustInherit&#39; et ne peut pas être créé.
Une instruction tente de convertir une expression lambda en un type qui peut être uniquement une classe de base et ne peut pas être utilisé pour créer une instance.  
  
 **ID d’erreur :** BC36626
