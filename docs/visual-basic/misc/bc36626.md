# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-type-lttypenamegt-is-declared-mustinherit-and-cannot-be-created"></a>Expression lambda ne peut pas être convertie en '&lt;typename&gt;', car type '&lt;typename&gt;' est déclaré 'MustInherit' et ne peut pas être créé.
Une instruction tente de convertir une expression lambda en un type qui peut être uniquement une classe de base et ne peut pas être utilisé pour créer une instance.  
  
 **ID d’erreur :** BC36626
