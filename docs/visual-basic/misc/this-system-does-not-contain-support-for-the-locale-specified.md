# <a name="this-system-does-not-contain-support-for-the-locale-specified"></a>Ce système n’offre pas de prise en charge des paramètres régionaux spécifiés
Vous essayez d’utiliser l’énumération `VbStrConv` , mais votre système ne prend pas en charge les paramètres régionaux spécifiés.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Mettez à niveau l’ordinateur pour prendre en charge les paramètres régionaux, ou exécutez l’application sur un ordinateur qui les prend en charge.
