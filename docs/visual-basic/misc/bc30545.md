# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Un accès à la propriété doit assigner la propriété ou utiliser sa valeur
Vous avez tenté d’accéder à une propriété sans lui assigner de valeur ou sans utiliser sa valeur.
  
 **ID d’erreur :** BC30545  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Affecter une valeur à la propriété.  
  
     \- ou -  
  
-   Utilisez la valeur de la propriété.
