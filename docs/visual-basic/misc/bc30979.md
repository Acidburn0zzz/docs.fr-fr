# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Option Infer&#39; peut être suivie que de &#39;sur&#39; ou &#39;hors tension&#39;
Les instructions`Option Infer` doivent spécifier `On` ou `Off`. D’autres valeurs ne sont pas valides.  
  
 **ID d’erreur :** BC30979  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Spécifiez soit `Option Infer On`, soit `Option Infer Off`.
