# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39; &lt;typename&gt;&#39; est un type de &#39;&lt; ContainerName&gt;&#39; et ne peut pas être utilisé en tant qu’expression
L’un des noms utilisés est déjà déclaré comme nom d’un type et ne peut pas être utilisé dans une expression.  
  
 **ID d’erreur :** BC30691  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Attribuez un nom unique au membre.  
  
-   Mettez le nom du membre entre crochets.
