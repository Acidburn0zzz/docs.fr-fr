# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>«&lt;typename&gt;'est un type dans'&lt;containername&gt;' et ne peut pas être utilisé en tant qu’expression
L’un des noms utilisés est déjà déclaré comme nom d’un type et ne peut pas être utilisé dans une expression.  
  
 **ID d’erreur :** BC30691  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Attribuez un nom unique au membre.  
  
-   Mettez le nom du membre entre crochets.
