# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39;MyClass&#39; ne peut pas être utilisé en dehors d’une classe
Vous avez utilisé le mot clé `MyClass` en dehors d’une structure de classe.  
  
 **ID d’erreur :** BC30470  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déplacez la déclaration `MyClass` vers votre structure de classe.
