# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Le nom d’une fonction d’agrégation ne peut pas être utilisé avec un caractère de type
Un caractère de type a été ajouté à la fin du nom d’une fonction d’agrégation.  
  
 **ID d’erreur :** BC36617  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Supprimez le caractère de type du nom de la fonction d’agrégation.
