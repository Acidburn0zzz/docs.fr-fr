# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>Le &#39; &lt;mot clé&gt; &#39; est utilisé pour surcharger les membres hérités ; n’utilisez pas le &#39; &lt;mot clé&gt; &#39; mot clé lors de la surcharge &#39;Sub New&#39;
Un constructeur est déclaré avec le mot clé `Overloads` .  
  
 Visual Basic ne prend pas en charge l’héritage ni la surcharge des constructeurs.  
  
 **ID d’erreur :** BC32040  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le mot clé `Overloads` de toutes les déclarations de constructeur.
