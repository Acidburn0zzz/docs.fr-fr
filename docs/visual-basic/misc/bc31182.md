# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Valeur d’attribut non valide. 'value1' ou 'value2' attendu.
Une valeur non valide pour un attribut dans un littéral de document XML a été spécifiée. Les valeurs d’attributs valides pour un littéral de document XML sont basées sur la spécification XML 1.0.  
  
 **ID d’erreur :** BC31182  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Remplacez la valeur d’attribut par une valeur conforme à la spécification XML 1.0.
