# <a name="nullable-types-are-not-supported-because-the-current-version-of-39mscorlibdll39-does-not-define-the-type-39nullableof-t39"></a>Types Nullable ne sont pas pris en charge, car la version actuelle de &#39;mscorlib.dll&#39; ne définit pas le type &#39;Nullable (of T)&#39;
La version de .NET Framework que vous utilisez ne prend pas en charge les types Nullable.  
  
 **ID d’erreur :** BC32130
