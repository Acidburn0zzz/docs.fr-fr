# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Using' ne sont pas valides dans la fenêtre Exécution
Les instructions`Using` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30186  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction `Using` de la fenêtre **Exécution** .
