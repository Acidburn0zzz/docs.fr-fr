# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>&#39;À l’aide de&#39; instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Using` peuvent uniquement être utilisées dans le code source.  
  
 **ID d’erreur :** BC30186  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction `Using` de la fenêtre **Exécution** .
