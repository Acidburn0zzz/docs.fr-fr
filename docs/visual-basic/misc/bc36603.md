# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="f9730-102">Impossible de convertir un tableau multidimensionnel en une arborescence d’expression</span><span class="sxs-lookup"><span data-stu-id="f9730-102">Multi-dimensional array cannot be converted to an expression tree</span></span>
<span data-ttu-id="f9730-103">La plupart des expressions peuvent être converties en arborescences d’expression, mais pas les tableaux multidimensionnels.</span><span class="sxs-lookup"><span data-stu-id="f9730-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="f9730-104">Par exemple, le code suivant génère l’erreur suivante :</span><span class="sxs-lookup"><span data-stu-id="f9730-104">For example, the following code causes this error:</span></span>  
  
```vb

    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _

    End Sub  
End Module  
```
