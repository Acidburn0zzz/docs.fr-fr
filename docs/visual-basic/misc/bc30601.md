# <a name="39-39-expected"></a>&#39;-&#39; attendu
La syntaxe de cette ligne est incorrecte.  
  
 **ID d’erreur :** BC30601  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez la syntaxe du code qui a déclenché cette erreur.
