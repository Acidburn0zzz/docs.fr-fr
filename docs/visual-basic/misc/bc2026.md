# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Numéro d’avertissement '&lt;nombre&gt;« pour l’option'&lt;NomOption&gt;' est non configurable ou non valide
Un ID d’avertissement n’est pas valide. Les ID doivent être des valeurs entières supérieures à 0.  
  
 **ID d’erreur :** BC2026  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez l’ID d’avertissement fourni pour vous assurer qu’il est valide.
