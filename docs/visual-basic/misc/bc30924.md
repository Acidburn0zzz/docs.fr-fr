# <a name="ltnamegt-cannot-expose-type-lttype1gt-used-in-the-underlying-delegate-type-ltdelegatetypegt-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-lttype2gt"></a>«&lt;nom&gt;'ne peut pas exposer le type'&lt;type1&gt;'utilisé dans le type délégué sous-jacent'&lt;type_délégué&gt;' de l’événement qu’il implémente en dehors du projet via &lt;spécificateur&gt; '&lt;type2&gt;'
Le code expose un type privé en dehors de la classe publique.  
  
 **ID d’erreur :** BC30924  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez le type `Public` ou utilisez un autre type.
