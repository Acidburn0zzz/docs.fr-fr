# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39;End SyncLock&#39; doit être précédé d’une mise en correspondance &#39;SyncLock&#39;
`SyncLock` commencent par le mot clé `SyncLock` et se terminent par la construction `End``SyncLock` .  
  
 **ID d’erreur :** BC30674  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que le bloc `SyncLock` commence par une instruction `SyncLock` .
