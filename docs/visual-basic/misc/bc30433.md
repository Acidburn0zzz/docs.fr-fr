# <a name="methods-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Les méthodes d’un Module ne peuvent pas être déclarés &#39; &lt;spécificateur&gt;&#39;
Vous avez utilisé un spécificateur non valide pour un méthode comprise dans une instruction `Module` . Les modules ne peuvent jamais être instanciés, ils ne prennent pas en charge l’héritage et ils ne peuvent pas implémenter d’interfaces.  
  
 **ID d’erreur :** BC30433  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
