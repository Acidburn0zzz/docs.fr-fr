# <a name="argument-ltargumentname1gt-must-be-less-than-or-equal-the-length-of-argument-ltargumentname2gt"></a>Argument '&lt;nom_argument1&gt;' doit être inférieure ou égale la longueur de l’argument '&lt;nom_argument2&gt;'
La longueur d’un argument fourni dépasse la longueur de l’argument qu’il remplace.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Faites en sorte que la longueur du premier argument soit inférieure ou égale à celle du second argument.
