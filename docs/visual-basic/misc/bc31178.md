# <a name="expected-closing-3939-for-xml-entity"></a>Fermant attendu &#39;; &#39; pour une entité XML
Il manque un point-virgule (;) fermant dans une référence d’entité XML (par exemple, `>`).  
  
 **ID d’erreur :** BC31178  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Incluez le point-virgule à la fin de la référence d’entité XML.
