# <a name="cannot-inherit-interface-39ltinterfacename1gt39-because-it-could-be-identical-to-interface-39ltinterfacename2gt39-from-which-the-interface-39ltinterfacename3gt39-inherits-for-some-type-arguments"></a><span data-ttu-id="a5c88-102">Ne peut pas hériter de l’interface &#39; &lt;nom_interface1&gt; &#39; , car il peut être identique à l’interface &#39; &lt;nom_interface2&gt; &#39; à partir de laquelle l’interface &#39; &lt;nom_interface3&gt; &#39; hérite pour certains arguments de type</span><span class="sxs-lookup"><span data-stu-id="a5c88-102">Cannot inherit interface &#39;&lt;interfacename1&gt;&#39; because it could be identical to interface &#39;&lt;interfacename2&gt;&#39; from which the interface &#39;&lt;interfacename3&gt;&#39; inherits for some type arguments</span></span>
<span data-ttu-id="a5c88-103">Une interface générique hérite d’au moins deux interfaces génériques, et deux des héritages peuvent entrer en conflit pour certaines valeurs d’arguments de type.</span><span class="sxs-lookup"><span data-stu-id="a5c88-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="a5c88-104">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="a5c88-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="a5c88-105">Si `derivedInterface` est construit ou implémenté en fournissant le même type à `t1` et à `t2`, il doit hériter de deux versions de `interfaceA` avec des arguments de type identiques.</span><span class="sxs-lookup"><span data-stu-id="a5c88-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="a5c88-106">Il en résulte une ambiguïté quant à la version à laquelle accéder.</span><span class="sxs-lookup"><span data-stu-id="a5c88-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="a5c88-107">**ID d’erreur :** BC32123</span><span class="sxs-lookup"><span data-stu-id="a5c88-107">**Error ID:** BC32123</span></span>
