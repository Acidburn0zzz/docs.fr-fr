# <a name="type-of-argument-ltargumentnamegt-is-lttypenamegt-which-is-not-numeric"></a>Type d’argument '&lt;nom_argument&gt;'is'&lt;typename&gt;», qui n’est pas numérique
Un argument non numérique apparaît dans une situation dans laquelle un argument numérique est attendu.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Assurez-vous que l’argument est numérique.
