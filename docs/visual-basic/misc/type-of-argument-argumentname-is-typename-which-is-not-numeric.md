# <a name="type-of-argument-39ltargumentnamegt39-is-39lttypenamegt39-which-is-not-numeric"></a>Type d’argument &#39; &lt;nom_argument&gt; &#39; est &#39; &lt;typename&gt;&#39;, qui n’est pas numérique
Un argument non numérique apparaît dans une situation dans laquelle un argument numérique est attendu.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Assurez-vous que l’argument est numérique.
