# <a name="property-or-field-39ltnamegt39-does-not-have-a-valid-attribute-type"></a>Propriété ou champ &#39; &lt;nom&gt; &#39; n’est pas un type d’attribut valide
Vous utilisez peut-être un constructeur d’attribut qui n’accepte pas un type d’attribut valide, même si l’attribut peut avoir d’autres constructeurs appropriés.  
  
 **ID d’erreur :** BC30659  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez l’utilisation de cet attribut.
