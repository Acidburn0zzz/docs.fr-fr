# <a name="unable-to-create-temp-file-in-path-39ltfilenamegt39-lterror-messagegt"></a>Impossible de créer le fichier temporaire dans le chemin d’accès '&lt;nom de fichier&gt;' : &lt;message d’erreur&gt;
Le compilateur [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] appelle Assembly Linker (Al.exe, également appelé Alink) pour générer un assembly avec un manifeste. L’éditeur de liens a signalé une erreur pendant la création d’un fichier ou pendant l’écriture dans celui-ci d’une ressource en mémoire. Il peut s’agir d’un problème de configuration.  
  
 **ID d’erreur :** BC30138  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Examinez le message d’erreur cité et consultez la rubrique [Al.exe Tool Errors and Warnings](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) pour davantage d’explications et de conseils.  
  
2.  Si l'erreur persiste, rassemblez des informations sur ses circonstances et avertissez les services de support technique Microsoft.
