# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>Une méthode ne peut pas contenir à la fois un &#39;essayez&#39; instruction et un &#39;en cas d’erreur&#39; ou &#39;reprise&#39; instruction
Vous avez associé une instruction `Try` à `On Error` ou `Resume`.  
  
 **ID d’erreur :** BC30544  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez `On Error` ou `Resume`.
