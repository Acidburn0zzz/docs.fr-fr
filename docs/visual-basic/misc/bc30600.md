# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>'Variable des WithEvents ne déclenche pas les événements d’instance accessibles par'\<nom_conteneur >'
L’objet auquel vous faites référence ne déclenche pas d’événements accessibles.  
  
 **ID d’erreur :** BC30600  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Modifiez les modificateurs d’accès pour l’événement ou choisissez un événement accessible dans le contexte actuel.
