# <a name="white-space-may-not-appear-here"></a>L’espace ne peut pas s’afficher ici
Le nom d’un littéral d’élément XML comprend un ou plusieurs espaces.  
  
 **ID d’erreur :** BC31177  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les espaces du nom du littéral d’élément XML.
