# <a name="project-39ltprojectnamegt39-cannot-generate-a-reference-to-file-39ltfilenamegt39"></a>Projet &#39; &lt;nom_projet&gt; &#39; Impossible de générer une référence au fichier &#39; &lt;nom de fichier&gt;&#39;
Projet ' <`projectname`>' ne peut pas générer une référence au fichier ' <`filename`>'. Vous devrez peut-être supprimer et rajouter cette référence, puis effectuer une régénération complète de la solution.  
  
 Une erreur s’est produite lors de la création d’une référence à un fichier.  
  
 **ID d’erreur :** BC31514  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez la référence en cliquant avec le bouton droit sur l’élément dans **Références du projet** dans l’ **Explorateur de solutions** puis en cliquant sur **Supprimer**.  
  
2.  Restaurez la référence en cliquant avec le bouton droit sur **Références** dans l’ **Explorateur de solutions** puis en cliquant sur **Ajouter une référence**.
