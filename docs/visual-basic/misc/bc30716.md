# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Enfin&#39; instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Finally` peuvent uniquement être utilisées dans le code source.  
  
 **ID d’erreur :** BC30716  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Finally` du code de débogage.
