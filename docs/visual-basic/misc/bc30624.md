# <a name="39structure39-statement-must-end-with-a-matching-39end-structure39"></a>&#39;Structure&#39; instruction doit se terminer par une mise en correspondance &#39;End Structure&#39;
Les structures commencent par une instruction `Structure` et se terminent par une construction `End Structure` .  
  
 **ID d’erreur :** BC30624  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que le bloc de structure se termine par une instruction `End Structure` .
