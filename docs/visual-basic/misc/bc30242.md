# <a name="39ltmodifiergt39-is-not-valid-on-a-method-declaration"></a>&#39;&lt;modificateur&gt; &#39; n’est pas valide dans une déclaration de méthode
Une instruction `Function` ou `Sub` contient un mot clé non valide.  
  
 **ID d’erreur :** BC30242  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le mot clé non valide de l’instruction `Function` ou `Sub` .
