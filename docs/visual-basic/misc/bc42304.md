-   [La balise de commentaire XML '\<balise >' n’est pas autorisée sur un «\<élément >' élément de langage](../../visual-basic/misc/bc42306.md)  
  
-   [Le paramètre de commentaire XML '\<paramètre >' ne correspond pas à un paramètre sur le correspondant \<mot clé > instruction](../../visual-basic/misc/bc42307.md)

-   [Balise de commentaire XML 'include' doit avoir un '\<attribut >' attribut](../../visual-basic/misc/bc42310.md)
