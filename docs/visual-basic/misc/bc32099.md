# <a name="comma-or--expected-type-parameter-list"></a>Virgule ou ’)’ attendue (liste de paramètres de type)
Une liste de paramètres de type contient un élément inattendu. Utilisez des virgules (`,`) pour séparer les paramètres de type dans ce genre de liste, et utilisez une parenthèse fermante (`)`) à la fin de la liste.  
  
 **ID d’erreur :** BC32099  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une virgule et continuez la liste de paramètres de type, ou ajoutez un caractère de parenthèse fermante pour terminer la liste.
