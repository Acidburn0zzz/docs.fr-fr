# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Référence au module &#39; requise &lt;modulename&gt;&#39; contenant la définition d’événements &#39;&lt; nom_événement&gt;&#39;
Une référence est requise pour le module ' <`modulename`>' contenant la définition pour l’événement ' <`eventname`>'. Ajoutez-en une à votre projet.  
  
 L’événement est défini dans un module qui n’est pas directement référencé dans votre projet. Le compilateur [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] nécessite une référence pour éviter toute ambiguïté au cas où l’événement serait défini dans plusieurs modules.  
  
 **ID d’erreur :** BC30006  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez le nom du module non référencé dans vos références de projet.
