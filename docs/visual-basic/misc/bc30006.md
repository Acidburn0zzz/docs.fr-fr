# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Une référence au module requise &#39; &lt;modulename&gt; &#39; contenant la définition d’événement &#39; &lt;eventname&gt;&#39;
Une référence est requise pour le module ' <`modulename`>' contenant la définition pour l’événement ' <`eventname`>'. Ajoutez-en une à votre projet.  
  
 L’événement est défini dans un module qui n’est pas directement référencé dans votre projet. Le compilateur Visual Basic requiert une référence afin d’éviter toute ambiguïté au cas où l’événement est défini dans plusieurs modules.  
  
 **ID d’erreur :** BC30006  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez le nom du module non référencé dans vos références de projet.
