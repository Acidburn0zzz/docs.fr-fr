# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Référence au module requise '&lt;modulename&gt;'contenant la définition pour l’événement'&lt;eventname&gt;»
Référence au module requise '`modulename`>' contenant la définition pour l’événement '`eventname`> ». Ajoutez-en une à votre projet.  
  
 L’événement est défini dans un module qui n’est pas directement référencé dans votre projet. Le [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilateur requiert une référence afin d’éviter toute ambiguïté au cas où l’événement est défini dans plusieurs modules.  
  
 **ID d’erreur :** BC30006  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez le nom du module non référencé dans vos références de projet.
