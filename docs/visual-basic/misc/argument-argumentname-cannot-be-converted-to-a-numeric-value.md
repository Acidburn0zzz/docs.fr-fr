# <a name="argument-argumentname-cannot-be-converted-to-a-numeric-value"></a>Argument '\<nom_argument >' ne peut pas être convertie en valeur numérique
Une conversion a tenté de convertir une variable, telle qu’une chaîne, en valeur numérique.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Recherchez une conversion significative.
