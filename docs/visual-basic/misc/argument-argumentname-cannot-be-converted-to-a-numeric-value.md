# <a name="argument-ltargumentnamegt-cannot-be-converted-to-a-numeric-value"></a>Argument '&lt;nom_argument&gt;' ne peut pas être convertie en valeur numérique
Une conversion a tenté de convertir une variable, telle qu’une chaîne, en valeur numérique.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Recherchez une conversion significative.
