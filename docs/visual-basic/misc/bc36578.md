# <a name="too-few-type-arguments"></a>Arguments de type insuffisants
Une méthode générique a été appelée avec moins d’arguments de type qu’il n’y a de paramètres de type.  
  
 Quand vous appelez une méthode générique, vous devez fournir un argument de type pour chaque paramètre de type défini par cette méthode.  
  
 **ID d’erreur :** BC36578  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez des arguments de type à votre liste d’arguments de type pour qu’il y en ait un pour chaque paramètre de type dans la méthode générique que vous appelez.
