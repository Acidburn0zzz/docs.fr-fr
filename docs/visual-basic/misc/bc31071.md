# <a name="structure-in-an-interface-cannot-be-declared-ltspecifiergt"></a>Une structure dans une interface ne peut pas être déclarée '&lt;spécificateur&gt;'
Un modificateur non valide a été spécifié avec une structure déclarée dans une interface.  
  
 **ID d’erreur :** BC31071  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les modificateurs tels que `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`ou `Overridable`.
