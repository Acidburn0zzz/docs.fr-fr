# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;Événement personnalisé&#39; doit se terminer par une mise en correspondance &#39;événement de fin&#39;
Une déclaration `Custom Event` doit se terminer par une instruction `End Event` .  
  
 **ID d’erreur :** BC31114  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que la déclaration `Custom Event` se termine par une instruction `End Event` .
