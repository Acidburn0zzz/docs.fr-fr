# <a name="method-ltmethodnamegt-cannot-be-declared-partial-because-only-one-method-ltmethodnamegt-can-be-marked-partial"></a>Méthode '&lt;nom_méthode&gt;' ne peut pas être déclaré 'Partial', car qu’une seule méthode '&lt;nom_méthode&gt;' peut être marquée comme 'Partial'
Vous ne pouvez avoir deux déclarations de méthode partielle ayant les mêmes nom et signature.  
  
 **ID d’erreur :** BC31433  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Modifiez le nom ou la signature de l’une des déclarations de méthode partielle.
