# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-accepts-this-number-of-arguments"></a>A échoué, car la résolution de surcharge n’accessible &#39; &lt;méthode&gt; &#39; accepte ce nombre d’arguments
Vous avez appelé une méthode surchargée, mais le compilateur ne peut pas trouver de méthode avec une liste d’arguments correspondante.  
  
 **ID d’erreur :** BC30516  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez la liste d’arguments que vous avez fournie.
