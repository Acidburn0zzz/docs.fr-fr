# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39; &lt;nom_interface1&gt;&#39; ne peut pas implémenter &#39;&lt; MethodName&gt;&#39; car il n’existe aucune correspondance &lt;méthode&gt; sur l’interface &#39;&lt; nom_interface2&gt;&#39;
Vous avez tenté d’implémenter une méthode sur une interface alors qu’il n’y a aucune méthode correspondante sur l’interface à partir de laquelle la première interface a été dérivée.  
  
 **ID d’erreur :** BC30401  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que vous avez correctement orthographié le nom de la méthode ou fournissez la méthode manquante.
