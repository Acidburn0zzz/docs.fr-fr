# <a name="ltinterfacename1gt-cannot-implement-ltmethodnamegt-because-there-is-no-matching-ltmethodgt-on-interface-ltinterfacename2gt"></a>«&lt;nom_interface1&gt;'ne peut pas implémenter'&lt;nom_méthode&gt;', car il n’existe aucune &lt;méthode&gt; sur l’interface '&lt;nom_interface2&gt;»
Vous avez tenté d’implémenter une méthode sur une interface alors qu’il n’y a aucune méthode correspondante sur l’interface à partir de laquelle la première interface a été dérivée.  
  
 **ID d’erreur :** BC30401  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que vous avez correctement orthographié le nom de la méthode ou fournissez la méthode manquante.
