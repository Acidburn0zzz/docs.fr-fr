# <a name="39variant39-is-no-longer-a-supported-type-use-the-39object39-type-instead"></a>&#39; Variante &#39; n’est plus un type pris en charge ; utiliser le &#39; objet &#39; type à la place
Le type de données `Object` remplace `Variant`.  
  
 **ID d’erreur :** BC30804  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez l’élément comme `Object`.
