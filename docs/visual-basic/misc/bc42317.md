# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Le paramètre de type de commentaire XML &lt;paramètre&gt; ne correspond pas à un paramètre de type sur le correspondant &lt;mot clé&gt; instruction
Le paramètre de commentaire XML \<paramètre > ne correspond pas à un paramètre de type sur le correspondant \<mot clé > instruction. Le commentaire XML est ignoré.  
  
 Une balise de commentaire XML a été spécifiée pour certains (mais pas tous) les paramètres de type d’une méthode.  
  
 **ID d’erreur :** BC42317  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez des balises de commentaire XML pour tous les paramètres de type de la méthode.
