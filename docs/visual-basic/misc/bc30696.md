# <a name="39ltmembername1gt39-and-39ltmembername2gt39-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>&#39; &lt;nom_membre1&gt;&#39; et &#39;&lt; nom_membre2&gt;&#39; ne peut pas se surcharger mutuellement, car seuls les types des paramètres optionnels les différencient
Les membres surchargés doivent avoir des types de données différents pour un ou plusieurs paramètres obligatoires.  
  
 **ID d’erreur :** BC30696  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Définissez au moins un paramètre obligatoire en tant que type de données différent.
