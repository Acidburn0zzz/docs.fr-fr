# <a name="error-embedding-win32-manifest-manifest-file-ltfilegt-cannot-be-found"></a>Erreur d’incorporation du manifeste Win32 : le fichier manifeste &lt;fichier&gt; ne peut pas être trouvé.
Le fichier spécifié pour l’option du compilateur `/win32manifest` est introuvable.  
  
 **ID d’erreur :** BC31192  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Indiquez un nom de fichier et un chemin valides pour l’option du compilateur `/win32manifest` .
