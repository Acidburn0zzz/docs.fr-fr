# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39; #End region &#39; doit être précédé d’une mise en correspondance &#39; #Region &#39;
`#Region` vous permet de spécifier un bloc de code que vous pouvez développer ou réduire lors de l’utilisation de la fonctionnalité mode Plan de l’éditeur de code [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] . Le début et la fin des instructions `#Region` doivent figurer dans le même bloc de code.  
  
 **ID d’erreur :** BC30680  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Insérez `#Region` à l’emplacement approprié avant l’instruction `#End``Region` correspondante.
