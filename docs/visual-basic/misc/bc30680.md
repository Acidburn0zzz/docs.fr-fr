# <a name="end-region-must-be-preceded-by-a-matching-region"></a>'#End Region' doit être précédé d’un '#Region' correspondant
Avec `#Region` vous pouvez spécifier un bloc de code pour développer ou réduire lors de l’utilisation de la fonctionnalité mode plan de l’éditeur de Code Visual Studio. Le début et la fin des instructions `#Region` doivent figurer dans le même bloc de code.  
  
 **ID d’erreur :** BC30680  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Insérez `#Region` à l’emplacement approprié avant l’instruction `#End Region` correspondante.
