# <a name="operator-is-not-valid-for-type-39lttypenamegt39"></a>Opérateur n’est pas valide pour le type &#39; &lt;typename&gt;&#39;
Un opérateur n’est pas valide pour les types.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que l’opérateur est compatible avec les types utilisés.
