# <a name="operator-is-not-valid-for-type-lttypenamegt"></a>Opérateur n’est pas valide pour le type '&lt;typename&gt;'
Un opérateur n’est pas valide pour les types.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que l’opérateur est compatible avec les types utilisés.
