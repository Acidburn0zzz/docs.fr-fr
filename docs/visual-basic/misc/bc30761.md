# <a name="project-39ltprojectnamegt39-cannot-reference-project-39ltprojectnamegt39-because-39ltprojectnamegt39-directly-or-indirectly-references-39ltprojectnamegt39"></a>Projet &#39; &lt;nom_projet&gt;&#39; ne peut pas référencer de projet &#39;&lt; nom du projet&gt;&#39; car &#39;&lt; nom du projet&gt;&#39; directement ou indirectement référence &#39;&lt; nom du projet&gt;&#39;
Un projet inclut une référence à un deuxième projet. Le deuxième projet inclut, à son tour, une référence au projet dans lequel il est référencé.  
  
 **ID d’erreur :** BC30761  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’une des références.
