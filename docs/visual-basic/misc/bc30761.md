# <a name="project-ltprojectnamegt-cannot-reference-project-ltprojectnamegt-because-ltprojectnamegt-directly-or-indirectly-references-ltprojectnamegt"></a>Projet «&lt;nom_projet&gt;'ne peut pas référencer le projet'&lt;nom_projet&gt;'car'&lt;nom_projet&gt;'référence directement ou indirectement'&lt;nom_projet &gt;'
Un projet inclut une référence à un deuxième projet. Le deuxième projet inclut, à son tour, une référence au projet dans lequel il est référencé.  
  
 **ID d’erreur :** BC30761  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’une des références.
