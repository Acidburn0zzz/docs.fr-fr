# <a name="ltmembernamegt-has-the-same-name-as-a-type-parameter"></a>«&lt;membername&gt; porte le même nom qu’un paramètre de type
Un membre d’une classe ou d’une structure générique porte le même nom qu’un paramètre de type de cette classe ou de cette structure.  
  
 Chaque variable, constante, énumération, procédure, propriété, événement et délégué doit avoir un nom différent de celui des paramètres de type.  
  
 **ID d’erreur :** BC32054  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez le nom du membre de la classe ou de la structure afin qu’il soit différent de ceux des paramètres de type.
