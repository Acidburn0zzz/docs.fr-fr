# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>Les spécificateurs et les attributs ne sont pas valides pour les instructions 'Namespace'.
Les espaces de noms sont utilisés pour l’organisation hiérarchique des blocs de programme et des éléments de programmation. Ils ne prennent aucun modificateur ou attribut.  
  
 **ID d’erreur :** BC32026  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez tous les mots clés et blocs d’attributs de l’instruction `Namespace` .
