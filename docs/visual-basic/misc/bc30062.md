# &#39;Me&#39; ne peut pas &#234;tre la cible d’une assignation
Le mot clé `Me` fait toujours référence à l’instance spécifique d’une classe dans laquelle le code en cours d’exécution. Sa valeur ne peut pas être modifiée.  
  
 **ID d’erreur :** BC30062  
  
### Pour corriger cette erreur  
  
-   Supprimez l’instruction qui assigne une valeur à `Me`.
