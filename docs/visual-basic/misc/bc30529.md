# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Si l’un des paramètres est explicitement typé, tous doivent l’être
Vous avez combiné différents types de déclarations de type dans une fonction.  
  
 **ID d’erreur :** BC30529  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que tous les paramètres sont explicitement typés.
