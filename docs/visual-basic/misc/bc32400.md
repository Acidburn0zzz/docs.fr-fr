# <a name="class-39ltclassnamegt39-could-not-be-created-lterrorgt"></a>La classe&lt;classname&gt;' a pas pu être créé : &lt;erreur&gt;
Une initialisation de classe a échoué. Une cause possible est un élément endommagé [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] installation.  
  
 **ID d’erreur :** BC32400  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Examinez le message d’erreur et prenez les mesures nécessaires.  
  
2.  Réinstallez le [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] et recompilez le programme.  
  
3.  Si l'erreur se produit à nouveau, réinstallez le compilateur [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
4.  Si l’erreur persiste après la réinstallation, rassemblez des informations sur ses circonstances, puis avertissez les services de support technique Microsoft.
