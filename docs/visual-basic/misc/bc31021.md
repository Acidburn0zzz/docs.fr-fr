# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Compilateur Visual Basic ne peut pas récupérer de l’erreur suivante : &lt;erreur&gt;
Le [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilateur ne peut pas récupérer de l’erreur suivante : `error`>.  
  
 Certaines fonctionnalités de l’IDE ont été désactivées.  
  
 Une erreur interne s’est produite dans le [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilateur.  
  
 **ID d’erreur :** BC31021  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
2.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
3.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
