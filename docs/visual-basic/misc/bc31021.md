# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Compilateur Visual Basic ne peut pas récupérer à partir de l’erreur suivante : &lt;erreur&gt;
Le [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compilateur ne peut pas récupérer à partir de l’erreur suivante : <`error`>.  
  
 Certaines fonctionnalités de l’IDE ont été désactivées.  
  
 Une erreur interne s’est produite dans le compilateur [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] .  
  
 **ID d’erreur :** BC31021  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
2.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
