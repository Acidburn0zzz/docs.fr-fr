# <a name="ltspecifier1gt-and-ltspecifier2gt-cannot-be-combined"></a>«&lt;spécificateur1&gt;'et'&lt;spécificateur2&gt;' ne peut pas être combiné
Deux spécificateurs incompatibles, tels que `Shadows` et `Overrides` , sont utilisés ensemble.  
  
 **ID d’erreur :** BC31408  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’un des deux spécificateurs.
