# <a name="specifier1-and-specifier2-cannot-be-combined"></a>«\<spécificateur1 >' et '\<spécificateur2 >' ne peut pas être combiné
Deux spécificateurs incompatibles, tels que `Shadows` et `Overrides` , sont utilisés ensemble.  
  
 **ID d’erreur :** BC31408  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’un des deux spécificateurs.
