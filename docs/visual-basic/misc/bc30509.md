# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;spécificateur1&gt; &lt;type&gt; ne peut pas hériter d’un &lt;spécificateur2&gt; &lt;type&gt; , car il étend l’accès de la base de &lt;type&gt;
Vous avez fait en sorte qu’une classe publique hérite d’une classe de base marquée `Private` ou `Friend`.  
  
 **ID d’erreur :** BC30509  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez la classe de base `Public` ou la classe héritière `Private` ou `Friend`.
