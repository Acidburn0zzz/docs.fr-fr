# <a name="parameter-already-declared-with-name-argumentname"></a>Paramètre déjà déclaré avec le nom '\<nom_argument >'
Une déclaration de procédure définit deux arguments avec le même nom. Tous les noms d’arguments dans une déclaration de procédure doivent être uniques.  
  
 **ID d’erreur :** BC30237  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez le nom de l’un des arguments.
