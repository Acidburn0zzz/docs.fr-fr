# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argument &#39; Accéder au &#39; n’est pas valide. Valeurs valides pour ajouter le mode sont &#39;OpenAccess.Write &#39; et &#39;OpenAccess.Default &#39;
Une valeur d’accès n’est pas valide en mode Append.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Remplacez la valeur par `OpenAccess.Write` ou `OpenAccess.Default`.
