# <a name="cannot-find-net-framework-directory-lterrorgt"></a>Impossible de trouver le répertoire .NET Framework : &lt;erreur&gt;
Le répertoire où le [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] sont stockés les assemblys n’a pas été trouvé.  
  
 **ID d’erreur :** BC31508  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Si l'erreur se produit encore, enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
3.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
4.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
