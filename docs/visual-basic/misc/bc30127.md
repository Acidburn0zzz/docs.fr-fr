# <a name="attribute-attributename-is-not-valid-error"></a>Attribut '\<attributename >' n’est pas valide : \<erreur >
Le Common Language Runtime n’accepte pas l’attribut dans le contexte actuel.  
  
 **ID d’erreur :** BC30127  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez l’attribut de la déclaration ou remplacez-le par un attribut valide pour le contexte.  
  
2.  Examinez le message d’erreur et prenez les mesures nécessaires.
