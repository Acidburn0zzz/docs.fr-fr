# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>Référence au module requise '&lt;modulename&gt;'contenant la classe de base'&lt;classname&gt;»
Référence au module requise '\<modulename >' contenant la classe de base\<classname > ». Ajoutez-en une à votre projet.  
  
 La classe est définie dans un module qui n’est pas directement référencé dans votre projet. Le [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilateur requiert une référence afin d’éviter toute ambiguïté au cas où la classe est définie dans plusieurs modules.  
  
 **ID d’erreur :** BC30008  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez le nom du module non référencé dans vos références de projet.
