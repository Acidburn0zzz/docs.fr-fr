# <a name="39ltspecifiergt39-is-not-valid-on-a-constant-declaration"></a>&#39; &lt;spécificateur&gt;&#39; n’est pas valide dans une déclaration de constante
Une instruction `Const` contient un mot clé non valide. Une instruction `Const` peut inclure uniquement les mots clés `Friend`, `Private`, `Protected`, `Public`ou `Shadows` .  
  
 **ID d’erreur :** BC30233  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que le mot clé est correctement orthographié.  
  
2.  Supprimez le mot clé non valide de l’instruction `Const` .
