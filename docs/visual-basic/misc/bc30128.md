# <a name="security-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Attribut de sécurité '&lt;attributename&gt;' n’est pas valide : &lt;erreur&gt;
Le Common Language Runtime n’accepte pas l’attribut dans un contexte de sécurité.

**ID d’erreur :** BC30128

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

1. Supprimez l’attribut de la déclaration ou remplacez-le par un attribut valide pour le contexte.

2. Examinez le message d’erreur mentionné et prenez les mesures nécessaires.
