# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Les initialiseurs de type anonyme ne sont pas valides dans les fenêtres de débogage
Un initialiseur de type anonyme a été entré dans la fenêtre **Espion** ou **Exécution** pendant une session de débogage. Les initialiseurs de type anonyme ne sont pas disponibles dans les fenêtres de débogage.  
  
 **ID d’erreur :** BC36577  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Testez les initialiseurs de type anonyme dans l’éditeur de code.
