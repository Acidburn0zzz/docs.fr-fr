<span data-ttu-id="21ecc-103">Impossible de déduire un type commun pour le premier et le second opérandes de l’opérateur 'If' binaire.</span><span class="sxs-lookup"><span data-stu-id="21ecc-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="21ecc-104">L’un doit avoir une conversion étendue vers l’autre.</span><span class="sxs-lookup"><span data-stu-id="21ecc-104">One must have a widening conversion to the other.</span></span>  
  
 <span data-ttu-id="21ecc-105">L’opérateur `If` binaire exige une conversion étendue entre l’un des arguments et l’autre argument.</span><span class="sxs-lookup"><span data-stu-id="21ecc-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="21ecc-106">Par exemple, comme il n’y aucune conversion étendue dans l’un ou l’autre sens entre `Integer` et `String`, le code suivant génère cette erreur.</span><span class="sxs-lookup"><span data-stu-id="21ecc-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb  
Dim second As String = "First is Nothing"  
'' Not valid.  
' Console.WriteLine(If(first, second))  
```  
  
 <span data-ttu-id="21ecc-107">**ID d’erreur :** BC33110</span><span class="sxs-lookup"><span data-stu-id="21ecc-107">**Error ID:** BC33110</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="21ecc-108">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="21ecc-108">To correct this error</span></span>

    ```

        Console.WriteLine(first)  
    Else  
        Console.WriteLine(second)  
    End If  
    ```
