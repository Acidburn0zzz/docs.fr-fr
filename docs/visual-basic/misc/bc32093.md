# <a name="of-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>’Of’ requis lors de la spécification des arguments de type pour un type ou une méthode générique
Une instruction essaie de construire un type à partir d’un type générique, ou d’appeler une méthode générique, sans utiliser de clause [Of](../../visual-basic/language-reference/statements/of-clause.md) .  
  
 La syntaxe Visual Basic pour les types génériques appelle pour les paramètres de type et les arguments de type soient introduits par le `Of` mot clé. De plus, la liste des paramètres ou arguments de type doit être mise entre parenthèses.  
  
 **ID d’erreur :** BC32093  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Incluez le mot clé `Of` au début de la liste d’arguments de type et mettez l’intégralité de la liste entre parenthèses.
