# <a name="for-loop-not-initialized-visual-basic"></a>Boucle for non initialisée (Visual Basic)
Vous avez effectué un saut au milieu d’une boucle `For...Next` .  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez le saut dans la boucle.
