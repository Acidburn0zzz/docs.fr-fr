# <a name="conflicts-with-ltname1gt-which-is-implicitly-created-for-ltname2gt-in-lttypegt-ltname3gt"></a>Est en conflit avec '&lt;name1&gt;', qui est créé implicitement pour'&lt;name2&gt;' dans &lt;type&gt; '&lt;Nom3&gt;'
Le nom d’un membre de type est en conflit avec un autre membre créé implicitement.  
  
 **ID d’erreur :** BC31058  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Renommez le membre déclaré explicitement pour supprimer le conflit de noms.
