# <a name="assembly-attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Attribut d’assembly &#39; &lt;attributename&gt; &#39; n’est pas valide : &lt;erreur&gt;
Le Common Language Runtime n’accepte pas l’attribut sur un assembly.

**ID d’erreur :** BC30129

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

1. Supprimez l’attribut de la déclaration ou remplacez-le par un attribut valide pour le contexte.

2. Examinez le message d’erreur cité et prenez les mesures nécessaires.
