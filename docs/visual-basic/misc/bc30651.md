# <a name="ltpropertygt-parameters-cannot-be-declared-byref"></a>&lt;Propriété&gt; paramètres ne peuvent pas être déclarés 'ByRef'
Les paramètres de propriété doivent être déclarés à l’aide du modificateur `ByVal` .  
  
 **ID d’erreur :** BC30651  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Définissez les paramètres de propriété en tant que `ByVal`.
