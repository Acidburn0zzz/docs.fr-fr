# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;Propriété&gt; les paramètres ne peuvent pas être déclarés &#39; ByRef &#39;
Les paramètres de propriété doivent être déclarés à l’aide du modificateur `ByVal` .  
  
 **ID d’erreur :** BC30651  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Définissez les paramètres de propriété en tant que `ByVal`.
