# <a name="39end-get39-must-be-the-first-statement-on-a-line"></a>&#39; Get de fin &#39; doit être la première instruction sur une ligne
Une instruction `End Get` suit un séparateur d’instruction deux-points (:). `End Get` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC32033  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Séparez les instructions en les plaçant sur des lignes différentes.
