# <a name="ltfilenamegt-cannot-be-referenced-because-it-is-not-an-assembly"></a>«&lt;filename&gt;' ne peut pas être référencé, car il n’est pas un assembly
Il a été fait référence à un fichier qui n’est pas un assembly .NET.  
  
 **ID d’erreur :** BC31519  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez le nom et l’emplacement de l’assembly.  
  
2.  Vérifiez que l’assembly n’est pas endommagé.
