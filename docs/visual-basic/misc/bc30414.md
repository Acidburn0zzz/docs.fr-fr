# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-the-array-types-have-different-numbers-of-dimensions"></a>Valeur de type &#39; &lt;type1&gt; &#39; ne peut pas être converti en &#39; &lt;type2&gt; &#39; , car les types tableau ont des nombres de dimensions différents
Vous avez essayé de convertir un tableau en un tableau avec un nombre de dimensions différent.  
  
 **ID d’erreur :** BC30414  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez l’un des tableaux ou les deux pour que le nombre de dimensions corresponde.
