# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-the-array-types-have-different-numbers-of-dimensions"></a>Valeur de type '&lt;type1&gt;« ne peut pas être converti en »&lt;type2&gt;', car les types tableau ont des nombres de dimensions différents
Vous avez essayé de convertir un tableau en un tableau avec un nombre de dimensions différent.  
  
 **ID d’erreur :** BC30414  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez l’un des tableaux ou les deux pour que le nombre de dimensions corresponde.
