# <a name="interface-can-inherit-only-from-another-interface"></a>Une interface ne peut hériter que d’une autre interface
Vous avez tenté d’hériter d’un élément qui n’est pas une interface.  
  
 **ID d’erreur :** BC30354  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que l’interface de base ou les interfaces à partir desquelles vous tentez d’hériter sont réellement des interfaces.
