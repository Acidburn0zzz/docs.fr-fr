# <a name="argument-ltargumentnamegt-cannot-be-converted-to-type-lttypenamegt"></a>Argument '&lt;nom_argument&gt;'ne peut pas être converti en type'&lt;typename&gt;'
Une conversion tente de modifier un argument en un type qui n’est pas valide.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Remplacez le type par un type valide.
