# <a name="properties-in-a-module-cannot-be-declared-ltspecifiergt"></a>Les propriétés d’un Module ne peuvent pas être déclarées '&lt;spécificateur&gt;'
Vous avez tenté d’utiliser un spécificateur, tel que `MustOverride`, `NotOverridable`ou `Overridable`, sur une propriété d’un module.  
  
 **ID d’erreur :** BC30503  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
