# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Argument &#39;accès&#39; n’est pas valide. Les valeurs valides pour le mode Input sont &#39;OpenAccess.Read&#39; et &#39;OpenAccess.Default.&#39;
Une valeur d’accès n’est pas valide en mode `Input` .  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Remplacez la valeur par `OpenAccess.Read` ou `OpenAccess.Default`.
