# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39;Hérite&#39; instruction doit précéder toutes les déclarations dans une classe
Les instructions `Inherits` doivent figurer au début des définitions de classe.  
  
 **ID d’erreur :** BC30683  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déplacez l’instruction `Inherits` au début de votre classe.
