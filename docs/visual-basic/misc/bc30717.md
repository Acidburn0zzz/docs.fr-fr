# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Essayez de &#39; les instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Try` peuvent uniquement être utilisées dans le code source.  
  
 **ID d’erreur :** BC30717  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Try` du code de débogage.
