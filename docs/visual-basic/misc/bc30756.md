# <a name="goto-ltlabelnamegt-is-not-valid-because-ltlabelnamegt-is-inside-a-with-statement-that-does-not-contain-this-statement"></a>' GoTo &lt;labelname&gt;' n’est pas valide, car '&lt;labelname&gt;' se trouve dans une instruction 'With' qui ne contient-elle pas cette instruction
Les instructions`GoTo` sont limitées à des sauts dans le bloc de code actuel.  
  
 **ID d’erreur :** BC30756  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Restructurez votre code pour que l’instruction `GoTo` et l’étiquette se trouvent à l’intérieur du bloc `With` .
