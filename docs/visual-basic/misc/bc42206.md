# <a name="maximum-number-of-warnings-has-been-exceeded"></a>Le nombre maximal d’avertissements a été dépassé.
Le nombre maximal d’avertissements de compilation s’élève à 100. Le compilateur impose cette limite pour réduire la taille des rapports d’erreurs et d’avertissements.  
  
 Par défaut, ce message est un avertissement. Pour plus d’informations sur le masquage des avertissements ou le traitement des avertissements comme des erreurs, consultez la page [configuration d’avertissements en Visual Basic](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID d’erreur :** BC42206  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Examinez les autres avertissements générés par cette compilation et corrigez-en autant que possible. Si le code source contient plus de 100 avertissements, d’autres avertissements risquent de figurer dans la compilation suivante.
