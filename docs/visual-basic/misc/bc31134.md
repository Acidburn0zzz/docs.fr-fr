# <a name="39addhandler39-and-39removehandler39-method-parameters-cannot-be-declared-39byref39"></a>&#39; AddHandler &#39; et &#39; RemoveHandler &#39; paramètres de méthode ne peut pas être déclarés &#39; ByRef &#39;
Les paramètres des méthodes `AddHandler` et `RemoveHandler` ne peuvent pas être déclarés avec le modificateur `ByRef` .  
  
 **ID d’erreur :** BC31134  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le mot clé `ByRef` du paramètre.
