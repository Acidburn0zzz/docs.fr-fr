# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>Objet référencé &#39; &lt;objectname&gt; &#39; a la valeur &#39;Nothing&#39;
L’objet référencé n’a pas été défini ou a été défini sur la valeur `Nothing`.  
  
 **ID d’erreur :** BC30705  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que le nom de l’objet est correct et que l’objet a été défini.
