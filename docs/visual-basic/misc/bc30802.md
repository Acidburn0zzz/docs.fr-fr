# <a name="39type39-statements-are-no-longer-supported-use-39structure39-statements-instead"></a>&#39;Type&#39; instructions ne sont plus prises en charge ; Utilisez &#39;Structure&#39; instructions à la place
Vous ne pouvez plus utiliser le mot clé `Type` pour définir des types définis par l’utilisateur.  
  
 **ID d’erreur :** BC30802  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez l’instruction `Structure` pour définir un nouveau type.
