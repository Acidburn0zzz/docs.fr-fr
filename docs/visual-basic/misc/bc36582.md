# <a name="too-many-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Arguments trop nombreux dans la méthode d’extension &#39; &lt;methodname&gt; &#39; définies dans &#39; &lt;typename&gt;&#39;
Le nombre d’arguments fournis dans un appel de méthode d’extension dépasse le nombre d’arguments définis pour la méthode.  
  
 **ID d’erreur :** BC36582  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Consultez la définition de la méthode d’extension et supprimez les arguments en trop de l’appel.
