# <a name="printer-error"></a>Erreur d'imprimante
Une erreur s’est produite au niveau de l’imprimante, mais aucune autre information n’a été renvoyée à l’ordinateur qui a envoyé le fichier.  
  
### <a name="to-correct-the-error"></a>Pour corriger l’erreur  
  
-   Examinez physiquement l’imprimante. Assurez-vous que toutes les connexions entre l’ordinateur et l’imprimante sont correctes. La plupart des imprimantes affichent les informations d’erreur telles que « Hors connexion » ou « Absence de papier ».
