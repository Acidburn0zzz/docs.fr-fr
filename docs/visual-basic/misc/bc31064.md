-   <span data-ttu-id="a1108-106">Attribuez la valeur `Set` au type de données du paramètre de sorte qu’il corresponde au type de données de la propriété.</span><span class="sxs-lookup"><span data-stu-id="a1108-106">Change the data type of the parameter to `Set` so that it matches the data type for the property.</span></span> <span data-ttu-id="a1108-107">Exemple :</span><span class="sxs-lookup"><span data-stu-id="a1108-107">For example:</span></span>

    ```  
       End Property

    ```
