## <a name="to-correct-this-error"></a><span data-ttu-id="bfae3-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="bfae3-107">To correct this error</span></span>  
  
1.  <span data-ttu-id="bfae3-108">Utilisez <xref:System.DBNull> à la place de `Null`.</span><span class="sxs-lookup"><span data-stu-id="bfae3-108">Use <xref:System.DBNull> instead of `Null`.</span></span> <span data-ttu-id="bfae3-109">Cela est illustré par l'exemple suivant.</span><span class="sxs-lookup"><span data-stu-id="bfae3-109">The following example demonstrates this.</span></span>  
  
    ```  
    Sub TestDBNull()  
        Dim t As DataTable  
        ' Assume the DataGrid is bound to a DataTable.  
        t = CType(DataGrid1.DataSource, DataTable)  
        Dim r As DataRow  
        r = t.Rows(datagrid1.CurrentCell.RowNumber)  
        r.BeginEdit  
        r(1) = System.DBNull.Value ' Assign DBNull to the record.  
        r.EndEdit  
        r.AcceptChanges  
        If r.IsNull(1) Then  
            MsgBox("")  
        End If  
    End Sub  
    ```  
  
2.  <span data-ttu-id="bfae3-110">Utiliser le [rien](../../visual-basic/language-reference/nothing.md) mot clé pour les assignations et les comparaisons lorsque vous utilisez des variables d’objet.</span><span class="sxs-lookup"><span data-stu-id="bfae3-110">Use the [Nothing](../../visual-basic/language-reference/nothing.md) keyword for assignments and comparisons when you use object variables.</span></span> <span data-ttu-id="bfae3-111">Cela est illustré par l'exemple suivant.</span><span class="sxs-lookup"><span data-stu-id="bfae3-111">The following example demonstrates this.</span></span>

    ```  
        ' cls is Nothing if it has not been assigned using the New keyword.  
        If (cls Is Nothing) Then  
            MsgBox("cls is Nothing")  
        End If  
        cls = Nothing ' Assign Nothing to the class variable cls.  
    End Sub  
    ```
