# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Les instructions de déclaration de méthode doivent figurer en premier sur une ligne logique.
Une instruction `Function` ou `Sub` suit un séparateur d’instruction deux-points (:). `Function` ou `Sub` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC32009  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Séparez les instructions en les plaçant sur des lignes différentes.
