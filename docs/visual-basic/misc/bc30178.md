# Sp&#233;cificateur en double
Un mot clé apparaît plusieurs fois dans une même déclaration.  
  
 **ID d’erreur :** BC30178  
  
### Pour corriger cette erreur  
  
-   Supprimez de la déclaration les mots clés en double.
