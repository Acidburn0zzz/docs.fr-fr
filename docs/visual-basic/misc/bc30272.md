# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39; &lt;nom_argument&gt;&#39; n’est pas un paramètre de &#39;&lt; MethodDeclaration&gt;&#39;
Un appel de procédure fournit un argument supplémentaire qui n’est pas défini par la procédure.  
  
 **ID d’erreur :** BC30272  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’argument superflu de la liste d’arguments.
