# <a name="ltargumentnamegt-is-not-a-parameter-of-ltmethoddeclarationgt"></a>«&lt;nom_argument&gt;'n’est pas un paramètre de'&lt;methoddeclaration&gt;»
Un appel de procédure fournit un argument supplémentaire qui n’est pas défini par la procédure.  
  
 **ID d’erreur :** BC30272  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’argument superflu de la liste d’arguments.
