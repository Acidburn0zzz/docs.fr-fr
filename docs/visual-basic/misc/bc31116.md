# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39;RemoveHandler&#39; déclaration doit se terminer par une mise en correspondance &#39;End RemoveHandler&#39;
Une déclaration `RemoveHandler` doit se terminer par une instruction `End RemoveHandler` .  
  
 **ID d’erreur :** BC31116  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que la déclaration `RemoveHandler` se termine par une instruction `End RemoveHandler` .
