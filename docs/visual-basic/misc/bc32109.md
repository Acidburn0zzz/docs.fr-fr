# <a name="indirect-constraint-ltconstraint1gt-obtained-from-the-type-parameter-constraint-lttypeparameter1gt-conflicts-with-the-indirect-constraint-ltconstraint2gt-obtained-from-the-type-parameter-constraint-lttypeparameter2gt"></a>La contrainte indirecte '&lt;contrainte1&gt;'obtenue de la contrainte de paramètre de type'&lt;paramètre_type1&gt;'est en conflit avec la contrainte indirecte'&lt;contrainte2&gt;' obtenu à partir de la contrainte de paramètre de type '&lt;paramètre_type2&gt;'
Un type générique est déclaré avec des contraintes en conflit en raison d’une combinaison de contraintes indirectes.  
  
 L’instruction suivante peut générer cette erreur.  
  
```  
Public Class testClass(Of t1 As {t2, t3}, t2 As Structure, t3 As Class)  
```  
  
 Les contraintes indirectes `Structure` et `Class` entraînent un conflit pour le paramètre de type `t1`, car la contrainte `Structure` exige que l’argument de type correspondant soit un type valeur, alors que `Class` exige qu’il soit un type référence.  
  
 **ID d’erreur :** BC32109  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez les contraintes de paramètre de type pour éviter les conflits entre contraintes.
