# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39; Nouveau &#39; ne peut pas être utilisé sur la classe &#39; &lt;classname&gt;&#39; car elle contient un &#39; MustOverride &#39; membre qui n’a pas été substitué
Vous avez tenté d’utiliser `New` sur une classe qui contient un membre `MustOverride` qui n’a pas été substitué.  
  
 **ID d’erreur :** BC30376  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction `New` .
