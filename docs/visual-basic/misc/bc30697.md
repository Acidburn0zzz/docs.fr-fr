# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>&#39; &lt;nom_membre1&gt;&#39; ne peut pas remplacer &#39;&lt; nom_membre2&gt;&#39; car les types des paramètres optionnels les différencient
Les membres surchargés doivent avoir des types de données différents pour un ou plusieurs paramètres obligatoires.  
  
 **ID d’erreur :** BC30697  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Définissez au moins un paramètre obligatoire en tant que type de données différent.
