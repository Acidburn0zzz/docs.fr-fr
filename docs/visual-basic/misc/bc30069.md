# <a name="for-loop-control-variable-ltvariablenamegt-already-in-use-by-an-enclosing-for-loop"></a>Variable de contrôle de boucle for '&lt;variablename&gt;' déjà en cours d’utilisation par une boucle For englobante
Les boucles `For` imbriquées doivent toutes utiliser des variables de contrôle uniques.  
  
 **ID d’erreur :** BC30069  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez l’un des noms de variables de contrôle.
