# <a name="for-loop-control-variable-39ltvariablenamegt39-already-in-use-by-an-enclosing-for-loop"></a>Variable de contrôle de boucle for &#39; &lt;variablename&gt; &#39; déjà en cours d’utilisation par une boucle For englobante
Les boucles `For` imbriquées doivent toutes utiliser des variables de contrôle uniques.  
  
 **ID d’erreur :** BC30069  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez l’un des noms de variables de contrôle.
