# <a name="39ltmodifiergt39-is-not-valid-on-a-declare"></a>&#39;&lt;modificateur&gt; &#39; n’est pas valide dans un Declare
Une instruction `Declare` contient un mot clé non valide, tel que `ReadOnly`.  
  
 **ID d’erreur :** BC30244  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le mot clé non valide de l’instruction `Declare` .
