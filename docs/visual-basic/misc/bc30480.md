# <a name="shared-sub-new-cannot-be-declared-ltspecifiergt"></a>Partagé 'Sub New' ne peut pas être déclaré '&lt;spécificateur&gt;'
Vous avez essayé d’utiliser un spécificateur dans une déclaration `Shared Sub New` , qui n’est pas valide.  
  
 **ID d’erreur :** BC30480  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
