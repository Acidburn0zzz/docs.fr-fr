# <a name="class-39ltclassnamegt39-does-not-implement-the-systemcollectionsicollection-interface"></a>La classe&lt;classname&gt;' n’implémente pas l’interface System.Collections.ICollection
La classe n’implémente pas l’interface `System.Collections.ICollection` , qui définit la taille, les énumérateurs et les méthodes de synchronisation de toutes les collections. Il peut être nécessaire de réinstaller [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Réinstallez [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].
