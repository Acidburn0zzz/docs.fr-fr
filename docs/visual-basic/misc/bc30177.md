# <a name="only-one-of-39notoverridable39-39mustoverride39-or-39overridable39-can-be-specified"></a>Seul un des &#39; NotOverridable &#39; &#39; MustOverride &#39; ou &#39; substituable &#39; peut être spécifié.
Un conflit de spécifications de substitution est présent dans une déclaration element. Les mots clés de substitution s’excluent mutuellement.  
  
 **ID d’erreur :** BC30177  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Choisissez un spécificateur de substitution, puis supprimez les autres mots clés de la déclaration.
