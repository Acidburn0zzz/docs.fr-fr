<span data-ttu-id="6e576-103">Le compilateur ne peut pas déduire le type de données d’une variable si celle-ci est utilisée pour établir sa valeur initiale dans la déclaration.</span><span class="sxs-lookup"><span data-stu-id="6e576-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="6e576-104">Par exemple, quand `Option Infer` a la valeur `On`, les exemples suivants ne sont pas compilés :</span><span class="sxs-lookup"><span data-stu-id="6e576-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
-   <span data-ttu-id="6e576-105">Déclarations</span><span class="sxs-lookup"><span data-stu-id="6e576-105">Declarations</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="6e576-106">Boucle`For` </span><span class="sxs-lookup"><span data-stu-id="6e576-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="6e576-107">Boucle`For Each` </span><span class="sxs-lookup"><span data-stu-id="6e576-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="6e576-108">**ID d’erreur :** BC30980</span><span class="sxs-lookup"><span data-stu-id="6e576-108">**Error ID:** BC30980</span></span>
