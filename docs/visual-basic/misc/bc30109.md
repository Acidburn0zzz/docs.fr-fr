# &#39;&lt;nom_type&gt;&#39; est un type de classe et ne peut pas &#234;tre utilis&#233; en tant qu’expression
Un nom de type apparaît lorsqu’une expression est exigée. Une expression doit se composer d’une combinaison de variables, de constantes, de littéraux, de propriétés et d’appels de procédure `Function`.  
  
 **ID d’erreur :** BC30109  
  
### Pour corriger cette erreur  
  
-   Supprimez le nom de type et construisez l’expression à l’aide d’éléments valides.
