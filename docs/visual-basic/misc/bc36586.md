# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Argument non spécifié pour le paramètre '&lt;nom_paramètre&gt;'de la méthode d’extension'&lt;nom_méthode&gt;'définie dans'&lt;typename&gt;'
Aucun argument n’a été indiqué pour un paramètre obligatoire de méthode d’extension.  
  
 **ID d’erreur :** BC36586  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez un argument pour le paramètre spécifié.
