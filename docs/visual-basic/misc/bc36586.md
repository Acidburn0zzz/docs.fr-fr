# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Argument non spécifié pour le paramètre &#39; &lt;nom_paramètre&gt; &#39; de méthode d’extension &#39; &lt;methodname&gt; &#39; définies dans &#39; &lt;typename&gt;&#39;
Aucun argument n’a été indiqué pour un paramètre obligatoire de méthode d’extension.  
  
 **ID d’erreur :** BC36586  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Fournissez un argument pour le paramètre spécifié.
