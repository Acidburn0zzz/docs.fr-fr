# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-declared-39public39"></a>Type &#39; &lt;typename&gt; &#39; ne peut pas être utilisé en tant qu’attribut, car il n’est pas déclaré &#39;Public&#39;
Les attributs doivent être publics.  
  
 **ID d’erreur :** BC31516  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Ajoutez le modificateur `Public` à la classe d’attributs.
