# <a name="type-lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Type '&lt;typename&gt;' ne peut pas être utilisé en tant qu’attribut, car il n’est pas déclaré 'Public'
Les attributs doivent être publics.  
  
 **ID d’erreur :** BC31516  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Ajoutez le modificateur `Public` à la classe d’attributs.
