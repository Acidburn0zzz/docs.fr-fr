# <a name="argument-ltargumentnamegt-cannot-be-converted-to-type-date"></a>Argument '&lt;nom_argument&gt;' ne peut pas être converti en type 'Date'
Une conversion a tenté de modifier une variable en type `Date`, mais la valeur n’est pas reconnue comme une date valide.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Recherchez une conversion significative.
