# <a name="39ltspecifiergt39-is-not-valid-on-a-structure-declaration"></a>&#39;&lt;spécificateur de&gt; &#39; n’est pas valide dans une déclaration de Structure
Vous avez utilisé un spécificateur, comme `Dim` ou `Override`, non valide dans une déclaration `Structure`  
  
 **ID d’erreur :** BC30395  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
