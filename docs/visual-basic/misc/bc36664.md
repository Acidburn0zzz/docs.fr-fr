# <a name="nesting-of-parentheses-in-lambda-expression39s-parameter-list-exceeds-maximum-allowable-depth"></a>Imbrication de parenthèses dans l’expression lambda&#39;liste de paramètres s dépasse la profondeur maximale autorisée
Les parenthèses imbriquées dans la liste des paramètres d’une expression lambda ne peut pas dépasser une profondeur de 128.  
  
 **ID d’erreur :** BC36664
