# <a name="nesting-of-parentheses-in-lambda-expressions-parameter-list-exceeds-maximum-allowable-depth"></a>L’imbrication de parenthèses dans la liste des paramètres de l’expression lambda dépasse la profondeur maximale autorisée
Les parenthèses imbriquées dans la liste des paramètres d’une expression lambda ne peut pas dépasser une profondeur de 128.  
  
 **ID d’erreur :** BC36664
