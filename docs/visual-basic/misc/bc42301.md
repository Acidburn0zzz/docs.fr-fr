# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Un seul bloc de commentaires XML est autorisé par élément de langage
Plusieurs blocs de commentaires XML ont été appliqués à un élément de langage.  
  
 **ID d’erreur :** BC42301  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le bloc de commentaires XML superflu.
