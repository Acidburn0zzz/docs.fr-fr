# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;Pour&#39; variable de contrôle de boucle déjà en cours d’utilisation par une englobante &#39;pour&#39; boucle
Les boucles `For` imbriquées doivent toutes utiliser des variables de contrôle uniques.  
  
 **ID d’erreur :** BC30975  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez l’un des noms de variables de contrôle.
