# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Option Explicit&#39; peut être suivie que de &#39;sur&#39; ou &#39;hors tension&#39;
Les instructions`Option Explicit` doivent spécifier `On` ou `Off`. Toutes les autres valeurs ne sont pas valides.  
  
 **ID d’erreur :** BC30640  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Spécifiez soit `Option Explicit On`, soit `Option Explicit Off`.
