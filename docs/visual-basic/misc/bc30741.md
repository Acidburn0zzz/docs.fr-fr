# <a name="lttypename1gt-cannot-be-converted-to-lttypename2gt"></a>«&lt;nom_type1&gt;« ne peut pas être converti en »&lt;nom_type2&gt;»
Une conversion de type a été tentée entre deux types incompatibles.  
  
 **ID d’erreur :** BC30741  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez les types qui font l’objet d’un cast.
