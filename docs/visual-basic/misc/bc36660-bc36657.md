<span data-ttu-id="a5d53-103">Types de données des ou des paramètres de type dans la méthode '\<nom_méthode >' ne peut pas être déduit à partir de ces arguments, car ils ne sont pas convertis vers le même type.</span><span class="sxs-lookup"><span data-stu-id="a5d53-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="a5d53-104">La spécification explicite du ou des types de données peut permettre de corriger cette erreur.</span><span class="sxs-lookup"><span data-stu-id="a5d53-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="a5d53-105">Une tentative a été faite d’utiliser l’inférence de type pour déterminer le ou les types de données du ou des paramètres de type pendant l’évaluation d’un appel à une procédure générique.</span><span class="sxs-lookup"><span data-stu-id="a5d53-105">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="a5d53-106">Le compilateur n’a pas pu trouver un type de données qui satisfasse aux contraintes de tous les arguments.</span><span class="sxs-lookup"><span data-stu-id="a5d53-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="a5d53-107">Par conséquent, il a signalé cette erreur.</span><span class="sxs-lookup"><span data-stu-id="a5d53-107">Therefore, it reported this error.</span></span>

>  <span data-ttu-id="a5d53-108">Quand la spécification d’arguments n’est pas une option (par exemple, pour les opérateurs de requête dans les expressions de requête), le message d’erreur apparaît sans la deuxième phrase.</span><span class="sxs-lookup"><span data-stu-id="a5d53-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>
