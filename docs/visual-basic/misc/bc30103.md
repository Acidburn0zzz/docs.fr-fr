# <a name="3939-requires-its-left-operand-to-have-a-type-parameter-class-or-interface-type-but-this-operand-has-the-type-39lttypegt39"></a>&#39;! &#39; nécessite sa gauche opérande ayant un paramètre de type, classe ou type d’interface, mais cet opérande a le type &#39; &lt;type&gt;&#39;
L’accès au membre de dictionnaire, qui utilise un point d’exclamation (`!`) au lieu d’un point (`.`), est uniquement valide sur une classe ou une interface.  
  
 **ID d’erreur :** BC30103  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Remplacez l’expression située à gauche du point d’exclamation par une expression qui correspond à un type de classe ou d’interface défini.
