# <a name="39optional39-and-39paramarray39-cannot-be-combined"></a>&#39;Facultatif&#39; et &#39;ParamArray&#39; ne peut pas être combiné
Les modificateurs `Optional` et `ParamArray` s’excluent mutuellement.  
  
 **ID d’erreur :** BC30642  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Spécifiez `ParamArray` ou `Optional`, mais pas les deux.
