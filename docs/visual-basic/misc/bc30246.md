# <a name="39ltmodifiergt39-is-not-valid-on-a-local-constant-declaration"></a>&#39; &lt;modificateur&gt;&#39; n’est pas valide dans une déclaration de constante locale
Une instruction `Const` à l’intérieur d’une procédure contient un mot clé non valide, tel que `Overloads`.  
  
 **ID d’erreur :** BC30246  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez le mot clé non valide de l’instruction `Const` .
