# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39;Partagé&#39; ne peut pas être combiné avec &#39; &lt;spécificateur&gt; &#39; sur une déclaration de propriété
Vous avez tenté d’associer le modificateur `Shared` à un spécificateur qui n’est pas valide dans une telle combinaison pour une déclaration de propriété.  
  
 **ID d’erreur :** BC30502  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
