# <a name="shared-cannot-be-combined-with-specifier-on-a-property-declaration"></a>'Shared' ne peut pas être combiné avec '\<spécificateur >' sur une déclaration de propriété
Vous avez tenté d’associer le modificateur `Shared` à un spécificateur qui n’est pas valide dans une telle combinaison pour une déclaration de propriété.  
  
 **ID d’erreur :** BC30502  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
