# <a name="ltmembergt-is-already-declared-in-this-structure"></a>«&lt;membre&gt;' est déjà déclaré dans cette structure
Un membre de l’énumération a été déclaré plusieurs fois dans cette structure.  
  
 **ID d’erreur :** BC31421  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez la déclaration superflue.
