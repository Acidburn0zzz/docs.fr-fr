# <a name="39if39-operands-cannot-be-named-arguments"></a><span data-ttu-id="044af-102">&#39;Si&#39; opérandes ne peuvent pas être des arguments nommés</span><span class="sxs-lookup"><span data-stu-id="044af-102">&#39;If&#39; operands cannot be named arguments</span></span>
<span data-ttu-id="044af-103">L’utilisation d’arguments nommés dans les opérandes de l’opérateur `If` n’est pas valide.</span><span class="sxs-lookup"><span data-stu-id="044af-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="044af-104">L’exemple suivant génère cette erreur :</span><span class="sxs-lookup"><span data-stu-id="044af-104">The following example causes this error:</span></span>  
  
```

Dim result As String

' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="044af-105">Il diffère de la fonction `IIf` qui autorise des arguments nommés, comme indiqué dans le code suivant :</span><span class="sxs-lookup"><span data-stu-id="044af-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>

