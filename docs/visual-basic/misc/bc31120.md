# &#39;End RaiseEvent&#39; doit &#234;tre la premi&#232;re instruction sur une ligne
Une instruction `End RaiseEvent` suit un séparateur d’instruction deux\-points \(:\).`End RaiseEvent` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC31120  
  
### Pour corriger cette erreur  
  
-   Séparez les différentes instructions en les plaçant sur des lignes différentes.
