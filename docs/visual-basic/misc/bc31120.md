# <a name="39end-raiseevent39-must-be-the-first-statement-on-a-line"></a>&#39; End RaiseEvent &#39; doit être la première instruction sur une ligne
Une instruction `End RaiseEvent` suit un séparateur d’instruction deux-points (:). `End RaiseEvent` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC31120  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Séparez les instructions en les plaçant sur des lignes différentes.
