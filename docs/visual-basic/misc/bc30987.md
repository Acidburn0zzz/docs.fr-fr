<span data-ttu-id="726e3-103">Pour déclarer une instance d’un type nommé ou anonyme à l’aide d’un initialiseur d’objet, vous devez placer la liste des champs ou des propriétés et leurs valeurs initiales entre accolades ({et}).</span><span class="sxs-lookup"><span data-stu-id="726e3-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>
