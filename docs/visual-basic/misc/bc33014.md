# <a name="operator-39ltoperatorgt39-must-have-one-parameter"></a>Opérateur &#39; &lt;opérateur&gt;&#39; doit avoir un paramètre
Un opérateur unaire est défini sans aucun paramètre ou avec plus d’un paramètre.  
  
 Un opérateur unaire doit avoir un seul paramètre.  
  
 **ID d’erreur :** BC33014  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez la définition de façon à spécifier un seul paramètre.  
  
-   Si vous avez besoin de deux paramètres, vous devez définir un opérateur binaire.  
  
-   Si vous avez besoin d’aucun paramètre ou plus de deux, vous devez utiliser le [Function, instruction](../../visual-basic/language-reference/statements/function-statement.md) pour définir un `Function` procédure au lieu d’un opérateur.
