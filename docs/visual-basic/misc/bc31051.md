# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace ou type &#39; &lt;nom&gt;&#39; a déjà été importé
L’utilisateur a tenté d’importer plusieurs fois un même espace de noms.  
  
 **ID d’erreur :** BC31051  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions redondantes `Imports` .
