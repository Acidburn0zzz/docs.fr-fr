# &#39;&lt;&#39; ouvrant attendu pour une balise XML
La balise XML pour un littéral XML ne possède pas le caractère '\<' de début requis.  
  
 **ID d’erreur :** BC31165  
  
### Pour corriger cette erreur  
  
-   Ajoutez le caractère '\<' au début de la balise XML pour le littéral XML.
