# <a name="expected-beginning-lt-for-an-xml-tag"></a>Ouvrant attendu '&lt;' pour une balise XML
La balise XML pour un littéral XML ne possède pas le caractère '<' de début requis.  
  
 **ID d’erreur :** BC31165  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez le caractère '<' au début de la balise XML pour le littéral XML.
