# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>&#39;RaiseEvent&#39; instructions ne sont pas valides dans la fenêtre exécution
Vous ne pouvez pas utiliser d’instructions`RaiseEvent` dans la fenêtre **Exécution** .  
  
 **ID d’erreur :** BC30739  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Testez les instructions `RaiseEvent` du code dans l’ **éditeur de code**.
