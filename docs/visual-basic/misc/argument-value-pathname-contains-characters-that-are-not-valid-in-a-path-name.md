# <a name="argument-value-ltpathnamegt-contains-characters-that-are-not-valid-in-a-path-name"></a>Valeur de l’argument '&lt;pathname&gt;' contient des caractères qui ne sont pas valides dans un nom de chemin d’accès
Un nom de chemin contient un ou plusieurs caractères qui ne sont pas valides.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le caractère du nom du chemin.
