# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>&#39; End RemoveHandler &#39; doit être la première instruction sur une ligne
Une instruction `End RemoveHandler` suit un séparateur d’instruction deux-points (:). `End RemoveHandler` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC31119  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Séparez les instructions en les plaçant sur des lignes différentes.
