# <a name="character-character-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Caractère 'caractère' (0 x&lt;hexadécimal&gt;) n’est pas autorisé au début d’un nom XML
Un caractère non pris en charge est inclus au début du nom d’un littéral XML ou d’une propriété d’axe XML.  
  
 **ID d’erreur :** BC31169  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le caractère non pris du début du nom du littéral XML ou de la propriété d’axe XML.
