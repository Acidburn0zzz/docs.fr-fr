# Le caract&#232;re &#39;character&#39; (0x&lt;hexad&#233;cimal&gt;) n’est pas autoris&#233; au d&#233;but d’un nom XML
Un caractère non pris en charge est inclus au début du nom d’un littéral XML ou d’une propriété d’axe XML.  
  
 **ID d’erreur :** BC31169  
  
### Pour corriger cette erreur  
  
-   Supprimez le caractère non pris du début du nom du littéral XML ou de la propriété d’axe XML.
