# <a name="39stop39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Arrêt &#39; les instructions ne sont pas valides dans la fenêtre exécution
Les instructions `Stop` et `End` interrompent l’exécution et ne sont pas autorisées dans un contexte de débogage.  
  
 **ID d’erreur :** BC30122  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   N’émettez pas d’instruction `Stop` ou `End` dans la fenêtre **Exécution** .
