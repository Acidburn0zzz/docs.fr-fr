# <a name="unable-to-find-entry-point-39ltnamegt39-in-dll-39ltfilenamegt39-lterrorgt"></a>Impossible de trouver le point d’entrée &#39; &lt;nom&gt; &#39; dans la DLL &#39; &lt;nom de fichier&gt;&#39;: &lt;erreur&gt;
Le nom de la procédure est introuvable dans la DLL spécifiée.  
  
 **ID d’erreur :** BC31014  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que le nom de la procédure est correctement orthographié et que la casse correspond à celle de la fonction dans la DLL.
