# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Une référence au module requise &#39; &lt;modulename&gt; &#39; contenant l’interface implémentée &#39; &lt;nom_interface&gt;&#39;
Une référence au module requise '\<nom_module >' contenant l’interface implémentée '\<nom_interface >'. Ajoutez-en une à votre projet.  
  
 L’interface est définie dans un module qui n’est pas directement référencé dans votre projet. Le compilateur Visual Basic requiert une référence afin d’éviter toute ambiguïté au cas où l’interface est définie dans plusieurs modules.  
  
 **ID d’erreur :** BC30010  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez le nom du module non référencé dans vos références de projet.
