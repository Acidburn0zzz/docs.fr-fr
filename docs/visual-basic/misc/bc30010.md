# <a name="reference-required-to-module-ltmodulenamegt-containing-the-implemented-interface-ltinterfacenamegt"></a>Une référence au module requise '&lt;modulename&gt;'contenant l’interface implémentée'&lt;interfacename&gt;'
Une référence au module requise '\<nom_module >' contenant l’interface implémentée '\<nom_interface >'. Ajoutez-en une à votre projet.  
  
 L’interface est définie dans un module qui n’est pas directement référencé dans votre projet. Le compilateur Visual Basic requiert une référence pour éviter toute ambiguïté au cas où l’interface est définie dans plusieurs modules.  
  
 **ID d’erreur :** BC30010  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez le nom du module non référencé dans vos références de projet.
