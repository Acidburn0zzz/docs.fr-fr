# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Lever&#39; instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Throw` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30725  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Throw` du code de débogage.
