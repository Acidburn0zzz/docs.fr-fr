# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>Une référence au module requise &#39; &lt;modulename&gt; &#39; contenant le type &#39; &lt;membername&gt;&#39;
Une référence au module requise '\<nom_module >' contenant le type '\<nom_membre >'. Ajoutez-en une à votre projet.  
  
 Le projet contient un type qui dépend d’un autre type d’un assembly qui n’est pas inclus dans les références de projet.  
  
 **ID d’erreur :** BC30653  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une référence de projet pour les types dépendants contenus dans le projet.
