# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>Référence au module requise '&lt;modulename&gt;'contenant le type'&lt;membername&gt;»
Référence au module requise '\<modulename >' contenant le type '\<membername > ». Ajoutez-en une à votre projet.  
  
 Le projet contient un type qui dépend d’un autre type d’un assembly qui n’est pas inclus dans les références de projet.  
  
 **ID d’erreur :** BC30653  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une référence de projet pour les types dépendants du projet.
