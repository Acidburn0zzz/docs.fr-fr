 `End Interface`  
  
 <span data-ttu-id="1f3d5-105">Si `derivedInterface` est construit ou implémenté en fournissant le même type à `t1` et à `t2`, il doit hériter de deux versions de `interfaceA` avec des arguments de type identiques.</span><span class="sxs-lookup"><span data-stu-id="1f3d5-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="1f3d5-106">Procéder ainsi est source d’ambiguïté quant à la version à laquelle accéder.</span><span class="sxs-lookup"><span data-stu-id="1f3d5-106">Doing so would produce an ambiguity about which version to access.</span></span>
