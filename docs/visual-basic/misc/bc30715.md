# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Catch&#39; instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Catch` peuvent uniquement être utilisées dans le code source.  
  
 **ID d’erreur :** BC30715  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Catch` du code de débogage.
