# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-xml-literal"></a>Caractère &#39;caractère&#39; (0 x&lt;hexadécimal&gt;) est interdite dans XML littéral
Un caractère non pris en charge a été utilisé dans un littéral XML.  
  
 **ID d’erreur :** BC31176  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le caractère non pris en charge.
