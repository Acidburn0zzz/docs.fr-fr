# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Caractère 'caractère' (0 x\<hexadécimal >) n’est pas autorisé dans XML littéral
Un caractère non pris en charge a été utilisé dans un littéral XML.  
  
 **ID d’erreur :** BC31176  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le caractère non pris en charge.
