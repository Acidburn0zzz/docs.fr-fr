# <a name="modifier-is-not-valid-on-an-event-declaration"></a>'\<modificateur >' n’est pas valide dans une déclaration event
Une instruction `Event` contient un mot clé non valide, tel que `ReadOnly`.  
  
 **ID d’erreur :** BC30243  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez le mot clé non valide de l’instruction `Event` .
