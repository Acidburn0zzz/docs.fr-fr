# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;&lt;modificateur&gt; &#39; n’est pas valide dans une déclaration event
Une instruction `Event` contient un mot clé non valide, tel que `ReadOnly`.  
  
 **ID d’erreur :** BC30243  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez le mot clé non valide de l’instruction `Event` .
