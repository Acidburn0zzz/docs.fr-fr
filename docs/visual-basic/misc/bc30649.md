# <a name="lttypenamegt-is-an-unsupported-type"></a>«&lt;typename&gt;' est un type non pris en charge
Le type spécifié ne peut pas être représenté par Visual Studio.  
  
 **ID d’erreur :** BC30649  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez un type pris en charge.  
  
-   Développez un nouveau type.
