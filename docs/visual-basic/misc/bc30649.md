# &#39;&lt;nom_type&gt;&#39; est un type non pris en charge
Le type spécifié ne peut pas être représenté par [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].  
  
 **ID d’erreur :** BC30649  
  
### Pour corriger cette erreur  
  
-   Utilisez un type pris en charge.  
  
-   Développez un nouveau type.
