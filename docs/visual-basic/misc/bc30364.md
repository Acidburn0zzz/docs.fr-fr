# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39;Sub New&#39; ne peut pas être déclaré &#39; &lt;modificateur&gt;&#39;
Vous avez tenté de déclarer un `Sub New` avec un modificateur, tel que `Overrides`, qui n’est pas valide.  
  
 **ID d’erreur :** BC30364  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le modificateur.
