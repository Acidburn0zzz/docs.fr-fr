# <a name="sub-new-cannot-be-declared-ltmodifiergt"></a>'Sub New' ne peut pas être déclaré '&lt;modificateur&gt;'
Vous avez tenté de déclarer un `Sub New` avec un modificateur, tel que `Overrides`, qui n’est pas valide.  
  
 **ID d’erreur :** BC30364  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le modificateur.
