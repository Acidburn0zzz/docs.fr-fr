# Le ConnectionTimeout doit &#234;tre sup&#233;rieur &#224;&#160;0
Quand vous chargez et téléchargez des fichiers avec l’[My.Computer.Network Object](../../visual-basic/language-reference/objects/my-computer-network-object.md), vous devez spécifier un `connectionTimeout` supérieur à `0`.  
  
### Pour corriger cette erreur  
  
-   Fournissez un `connectionTimeout` supérieur à `0`.
