# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39; &lt;nom&gt;&#39; est déclaré dans le projet &#39;&lt; nom_projet1&gt;&#39; qui n’est pas référencé par projet &#39;&lt; nom_projet2&gt;&#39;
Un élément de programmation est défini dans un projet auquel le projet actuel ne fait pas référence.  
  
 **ID d’erreur :** BC32004  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Dans le projet actif, ajoutez une référence au projet qui contient l’élément souhaité.
