# <a name="ltnamegt-is-declared-in-project-ltprojectname1gt-which-is-not-referenced-by-project-ltprojectname2gt"></a>«&lt;nom&gt;'est déclaré dans le projet'&lt;nom_projet1&gt;', qui n’est pas référencé par le projet'&lt;nom_projet2&gt;»
Un élément de programmation est défini dans un projet auquel le projet actuel ne fait pas référence.  
  
 **ID d’erreur :** BC32004  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Dans le projet actif, ajoutez une référence au projet qui contient l’élément souhaité.
