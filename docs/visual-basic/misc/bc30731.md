# <a name="addressof-expressions-are-not-valid-in-debug-windows"></a>Les expressions 'AddressOf' ne sont pas valides dans les fenêtres de débogage
Les instructions`AddressOf` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30731  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez les instructions `AddressOf` du code de débogage.
