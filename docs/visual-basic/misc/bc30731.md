# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>&#39;AddressOf&#39; expressions ne sont pas valides dans les fenêtres de débogage
Les instructions`AddressOf` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30731  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez les instructions `AddressOf` du code de débogage.
