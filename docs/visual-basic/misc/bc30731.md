# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>Les expressions 'AddressOf' ne sont pas valides dans les fenêtres de débogage
`AddressOf`instructions ne peuvent pas être utilisées dans le code source.  
  
 **ID d’erreur :** BC30731  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez les instructions `AddressOf` du code de débogage.
