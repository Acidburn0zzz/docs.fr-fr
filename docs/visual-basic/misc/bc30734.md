# <a name="39ltnamegt39-is-already-declared-as-a-parameter-of-this-method"></a>&#39;&lt;nom&gt; &#39; est déjà déclaré en tant que paramètre de cette méthode
Le nom de paramètre spécifié a déjà été déclaré pour cette méthode.  
  
 **ID d’erreur :** BC30734  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Renommez le paramètre pour qu’il soit unique.
