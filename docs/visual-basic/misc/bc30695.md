# <a name="lttypenamegt-39ltmembernamegt39-must-be-declared-39shadows39-because-another-member-with-this-name-is-declared-39shadows39"></a>&lt;TypeName&gt; &#39; &lt;membername&gt; &#39; doit être déclaré &#39;ombres&#39; , car un autre membre portant ce nom est déclaré &#39;ombres&#39;
Cette propriété ou procédure porte le même nom qu’une autre propriété ou procédure marquée avec le modificateur `Shadows` .  
  
 **ID d’erreur :** BC30695  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez le modificateur `Shadows` .
