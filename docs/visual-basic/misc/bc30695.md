# <a name="lttypenamegt-ltmembernamegt-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>&lt;TypeName&gt; '&lt;membername&gt;' doit être déclaré 'Shadows', car un autre membre portant ce nom est déclaré 'Shadows'
Cette propriété ou procédure porte le même nom qu’une autre propriété ou procédure marquée avec le modificateur `Shadows` .  
  
 **ID d’erreur :** BC30695  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez le modificateur `Shadows` .
