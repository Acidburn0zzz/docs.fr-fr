# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39;Obtenir&#39; instruction doit se terminer par une mise en correspondance &#39;End Get&#39;
Les procédures de propriété`Get` doivent se terminer par une instruction `End Get` .  
  
 **ID d’erreur :** BC30631  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que la procédure de propriété `Get` se termine par une construction `End Get` .
