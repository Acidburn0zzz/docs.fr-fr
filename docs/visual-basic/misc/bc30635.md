# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39;En lecture seule&#39; et &#39;WriteOnly&#39; ne peut pas être combiné
Les modificateurs `ReadOnly` et `WriteOnly` s’excluent mutuellement.  
  
 **ID d’erreur :** BC30635  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Spécifiez `ReadOnly` ou `WriteOnly`, mais pas les deux.
