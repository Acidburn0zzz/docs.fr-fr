# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Instructions de commentaire ne peut pas être évaluées (pas dans la table des matières)
Une instruction comment a été entrée dans la fenêtre Exécution pendant une session de débogage. Les instructions comment dans la fenêtre Exécution ne sont pas évaluées.  
  
 **ID d’erreur :** BC30974
