# <a name="method-in-a-structure-cannot-be-declared-39protected39-or-39protected-friend39"></a>Méthode dans une structure ne peuvent pas être déclarée &#39;protégé&#39; ou &#39;Protected Friend&#39;
Une méthode dans une structure a été déclarée en tant que `Protected` ou `Protected Friend`.  
  
 **ID d’erreur :** BC31067  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez les méthodes à l’intérieur de structures en tant que `Public` ou `Private`.
