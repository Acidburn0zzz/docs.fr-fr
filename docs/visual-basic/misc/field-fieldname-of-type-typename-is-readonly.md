# <a name="field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-39readonly39"></a>Champ &#39; &lt;fieldname&gt; &#39; de type &#39; &lt;typename&gt; &#39; est &#39;en lecture seule&#39;
Vous avez tenté d’écrire dans un champ qui est déclaré `ReadOnly`.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez le champ pour qu’il ne soit plus déclaré `ReadOnly`.  
  
-   - ou -  
  
-   N’essayez pas d’écrire dans ce champ.
