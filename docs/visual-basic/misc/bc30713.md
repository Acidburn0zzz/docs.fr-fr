# <a name="exit-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Exit' ne sont pas valides dans la fenêtre Exécution
Les instructions`Exit` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30713  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Exit` du code de débogage.
