# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Sortie&#39; instructions ne sont pas valides dans la fenêtre exécution
Les instructions`Exit` peuvent uniquement être utilisées dans le code source.  
  
 **ID d’erreur :** BC30713  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Exit` du code de débogage.
