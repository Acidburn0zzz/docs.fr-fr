# <a name="end-event-must-be-preceded-by-a-matching-custom-event"></a>'End Event' doit être précédé d’un 'Custom Event' correspondant
Une instruction `End Event` se produit sans instruction `Custom Event` correspondante. `End Event` doit être précédé d’une instruction `Custom Event` correspondante.  
  
 **ID d’erreur :** BC31123  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que l’instruction `Custom Event` précédente est valide et correctement orthographiée.
