# <a name="structure-ltstructurenamegt-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Structure '&lt;nom_structure&gt;' doit contenir au moins une variable membre d’instance ou une déclaration d’événement
Une structure fonctionnellement vide a été détectée. Au moins un membre de variable doit être déclaré entre l’instruction `Structure` et l’instruction `End Structure` .  
  
 **ID d’erreur :** BC30281  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une instruction `Dim` ou une instruction `Event` à la structure.
