# <a name="structure-39ltstructurenamegt39-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Structure &#39; &lt;nom_structure&gt; &#39; doit contenir au moins une variable membre d’instance ou une déclaration Event
Une structure fonctionnellement vide a été détectée. Au moins un membre de variable doit être déclaré entre l’instruction `Structure` et l’instruction `End Structure` .  
  
 **ID d’erreur :** BC30281  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une instruction `Dim` ou une instruction `Event` à la structure.
