# <a name="events-in-a-module-cannot-be-declared-ltspecifiergt"></a>Événements d’un Module ne peuvent pas être déclarés '&lt;spécificateur&gt;'
Vous avez utilisé un spécificateur non valide pour un événement dans une instruction `Module` . Les modules ne peuvent jamais être instanciés, ils ne prennent pas en charge l’héritage et ils ne peuvent pas implémenter d’interfaces.  
  
 **ID d’erreur :** BC30434  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
