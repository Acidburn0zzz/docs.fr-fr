# <a name="too-many-dll-application-clients"></a>Trop de clients pour cette DLL
La bibliothèque de liens dynamiques (DLL) pour Visual Basic autorise seulement l’accès d’un nombre limité d’applications hôtes. Votre application et autres applications qui sont des hôtes Visual Basic (certains d'entre eux sont accessibles par votre application) tentent d’accéder à la DLL de Visual Basic en même temps.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Réduire le nombre d’applications ouvertes accédant à Visual Basic.
