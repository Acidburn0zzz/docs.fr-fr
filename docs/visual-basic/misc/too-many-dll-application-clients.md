# <a name="too-many-dll-application-clients"></a>Trop de clients pour cette DLL
La bibliothèque de liens dynamiques (DLL) pour Visual Basic autorise seulement l’accès à un nombre limité d’applications hôtes. Votre application et autres applications qui sont des hôtes Visual Basic (dont certaines sont accessibles par votre application) tentent toutes d’à accéder à la DLL de Visual Basic en même temps.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Réduisez le nombre d’applications ouvertes accédant à Visual Basic.
