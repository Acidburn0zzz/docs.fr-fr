# <a name="39ltmodifiergt39-is-not-valid-on-a-local-variable-declaration"></a>&#39;&lt;modificateur&gt; &#39; n’est pas valide dans une déclaration de variable locale
Une instruction `Dim` à l’intérieur d’une procédure contient un mot clé non valide, tel que `Overloads`.  
  
 **ID d’erreur :** BC30247  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le mot clé non valide de l’instruction `Dim` .
