# <a name="39ltmodulenamegt39-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>&#39;&lt;nom du module&gt; &#39; est un module et ne peut pas être référencé en tant qu’assembly
Une tentative d’utiliser un module en tant qu’assembly a été effectuée.  
  
 **ID d’erreur :** BC31076  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez la référence à un assembly.
