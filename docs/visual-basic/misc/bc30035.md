# Erreur de syntaxe (erreur Visual Basic)
La ligne de code source génère une erreur de syntaxe non spécifique.  
  
 **ID d’erreur :** BC30035  
  
### Pour corriger cette erreur  
  
1.  Examinez la documentation de chacun des mots clés utilisés dans la ligne de code source. Comparez l’exemple de code dans la documentation à la ligne source générant cette erreur.  
  
2.  Si vous ne parvenez pas à identifier la cause de l’erreur, rassemblez des informations sur ses circonstances et avertissez les Services de support technique Microsoft.
