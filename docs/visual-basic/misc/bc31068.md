# <a name="delegate-in-an-interface-cannot-be-declared-specifier"></a>Un délégué dans une interface ne peut pas être déclaré '\<spécificateur >'
Vous avez spécifié un modificateur non valide avec un délégué déclaré dans une interface.  
  
 **ID d’erreur :** BC31068  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez les modificateurs tels que `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`et `Overridable`.
