# <a name="the-value-value-is-invalid-for-option-optionname"></a>la valeur '\<valeur >' n’est pas valide pour l’option '\<nom_option >'
Une valeur non valide a été fournie pour une option de ligne de commande.  
  
 **ID d’erreur :** BC2014  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que toutes les valeurs sont valides.
