# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="db978-102">Un littéral XML ne peut pas s’afficher ici à moins qu’il ne soit mis entre parenthèses</span><span class="sxs-lookup"><span data-stu-id="db978-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="db978-103">Une déclaration de littéral XML est utilisée dans une expression, à un emplacement ambigu pour le compilateur [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] .</span><span class="sxs-lookup"><span data-stu-id="db978-103">An XML literal declaration is used in an expression in a location that is ambiguous to the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler.</span></span> <span data-ttu-id="db978-104">Autrement dit, le [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compilateur ne peut pas déterminer si le moins-de caractères (<) est conçue comme un opérateur de comparaison ou d’un littéral XML.</span><span class="sxs-lookup"><span data-stu-id="db978-104">That is, the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="db978-105">Le code suivant est fourni à titre d'exemple :</span><span class="sxs-lookup"><span data-stu-id="db978-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="db978-106">**ID d’erreur :** BC31198</span><span class="sxs-lookup"><span data-stu-id="db978-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="db978-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="db978-107">To correct this error</span></span>  
  
-   <span data-ttu-id="db978-108">Placez la déclaration de littéral XML entre parenthèses, comme illustré dans l’exemple suivant :</span><span class="sxs-lookup"><span data-stu-id="db978-108">Enclose the XML literal declaration in parentheses, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
-   <span data-ttu-id="db978-109">Modifiez votre code pour référencer un littéral XML existant, comme illustré dans l’exemple suivant :</span><span class="sxs-lookup"><span data-stu-id="db978-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
