<span data-ttu-id="2e61d-103">Une déclaration de littéral XML est utilisée dans une expression dans un emplacement est ambigu pour le compilateur Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="2e61d-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="2e61d-104">Autrement dit, le compilateur Visual Basic ne peut pas déterminer si le moins-de caractères (<) est conçue comme un opérateur de comparaison ou d’un littéral XML.</span><span class="sxs-lookup"><span data-stu-id="2e61d-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="2e61d-105">Le code suivant est fourni à titre d'exemple :</span><span class="sxs-lookup"><span data-stu-id="2e61d-105">The following code provides an example:</span></span>  

```vb

                  Select element  
```
