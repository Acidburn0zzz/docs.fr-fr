# <a name="xml-comment-tag-39lttaggt39-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>La balise de commentaire XML &#39; &lt;balise&gt; &#39; s’affiche avec des attributs identiques plusieurs fois dans le même bloc de commentaire XML
La balise de commentaire XML \<balise > apparaît avec des attributs identiques plusieurs fois dans le même bloc de commentaire XML. Le commentaire XML est ignoré.  
  
 Le commentaire XML contient plusieurs balises avec des attributs identiques.  
  
 **ID d’erreur :** BC42305  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez la balise en double.
