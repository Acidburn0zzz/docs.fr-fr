# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-from-39lttype1gt39-to-39lttype2gt39"></a>Argument correspondant au paramètre &#39; &lt;nom_paramètre&gt; &#39; passe de &#39; &lt;type1&gt; &#39; à &#39; &lt;type2&gt;&#39;
Vous avez effectué un appel vers une méthode surchargée, mais le compilateur ne peut pas trouver les méthodes qui peuvent être appelées sans conversion restrictive. Une conversion restrictive remplace une valeur par un type de données susceptible de ne pas pouvoir contenir certaines des valeurs possibles.  
  
 **ID d’erreur :** BC30520  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Spécifiez `Option Strict Off`.
