# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>Fichier source &lt;nom de fichier&gt; spécifié plusieurs fois
Un fichier a été spécifié plusieurs fois.  
  
 **ID d’erreur :** BC2002  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les spécificateurs de fichiers redondants.
