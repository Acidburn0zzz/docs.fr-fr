# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Erreur &#39; les instructions ne sont pas valides dans la fenêtre exécution
Vous ne pouvez pas utiliser d’instructions`Error` dans le code source.  
  
 **ID d’erreur :** BC30724  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Error` du code de débogage.
