# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>«&lt;NomProcédure&gt;' a un type de retour n’est pas pris en charge ou des types de paramètres qui ne sont pas pris en charge
Une procédure est déclarée avec des paramètres ou un type de retour ne peut pas être représentée par [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
 **ID d’erreur :** BC30657  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Définissez les types de retour ou les paramètres comme des types pris en charge.  
  
-   Développez un nouveau type décrivant les données utilisées par cette procédure.
