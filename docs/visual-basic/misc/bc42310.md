# <a name="xml-comment-tag-39include39-must-have-a-39ltattributegt39-attribute"></a>XML &#39;de la balise de commentaire ; incluent &#39; doit avoir un &#39; &lt;attribut&gt;&#39; attribut
Balise de commentaire XML 'include' doit avoir un \<attribut > attribut. Le commentaire XML est ignoré.  
  
 Un attribut `file` ou `path` , passé à la balise `<`include`>` , est manquant ou incomplet.  
  
 **ID d’erreur :** BC42310  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez ou complétez l’attribut requis pour la \<inclure > balise.
