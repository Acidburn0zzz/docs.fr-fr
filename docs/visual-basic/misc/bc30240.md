# <a name="39exit39-must-be-followed-by-39sub39-39function39-39property39-39do39-39for39-39while39-39select39-or-39try39"></a>&#39;Quitter&#39; doit être suivi par &#39;Sub&#39;, &#39;fonction&#39;, &#39;propriété&#39;, &#39;faire&#39;, &#39;pour&#39;, &#39;tandis que&#39;, &#39;sélectionnez&#39;, ou &#39;essayez&#39;
Une instruction `Exit` contient un mot clé non valide. `Exit` doit spécifier le bloc à partir duquel le contrôle doit être transféré à l’instruction qui suit le bloc. Par exemple : `End While`.  
  
 **ID d’erreur :** BC30240  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez un mot clé valide après `Exit`ou supprimez l’instruction `Exit` .
