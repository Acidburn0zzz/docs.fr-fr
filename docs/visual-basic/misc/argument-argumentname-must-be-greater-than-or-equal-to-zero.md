# <a name="argument-39ltargumentnamegt39-must-be-greater-than-or-equal-to-zero"></a>Argument &#39; &lt;nom_argument&gt; &#39; doit être supérieur ou égal à zéro
Un argument n’est pas valide, car il est inférieur à zéro.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez l’orthographe des arguments. Un nom de variable mal orthographié peut créer implicitement une variable numérique initialisée à zéro.  
  
2.  Vérifiez les opérations précédentes sur les variables de l’expression, surtout celles passées dans la procédure en tant qu’arguments à partir d’autres procédures.
