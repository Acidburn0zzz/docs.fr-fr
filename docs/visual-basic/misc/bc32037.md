<span data-ttu-id="20d41-103">Les boucles imbriquées sont terminées avec une seule instruction `Next` qui spécifie plus de variables de boucle qu’il n’y en a dans l’imbrication, comme dans l’exemple suivant :</span><span class="sxs-lookup"><span data-stu-id="20d41-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>

```  
   For J = 1 To 5  
      ' ...

```
