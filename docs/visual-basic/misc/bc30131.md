# <a name="security-attribute-39ltattributenamegt39-cannot-be-applied-to-a-module"></a>Attribut de sécurité &#39; &lt;attributename&gt;&#39; ne peut pas être appliqué à un module
Le Common Language Runtime n’accepte pas l’attribut de sécurité sur un module.

**ID d’erreur :** BC30131

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Supprimez l’attribut de la déclaration.
