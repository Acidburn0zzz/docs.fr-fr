# <a name="comma--or-a-valid-expression-continuation-expected"></a>Virgule, ')' ou continuation d’expression valide attendue
Une liste, telle qu’une liste d’arguments dans une définition de procédure ou un appel de procédure, ne sépare pas ou ne met pas fin correctement à ses membres. Il est possible que deux membres de la liste soient séparés uniquement par un ou plusieurs espaces.  
  
 **ID d’erreur :** BC32017  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez une virgule (,) pour séparer les membres de la liste, ainsi qu’une parenthèse fermante ()) pour terminer la liste.
