# <a name="bracketed-identifier-is-missing-closing-3939"></a>Identificateur entre crochets fermant manquant &#39;]&#39;
Dans un nom échappé, les crochets doivent apparaître par paires.  
  
 **ID d’erreur :** BC30034  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Placez un crochet fermant à la fin du nom échappé.
