# <a name="ltnamegt-cannot-expose-the-underlying-delegate-type-ltdelegatetypegt-of-the-event-it-is-implementing-to-ltspecifier1gt-lttype1gt-through-ltspecifier2gt-lttype2gt"></a>«&lt;nom&gt;'ne peut pas exposer le type délégué sous-jacent'&lt;type_délégué&gt;' de l’événement qu’il implémente à &lt;spécificateur1&gt; '&lt;type1&gt;» via &lt;spécificateur2&gt; '&lt;type2&gt;'
Le code expose un type privé en dehors de la classe publique.  
  
 **ID d’erreur :** BC30914  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez le type `Public` ou utilisez un autre type.
