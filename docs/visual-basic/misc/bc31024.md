# <a name="unable-to-create-a-net-runtime-interface-lterrorgt"></a>Impossible de créer une interface Runtime .NET : &lt;erreur&gt;
Une erreur interne s’est produite lors de la création d’une interface, ou un appel de méthode sur une interface a échoué.  
  
 **ID d’erreur :** BC31024  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Enregistrez votre travail et redémarrez Visual Studio.  
  
2.  Si l’erreur persiste, réinstallez Visual Studio.  
  
3.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
