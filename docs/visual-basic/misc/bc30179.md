# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; '&lt;typename&gt;' et &lt;type2&gt; '&lt;typename&gt;' sont en conflit dans &lt;espace de noms&gt; '&lt;namespacename &gt;'
Deux éléments déclarés, tels qu’une classe et un module, ont le même nom dans un même espace de noms.  
  
 **ID d’erreur :** BC30179  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déplacez la déclaration de l’un des éléments vers un autre espace de noms.  
  
-   Modifiez le nom de l’un des éléments.
