# <a name="39ltelementnamegt39-is-not-a-parameter-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>&#39;&lt;ElementName&gt; &#39; n’est pas un paramètre de méthode d’extension &#39; &lt;methodname&gt; &#39; définies dans &#39; &lt;typename&gt;&#39;
Un appel de procédure fournit un argument supplémentaire qui n’est pas défini par la méthode d’extension.  
  
 **ID d’erreur :** BC36585  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’argument superflu de la liste d’arguments.
