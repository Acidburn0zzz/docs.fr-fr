# <a name="event-39lteventnamegt39-cannot-be-handled-because-it-is-not-accessible-from-39ltnamegt39"></a>Événement &#39; &lt;eventname&gt; &#39; ne peut pas être géré car il n’est pas accessible à partir de &#39; &lt;nom&gt;&#39;
Vous avez tenté de gérer un événement qui n’est pas accessible. Par exemple, si une variable `Handles` est partagée, la méthode gérant les événements doit également être partagée.  
  
 **ID d’erreur :** BC30585  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que l’événement est accessible.
