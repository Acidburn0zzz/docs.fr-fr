# <a name="mylog-object"></a>My.Log, objet
Fournit une propriété et des méthodes pour écrire des informations d’événements et les exceptions dans les écouteurs de journalisation de l’application.  
  
## <a name="remarks"></a>Notes  
 Pour plus d’informations sur les méthodes et propriétés de la `My.Log` , consultez <xref:Microsoft.VisualBasic.Logging.AspLog>.</xref:Microsoft.VisualBasic.Logging.AspLog>  
  
 Le `My.Log` est disponible pour que les applications ASP.NET. Pour les applications clientes, utilisez [My.Application.Log (objet)](../../../visual-basic/language-reference/objects/my-application-log-object.md).
