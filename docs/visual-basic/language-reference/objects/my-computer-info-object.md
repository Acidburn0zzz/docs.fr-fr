# My.Computer.Info Object
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Fournit des propriétés pour obtenir des informations concernant la mémoire, les assemblys chargés, le nom et le système d'exploitation de l'ordinateur.  
  
## Notes  
 Pour plus d'informations sur les propriétés de l'objet d' `My.Computer.Info` , consultez l' <xref:Microsoft.VisualBasic.Devices.ComputerInfo>.
