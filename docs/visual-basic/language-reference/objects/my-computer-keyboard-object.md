# My.Computer.Keyboard Object
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

Fournit des propriétés pour accéder à l'état actuel du clavier, par exemple pour savoir quelles touches sont actuellement utilisées, et fournit une méthode pour envoyer des séquences de touches à la fenêtre active.  
  
## Notes  
 Pour plus d'informations sur les méthodes et les propriétés de l'objet d' `My.Computer.Keyword` , consultez l' <xref:Microsoft.VisualBasic.Devices.Keyboard>.  
  
 Pour plus d'informations, consultez [Accessing the Keyboard](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-keyboard.md).  
  
## Configuration requise  
 **Espace de noms :** <xref:Microsoft.VisualBasic.Devices>  
  
 **Classe :** <xref:Microsoft.VisualBasic.Devices.Keyboard>  
  
 **Assembly :** bibliothèque Runtime Visual Basic \(dans Microsoft.VisualBasic.dll\)
