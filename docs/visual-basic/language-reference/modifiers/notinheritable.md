# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
Spécifie qu’une classe ne peut pas être utilisée comme classe de base.  
  
## <a name="remarks"></a>Notes  
 **Autres conditions**. Une classe ne peut pas être héritée est parfois appelée un *sealed* classe.  
  
 Le `NotInheritable` modificateur peut être utilisé dans ce contexte :  
  
 [Class (instruction)](../../../visual-basic/language-reference/statements/class-statement.md)
