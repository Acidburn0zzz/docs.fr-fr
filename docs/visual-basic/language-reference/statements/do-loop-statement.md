```  
Do { While | Until } condition  
    [ statements ]  
    [ Continue Do ]

    [ Exit Do ]

    [ statements ]  
    [ Continue Do ]  
    [ statements ]  
    [ Exit Do ]  
    [ statements ]  
Loop { While | Until } condition  
```
