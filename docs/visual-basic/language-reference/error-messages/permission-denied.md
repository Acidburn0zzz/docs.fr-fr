# <a name="permission-denied-visual-basic"></a>Autorisation refusée (Visual Basic)
Un utilisateur a tenté d’écrire sur un disque protégé en écriture ou accéder à un fichier verrouillé.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Pour ouvrir un fichier protégé en écriture, modifiez l’attribut de protection en écriture du fichier.  
  
2.  Assurez-vous qu’un autre processus n’a pas verrouillé le fichier et attendez avant d’ouvrir le fichier que l’autre processus libère.  
  
3.  Pour accéder au Registre, vérifiez que vos autorisations d’utilisateur incluent ce type d’accès au Registre.
