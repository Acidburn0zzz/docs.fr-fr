# <a name="expression-does-not-produce-a-value"></a>Cette expression ne produit pas de valeur
Vous avez tenté d’utiliser une expression qui ne produit pas une valeur dans un contexte produisant des valeurs, comme l’appel une `Sub` dans un contexte où une `Function` est attendu.  
  
 **ID d’erreur :** BC30491  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez l’expression pour qu’il produit une valeur.
