# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>&#39;&lt;nom1&gt; &#39; est ambigu, importé des espaces de noms ou des types &#39; &lt;name2&gt;&#39;
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le compilateur Visual Basic n’a pas les règles de résolution de conflit ; Vous devez lever l’ambiguïté des noms vous-même.  
  
 **ID d’erreur :** BC30561  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Lever l’ambiguïté du nom en supprimant les importations d’espace de noms.  
  
2.  Utilisez un nom qualifié complet.
