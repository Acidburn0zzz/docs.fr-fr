# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>«&lt;nom1&gt;'est ambigu, importé des espaces de noms ou des types'&lt;name2&gt;»
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilateur n’a pas les règles de résolution de conflit, vous devez distinguer des noms vous-même.  
  
 **ID d’erreur :** BC30561  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Lever l’ambiguïté du nom en supprimant les importations d’espace de noms.  
  
2.  Utilisez un nom qualifié complet.
