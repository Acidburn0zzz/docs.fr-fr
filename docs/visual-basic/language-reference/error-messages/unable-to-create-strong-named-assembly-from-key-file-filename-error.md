# <a name="unable-to-create-strong-named-assembly-from-key-file-39ltfilenamegt39-lterrorgt"></a>Impossible de créer un assembly avec nom fort à partir du fichier de clé &#39; &lt;filename&gt;&#39;: &lt;erreur&gt;
Un assembly avec nom fort n’a pas pu être créé à partir du fichier de clé spécifié.  
  
 **ID d’erreur :** BC31026  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que le fichier de clé correct a été spécifié, et qu’il n’est pas verrouillé par une autre application.
