# <a name="unable-to-create-strong-named-assembly-from-key-file-filename-error"></a>Impossible de créer un assembly avec nom fort à partir du fichier de clé '\<nom_fichier >': \<erreur >
Un assembly avec nom fort n’a pas pu être créé à partir du fichier de clé spécifié.  
  
 **ID d’erreur :** BC31026  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez que le fichier de clé correct a été spécifié, et qu’il n’est pas verrouillé par une autre application.
