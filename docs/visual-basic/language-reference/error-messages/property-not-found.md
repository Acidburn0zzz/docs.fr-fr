# <a name="property-not-found"></a>Propriété introuvable
Cet objet ne prend pas en charge la propriété spécifiée.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez l’orthographe du nom de la propriété.  
  
2.  Consultez la documentation de l’objet pour vous assurer que vous ne tentez pas d’accéder à un élément comme une propriété « text » lorsque l’objet prend en charge un « caption » ou une propriété portant le même nom.
