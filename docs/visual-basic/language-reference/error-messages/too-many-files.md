# <a name="too-many-files"></a>Trop de fichiers
Plusieurs fichiers ont été créés dans le répertoire racine autorisés par le système d’exploitation ou d’autres fichiers que le nombre spécifié dans le **fichiers =** définissant dans votre configuration. Fichier SYS.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Si votre programme est ouvrir, fermer ou enregistrer des fichiers dans le répertoire racine, modifiez votre programme afin qu’il utilise un sous-répertoire.  
  
2.  Augmenter le nombre de fichiers spécifiés dans votre **fichiers =** définissant dans votre configuration. SYS et redémarrez votre ordinateur.
