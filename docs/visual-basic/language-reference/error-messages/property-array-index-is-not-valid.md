# <a name="property-array-index-is-not-valid"></a>L'index de tableau de propriétés est non valide
La valeur fournie n’est pas valide pour un index de tableau de propriétés.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez la documentation du composant pour vous assurer que votre index se trouve dans la plage valide pour la propriété spécifiée.
