# <a name="39optional39-expected"></a>&#39; facultatif &#39; attendu
Un argument facultatif dans une déclaration de procédure est suivi d’un argument obligatoire. Chaque argument d’un argument facultatif doit également être facultatif.  
  
 **ID d’erreur :** BC30202  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Si l’argument est destiné à être nécessaire, déplacez afin qu’il précède le premier argument facultatif dans la liste d’arguments.  
  
2.  Si l’argument est destiné à être facultatif, utilisez le `Optional` (mot clé).
