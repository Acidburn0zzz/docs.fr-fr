# <a name="unable-to-find-required-file-39ltfilenamegt39"></a>Impossible de trouver le fichier requis &#39; &lt;nom de fichier&gt;&#39;

 **ID d’erreur :** BC30655  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Réinstallez Visual Studio.
