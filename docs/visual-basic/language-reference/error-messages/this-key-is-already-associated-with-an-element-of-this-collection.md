# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>Cette clé est déjà associée à un élément de cette collection
La clé spécifiée pour un membre de collection identifie déjà un autre membre de la collection. Une clé est une chaîne spécifiée dans le `Add` (méthode) qui identifie de façon unique un membre spécifique d’une collection.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utiliser une autre clé pour ce membre.
