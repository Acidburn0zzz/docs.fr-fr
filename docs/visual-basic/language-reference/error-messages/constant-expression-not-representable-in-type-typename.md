# Constant expression not representable in type &#39;&lt;typename&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

Vous tentez d'évaluer une constante qui ne correspondra pas au type cible, généralement car elle dépasse la plage.  
  
 **ID d'erreur :** BC30439  
  
### Pour corriger cette erreur  
  
1.  Modifiez le type cible en un type capable de gérer la constante.
