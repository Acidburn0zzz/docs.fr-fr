# &#39;&lt;name&gt;&#39; is ambiguous in the namespace &#39;&lt;namespacename&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

Vous avez fourni un nom ambigu qui entre par conséquent en conflit avec un autre nom.  Le compilateur [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] ne possède aucune règle de résolution de conflit ; vous devez lever vous\-même toute ambiguïté sur les noms.  
  
 **ID d'erreur :** BC30560  
  
### Pour corriger cette erreur  
  
-   Qualifiez le nom de manière complète.
