# <a name="help--visual-basic"></a>/help, /? (Visual Basic)
Affiche les options du compilateur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
/help  
' -or-  
/?  
```  
  
## <a name="remarks"></a>Notes  
 Si vous incluez cette option dans une compilation, aucun fichier de sortie n’est créé et aucune compilation ne se produit.  
  
> [!NOTE]
>  Le `/help` option n’est pas accessible à partir de la [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] l’environnement de développement ; il est disponible uniquement lors de la compilation à partir de la ligne de commande.  
  
## <a name="example"></a>Exemple  
 Le code suivant affiche l’aide de la ligne de commande.  
  
```  
vbc /help  
```
