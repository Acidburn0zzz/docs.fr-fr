# <a name="visual-basic-language-specification"></a>Spécification du langage Visual Basic
Le [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] spécification du langage est la source faisant autorité pour obtenir des réponses à toutes les questions sur [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] grammaire et la syntaxe. Il contient des informations détaillées sur le langage, y compris de nombreux points non couverts dans la [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] documentation de référence.  
  
 La spécification est disponible sur le [Microsoft Download Center](http://go.microsoft.com/fwlink/?LinkId=188623).
