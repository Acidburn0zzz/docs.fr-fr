# WCF et .NET&#160;Framework Client Profile
[!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] est une version légère du .Net Framework complet destinée aux clients qui n'ont pas besoin de l'infrastructure entière.  Windows Communication Foundation n'est pas intégralement pris en charge par l'infrastructure cliente.  
  
## Fonctionnalités WCF prises en charge par .Net Framework Client Profile  
 Les fonctionnalités suivantes de Windows Communication Foundation sont prises en charge par [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] :  
  
-   WCF est intégralement pris en charge, à l'exception de Cardspace et l'hébergement Web.  
  
-   Les canaux TCP\/IP de communication à distance sont pris en charge.  
  
-   Les services Web Asmx ne sont pas pris en charge.
