# H&#233;bergement
Les rubriques de cette section décrivent l'hébergement de service.Un service peut être hébergé par les services IIS \(Internet Information Services\), le service WAS \(Windows Process Activation Service\), Windows Server AppFabric, un service Windows ou par une application managée. Cette option est souvent connue sous le nom d'*auto\-hébergement*.  
  
 Il est important de noter que l'exécution d'un service ou d'une extension à partir d'un hôte non fiable compromet la sécurité.
