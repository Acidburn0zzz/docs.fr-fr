# <a name="data-contract-serializer"></a>Sérialiseur de contrat de données
L'infrastructure [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] utilise la classe <xref:System.Runtime.Serialization.DataContractSerializer> comme sérialiseur par défaut. Les rubriques de cette section fournissent davantage d'informations sur son utilisation en tant que sérialiseur autonome.  
  
## <a name="in-this-section"></a>Dans cette section  
 [Sérialisation et désérialisation](../../../../docs/framework/wcf/feature-details/serialization-and-deserialization.md)  
 Explique comment utiliser la classe <xref:System.Runtime.Serialization.DataContractSerializer> pour sérialiser des graphiques d'objet et désérialiser des documents XML dans des objets.  
  
 [Exportation et importation de schéma](../../../../docs/framework/wcf/feature-details/schema-import-and-export.md)  
 Explique comment générer un schéma en langage XSD (XML Schema Definition) XML à partir d'une classe [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)], ou comment générer du code à partir d'un document de schéma XSD.
