# Interop&#233;rabilit&#233; avec Web Services Enhancements&#160;3.0
Les clients et services [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] sont compatible au niveau câble avec les clients et les services Web Services Enhancements 3.0 for Microsoft .NET \(WSE\) lorsque les clients et les services [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] sont configurés pour utiliser la version d'août 2004 de la spécification WS\-Addressing.
