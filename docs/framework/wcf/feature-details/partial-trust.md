# Confiance partielle
À partir du [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)], les appelants d'un niveau de confiance partiel peuvent accéder à des types et des méthodes publics implémentés dans <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>et <xref:System.ServiceModel.Web>.Cette section décrit des scénarios pris en charge pour utiliser [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] dans une application d'un niveau de confiance partiel ainsi que le sous\-ensemble limité des fonctionnalités [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] disponible aux applications qui s'exécutent avec des autorisations réduites de la sécurité d'accès du code.

 Décrit les principaux scénarios de confiance partielle pour exécuter [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)].

 Décrit les fonctionnalités [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] qui ne peuvent pas être utilisées avec la confiance partielle.
