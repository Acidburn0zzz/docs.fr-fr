# Jetons personnalis&#233;s
Bien que [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] prenne en charge en mode natif les certificats X.509, les jetons de contexte de sécurité, les tickets Kerberos et les jetons de nom d'utilisateur comme informations d'identification, il procure une flexibilité suffisante pour vous permettre d'utiliser vos propres informations d'identification personnalisées.

 Décrit comment créer un jeton de sécurité personnalisé à l'aide de la classe <xref:System.IdentityModel.Tokens.SecurityToken> et comment l'intégrer à un fournisseur et un authentificateur de jetons de sécurité personnalisés.
