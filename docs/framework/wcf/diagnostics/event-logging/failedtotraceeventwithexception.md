# <a name="failedtotraceeventwithexception"></a>FailedToTraceEventWithException
Id : 105  
  
 Gravité : Error  
  
 Catégorie : Traçage  
  
## <a name="description"></a>Description  
 Un événement ou des événements n'ont pas été tracés.  
  
 L'événement répertorie la chaîne d'événement, l'exception, le nom de processus et l'ID de processus d'origine.
