# ImpersonationFailure
ID : 176  
  
 Gravité : Erreur  
  
 Catégorie : SecurityAudit  
  
## Description  
 Cet événement indique que l'emprunt d'identité a échoué et répertorie le nom de méthode, l'identité du client et l'ID d'activité.
