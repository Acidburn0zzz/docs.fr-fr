# ParticipantRecoveryLogEntryCorrupt
ID : 138  
  
 Gravité : Erreur  
  
 Catégorie : TransactionBridge  
  
## Description  
 Cet événement indique qu'une entrée de journal de récupération de participant a été endommagée et n'a pas pu être désérialisée.Cette erreur risque d'entraîner une perte de données.L'événement répertorie l'ID de transaction, les données de récupération \(encodage Base64\), l'exception, le nom de processus et l'ID de processus.
