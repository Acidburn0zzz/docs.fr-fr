# <a name="wmiputinstancefailed"></a>WmiPutInstanceFailed
ID : 121  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique que la demande PutInstance WMI n'a pas été traitée. L'événement répertorie l'objet WMI, l'exception, le nom de processus et l'ID de processus.
