# WmiPutInstanceFailed
ID : 121  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique que la demande PutInstance WMI n'a pas été traitée.Il répertorie l'objet WMI, l'exception, le nom de processus et l'ID de processus.
