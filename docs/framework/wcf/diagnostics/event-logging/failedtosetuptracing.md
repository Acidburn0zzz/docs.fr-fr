# FailedToSetupTracing
ID : 100  
  
 Gravité : Erreur  
  
 Catégorie : suivi  
  
## Description  
 Échec de la tentative d'installation du suivi.  Le suivi est désactivé.  
  
 L'événement répertorie l'exception, le nom de processus et l'ID de processus.
