# WmiDeleteInstanceFailed
ID : 122  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique que la demande DeleteInstance WMI n'a pas été traitée.Il répertorie l'exception, le nom de processus et l'ID de processus.
