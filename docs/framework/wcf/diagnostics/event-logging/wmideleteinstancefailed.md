# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
ID : 122  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique que la demande DeleteInstance WMI n'a pas été traitée. L'événement répertorie l'exception, le nom de processus et l'ID de processus.
