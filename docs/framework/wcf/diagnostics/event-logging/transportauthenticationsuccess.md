# TransportAuthenticationSuccess
ID : 173  
  
 Gravité : Informatif  
  
 Catégorie : SecurityAudit  
  
## Description  
 Cet événement indique que l'authentification de transport a réussi.L'événement répertorie le service, l'identité du client et l'ID d'activité.
