# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
Id : 106  
  
 Gravité : Error  
  
 Catégorie : Traçage  
  
## <a name="description"></a>Description  
 Cet événement indique qu'une assertion de type invariant a la valeur false. L'événement répertorie le message, le nom de processus et l'ID de processus.
