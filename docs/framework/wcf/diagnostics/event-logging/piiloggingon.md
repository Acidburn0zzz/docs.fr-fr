# PiiLoggingOn
ID : 107  
  
 Gravité : Erreur  
  
 Catégorie : suivi  
  
## Description  
 Cet événement indique que la journalisation PII a été activée.Les informations sensibles seront enregistrées.Cet événement répertorie le nom et l'ID du processus.
