# MessageLoggingOff
ID : 117  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique que la journalisation des messages a été désactivée.Cet événement répertorie le nom et l'ID de processus.
