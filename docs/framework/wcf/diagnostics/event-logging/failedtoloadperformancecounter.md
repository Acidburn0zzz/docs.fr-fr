# <a name="failedtoloadperformancecounter"></a>FailedToLoadPerformanceCounter
ID : 118  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique qu'un compteur de performance n'a pas été chargé. L'événement répertorie le nom de la catégorie, le nom de compteur, le nom de l'instance, l'exception, le nom de processus et l'ID de processus.
