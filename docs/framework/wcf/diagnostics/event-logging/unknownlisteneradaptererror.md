# UnknownListenerAdapterError
ID : 161  
  
 Gravité : Erreur  
  
 Catégorie : SMSvcHost  
  
## Description  
 Cet événement indique qu'une erreur inattendue a eu lieu dans l'adaptateur d'écouteur en gérant une notification WAS \(Windows Processus Activation Service\).  Le processus est terminé.  L'événement répertorie le protocole, l'exception, le nom de processus et l'ID de processus.
