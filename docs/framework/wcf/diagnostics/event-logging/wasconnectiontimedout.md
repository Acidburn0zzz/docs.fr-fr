# WasConnectionTimedout
ID : 163  
  
 Gravité : Erreur  
  
 Catégorie : SMSvcHost  
  
## Description  
 Cet événement indique l'expiration d'une connexion que nécessite le service WAS \(Windows Process Activation Service\), et répertorie le nom de processus et l'ID.
