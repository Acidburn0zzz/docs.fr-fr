# MessageQueueDuplicatedPipeLeak
ID : 166  
  
 Gravité : Erreur  
  
 Catégorie : SMSvcHost  
  
## Description  
 Cet événement indique que l'erreur s'est produite lors de la distribution du canal nommé dupliqué.Cet handle est à présent manquant dans le processus.Cet événement répertorie la source, l'exception, l'ID et le nom du processus.
