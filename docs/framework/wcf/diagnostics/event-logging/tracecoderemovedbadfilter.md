# TraceCodeRemovedBadFilter
ID : 114  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique qu'un filtre de journalisation des messages incorrect qui accédait au corps du message a été retiré.Il répertorie le XPath, le nom de processus et l'ID de processus.
