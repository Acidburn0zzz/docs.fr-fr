# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
ID : 160  
  
 Gravité : Erreur  
  
 Catégorie : SMSvcHost  
  
## <a name="description"></a>Description  
 Cet événement indique qu'une erreur s'est produite dans le service d'activation spécifié du protocole spécifié, lors d'une tentative d'écoute du site spécifié ; le protocole est par conséquent désactivé momentanément pour le site. L'événement répertorie l'URL, l'état, l'exception, le nom du processus et l'ID du processus.
