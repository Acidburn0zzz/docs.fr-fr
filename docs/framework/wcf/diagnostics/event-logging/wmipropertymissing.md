# WmiPropertyMissing
ID : 129  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique qu'une non\-correspondance de type est survenue.La classe CIM spécifiée ne dispose pas de la propriété spécifiée pour le type spécifié.Cet événement répertorie le nom et l'ID du processus.
