# <a name="wmipropertymissing"></a>WmiPropertyMissing
ID : 129  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique qu'une non-correspondance de type s'est produite. La classe CIM spécifiée ne dispose pas de la propriété spécifiée pour le type spécifié. Cet événement répertorie le nom et l'ID de processus.
