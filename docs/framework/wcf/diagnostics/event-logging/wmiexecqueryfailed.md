# <a name="wmiexecqueryfailed"></a>WmiExecQueryFailed
Id : 124  
  
 Gravité : Error  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique que la requête WMI ExecQuery n'a pas été traitée. L'événement répertorie l'exception, le nom de processus et l'ID de processus.
