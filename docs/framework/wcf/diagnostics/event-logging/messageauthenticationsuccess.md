# MessageAuthenticationSuccess
ID : 169  
  
 Gravité : Informatif  
  
 Catégorie : SecurityAudit  
  
## Description  
 Cet événement indique que l'authentification du message a réussi.Cet événement répertorie le service, l'action, l'identité du client et l'ID d'activité.
