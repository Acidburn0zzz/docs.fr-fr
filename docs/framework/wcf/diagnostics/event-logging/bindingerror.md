# <a name="bindingerror"></a>BindingError
ID : 159  
  
 Gravité : Erreur  
  
 Catégorie : SMSvcHost  
  
## <a name="description"></a>Description  
 Cet événement indique qu’une erreur s’est produite lors de l’analyse de la liaison spécifique du site donné, et que par conséquent, le protocole est temporairement désactivé pour ce site. Il répertorie la liaison, la source, l’exception, le nom de processus et l’ID de processus.
