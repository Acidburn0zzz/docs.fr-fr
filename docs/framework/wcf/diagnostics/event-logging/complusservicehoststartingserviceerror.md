# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
ID : 130  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique qu'une erreur est survenue lors du démarrage du service COM+. L'événement répertorie l'AppID, CLSID, le substitut, l'exception, le nom et l'ID de processus.
