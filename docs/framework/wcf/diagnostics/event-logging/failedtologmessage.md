# FailedToLogMessage
ID : 113  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique qu'un message n'a pas été enregistré.Il répertorie l'exception, le nom de processus et l'ID de processus.
