# <a name="failedtologmessage"></a>FailedToLogMessage
ID : 113  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique qu'un message n'a pas été enregistré. L'événement répertorie l'exception, le nom de processus et l'ID de processus.
