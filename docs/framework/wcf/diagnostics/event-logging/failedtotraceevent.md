# FailedToTraceEvent
ID : 104  
  
 Gravité : Erreur  
  
 Catégorie : suivi  
  
## Description  
 Un ou plusieurs événements n'ont pas été suivis.L'événement répertorie la chaîne de l'événement d'origine, ainsi que le nom et l'ID de processus.
