# <a name="protocolrecoverycomplete"></a>ProtocolRecoveryComplete
ID : 150  
  
 Gravité : Erreur  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  
 Cet événement indique que le démarrage et la récupération du service du protocole de transaction WS-Atomic se sont déroulés avec succès. L'événement répertorie l'ID de protocole, le nom de protocole, le nom de processus et l'ID de processus.
