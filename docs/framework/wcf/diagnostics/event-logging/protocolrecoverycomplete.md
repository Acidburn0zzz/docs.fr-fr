# ProtocolRecoveryComplete
ID : 150  
  
 Gravité : Erreur  
  
 Catégorie : TransactionBridge  
  
## Description  
 Cet événement indique que le démarrage et la récupération du service du protocole de transaction WS\-Atomic se sont déroulés avec succès.Cet événement répertorie l'ID et le nom du protocole ainsi que l'ID et le nom du processus.
