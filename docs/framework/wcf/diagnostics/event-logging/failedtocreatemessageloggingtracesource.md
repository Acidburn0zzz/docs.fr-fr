# FailedToCreateMessageLoggingTraceSource
ID : 115  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique que l'enregistrement des messages n'est pas parvenu à créer la source du suivi.Cet événement répertorie l'exception, l'ID et le nom du processus.
