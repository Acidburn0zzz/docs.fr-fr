# <a name="failedtocreatemessageloggingtracesource"></a>FailedToCreateMessageLoggingTraceSource
ID : 115  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique que l'enregistrement des messages n'est pas parvenu à créer la source du suivi. L'événement répertorie l'exception, le nom de processus et l'ID de processus.
