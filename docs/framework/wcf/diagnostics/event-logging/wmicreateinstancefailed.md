# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
Id : 123  
  
 Gravité : Error  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique que la demande CreateInstance WMI n'a pas été traitée. L'événement répertorie le nom de la classe, l'exception, le nom de processus et l'ID de processus.
