# WmiRegistrationFailed
ID : 126  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique que le fournisseur WMI n'a pas été enregistré.Cet événement répertorie l'objet WMI, l'erreur, le nom et l'ID du processus.
