# <a name="impersonationsuccess"></a>ImpersonationSuccess
ID : 175  
  
 Gravité : Informatif  
  
 Catégorie : SecurityAudit  
  
## <a name="description"></a>Description  
 Cet événement indique que l'emprunt d'identité a réussi et répertorie le nom de méthode, l'identité du client et l'ID d'activité.
