# WebHostFailedToProcessRequest
ID : 111  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique que WebHost n'a pas pu traiter une demande.L'événement répertorie les informations sur l'émetteur, l'exception, le nom de processus et l'ID de processus.
