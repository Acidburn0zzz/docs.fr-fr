# WmiUnregistrationFailed
ID : 127  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique que le fournisseur WMI n'a pas été désinscrit.L'événement répertorie l'objet WMI, l'erreur, le nom de processus et l'ID de processus.
