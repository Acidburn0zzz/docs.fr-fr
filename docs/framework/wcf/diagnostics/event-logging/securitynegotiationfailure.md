# <a name="securitynegotiationfailure"></a>SecurityNegotiationFailure
ID : 172  
  
 Gravité : Erreur  
  
 Catégorie : SecurityAudit  
  
## <a name="description"></a>Description  
 Cet événement indique que la négociation de sécurité a échoué. Il répertorie le service, l'action, l'identité du client, l'ID d'activité et la négociation.
