# StartErrorPublish
ID : 158  
  
 Gravité : Erreur  
  
 Catégorie : SMSvcHost  
  
## Description  
 Cet événement indique que le démarrage du service NT a échoué.  Le point de terminaison d'écoute n'a pas pu être publié.  Cet événement répertorie le nom et l'ID de processus.
