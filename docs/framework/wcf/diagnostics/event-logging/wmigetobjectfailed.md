# WmiGetObjectFailed
ID : 120  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## Description  
 Cet événement indique que la requête WMI GetObject n'a pas été traitée.L'événement répertorie l'objet WMI, l'exception, le nom de processus et l'ID de processus.
