# <a name="securitynegotiationsuccess"></a>SecurityNegotiationSuccess
ID : 171  
  
 Gravité : Informatif  
  
 Catégorie : SecurityAudit  
  
## <a name="description"></a>Description  
 Cet événement indique que la négociation de sécurité a réussi et répertorie le nom du service, l'action, l'identité du client, l'ActivityId et la négociation.
