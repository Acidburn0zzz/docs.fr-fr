# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
Id : 144  
  
 Gravité : Error  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  
 Cet événement indique que le protocole MSDTC WS-Atomic Transaction a échoué au début de la récupération. Par conséquent, les fonctionnalités WS-Atomic Transaction sont désactivées. L'événement répertorie l'ID de protocole, le nom de protocole, l'exception, le nom de processus et l'ID de processus.
