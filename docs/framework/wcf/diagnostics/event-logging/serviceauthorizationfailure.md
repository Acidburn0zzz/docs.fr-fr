# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure
Id : 168  
  
 Gravité : Error  
  
 Catégorie : SecurityAudit  
  
## <a name="description"></a>Description  
 Cet événement indique que l'autorisation de service a échoué. L'événement répertorie le service, l'action, l'identité du client, AuthorizationContext, l'ID d'activité et ServiceAuthorizationManager.
