# <a name="wmiadmintypemismatch"></a>WmiAdminTypeMismatch
ID : 128  
  
 Gravité : Erreur  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique qu'une non-correspondance de type s'est produite. La propriété de classe CIM spécifiée ne peut pas accepter la valeur du type spécifique. Cet événement répertorie le nom et l'ID de processus.
