# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage
ID : 156  
  
 Gravité : Erreur  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  
 Cet événement indique que le certificat d'identité ayant le nom de sujet et l'empreinte numérique spécifiés ne fournit pas des données spécifiques parmi ses KeyUsages. Il répertorie également le nom de processus et l'ID de processus.
