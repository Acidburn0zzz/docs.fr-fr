# <a name="core-communications-channel-framework"></a>Communications principales : infrastructure de canal
Cette rubrique répertorie toutes les exceptions générées par l'infrastructure de canal [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
## <a name="exception-list"></a>Liste des exceptions  
  
|Code de la ressource|Chaîne de la ressource|  
|-------------------|---------------------|  
|ReceiveRequestTimedOut|La demande de réception sur l'adresse locale spécifiée a dépassé le délai imparti. Le temps alloué à cette opération peut avoir été une partie d'un délai d'expiration plus long.|  
|ReceiveTimedOutNoLocalAddress|L'opération de réception a dépassé le délai imparti. Le temps alloué à cette opération peut avoir été une partie d'un délai d'expiration plus long.|  
|UnableToDemuxChannel|Aucun canal n'est disponible pour accepter le message avec l'action spécifiée.|
