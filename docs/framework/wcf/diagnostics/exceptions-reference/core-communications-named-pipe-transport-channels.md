# <a name="core-communications-named-pipe-transport-channels"></a>Communications principales : canaux de transport du canal nommé
Cette rubrique répertorie toutes les exceptions générées par les canaux de transport du canal nommé.  
  
## <a name="exception-list"></a>Liste des exceptions  
  
|Code de la ressource|Chaîne de la ressource|  
|-------------------|---------------------|  
|PipeCantCloseWithPendingWrite|Le canal ne peut pas être fermé alors qu'une opération d'écriture sur le canal est en attente.|  
|PipeReadPending|Une opération de lecture est en cours pour le canal.|  
|PipeShutdownReadError|L'opération de lecture de l'indicateur 'shutdown' du canal a échoué.|  
|PipeShutdownWriteError|L'opération d'écriture de l'indicateur 'shutdown' du canal a échoué.|  
|PipeWritePending|Une opération d'écriture est en cours pour le canal.|
