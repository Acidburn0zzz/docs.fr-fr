# Communications principales&#160;: utilitaires
Cette rubrique répertorie toutes les exceptions générées par les utilitaires de la communication principale.  
  
## Liste des exceptions  
  
|Code de la ressource|Chaîne de la ressource|  
|--------------------------|----------------------------|  
|AsyncObjectAlreadyEnded|La méthode End a déjà été appelée sur cet objet de résultat asynchrone.|
