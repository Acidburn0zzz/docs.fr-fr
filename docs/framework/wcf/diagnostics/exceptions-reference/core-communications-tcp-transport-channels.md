# <a name="core-communications-tcp-transport-channels"></a>Communications principales : canaux de transport TCP
Cette rubrique répertorie toutes les exceptions générées par les canaux de transport TCP.  
  
## <a name="exception-list"></a>Liste des exceptions  
  
|Code de la ressource|Chaîne de la ressource|  
|-------------------|---------------------|  
|SocketCloseReadTimeout|Le point de terminaison distant du socket spécifié n'a pas répondu à une demande de fermeture dans le délai imparti. Il est possible que le point de terminaison distant n'appelle pas Close après avoir reçu le signal EOF (null) de l'opération de réception. Le temps alloué à cette opération peut avoir été une partie d'un délai d'expiration plus long.|
