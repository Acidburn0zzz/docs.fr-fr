# Messagerie fiable
Cette rubrique répertorie toutes les exceptions levées par la messagerie fiable.  
  
## Liste des exceptions  
  
|Code de la ressource|Chaîne de la ressource|  
|--------------------------|----------------------------|  
|InvalidBufferRemaining|Un message a été reçu par la mémoire tampon avec la taille restante spécifiée.La mémoire tampon restante ne peut pas être inférieure à zéro.|
