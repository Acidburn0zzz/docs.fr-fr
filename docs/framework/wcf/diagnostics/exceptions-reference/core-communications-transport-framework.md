# <a name="core-communications-transport-framework"></a>Communications principales : infrastructure de transport
Cette rubrique répertorie toutes les exceptions générées par l'infrastructure de transport.  
  
## <a name="exception-list"></a>Liste des exceptions  
  
|Code de la ressource|Chaîne de la ressource|  
|-------------------|---------------------|  
|Hosting_TransportBindingNotFound|Aucune liaison de protocole ne correspond à l'adresse spécifiée. Les liaisons de protocole sont configurées au niveau du site dans la configuration des services IIS (Internet Information Services) ou WAS (Windows Process Activation Services).|
