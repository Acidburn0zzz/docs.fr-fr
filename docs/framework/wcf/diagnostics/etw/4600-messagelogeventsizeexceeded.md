# 4600 - MessageLogEventSizeExceeded
## Propriétés  
  
|||  
|-|-|  
|ID|4600|  
|Mots clés|WCFMessageLogging|  
|Niveau|Avertissement|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Cet événement est émis lorsque le message n'a pas pu être enregistré, car il dépasse la taille d'événements ETW.  
  
## Message  
 Impossible de journaliser le message, car il dépasse la taille des événements ETW
