## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|ID|4821|  
|Mots clés|Découverte|  
|Niveau|Information|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  
 Cet événement est émis lorsque DiscoveryClientChannel a attribué à SynchronizationContext la valeur Null avant le lancement de l'opération Find.  
  
## <a name="message"></a>Message  
 DiscoveryClientChannel a attribué à SynchronizationContext la valeur Null avant le lancement de l'opération Find.
