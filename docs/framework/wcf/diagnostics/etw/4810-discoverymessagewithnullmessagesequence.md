# 4810 - DiscoveryMessageWithNullMessageSequence
## Propriétés  
  
|||  
|-|-|  
|ID|4810|  
|Mots clés|Découverte|  
|Niveau|Avertissement|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Cet événement est émis lorsque le message de découverte a été supprimé par DiscoveryClient, car il n'avait pas de propriété DiscoveryMessageSequence.  
  
## Message  
 Un message %1 avec messageId\='%2' a été supprimé par DiscoveryClient, car il n'avait pas la propriété DiscoveryMessageSequence.
