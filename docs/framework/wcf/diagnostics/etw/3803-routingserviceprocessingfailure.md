# 3803 - RoutingServiceProcessingFailure
## Propriétés  
  
|||  
|-|-|  
|ID|3803|  
|Mots clés|RoutingServices|  
|Niveau|Erreur|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Cet événement est émis lorsque le service de routage n'est pas parvenu à traiter un message sur le point de terminaison.  
  
## Message  
 Échec du service de routage lors du traitement d'un message sur le point de terminaison avec l'adresse « %1 ».
