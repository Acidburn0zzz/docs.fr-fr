# <a name="3825---routingserviceabandoningreceivecontext"></a>3825 - RoutingServiceAbandoningReceiveContext
## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|ID|3825|  
|Mots clés|RoutingServices|  
|Niveau|Avertissement|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  
 Cet événement est émis lorsque le service de routage appelle ReceiveContext.Abandon pour un message avec un ID spécifié.  
  
## <a name="message"></a>Message  
 Le service de routage appelle ReceiveContext.Abandon pour le message d'ID : « %1 ».
