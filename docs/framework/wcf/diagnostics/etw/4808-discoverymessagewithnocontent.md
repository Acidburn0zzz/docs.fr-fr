# 4808 - DiscoveryMessageWithNoContent
## Propriétés  
  
|||  
|-|-|  
|ID|4808|  
|Mots clés|Découverte|  
|Niveau|Avertissement|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Cet événement est émis lorsque le message de découverte a été supprimé, car son adresse de réponse ReplyTo n'était pas valide.  
  
## Message  
 Un message %1 a été supprimé, car il n'avait aucun contenu.
