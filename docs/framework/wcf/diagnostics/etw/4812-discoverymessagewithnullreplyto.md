# 4812 - DiscoveryMessageWithNullReplyTo
## Propriétés  
  
|||  
|-|-|  
|ID|4812|  
|Mots clés|Découverte|  
|Niveau|Avertissement|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Cet événement est émis lorsque le message de découverte a été supprimé par DiscoveryClient, car il n'avait pas d'adresse de réponse ReplyTo.  
  
## Message  
 Un message de demande de découverte avec messageId\='%1' a été supprimé, car il n'avait pas d'adresse de réponse ReplyTo.
