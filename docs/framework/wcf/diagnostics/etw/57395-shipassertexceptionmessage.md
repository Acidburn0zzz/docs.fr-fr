# 57395 - ShipAssertExceptionMessage
## Propriétés  
  
|||  
|-|-|  
|ID|57395|  
|Mots clés|Infrastructure|  
|Niveau|Erreur|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Analyse|  
  
## Description  
 Cet événement est émis lorsqu'une erreur inattendue se produit.  
  
## Message  
 Une erreur inattendue s'est produite.  Les applications ne doivent pas essayer de gérer cette erreur.  À des fins de diagnostic, ce message en anglais est associé à l'échec : %1.
