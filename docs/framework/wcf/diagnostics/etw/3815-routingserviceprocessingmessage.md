# 3815 - RoutingServiceProcessingMessage
## Propriétés  
  
|||  
|-|-|

|Niveau|Information|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Cet événement est émis lorsque le service de routage traite un message.  
  
## Message  
 Le service de routage traite un message avec l'ID : « %1 », action : « %2 », URL entrante : « %3 », Reçu dans la transaction : %4.
