## <a name="description"></a>Description  
 Cet événement est émis lorsque la fermeture de toutes les instances de canal d'écouteur est terminée.  
  
## <a name="message"></a>Message  
 Fermeture de toutes les instances du canal de l'écouteur à l'aide de WAS terminée.
