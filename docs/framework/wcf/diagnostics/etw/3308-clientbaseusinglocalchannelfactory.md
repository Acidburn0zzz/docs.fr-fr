# 3308 - ClientBaseUsingLocalChannelFactory

## Description  
 Cet événement est émis lorsqu'aucune fabrique de canaux n'est utilisée dans le cache.  
  
## Message  
 Ne pas utiliser la fabrique de canaux du cache, c'est\-à\-dire,  mise en cache désactivée pour l'instance.
