# <a name="3820---routingservicecreatingtransaction"></a>3820 - RoutingServiceCreatingTransaction
## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|ID|3820|  
|Mots clés|RoutingServices|  
|Niveau|Information|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  
 Cet événement est émis lorsque le service de routage a créé une nouvelle transaction pour traiter un message.  
  
## <a name="message"></a>Message  
 Le service de routage a créé une nouvelle transaction avec l'id « %1 »' pour le traitement des messages.
