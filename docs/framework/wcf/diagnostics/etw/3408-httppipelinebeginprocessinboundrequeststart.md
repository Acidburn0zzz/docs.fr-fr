## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|ID|3408|  
|Mots clés|HTTP|  
|Niveau|Verbose|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  
 Cet événement est émis lorsque le gestionnaire de messages HTTP a commencé le traitement de la demande entrante de façon asynchrone.  
  
## <a name="message"></a>Message  
 Le gestionnaire de messages HTTP a démarré le traitement asynchrone de la demande entrante.
