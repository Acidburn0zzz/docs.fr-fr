# 3305 - ClientBaseCachedChannelFactoryCount
## Propriétés  
  
|||

## Description  
 Cet événement est émis pour indiquer le nombre de fabriques de canaux mises en cache par base cliente.  
  
## Message  
 Le nombre de fabriques de canaux mises en cache est de %1.  Seules «%2 » fabriques de canaux peuvent être mises en cache.
