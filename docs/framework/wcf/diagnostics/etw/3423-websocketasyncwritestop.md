## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|ID|3423|  
|Mots clés|HTTP|  
|Niveau|Verbose|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  
 Cet événement est émis lorsque l'écriture asynchrone WebSocket a cessé.  
  
## <a name="message"></a>Message  
 Arrêt de l'écriture asynchrone de l'élément WebSocketId %1.
