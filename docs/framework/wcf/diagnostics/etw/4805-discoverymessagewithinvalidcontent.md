# 4805 - DiscoveryMessageWithInvalidContent
## Propriétés  
  
|||  
|-|-|  
|ID|4805|  
|Mots clés|Découverte|  
|Niveau|Avertissement|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Cet événement est émis lorsque le message de découverte contient du contenu non valide.  
  
## Message  
 Un message %1 avec messageId\='%2' a été supprimé, car son contenu n'était pas valide.
