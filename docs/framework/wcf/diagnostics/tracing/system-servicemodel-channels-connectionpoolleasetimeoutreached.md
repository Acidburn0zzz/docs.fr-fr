# System.ServiceModel.Channels.ConnectionPoolLeaseTimeoutReached
System.ServiceModel.Channels.ConnectionPoolLeaseTimeoutReached  
  
## Description  
 Une connexion a dépassé le délai d'attente du bail de la connexion de ce pool de connexions et a été fermée.
