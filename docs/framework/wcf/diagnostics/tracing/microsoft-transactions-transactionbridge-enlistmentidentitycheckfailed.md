# Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
L'identité de l'expéditeur du message entrant ne correspond pas à celle dérivée de la référence du point de terminaison de l'inscription.  
  
## Description  
 Un suivi est généré si la validation de la sécurité de niveau message a échoué.  
  
## Résolution des problèmes  
 Assurez\-vous que la sécurité de la transaction est correctement configurée pour tous les ordinateurs y participant.
