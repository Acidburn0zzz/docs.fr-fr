# <a name="systemservicemodeltxcompletionstatuscompletedforerror"></a>System.ServiceModel.TxCompletionStatusCompletedForError
La transaction spécifiée pour l’opération spécifiée a été effectuée en raison d’une exception d’exécution non prise en charge.  
  
## <a name="description"></a>Description  
 Suivi lorsqu'une erreur se produit durant une tentative d'exécution de la transaction actuelle. Cela se produit avant qu'une réponse ou une erreur ne soit envoyée à l'appelant.  
  
## <a name="troubleshooting"></a>Résolution des problèmes  
 Inspectez le message tracé pour le message d'exception et tout élément sur lequel une action peut être effectuée.
