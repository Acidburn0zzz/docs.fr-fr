# System.Runtime.Serialization.ObjectWithLargeDepth
System.Runtime.Serialization.ObjectWithLargeDepth  
  
## Description  
 L'écriture d'un objet à l'aide d'une structure récursive présente certaines limites lorsque la profondeur est importante.Réduisez la profondeur de l'objet.
