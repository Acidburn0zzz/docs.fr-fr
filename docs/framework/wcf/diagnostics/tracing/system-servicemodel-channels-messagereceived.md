# <a name="systemservicemodelchannelsmessagereceived"></a>System.ServiceModel.Channels.MessageReceived
System.ServiceModel.Channels.MessageReceived  
  
## <a name="description"></a>Description  
 Réception d'un message sur un canal.  
  
 Ce suivi est émis lorsqu’un message WCF est reçu si le `System.ServiceModel` source de suivi est activé au niveau des informations ou commentaires. Ce suivi est nécessaire pour consulter la flèche de corrélation du message dans la vue du graphique d'activité.
