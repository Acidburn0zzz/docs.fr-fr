# D&#233;bogage sur le client
Pour permettre aux utilisateurs d'écrire facilement des applications clientes pour votre service [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)], vous pouvez ajouter le comportement de service [\<serviceDebug\>](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) au fichier de configuration de votre service.Ce comportement permet de publier des pages d'aide et de retourner des informations d'exception gérées dans les détails des erreurs SOAP renvoyées au client.
