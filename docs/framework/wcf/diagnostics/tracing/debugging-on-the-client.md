# <a name="debugging-on-the-client"></a>Débogage sur le client
Pour faciliter l’écriture d’applications client pour votre [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] service, vous pouvez ajouter la [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) comportement dans le fichier de configuration de votre service de service. Ce comportement permet de publier des pages d'aide et de retourner des informations d'exception gérées dans les détails des erreurs SOAP renvoyées au client.
