# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
Un certificat client est nécessaire. Aucun certificat n'a été trouvé dans la demande.  
  
## <a name="description"></a>Description  
 Ce suivi indique que l'écouteur HTTPS a reçu une demande HTTPS qui n'était pas associée à un certificat client. Comme l'écouteur était configuré pour exiger des certificats clients sur toutes les demandes HTTPS, l'écouteur n'a pas pu valider l'authenticité du client.
