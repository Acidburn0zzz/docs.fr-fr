# System.ServiceModel.Channels.PeerServiceOpened
Le module du service homologue a été ouvert et a commencé à écouter à l'adresse spécifiée.  
  
## Description  
 Ce suivi se produit lors de l'ouverture du module du service homologue d'un PeerNode.
