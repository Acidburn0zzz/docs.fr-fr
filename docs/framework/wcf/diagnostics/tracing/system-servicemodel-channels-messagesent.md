# <a name="systemservicemodelchannelsmessagesent"></a>System.ServiceModel.Channels.MessageSent
System.ServiceModel.Channels.MessageSent  
  
## <a name="description"></a>Description  
 Envoi d'un message sur un canal.  
  
 Ce suivi est émis lorsqu’un message WCF est envoyé si le `System.ServiceModel` source de suivi est activé au niveau des informations ou commentaires. Ce suivi est nécessaire pour consulter la flèche de corrélation du message dans la vue du graphique d'activité.
