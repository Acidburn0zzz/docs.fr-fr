# <a name="systemservicemodelchannelsmessagesent"></a>System.ServiceModel.Channels.MessageSent
System.ServiceModel.Channels.MessageSent  
  
## <a name="description"></a>Description  
 Envoi d'un message sur un canal.  
  
 Ce suivi est émis lorsqu'un message [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] est envoyé si la source du suivi `System.ServiceModel` est activée au niveau des informations ou des commentaires. Ce suivi est nécessaire pour consulter la flèche de corrélation du message dans la vue du graphique d'activité.
