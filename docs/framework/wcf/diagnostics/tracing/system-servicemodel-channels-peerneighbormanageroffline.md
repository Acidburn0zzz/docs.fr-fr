# <a name="systemservicemodelchannelspeerneighbormanageroffline"></a>System.ServiceModel.Channels.PeerNeighborManagerOffline
PeerNode est maintenant hors connexion, ce qui indique qu'aucun voisin n'est actuellement connecté.  
  
## <a name="description"></a>Description  
 Ce suivi se produit lors de la suppression d'un voisin de la liste de voisins du nœud local.
