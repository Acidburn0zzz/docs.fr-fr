# System.ServiceModel.Channels.SystemTimeResolution
System.ServiceModel.Channels.SystemTimeResolution  
  
## Description  
 La résolution de l'horloge du système d'exploitation a été détectée étant égale à *n* tics, soit environ *m* millisecondes.  Où *n* et *m* sont des valeurs numériques.
