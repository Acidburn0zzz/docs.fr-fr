# System.ServiceModel.TxCompletionStatusCompletedForAutocomplete
La transaction spécifiée pour l'opération spécifiée a été effectuée car le membre `OperationBehaviorAttribute.TransactionAutoComplete` a la valeur `true`.  
  
## Description  
 Suivi lorsque l'infrastructure effectue automatiquement la transaction pour le compte de l'utilisateur.
