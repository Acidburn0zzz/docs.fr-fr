# System.ServiceModel.Channels.PeerNeighborNotFound
Impossible d'effectuer l'opération sur le voisin spécifié, celui\-ci étant introuvable dans la liste des voisins.  
  
## Description  
 Cette erreur se produit lors de la connexion à ou de la fermeture d'un voisin.
