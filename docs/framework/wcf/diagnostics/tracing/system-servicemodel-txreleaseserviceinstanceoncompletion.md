# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
L’instance de service a été libérée lors de l’aboutissement de la transaction « {0} » car ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute a pris la valeur true.  
  
## <a name="description"></a>Description  
 Suivi lorsque l’instance de service actuelle est libérée ou supprimée en raison de l’achèvement de la transaction active actuelle et que ReleaseServiceInstanceOnTransactionComplete a la valeur `true`.
