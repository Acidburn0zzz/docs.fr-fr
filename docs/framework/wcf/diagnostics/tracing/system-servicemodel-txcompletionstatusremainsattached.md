# <a name="systemservicemodeltxcompletionstatusremainsattached"></a>System.ServiceModel.TxCompletionStatusRemainsAttached
La transaction spécifiée pour l’opération spécifiée demeure attachée à InstanceContext.  
  
## <a name="description"></a>Description  
 Suivi généré lorsqu’une ServiceOperation se termine mais que la transaction n’est pas configurée pour se terminer.
