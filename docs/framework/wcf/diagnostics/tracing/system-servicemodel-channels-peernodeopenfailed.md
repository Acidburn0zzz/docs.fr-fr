# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Échec de la tentative d'ouverture d'une connexion de voisin.  
  
## <a name="description"></a>Description  
 Ce suivi se produit après la tentative d'établissement d'une connexion de voisin. Les raisons possibles sont les suivantes : un programme de résolution qui ne répond pas, un paramètre de liaison incorrect ou des défaillances temporaires du réseau.  
  
## <a name="troubleshooting"></a>Résolution des problèmes  
 Vérifiez que les deux nœuds ont des adresses valides et sont configurés de manière appropriée.
