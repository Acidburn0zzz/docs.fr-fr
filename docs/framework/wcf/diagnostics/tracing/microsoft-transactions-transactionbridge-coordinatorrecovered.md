# <a name="microsofttransactionstransactionbridgecoordinatorrecovered"></a>Microsoft.Transactions.TransactionBridge.CoordinatorRecovered
Un enrôlement de coordinateur a été récupéré à partir du journal du gestionnaire de transactions.  
  
## <a name="description"></a>Description  
 Tracé lorsque le gestionnaire de transactions local a exécuté avec succès la récupération pour un enrôlement de coordinateur supérieur donné.
