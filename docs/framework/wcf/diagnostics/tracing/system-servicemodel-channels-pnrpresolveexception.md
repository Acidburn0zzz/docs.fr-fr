# System.ServiceModel.Channels.PnrpResolveException
Une exception a été rencontrée pendant que le PnrpResolverService essaie d'exécuter une résolution.  
  
## Description  
 Ce suivi se produit lors d'une opération de résolution effectuée par le programme de résolution PNRP.
