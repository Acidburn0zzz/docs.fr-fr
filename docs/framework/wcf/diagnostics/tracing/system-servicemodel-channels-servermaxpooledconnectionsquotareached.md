# System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached
System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached  
  
## Description  
 Le quota MaxOutboundConnectionsPerEndpoint ayant été atteint, la connexion a été fermée et celle\-ci n'a pas été réutilisée par l'écouteur.
