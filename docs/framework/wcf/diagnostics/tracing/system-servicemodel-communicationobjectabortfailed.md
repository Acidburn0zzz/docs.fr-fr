# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
Échec d'abandon d'un objet de communication.  
  
## <a name="description"></a>Description  
 Un objet de communication n'a pas pu abandonner parce qu'une exception a été levée pendant l'exécution de sa méthode de Abort. Un autre suivi peut contenir des détails à propos de l'exception.
