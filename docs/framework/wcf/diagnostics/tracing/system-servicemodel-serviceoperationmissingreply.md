# <a name="systemservicemodelserviceoperationmissingreply"></a>System.ServiceModel.ServiceOperationMissingReply
System.ServiceModel.ServiceOperationMissingReply  
  
## <a name="description"></a>Description  
 L'opération demande-réponse ne comporte pas de message de réponse.  
  
 Il s'agit d'une opération bidirectionnelle, mais il n'y a aucune réponse à envoyer au client. Vous pouvez utiliser ce suivi à des fins de débogage en cours de développement.
