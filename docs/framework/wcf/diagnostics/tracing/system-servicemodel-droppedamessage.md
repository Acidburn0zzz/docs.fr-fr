# System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## Description  
 Un message de réponse a été reçu, mais aucune demande en attente n'attend ce message.Le message est supprimé par l'exécution de [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
 Ce suivi peut être utilisé pour déboguer des messages perdus.
