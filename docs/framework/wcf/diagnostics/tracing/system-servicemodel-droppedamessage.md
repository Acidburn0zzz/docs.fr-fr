# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Description  
 Un message de réponse a été reçu, mais aucune demande en attente n'attend ce message. Le message est supprimé par le runtime de Windows Communication Foundation (WCF).  
  
 Ce suivi peut être utilisé pour déboguer des messages perdus.
