# <a name="systemservicemodelchannelspeerneighbormessagereceived"></a>System.ServiceModel.Channels.PeerNeighborMessageReceived
Un message a été reçu d'un voisin pair.  
  
## <a name="description"></a>Description  
 Cette trace se produit pendant le traitement d'un message entrant depuis un nœud figurant dans la liste de voisins.
