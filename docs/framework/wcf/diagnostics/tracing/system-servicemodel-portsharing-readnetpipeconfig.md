# System.ServiceModel.PortSharing.ReadNetPipeConfig
System.ServiceModel.PortSharing.ReadNetPipeConfig  
  
## Description  
 Les données de configuration pour NetPipeSection ont été lues avec les données spécifiées : maxConnectionsPendingDispatch, maxPendingAccepts et receiveTimeout.
