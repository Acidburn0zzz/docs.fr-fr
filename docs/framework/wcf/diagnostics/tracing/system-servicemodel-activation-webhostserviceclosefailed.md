# System.ServiceModel.Activation.WebHostServiceCloseFailed
Se produit lorsqu'un service ne peut pas être fermé normalement et est abandonné.  
  
## Description  
 Ce code d'erreur apparaît seulement dans le fichier journal.Il indique généralement une erreur de programmation, par exemple lorsque vous essayez de fermer un service après l'appel d'Abort.  
  
## Résolution des problèmes  
 Vérifiez le code source de l'application.
