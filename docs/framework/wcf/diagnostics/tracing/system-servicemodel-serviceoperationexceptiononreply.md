# System.ServiceModel.ServiceOperationExceptionOnReply
System.ServiceModel.ServiceOperationExceptionOnReply  
  
## Description  
 Répondre à une opération de demande\/réponse lève une exception.  Ce suivi est utile pour déboguer pendant le développement.
