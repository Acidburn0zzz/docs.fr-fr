# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Échec de la fermeture d'un objet de communication.  
  
## <a name="description"></a>Description  
 Un objet de communication n'a pas pu être fermé parce qu'une exception a été levée pendant l'exécution de sa méthode Close. Un autre suivi peut contenir des détails à propos de l'exception.
