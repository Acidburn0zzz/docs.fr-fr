# System.ServiceModel.Activation.WebHostServiceActivated
System.ServiceModel.Activation.WebHostServiceActivated  
  
## Description  
 Se produit lorsqu'un service hébergé sur les services IIS \(Internet Information Services\) ou WAS \(Windows Process Activation Service\) est activé pleinement et se met à écouter les messages.
