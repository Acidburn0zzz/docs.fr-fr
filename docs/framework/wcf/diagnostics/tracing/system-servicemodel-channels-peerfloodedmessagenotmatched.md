# System.ServiceModel.Channels.PeerFloodedMessageNotMatched
Un nœud d'homologue a reçu un message qui ne correspondait pas à tous les canaux locaux.  
  
## Description  
 Ce suivi se produit pendant le traitement d'un message de saturation entrant.
