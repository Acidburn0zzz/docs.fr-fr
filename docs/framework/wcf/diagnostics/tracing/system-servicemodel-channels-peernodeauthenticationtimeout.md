# System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
Le protocole de transfert de sécurité n'a pas réussi dans la limite de temps spécifiée.  
  
## Description  
 Cette trace se produit lors de la tentative d'établissement d'une connexion de voisin sécurisée.  
  
## Résolution des problèmes  
 Veillez à ce que tous les voisins aient les informations d'identification de sécurité appropriées et répondent à tous les messages en moins d'une minute.
