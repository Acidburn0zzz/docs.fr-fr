# System.ServiceModel.Diagnostics.FilterNotMatchedNodeQuotaExceeded
System.ServiceModel.Diagnostics.FilterNotMatchedNodeQuotaExceeded  
  
## Description  
 L'évaluation du filtre de journalisation des messages par rapport au message a dépassé le quota de nœuds défini sur le filtre.
