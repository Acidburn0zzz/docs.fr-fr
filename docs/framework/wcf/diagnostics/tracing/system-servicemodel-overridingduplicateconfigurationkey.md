# System.ServiceModel.OverridingDuplicateConfigurationKey
System.ServiceModel.OverridingDuplicateConfigurationKey  
  
## Description  
 Le système de configuration a détecté une clé en double dans une étendue de configuration différente et la remplace par la valeur la plus récente.
