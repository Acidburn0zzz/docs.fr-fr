# <a name="systemservicemodelchannelspeerreceivemessageauthenticationfailure"></a>System.ServiceModel.Channels.PeerReceiveMessageAuthenticationFailure
La source du message n'a pas pu être authentifiée.  
  
## <a name="description"></a>Description  
 Cette trace se produit en essayant d’authentifier des messages entrants et l’authentification de message est activée sur la liaison.
