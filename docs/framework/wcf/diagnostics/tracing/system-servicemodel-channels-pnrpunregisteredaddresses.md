# System.ServiceModel.Channels.PnrpUnregisteredAddresses
Le programme de résolution Pnrp a désinscrit les adresses du nœud local.  
  
## Description  
 Ce suivi se produit lorsque le service PNRP réussit à désinscrire les adresses du nœud local.
