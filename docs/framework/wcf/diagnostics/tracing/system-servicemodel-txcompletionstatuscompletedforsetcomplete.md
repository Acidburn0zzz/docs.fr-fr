# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
La transaction spécifiée pour l’opération indiquée a été effectuée en raison d’un appel à SetTransactionComplete.  
  
## <a name="description"></a>Description  
 Suivi lorsque l’utilisateur a décidé explicitement d’exécuter la transaction en cours via OperationContext.
