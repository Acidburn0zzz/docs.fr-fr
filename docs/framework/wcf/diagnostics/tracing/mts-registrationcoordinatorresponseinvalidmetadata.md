# Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorResponseInvalidMetadata
Le service du protocole WS\-AT a reçu un message RegisterResponse de son coordinateur contenant une référence de point de terminaison avec des métadonnées non valides ou incompatibles.  
  
## Description  
 Suivi généré lorsque le Gestionnaire de transactions local essaie de s'inscrire auprès de son Gestionnaire de transactions supérieur et que le supérieur retourne une adresse non valide dans le message RegisterResponse.
