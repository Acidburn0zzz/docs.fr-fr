# Behavior, classe
## Syntaxe  
  
```  
class Behavior  
{  
  string Type;  
};  
```  
  
## Méthodes  
 La classe Behavior ne définit aucune méthode.  
  
## Propriétés  
 La classe Behavior dispose de la propriété suivante :  
  
### Type  
 Type de données : chaîne  
  
 Type d'accès : lecture seule  
  
 Type de comportement.
