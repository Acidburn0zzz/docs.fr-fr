# GetOperationCounterInstanceName
Récupère le nom d'instance du compteur de performance d'opération.  
  
## Syntaxe  
  
```  
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## Paramètres  
  
### Opération  
 Chaîne \[in\]
