# <a name="transportsecuritybindingelement"></a>TransportSecurityBindingElement
TransportSecurityBindingElement  
  
## <a name="syntax"></a>Syntaxe  
  
```  
class TransportSecurityBindingElement : SecurityBindingElement  
{  
};  
```  
  
## <a name="methods"></a>Méthodes  
 La classe TransportSecurityBindingElement hérite de SecurityBindingElement mais ne définit pas de propriété ou de méthode supplémentaire.
