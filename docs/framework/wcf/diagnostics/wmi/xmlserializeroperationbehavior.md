# <a name="xmlserializeroperationbehavior"></a>XmlSerializerOperationBehavior
XmlSerializerOperationBehavior  
  
## <a name="syntax"></a>Syntaxe  
  
```csharp
class XmlSerializerOperationBehavior : Behavior  
{  
  string Style;  
  string Use;  
};  
```  
  
## <a name="methods"></a>Méthodes  
 La classe XmlSerializerOperationBehavior ne définit pas de méthodes.  
  
## <a name="properties"></a>Properties  
 La classe XmlSerializerOperationBehavior a les propriétés suivantes :  
  
### <a name="style"></a>Style  
 Type de données : chaîne  
  
 Type d’accès : Propriétés en lecture seule  
  
 Définit le style du message SOAP.  
  
### <a name="use"></a>Utilisez  
 Type de données : chaîne  
  
 Type d’accès : Propriétés en lecture seule  
  
 Spécifie le style d'encodage SOAP.
