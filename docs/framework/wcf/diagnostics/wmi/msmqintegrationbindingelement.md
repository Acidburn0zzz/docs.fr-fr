# MsmqIntegrationBindingElement
MsmqIntegrationBindingElement  
  
## Syntaxe  
  
```  
class MsmqIntegrationBindingElement : MsmqBindingElementBase  
{  
  string SerializationFormat;  
};  
```  
  
## Méthodes  
 La classe MsmqIntegrationBindingElement ne définit pas de méthode.  
  
## Propriétés  
 La classe MsmqIntegrationBindingElement a la propriété suivante :  
  
### SerializationFormat  
 Type de données : chaîne  
  
 Type d'accès : lecture seule  
  
 Format utilisé par les liaisons pour sérialiser les messages.
