# TraceListener
TraceListener.  
  
## Syntaxe  
  
```  
class TraceListener  
{  
  string Name;  
  TraceListenerArgument TraceListenerArguments[];  
};  
```  
  
## Méthodes  
 La classe TraceListener ne définit pas de méthodes.  
  
## Propriétés  
 La classe TraceListener a les propriétés suivantes :  
  
### Nom  
 Type de données : chaîne  
  
 Type d'accès : lecture seule  
  
 Nom de l'écouteur de suivi.  
  
### TraceListenerArguments  
 Type de données : tableau TraceListenerArgument  
  
 Type d'accès : lecture seule  
  
 Arguments de l'écouteur de suivi.
