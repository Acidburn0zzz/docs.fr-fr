# <a name="service-calls-duration"></a>Service : Durée des appels
Nom du compteur : Durée des appels  
  
## <a name="description"></a>Description  
 Durée moyenne des appels à ce service. La durée moyenne est calculée en fonction de cette équation : (N1-N0)/(D1-D0).
