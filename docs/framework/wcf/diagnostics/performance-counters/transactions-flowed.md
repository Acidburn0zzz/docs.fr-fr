# <a name="transactions-flowed"></a>Transactions passées
Nom du compteur : Transactions passées  
  
## <a name="description"></a>Description  
 Nombre de transactions passées à cette opération. Ce compteur est incrémenté pour tout message envoyé à l'opération qui contient un ID de transaction.
