# <a name="endpoint-transactions-flowed"></a>Point de terminaison : transactions passées
Nom du compteur : Transactions passées.  
  
## <a name="description"></a>Description  
 Nombre de transactions passées aux opérations au niveau de ce point de terminaison. Ce compteur est incrémenté à chaque ID de transaction présent dans un message envoyé vers le point de terminaison.
