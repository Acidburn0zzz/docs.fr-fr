# Messages incoh&#233;rents mis en file d&#39;attente
Nom du compteur : Messages incohérents mis en file d'attente.  
  
## Description  
 Nombre de messages vers ce service marqués comme étant incohérents par le transport de mise en file d'attente.  
  
 Consultez [Gestion des messages incohérents](http://go.microsoft.com/fwlink/?LinkID=96233) \(page pouvant être en anglais\) pour plus d'informations sur le moment auquel les messages sont marqués comme incohérents.
