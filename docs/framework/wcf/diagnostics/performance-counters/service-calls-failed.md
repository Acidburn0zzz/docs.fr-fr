# <a name="service-calls-failed"></a>Service : Appels ayant échoué
Nom du compteur : Appels ayant échoué.  
  
## <a name="description"></a>Description  
 Nombre d'appels comportant des exceptions non gérées sur ce service. Ce compteur est incrémenté à chaque fois qu'une exception non gérée est levée sur ce service.
