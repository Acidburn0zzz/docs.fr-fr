# Dur&#233;e des appels
Nom du compteur : Durée des appels  
  
## Description  
 La durée moyenne des appels à cette opération.  La durée moyenne est calculée selon cette équation : \(N1\-N0\)\/\(D1\-D0\).  
  
> [!WARNING]
>  S'il est utilisé sur un service WCF asynchrone, le compteur Durée des appels retourne toujours \-1.
