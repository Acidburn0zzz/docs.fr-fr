# <a name="calls-duration"></a>Durée des appels
Nom du compteur : Durée des appels  
  
## <a name="description"></a>Description  
 La durée moyenne des appels à cette opération. La durée moyenne est calculée en fonction de cette équation : (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Lorsqu’il est utilisé sur un service WCF asynchrone le compteur durée des appels retourne toujours -1.
