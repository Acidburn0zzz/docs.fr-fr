# Configuration requise WCF
[!INCLUDE[indigo1](../../../includes/indigo1-md.md)] est une infrastructure de communication utilisée pour créer des applications distribuées.  La section suivante répertorie les spécifications qui permettent à [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] de s'exécuter.  
  
## Configuration requise  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] est installé par défaut sur [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] peut également être installé sur [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)] ou [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Remarque** Les fonctionnalités Message Queuing \(MSMQ\) de [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] sont prises en charge uniquement sur [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] et [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
