# Activation des processus Windows
Cette section contient des exemples qui illustrent l'activation de services via des protocoles réseau.  
  
## Dans cette section  
 [NamedPipe Activation](../../../../docs/framework/wcf/samples/namedpipe-activation.md)  
 Illustre l'hébergement d'un service qui utilise le service d'activation des processus Windows \(WAS\) pour activer un service qui communique sur des canaux nommés.  
  
 [TCP Activation](../../../../docs/framework/wcf/samples/tcp-activation.md)  
 Illustre l'hébergement d'un service utilisant les services d'activation des processus Windows \(WAS\) pour activer un service qui communique à l'aide du protocole net.tcp.
