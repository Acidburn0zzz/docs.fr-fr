# Constantes (r&#233;f&#233;rence des API non manag&#233;es)
Cette rubrique décrit le type de langage, fournisseur de langage et les constantes de type de document qui sont définies dans CorSym.idl.  
  
## Constantes de Type de langage  
 Le tableau suivant affiche des constantes de type, qui représentent les GUID qui identifient les langages de programmation langage.  
  
|Symbole|Description|  
|-------------|-----------------|  
|CorSym\_LanguageType\_C|Indique le langage C.|  
|CorSym\_LanguageType\_CPlusPlus|Indique le langage C\+\+.|  
|CorSym\_LanguageType\_CSharp|Indique le langage c\#.|  
|CorSym\_LanguageType\_Basic|Indique la langue de base.|  
|CorSym\_LanguageType\_Java|Indique le langage Java.|  
|CorSym\_LanguageType\_Cobol|Indique le langage COBOL.|  
|CorSym\_LanguageType\_Pascal|Indique le langage Pascal.|  
|CorSym\_LanguageType\_ILAssembly|Indique le code d’assembly Microsoft intermediate language \(MSIL\).|  
|CorSym\_LanguageType\_JScript|Indique le langage JScript.|  
|CorSym\_LanguageType\_SMC|Indique le langage SMC.|  
|CorSym\_LanguageType\_MCPlusPlus|Indique le langage C\+\+ activé pour le .NET Framework.|  
  
## Constantes de fournisseur de langage  
 Le tableau suivant affiche des constantes de fournisseur, qui représentent les GUID qui identifient les éditeurs de langage de programmation de langage.  
  
|Symbole|Description|  
|-------------|-----------------|  
|CorSym\_LanguageVendor\_Microsoft|Indique à Microsoft.|  
  
## Constantes de Type de document  
 Le tableau suivant présente les documents des constantes de type, qui représentent les GUID qui identifient les types de documents.  
  
|Symbole|Description|  
|-------------|-----------------|  
|CorSym\_DocumentType\_Text|Indique un document texte.|  
|CorSym\_DocumentType\_MC|Indique un document non textuels.|
