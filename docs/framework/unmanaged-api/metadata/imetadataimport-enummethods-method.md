# <a name="imetadataimportenummethods-method"></a>IMetaDataImport::EnumMethods, méthode
Énumère les jetons MethodDef représentant les méthodes du type spécifié.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT EnumMethods (  
   [in, out] HCORENUM   *phEnum,   
   [in]  mdTypeDef      cl,   
   [out] mdMethodDef    rMethods[],   
   [in]  ULONG          cMax,   
   [out] ULONG          *pcTokens  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `phEnum`  
 [dans, out] Pointeur vers l’énumérateur. Cela doit être NULL pour le premier appel de cette méthode.  
  
 `cl`  
 [in] Un jeton TypeDef représentant le type dont les méthodes à énumérer.  
  
 `rMethods`  
 [out] Le tableau pour stocker les jetons MethodDef.  
  
 `cMax`  
 [in] La taille maximale de la MethodDef `rMethods` tableau.  
  
 `pcTokens`  
 [out] Le nombre de jetons MethodDef retournés dans `rMethods`.  
  
## <a name="return-value"></a>Valeur de retour  
  
|HRESULT|Description|  
|-------------|-----------------|  
|`S_OK`|`EnumMethods`retourné avec succès.|  
|`S_FALSE`|Il n’existe pas de jetons MethodDef à énumérer. Dans ce cas, `pcTokens` est égal à zéro.|
