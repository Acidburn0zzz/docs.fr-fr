# <a name="metadata-global-static-functions"></a>Fonctions statiques globales des métadonnées
À partir de la [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)], la documentation relative à la `_Cor*`, `CoInitialize*`, `CoUninitialize*`, et `CoEEShutDownCOM` fonctions relatives à l’hébergement, se trouve dans [déconseillée des fonctions d’hébergement CLR](../../../../docs/framework/unmanaged-api/hosting/deprecated-clr-hosting-functions.md). Il n'existe aucune fonction statique globale relative aux métadonnées.
