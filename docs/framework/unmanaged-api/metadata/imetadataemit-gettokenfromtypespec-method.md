# <a name="imetadataemitgettokenfromtypespec-method"></a>IMetaDataEmit::GetTokenFromTypeSpec, méthode
Obtient les métadonnées jeton pour le type avec la signature de métadonnées spécifiée.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetTokenFromTypeSpec (   
    [in]  PCCOR_SIGNATURE       pvSig,   
    [in]  ULONG                 cbSig,   
    [out] mdTypeSpec            *ptypespec   
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `pvSig`  
 [in] La signature est définie.  
  
 `cbSig`  
 [in] Le nombre d’octets dans `pvSig`.  
  
 `ptypespec`  
 [out] Le `mdTypeSpec` jeton assigné.
