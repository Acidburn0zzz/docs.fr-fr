# <a name="imetadataemitdefinemethodimpl-method"></a>IMetaDataEmit::DefineMethodImpl, méthode
Crée une définition pour l’implémentation d’une méthode héritée d’une interface et retourne un jeton pour cette définition d’implémentation de méthode.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT DefineMethodImpl (   
    [in]  mdTypeDef         td,   
    [in]  mdToken           tkBody,   
    [in]  mdToken           tkDecl  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `td`  
 [in] Le `mdTypedef` jeton de la classe d’implémentation.  
  
 `tkBody`  
 [in] Le `mdMethodDef` ou `mdMethodRef` jeton du corps du code.  
  
 `tkDecl`  
 [in] Le `mdMethodDef` ou `mdMethodRef` jeton de la méthode d’interface implémentée.
