# <a name="icorprofilerfunctionenumreset-method"></a>ICorProfilerFunctionEnum::Reset, méthode
Déplace le curseur de l'énumérateur à la position de départ de la séquence.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT Reset();  
```
