# <a name="cleardownloadcache-function"></a>ClearDownloadCache, fonction
Efface le global assembly cache d’assemblys téléchargés.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT ClearDownloadCache ();  
```
