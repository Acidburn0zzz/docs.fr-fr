# <a name="icordebugmodulegetname-method"></a>ICorDebugModule::GetName, méthode
Obtient le nom de fichier du module.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetName(  
    [in] ULONG32 cchName,  
    [out] ULONG32 *pcchName,  
    [out, size_is(cchName), length_is(*pcchName)] WCHAR szName[]  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `cchname`  
 [in] Taille du tableau `szName`.  
  
 `pcchName`  
 [in] Pointeur vers la longueur du nom retourné.  
  
 `szName`  
 [out] Tableau qui stocke le nom retourné.  
  
## <a name="remarks"></a>Notes  
 Le `GetName` méthode retourne un HRESULT S_OK si le nom de fichier du module correspond au nom sur le disque. `GetName`Retourne une valeur HRESULT S_FALSE si le nom est créé, comme pour un module dynamique ou en mémoire.
