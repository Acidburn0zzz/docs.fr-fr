# <a name="icordebugmanagedcallbackevalexception-method"></a>ICorDebugManagedCallback::EvalException, méthode
Notifie le débogueur qu’une évaluation s’est terminé avec une exception non gérée.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT EvalException (  
    [in] ICorDebugAppDomain *pAppDomain,  
    [in] ICorDebugThread    *pThread,  
    [in] ICorDebugEval      *pEval  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `pAppDomain`  
 [in] Pointeur vers un objet ICorDebugAppDomain qui représente le domaine d’application dans lequel l’évaluation s’est terminée.  
  
 `pThread`  
 [in] Pointeur vers un objet ICorDebugThread qui représente le thread dans lequel l’évaluation s’est terminée.  
  
 `pEval`  
 [in] Pointeur vers un objet ICorDebugEval qui représente le code qui a effectué l’évaluation.
