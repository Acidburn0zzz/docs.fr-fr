# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 (interface)
Étend logiquement l’interface ICorDebugAppDomain pour obtenir un objet managé à partir d’un wrapper CCW.  
  
## <a name="method"></a>Méthode  
  
|Méthode|Description|  
|------------|-----------------|  
|[GetObjectForCCW, méthode](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain4-getobjectforccw-method.md)|Obtient un objet managé à partir d'un pointeur de wrapper CCW (COM Callable Wrapper).|
