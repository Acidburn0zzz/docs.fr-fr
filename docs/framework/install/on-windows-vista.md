# <a name="install-the-net-framework-on-windows-vista"></a>Installer le .NET Framework sur Windows Vista

Le .NET Framework peut être nécessaire pour exécuter une application sur Windows Vista. Vous pouvez utiliser les instructions suivantes pour vous aider. Les instructions dépendent de la version du .NET Framework dont vous avez besoin.

## <a name="net-framework-461-and-later"></a>.NET Framework 4.6.1 et ultérieur

Vous ne pouvez pas installer le .NET Framework 4.6.1 ou ultérieur sur Windows Vista. Au lieu de cela, vous devez passer à une version ultérieure de Windows pour utiliser le .NET Framework 4.6.1 ou ultérieur.

Le .NET Framework 4.6.2 est inclus avec la [Mise à jour anniversaire Windows 10](https://www.microsoft.com/software-download/windows10).

## <a name="net-framework-46"></a>.NET Framework 4.6

Vous pouvez installer le [.NET Framework 4.6](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) sur Windows Vista.

## <a name="net-framework-35"></a>.NET Framework 3.5

Vous pouvez installer le [.NET Framework 3.5](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) sur Windows Vista.

Le .NET Framework 3.5 peut être utilisé pour exécuter des applications créées pour le .NET Framework 1.0 à 3.5.

## <a name="help"></a>Help

Vous pouvez [contacter Microsoft pour obtenir de l’aide](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework) si vous ne pouvez installer la version correcte du .NET Framework.
