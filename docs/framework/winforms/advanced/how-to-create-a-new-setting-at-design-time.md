# Comment&#160;: cr&#233;er un nouveau param&#232;tre au moment du design
Vous pouvez créer un paramètre au moment du design en utilisant le concepteur de paramètres.  Le concepteur de paramètres est une interface sous forme de grille qui vous permet de créer des paramètres et d'en spécifier les propriétés.  Vous devez spécifier le nom, la valeur, le type et la portée de vos nouveaux paramètres.  Une fois créé, un paramètre est accessible dans le code.  
  
### Pour créer un paramètre au moment du design en C\#  
  
1.  Dans **Explorateur de solutions**, développez le nœud **Propriétés** de votre projet.  
  
2.  Double\-cliquez sur le fichier .settings dans lequel vous souhaitez ajouter un nouveau paramètre.  Le nom par défaut de ce fichier est Settings.settings.  
  
3.  Dans le concepteur de paramètres, définissez le nom, la valeur, le type et la portée de votre paramètre.  Chaque ligne représente un paramètre unique.  
  
### Pour créer un paramètre au moment du design dans Visual Basic  
  
1.  Dans l'**Explorateur de solutions**, cliquez avec le bouton droit sur le nœud de votre projet et sélectionnez **Propriétés**.  
  
2.  Dans la page **Propriétés**, sélectionnez l'onglet **Paramètres**.  
  
3.  Dans le concepteur de paramètres, définissez le nom, la valeur, le type et la portée de votre paramètre.  Chaque ligne représente un paramètre unique.
