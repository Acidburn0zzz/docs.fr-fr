# Comment&#160;: modifier la valeur d&#39;un param&#232;tre existant au moment du design
Vous pouvez utiliser Visual Studio pour modifier les valeurs de paramètres existants dans votre projet.  
  
### Pour modifier la valeur d'un paramètre existant au moment du design en C\#  
  
1.  Dans **Explorateur de solutions**, développez le nœud **Propriétés** de votre projet.  
  
2.  Double\-cliquez sur le fichier .settings dans lequel vous souhaitez ajouter un nouveau paramètre.  Le nom par défaut de ce fichier est Settings.settings.  
  
3.  Dans le concepteur de paramètres, recherchez le paramètre pour lequel vous souhaitez modifier la valeur et saisissez la nouvelle valeur dans la colonne Valeur.  
  
### Pour modifier la valeur d'un paramètre existant au moment du design dans Visual Basic  
  
1.  Dans l'**Explorateur de solutions**, cliquez avec le bouton droit sur le nœud de votre projet et sélectionnez **Propriétés**.  
  
2.  Dans la page **Propriétés**, sélectionnez l'onglet **Paramètres**.  
  
3.  Dans le concepteur de paramètres, recherchez le paramètre pour lequel vous souhaitez modifier la valeur et saisissez la nouvelle valeur dans la colonne Valeur.
