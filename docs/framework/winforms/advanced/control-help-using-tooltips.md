# Affichage sous forme d&#39;info-bulles de l&#39;aide relative aux contr&#244;les
Le composant <xref:System.Windows.Forms.ToolTip> peut servir à afficher des messages d'aide courts et spécialisés relatifs à des contrôles individuels dans les Windows Forms.  Le composant <xref:System.Windows.Forms.ToolTip> fournit une propriété qui précise le texte affiché pour chaque contrôle du formulaire.  Pour plus d'informations sur cette utilisation du composant <xref:System.Windows.Forms.ToolTip>, consultez [Comment : définir des info\-bulles pour les contrôles d'un Windows Form au moment du design](../../../../docs/framework/winforms/controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md).  Par ailleurs, vous pouvez configurer le composant <xref:System.Windows.Forms.ToolTip> de telle sorte qu'il s'affiche après un certain délai.  Pour plus d'informations, consultez [Comment : modifier la durée avant affichage du composant ToolTip Windows Forms](../../../../docs/framework/winforms/controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md).
