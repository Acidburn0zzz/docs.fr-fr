# <a name="using-regions"></a>Utilisation de régions
Le [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Region> classe vous permet de définir une forme personnalisée. La forme peut être constituée de lignes, les polygones et les courbes.  
  
 Deux utilisations courantes pour les régions sont atteints de test et le découpage. Un test de positionnement consiste à déterminer si l’utilisateur a cliqué sur la souris dans une région donnée de l’écran. L’extrait est limité au dessin dans une région donnée.
