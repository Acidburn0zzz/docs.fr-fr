# Utilisation de r&#233;gions
La classe [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)]<xref:System.Drawing.Region> vous permet de définir une forme personnalisée.  Cette forme est constituée de traits, de polygones et de courbes.  
  
 Deux utilisations classiques des régions sont le test d'atteinte et le découpage.  Le test d'atteinte détermine s'il y a eu un clic de souris dans une région déterminée de l'écran.  Le découpage est limité au dessin dans une région déterminée.
