# Vue d&#39;ensemble des graphismes (Windows Forms)
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] est un Graphics Device Interface qui permet aux programmeurs d'écrire des applications indépendantes du périphérique.  Les services de [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] sont exposés via un jeu de classes managées.  
  
## Dans cette section  
 [Vue d'ensemble des graphismes](../../../../docs/framework/winforms/advanced/overview-of-graphics.md)  
 Fournit une introduction générale à [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
 [Trois catégories de services graphiques](../../../../docs/framework/winforms/advanced/three-categories-of-graphics-services.md)  
 Décrit les trois catégories qui constituent la programmation avec [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
 [Structure de l'interface graphique](../../../../docs/framework/winforms/advanced/structure-of-the-graphics-interface.md)  
 Décrit l'interface de classes managées de [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
## Référence  
 <xref:System.Drawing>  
 Permet d'accéder aux fonctionnalités graphiques de base de [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
 <xref:System.Drawing.Drawing2D>  
 Fournit des fonctionnalités avancées de graphismes à deux dimensions et vectoriels.  
  
 <xref:System.Drawing.Imaging>  
 Fournit les fonctionnalités d'imagerie avancées de [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
 <xref:System.Drawing.Text>  
 Fournit les fonctionnalités typographiques avancées de [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)].  
  
 <xref:System.Drawing.Printing>  
 Fournit des services d'impression.  
  
 <xref:System.Windows.Forms.TextRenderer>  
 Fournit les fonctionnalités de mesure et de dessin de texte [!INCLUDE[ndptecgdi](../../../../includes/ndptecgdi-md.md)].
