# Mise en r&#233;seau dans les applications Windows Forms
Le .NET Framework fournit des classes pour afficher des pages Web, télécharger des contenus Web, interagir avec des sites FTP et consommer des Services Web, facilitant l'intégration des fonctionnalités de réseau dans votre application.  Les ressources suivantes vous aideront à comprendre les technologies de mise en réseau du .NET Framework et à comprendre comment les intégrer dans des Windows Forms.  
  
## Référence  
 <xref:System.Net>  
 L'espace de noms racine pour les classes dans le .NET Framework qui gère la connectivité réseau.  
  
 <xref:System.Net.WebClient>  
 Une classe pratique pour récupérer les contenus Web ou HTTP par programme.  
  
 <xref:System.Net.FtpWebRequest>  
 Une classe pour récupérer et envoyer des fichiers avec FTP.  
  
 <xref:System.Windows.Forms.WebBrowser>  
 Une classe wrapper managée pour le contrôle `WebBrowser` qui est inclus avec Windows.
