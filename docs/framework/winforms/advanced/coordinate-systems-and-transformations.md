# Syst&#232;mes de coordonn&#233;es et transformations
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] fournit une transformation universelle et une transformation de page afin que vous puissiez transformer \(faire pivoter, mettre à l'échelle, traduire, etc.\) les éléments vous dessinez.  Ces deux types de transformations vous permettent également de travailler dans différents systèmes de coordonnées.
