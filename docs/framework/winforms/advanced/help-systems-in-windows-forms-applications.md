# Syst&#232;mes d&#39;aide dans les applications Windows&#160;Forms
En tant que développeur d'applications, il est primordial que vous fournissiez à vos utilisateurs un système d'aide efficace.  C'est en effet vers lui qu'ils se tourneront en cas de problème.  Fournir un système d'aide dans une application Windows est chose facile en utilisant le [HelpProvider, composant](../../../../docs/framework/winforms/controls/helpprovider-component-windows-forms.md).  
  
## Types différents d'aide  
 Le composant <xref:System.Windows.Forms.HelpProvider> Windows Forms permet d'associer un fichier d'aide HTML Help 1.x \(fichier .chm généré par HTML Help Workshop ou fichier .htm\) à votre application Windows.  Vous pouvez utiliser le composant <xref:System.Windows.Forms.HelpProvider> pour offrir une aide contextuelle relative aux contrôles des Windows Forms ou à des contrôles spécifiques.  En outre, le composant <xref:System.Windows.Forms.HelpProvider> peut ouvrir un fichier d'aide dans des zones spécifiques \(page principale d'une table des matières, index ou fonction de recherche, par exemple\).  Pour des informations d'ordre général sur le composant <xref:System.Windows.Forms.HelpProvider>, consultez [Vue d'ensemble du composant HelpProvider](../../../../docs/framework/winforms/controls/helpprovider-component-overview-windows-forms.md).  Pour des informations sur la façon d'utiliser le composant <xref:System.Windows.Forms.HelpProvider> pour afficher l'aide contextuelle dans les Windows Forms, consultez [Comment : afficher l'aide contextuelle](../../../../docs/framework/winforms/advanced/how-to-display-pop-up-help.md).  Pour plus d'informations sur l'utilisation du composant <xref:System.Windows.Forms.ToolTip> pour afficher l'aide spécifique à un contrôle, consultez [Affichage sous forme d'info\-bulles de l'aide relative aux contrôles](../../../../docs/framework/winforms/advanced/control-help-using-tooltips.md).  
  
 Vous pouvez générer des fichiers d'aide HTML Help 1.x à l'aide de HTML Help Workshop.  Pour plus d'informations sur HTML Help, consultez « HTML Help Workshop » ou les autres rubriques « HTML Help » dans MSDN.
