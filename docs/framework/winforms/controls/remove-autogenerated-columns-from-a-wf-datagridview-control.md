# <a name="how-to-remove-autogenerated-columns-from-a-windows-forms-datagridview-control"></a>Comment : supprimer les colonnes générées automatiquement d'un contrôle DataGridView Windows Forms
Lorsque votre <xref:System.Windows.Forms.DataGridView> contrôle est défini pour générer automatiquement ses colonnes en fonction des données à partir de sa source de données, vous pouvez omettre sélectivement certaines colonnes. Vous pouvez le faire en appelant le <xref:System.Windows.Forms.DataGridViewColumnCollection.Remove%2A> méthode sur le <xref:System.Windows.Forms.DataGridView.Columns%2A> collection. Vous pouvez également masquer les colonnes de la vue en définissant le <xref:System.Windows.Forms.DataGridViewColumn.Visible%2A> propriété `false`. Cette technique est utile lorsque vous souhaitez afficher les colonnes masquées dans certaines conditions, ou lorsque vous avez besoin accéder aux données dans les colonnes sans l’afficher.  
  
### <a name="to-remove-autogenerated-columns"></a>Pour supprimer les colonnes générées automatiquement  
  
-   Appelez le <xref:System.Windows.Forms.DataGridViewColumnCollection.Remove%2A> méthode sur le <xref:System.Windows.Forms.DataGridView.Columns%2A> collection.
