# Comment&#160;: fractionner une fen&#234;tre horizontalement &#224; l&#39;aide du concepteur
Cet exemple fait en sorte que le séparateur qui divise le contrôle <xref:System.Windows.Forms.SplitContainer> soit horizontal.  
  
> [!NOTE]
>  La propriété <xref:System.Windows.Forms.SplitContainer.Orientation%2A> du contrôle <xref:System.Windows.Forms.SplitContainer> détermine la direction du séparateur et non du contrôle lui\-même.  Les boîtes de dialogue et les commandes de menu qui s'affichent peuvent être différentes de celles qui sont décrites dans l'aide, en fonction de vos paramètres actifs ou de l'édition utilisée.  Pour modifier vos paramètres, choisissez **Importation et exportation de paramètres** dans le menu **Outils**.  Pour plus d'informations, consultez [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/fr-fr/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
### Pour fractionner une fenêtre horizontalement  
  
1.  Dans la fenêtre **Propriétés**, affectez à la propriété <xref:System.Windows.Forms.SplitContainer.Orientation%2A> du contrôle <xref:System.Windows.Forms.SplitContainer> la valeur <xref:System.Windows.Forms.Orientation>.
