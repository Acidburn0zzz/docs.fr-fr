# Comment&#160;: aligner plusieurs contr&#244;les dans des Windows Forms
Pour obtenir une présentation homogène de l'interface utilisateur dans toutes vos applications Windows, vous pouvez positionner des groupes de contrôles par une seule et unique commande.  
  
> [!NOTE]
>  Les boîtes de dialogue et les commandes de menu qui s'affichent peuvent être différentes de celles qui sont décrites dans l'aide, en fonction de vos paramètres actifs ou de l'édition utilisée.  Pour modifier vos paramètres, choisissez **Importation et exportation de paramètres** dans le menu **Outils**.  Pour plus d'informations, consultez [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/fr-fr/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
### Pour aligner plusieurs contrôles dans un formulaire  
  
1.  Ouvrez le formulaire contenant les contrôles à positionner dans le **Concepteur Windows Forms**.  
  
2.  Sélectionnez les contrôles en veillant à choisir en premier celui sur lequel tous les autres doivent être alignés.  
  
3.  Dans le menu **Format**, pointez sur **Aligner** puis sur l'une des sept options proposées.
