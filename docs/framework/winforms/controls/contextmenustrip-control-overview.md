# Vue d&#39;ensemble du contr&#244;le ContextMenuStrip
> [!NOTE]
>  Le contrôle <xref:System.Windows.Forms.ContextMenuStrip> remplace le contrôle <xref:System.Windows.Forms.ContextMenu> et lui ajoute des fonctionnalités ; toutefois, le contrôle <xref:System.Windows.Forms.ContextMenu> est conservé pour la compatibilité descendante et l'utilisation future si tel est votre choix.  
  
 Les menus contextuels, également appelés menus de raccourcis, apparaissent à la position de souris lorsque l'utilisateur clique avec le bouton droit de la souris.  Les *menus contextuels* fournissent des options pour la zone cliente ou le contrôle à la position du pointeur de souris.  
  
 Le contrôle <xref:System.Windows.Forms.ContextMenuStrip> est conçu pour travailler de façon homogène avec les nouveaux contrôles <xref:System.Windows.Forms.ToolStrip> et contrôles connexes, mais vous pouvez associer un <xref:System.Windows.Forms.ContextMenuStrip> à d'autres contrôles tout aussi facilement.  
  
 Le tableau suivant affiche les classes auxiliaires <xref:System.Windows.Forms.ContextMenuStrip> importantes.  
  
|Classe|Description|  
|------------|-----------------|  
|<xref:System.Windows.Forms.ToolStripMenuItem>|Représente une option pouvant être sélectionnée, qui est affichée sur un <xref:System.Windows.Forms.MenuStrip> ou un <xref:System.Windows.Forms.ContextMenuStrip>.|  
|<xref:System.Windows.Forms.ToolStripDropDown>|Représente un contrôle qui permet à l'utilisateur de sélectionner un seul élément dans une liste affichée lorsqu'il clique sur <xref:System.Windows.Forms.ToolStripDropDownButton> ou sur un élément de menu de niveau supérieur.|  
|<xref:System.Windows.Forms.ToolStripDropDownItem>|Fournit les fonctionnalités de base pour les contrôles dérivés de <xref:System.Windows.Forms.ToolStripItem> qui affichent des éléments déroulants lorsque l'utilisateur clique dessus.|
