# DataGrid, contr&#244;le (Windows Forms)
> [!NOTE]
>  Le contrôle <xref:System.Windows.Forms.DataGridView> remplace le contrôle `DataGrid` et lui ajoute des fonctionnalités ; toutefois, le contrôle `DataGrid` est conservé pour la compatibilité descendante et l'utilisation future si tel est votre choix.  Pour plus d'informations, consultez [Différences entre les contrôles DataGridView et DataGrid Windows Forms](../../../../docs/framework/winforms/controls/differences-between-the-windows-forms-datagridview-and-datagrid-controls.md).  
  
 Le contrôle `DataGrid` Windows Forms fournit une interface utilisateur aux groupes de données [!INCLUDE[vstecado](../../../../includes/vstecado-md.md)], affichant les données sous forme de tableau et permettant les mises à jour de la source de données.  
  
 Lorsque le contrôle `DataGrid` a pour valeur une source de données valide, il est automatiquement rempli et des colonnes et des lignes sont créées en fonction de la forme des données.  Le contrôle `DataGrid` permet d'afficher une table unique ou les relations hiérarchiques existant entre plusieurs tables.
