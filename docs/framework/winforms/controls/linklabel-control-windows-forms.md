# LinkLabel, contr&#244;le (Windows Forms)
Le contrôle `LinkLabel` Windows Forms vous permet de créer des liens Web dans les applications Windows Forms.  Vous pouvez utiliser le contrôle `LinkLabel` pour effectuer toutes les tâches que vous exécutez à l'aide du contrôle <xref:System.Windows.Forms.Label> ; vous pouvez également définir une partie du texte en tant que lien vers un objet ou une page Web.
