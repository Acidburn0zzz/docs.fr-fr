# FileDialog, classe
La classe <xref:System.Windows.Forms.FileDialog> Windows Forms est la classe de base commune des composants <xref:System.Windows.Forms.OpenFileDialog> et <xref:System.Windows.Forms.SaveFileDialog>.  Vous pouvez apporter des modifications à la classe <xref:System.Windows.Forms.FileDialog> qui affectent l'apparence et le comportement de ces boîtes de dialogue, selon la version de Windows sur laquelle l'application s'exécute.
