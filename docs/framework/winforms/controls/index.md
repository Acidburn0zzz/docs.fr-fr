# contr&#244;les Windows Forms
À mesure que vous concevez et modifiez l'interface utilisateur de vos applications Windows Forms, vous devez ajouter, aligner et positionner des contrôles.  Les contrôles sont des objets contenus dans des objets de type formulaire.  Chaque type de contrôle possède son propre ensemble de propriétés, de méthodes et d'événements qui le conditionne en vue d'atteindre un objectif spécifique.  Vous pouvez manipuler des contrôles dans le concepteur et écrire du code pour les ajouter dynamiquement au moment de l'exécution.
