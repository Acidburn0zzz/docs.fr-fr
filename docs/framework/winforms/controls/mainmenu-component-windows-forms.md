# MainMenu, composant (Windows Forms)
> [!NOTE]
>  Bien que `MenuStrip` et `ContextMenuStrip` remplacent les contrôles `MainMenu` et <xref:System.Windows.Forms.ContextMenu> des versions précédentes et leur ajoutent des fonctionnalités, `MainMenu` et <xref:System.Windows.Forms.ContextMenu> sont conservés pour la compatibilité descendante et une utilisation future, si tel est votre choix.  
  
 Le composant `MainMenu` Windows Forms permet d'afficher un menu au moment de l'exécution.
