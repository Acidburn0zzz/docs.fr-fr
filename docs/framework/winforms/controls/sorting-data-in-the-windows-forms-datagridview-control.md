# Tri des donn&#233;es dans le contr&#244;le DataGridView Windows Forms
Par défaut, les utilisateurs peuvent trier les données dans un contrôle `DataGridView` en cliquant sur l'en\-tête d'une colonne de zone de texte.  Vous pouvez modifier la propriété `SortMode` de colonnes spécifiques de manière à permettre aux utilisateurs de trier au besoin sur d'autres types de colonne.  Vous pouvez également trier les données par programme sur n'importe quelle colonne, ou sur plusieurs colonnes.
