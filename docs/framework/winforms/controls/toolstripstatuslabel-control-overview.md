# Vue d&#39;ensemble du contr&#244;le ToolStripStatusLabel
Le contrôle <xref:System.Windows.Forms.ToolStripStatusLabel> est une étiquette pour une <xref:System.Windows.Forms.StatusStrip>.  Comme les contrôles <xref:System.Windows.Forms.Label> ou <xref:System.Windows.Forms.ToolStripLabel>, le contrôle <xref:System.Windows.Forms.ToolStripStatusLabel> fournit une zone d'affichage non interactive pour du texte, des images ou les deux.  Le contrôle <xref:System.Windows.Forms.ToolStripStatusLabel> est hébergé par une <xref:System.Windows.Forms.StatusStrip>.  
  
### Membres ToolStripStatusLabel importants  
  
|Nom|Description|  
|---------|-----------------|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.Spring%2A>|Obtient ou définit une valeur qui indique si le contrôle <xref:System.Windows.Forms.ToolStripStatusLabel> remplit automatiquement l'espace disponible sur la <xref:System.Windows.Forms.StatusStrip> à mesure que le formulaire est redimensionné|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderSides%2A>|Obtient ou définit une valeur qui indique les côtés de <xref:System.Windows.Forms.ToolStripStatusLabel> qui affichent les bordures.|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderStyle%2A>|Obtient ou définit le style de bordure de <xref:System.Windows.Forms.ToolStripStatusLabel>.|
