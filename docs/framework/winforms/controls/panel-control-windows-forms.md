# Panel, contr&#244;le (Windows Forms)
Les contrôles `Panel` Windows Forms permettent de fournir un mode de groupement identifiable pour les autres contrôles.  Les zones sont principalement utilisées pour diviser un formulaire par fonctions.  Le contrôle `Panel` est similaire au contrôle <xref:System.Windows.Forms.GroupBox> ; toutefois, seul le contrôle `Panel` peut disposer de barres de défilement et seul le contrôle <xref:System.Windows.Forms.GroupBox> peut afficher une légende.
