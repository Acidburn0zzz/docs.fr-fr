# Vue d&#39;ensemble du contr&#244;le TabControl (Windows Forms)
Le contrôle <xref:System.Windows.Forms.TabControl> Windows Forms affiche plusieurs onglets comparables aux intercalaires d'un classeur.  Les onglets peuvent contenir des images ou d'autres contrôles.  Vous pouvez utiliser ce contrôle Tab pour créer des boîtes de dialogue à plusieurs pages semblables à celles qui s'affichent à divers endroits du système d'exploitation Windows et notamment dans la boîte de dialogue Propriétés de Affichage du Panneau de configuration.  En outre, le <xref:System.Windows.Forms.TabControl> peut être utilisé pour créer des pages de propriétés qui sont utilisées pour définir un groupe de propriétés connexes.  
  
## Utilisation du TabControl  
 La principale propriété du contrôle <xref:System.Windows.Forms.TabControl> est <xref:System.Windows.Forms.TabControl.TabPages%2A> qui contient les différents onglets.  Chaque onglet constitue un objet <xref:System.Windows.Forms.TabPage>.  Lorsqu'un utilisateur clique sur un onglet, il déclenche l'événement <xref:System.Windows.Forms.Control.Click> pour cet objet <xref:System.Windows.Forms.TabPage>.
