# ErrorProvider, composant (Windows Forms)
Le composant `ErrorProvider` Windows Forms permet d'indiquer discrètement un problème à l'utilisateur.  Il est en général utilisé avec la validation des données entrées par l'utilisateur dans un formulaire ou avec l'affichage des erreurs dans un groupe de données.
