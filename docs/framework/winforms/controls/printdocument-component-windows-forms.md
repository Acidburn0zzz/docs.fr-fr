# PrintDocument, composant (Windows Forms)
Le composant Windows Forms `PrintDocument` sert à définir les propriétés qui décrivent les éléments à imprimer, puis à imprimer le document dans des applications Windows.  Il peut être utilisé conjointement avec le composant <xref:System.Windows.Forms.PrintDialog> pour contrôler tous les aspects de l'impression du document.  
  
## Dans cette section  
 [Vue d'ensemble du composant PrintDocument](../../../../docs/framework/winforms/controls/printdocument-component-overview-windows-forms.md)  
 Présente les concepts généraux du composant `PrintDocument`, qui vous permet de définir les propriétés qui décrivent les éléments à imprimer et lance l'impression dans une application Windows.
