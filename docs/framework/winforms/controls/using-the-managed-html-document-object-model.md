# Utilisation du mod&#232;le objet de document HTML manag&#233;
Le modèle objet de document HTML managé fournit un wrapper basé sur le [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] pour les classes HTML exposées par Internet Explorer.  Utilisez ces classes pour manipuler des pages HTML hébergées dans le contrôle <xref:System.Windows.Forms.WebBrowser> ou encore pour créer totalement de nouvelles pages.  
  
## Dans cette section  
 [Comment : accéder au modèle objet de document HTML managé](../../../../docs/framework/winforms/controls/how-to-access-the-managed-html-document-object-model.md)  
 Décrit comment obtenir une instance valide de <xref:System.Windows.Forms.HtmlDocument> à partir d'une application Windows Forms ou d'un <xref:System.Windows.Forms.UserControl> contenu dans Internet Explorer.  
  
 [Comment : accéder à la source HTML dans le modèle objet de document HTML managé](../../../../docs/framework/winforms/controls/how-to-access-the-html-source-in-the-managed-html-document-object-model.md)  
 Décrit comment obtenir la source HTML non modifiée d'origine et la source « active » qui reflète l'état actuel du modèle DOM.  
  
 [Comment : modifier des styles d'un élément dans le modèle objet de document HTML managé](../../../../docs/framework/winforms/controls/how-to-change-styles-on-an-element-in-the-managed-html-document-object-model.md)  
 Décrit comment manipuler les styles utilisés pour contrôler l'affichage d'éléments.  
  
 [Accès aux frames dans le modèle objet de document HTML managé](../../../../docs/framework/winforms/controls/accessing-frames-in-the-managed-html-document-object-model.md)  
 Décrit les frames et les jeux de frames ainsi que les moyens d'accès au modèle DOM d'un frame.  
  
 [Accès aux membres non exposés sur le modèle objet de document HTML managé](../../../../docs/framework/winforms/controls/accessing-unexposed-members-on-the-managed-html-document-object-model.md)  
 Décrit comment accéder aux membres du modèle DOM sous\-jacent qui ne possèdent pas d'équivalent managé.
