# Utilisation du concepteur avec le contr&#244;le DataGridView Windows Forms
Visual Studio assure la prise en charge du concepteur pour le contrôle `DataGridView` qui vous permet d'effectuer de nombreuses tâches d'installation sans écrire de code.  Par ces tâches, citons la liaison du contrôle à une source de données, la modification des colonnes utilisées pour afficher des données et l'ajustement de l'apparence et du comportement de base du contrôle.
