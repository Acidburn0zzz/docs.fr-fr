# RadioButton, contr&#244;le (Windows Forms)
Le contrôle `RadioButton` Windows Forms présente à l'utilisateur plusieurs choix mutuellement exclusifs.  Les cases d'option et les cases à cocher ont à première vue un fonctionnement similaire, mais il existe entre elles une différence importante ; en effet, lorsqu'un utilisateur sélectionne dans un groupe une case d'option, les autres ne peuvent plus être sélectionnées.
