# FolderBrowserDialog, composant (Windows Forms)
Le composant `FolderBrowserDialog` Windows Forms affiche une interface permettant aux utilisateurs de rechercher et de sélectionner un dossier, ou d'en créer un.  Il s'agit un complément du composant [OpenFileDialog, composant](../../../../docs/framework/winforms/controls/openfiledialog-component-windows-forms.md) utilisé pour parcourir et sélectionner des fichiers.
