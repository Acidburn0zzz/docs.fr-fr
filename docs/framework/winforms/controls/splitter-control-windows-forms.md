# Splitter, contr&#244;le (Windows Forms)
Les contrôles `Splitter` Windows Forms servent à redimensionner des contrôles ancrés au moment de l'exécution.  Le contrôle `Splitter` est souvent utilisé dans les formulaires dont les contrôles doivent présenter des données de longueurs variables, comme l'Explorateur Windows, dont les volets de données contiennent des informations de longueur différente à différents moments.  
  
> [!NOTE]
>  Bien que <xref:System.Windows.Forms.SplitContainer> remplace et ajoute des fonctionnalités au contrôle `Splitter` des versions antérieures, `Splitter` est conservé pour la compatibilité descendante et l'utilisation future si tel est votre choix.
