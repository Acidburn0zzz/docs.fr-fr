# CheckedListBox, contr&#244;le (Windows Forms)
Le contrôle `CheckedListBox` Windows Forms affiche une liste d'éléments, tout comme le contrôle <xref:System.Windows.Forms.ListBox>, et peut également afficher une coche en regard d'éléments de la liste.
