<span data-ttu-id="07ac5-103">Vous pouvez redimensionner des contrôles individuels, et vous pouvez redimensionner plusieurs contrôles du type identique ou différent, tel que <xref:System.Windows.Forms.Button> et <xref:System.Windows.Forms.GroupBox> contrôles.</span><span class="sxs-lookup"><span data-stu-id="07ac5-103">You can resize individual controls, and you can resize multiple controls of the same or different kind, such as <xref:System.Windows.Forms.Button> and <xref:System.Windows.Forms.GroupBox> controls.</span></span>  
  
### <a name="to-resize-a-control"></a><span data-ttu-id="07ac5-104">Pour redimensionner un contrôle</span><span class="sxs-lookup"><span data-stu-id="07ac5-104">To resize a control</span></span>  
  
1.  <span data-ttu-id="07ac5-105">Cliquez sur le contrôle pour redimensionner et faites glisser une des huit poignées de redimensionnement.</span><span class="sxs-lookup"><span data-stu-id="07ac5-105">Click the control to be resized and drag one of the eight sizing handles.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="07ac5-106">Sélectionnez le contrôle et appuyez sur les touches de direction tout en maintenant enfoncée la touche MAJ pour redimensionner le contrôle un pixel à la fois.</span><span class="sxs-lookup"><span data-stu-id="07ac5-106">Select the control and press the ARROW keys while holding down the SHIFT key to resize the control one pixel at a time.</span></span> <span data-ttu-id="07ac5-107">Appuyez sur les touches de direction bas ou droite tout en maintenant les touches MAJ et CTRL pour redimensionner le contrôle par incréments de grande taille.</span><span class="sxs-lookup"><span data-stu-id="07ac5-107">Press the DOWN or RIGHT arrow keys while holding down the SHIFT and CTRL keys to resize the control in large increments.</span></span>  
  
### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="07ac5-108">Pour redimensionner plusieurs contrôles sur un formulaire</span><span class="sxs-lookup"><span data-stu-id="07ac5-108">To resize multiple controls on a form</span></span>  
  
1.  <span data-ttu-id="07ac5-109">Maintenez enfoncée la touche CTRL ou la touche MAJ ENFONCÉE et sélectionnez les contrôles que vous voulez redimensionner.</span><span class="sxs-lookup"><span data-stu-id="07ac5-109">Hold down the CTRL or SHIFT key and select the controls you want to resize.</span></span> <span data-ttu-id="07ac5-110">La taille du premier contrôle que vous sélectionnez est utilisée pour les autres contrôles.</span><span class="sxs-lookup"><span data-stu-id="07ac5-110">The size of the first control you select is used for the other controls.</span></span>  
  
2.  <span data-ttu-id="07ac5-111">Sur le **Format** menu, choisissez **Uniformiser la taille**, puis sélectionnez une des quatre options.</span><span class="sxs-lookup"><span data-stu-id="07ac5-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="07ac5-112">Les trois premières commandes modifient les dimensions des contrôles du contrôle sélectionné en premier.</span><span class="sxs-lookup"><span data-stu-id="07ac5-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
