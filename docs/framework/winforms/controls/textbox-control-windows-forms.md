# <a name="textbox-control-windows-forms"></a>TextBox, contrôle (Windows Forms)
Zones de texte Windows Forms sont utilisées pour obtenir des données à partir de l’utilisateur ou pour afficher le texte. Le `TextBox` contrôle est généralement utilisé pour le texte modifiable, bien qu’il puisse également être en lecture seule. Zones de texte peuvent afficher plusieurs lignes, renvoi à la taille du contrôle et ajouter une mise en forme. Le `TextBox` contrôle permet à un seul format de texte affiché ou entré dans le contrôle.
