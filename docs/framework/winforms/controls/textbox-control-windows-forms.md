# TextBox, contr&#244;le (Windows Forms)
Les zones de texte Windows Forms sont utilisées pour obtenir des informations de la part de l'utilisateur ou pour afficher du texte.  Le contrôle `TextBox` est généralement utilisé pour le texte modifiable bien qu'il puisse également être en lecture seule.  Les zones de texte peuvent afficher plusieurs lignes, renvoyer le texte à la ligne selon la taille du contrôle et ajouter une mise en forme de base.  Le contrôle `TextBox` ne permet d'entrer ou d'afficher qu'un seul format de texte.
