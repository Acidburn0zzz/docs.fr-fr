# Entr&#233;e au clavier dans une application Windows Forms
Windows Forms inclut des événements de clavier standard qui vous permettent de répondre à des activations de touches spécifiques, et vous fournit également des moyens d'intercepter, de modifier et de consommer des activations de touches au niveau du contrôle, du formulaire et de l'application.

 Fournit des informations sur les types d'événement de clavier et les informations qui sont reçues par les gestionnaires d'événement de clavier.

 Montre comment savoir si la touche MAJ, ALT ou CTRL a été enfoncée en plus d'une autre touche.  
  
 [Comment : gérer l'entrée au clavier au niveau du formulaire](../../../docs/framework/winforms/how-to-handle-keyboard-input-at-the-form-level.md)  
 Présente un exemple de code qui montre comment intercepter des touches avant qu'elles n'atteignent un contrôle.
