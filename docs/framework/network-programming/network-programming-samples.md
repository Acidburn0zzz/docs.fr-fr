# Exemples de programmation r&#233;seau
Cette section contient des descriptions et des liens vers des exemples téléchargeables de programmation réseau qui utilisent des classes dans <xref:System.Net>, <xref:System.Net.Cache>, <xref:System.Net.Configuration>, <xref:System.Net.Mail>, <xref:System.Net.Mime>, <xref:System.Net.NetworkInformation>, <xref:System.Net.PeerToPeer>, <xref:System.Net.Security>, <xref:System.Net.Sockets>, les espaces de noms associés.  
  
## Dans cette section  
 [Exemple de technologie d'indicateur de progression du téléchargement](http://go.microsoft.com/fwlink/?LinkID=179556)  
 Montre comment afficher la progression d'un téléchargement du fichier.  
  
 [Exemple de technologie cliente FTP](http://go.microsoft.com/fwlink/?LinkID=179557)  
 Montre comment télécharger et des fichiers téléchargés vers et à partir d'un serveur FTP.  
  
 [Exemple de technologie de HttpListener](http://go.microsoft.com/fwlink/?LinkID=179558)  
 Montre comment gérer des requêtes HTTP d'une application.  
  
 [Exemple d'application hôte de HttpListener ASPX](http://go.microsoft.com/fwlink/?LinkID=179560)  
 Explique comment utiliser les fonctionnalités de la classe d' <xref:System.Net.HttpListener> pour créer un serveur HTTP qui conduit les appels à une application hébergée ASP.NET.  
  
 [Exemple de technologie de messagerie](http://go.microsoft.com/fwlink/?LinkID=179561)  
 Montre comment envoyer des messages électroniques d'une application cliente.  
  
 [Exemple de technologie d'outils de netstat](http://go.microsoft.com/fwlink/?LinkID=179562)  
 Explique l'outil d'informations réseau de NCLNetStat.  
  
 [Exemple de technologie informatique d'informations réseau](http://go.microsoft.com/fwlink/?LinkID=179564)  
 Indique comment contrôler et afficher les informations sur le réseau.  
  
 [Exemple de client technologie de ping](http://go.microsoft.com/fwlink/?LinkID=179565)  
 Illustre une application cliente qui peut exécuter la commande PING sur un hôte distant.  
  
 [Exemple de technologie webclient](http://go.microsoft.com/fwlink/?LinkID=179566)  
 Explique comment exécuter des opérations courantes, telles que le téléchargement ou le téléchargement des fichiers ou des données.  
  
 [Définissez l'exemple de flux](http://go.microsoft.com/fwlink/?LinkID=179567)  
 Montre comment utiliser un flux sécurisé pour communiquer entre un client et un serveur.  
  
 [Exemple de sockets de IPv6](http://go.microsoft.com/fwlink/?LinkID=179568)  
 Explique comment utiliser des sockets IPv6 lorsque est activé.  
  
 [Exemple de technologie de l'explorateur FTP](http://go.microsoft.com/fwlink/?LinkID=179569)  
 Montre comment répertorier le contenu d'un serveur FTP.  
  
 [Exemple de technologie de performances de socket](http://go.microsoft.com/fwlink/?LinkID=179570)  
 Indique comment utiliser des améliorations dans la classe d' <xref:System.Net.Sockets.Socket> pour générer une application serveur qui utilise l'E\/S asynchrone de réseau pour accomplir performances élevées.  
  
 [Exemple PeerToPeer de technologie](http://go.microsoft.com/fwlink/?LinkID=179571)  
 Montre comment utiliser les classes dans l'espace de noms d' <xref:System.Net.PeerToPeer> pour enregistrer et publier un nom homologue puis pour résoudre un nom homologue.
