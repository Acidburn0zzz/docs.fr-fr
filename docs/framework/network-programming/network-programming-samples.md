# <a name="network-programming-samples"></a>Exemples de programmation réseau
Cette section contient des descriptions et des liens vers des exemples de programmation réseau téléchargeables qui utilisent des classes dans les espaces de noms <xref:System.Net>, <xref:System.Net.Cache>, <xref:System.Net.Configuration>, <xref:System.Net.Mail>, <xref:System.Net.Mime>, <xref:System.Net.NetworkInformation>, <xref:System.Net.PeerToPeer>, <xref:System.Net.Security> et <xref:System.Net.Sockets>, et autres espaces de noms associés.  
  
## <a name="in-this-section"></a>Dans cette section  
 [Exemple de technologie d’indicateur de progression de téléchargement](http://go.microsoft.com/fwlink/?LinkID=179556)  
 Montre comment afficher la progression du téléchargement d’un fichier.  
  
 [Exemple de technologie cliente FTP](http://go.microsoft.com/fwlink/?LinkID=179557)  
 Montre comment charger et télécharger des fichiers vers et à partir d’un serveur FTP.  
  
 [Exemple de technologie HttpListener](http://go.microsoft.com/fwlink/?LinkID=179558)  
 Montre comment traiter les requêtes HTTP à partir d’une application.  
  
 [Exemple d’application hôte ASPX HttpListener](http://go.microsoft.com/fwlink/?LinkID=179560)  
 Montre comment utiliser les fonctionnalités de la classe <xref:System.Net.HttpListener> pour créer un serveur HTTP qui route les appels vers une application ASP.NET hébergée.  
  
 [Exemple de technologie de messagerie](http://go.microsoft.com/fwlink/?LinkID=179561)  
 Montre comment envoyer des e-mails à partir d’une application cliente.  
  
 [Exemple de technologie NetStat](http://go.microsoft.com/fwlink/?LinkID=179562)  
 Explique l’utilisation de l’outil d’informations réseau NCLNetStat.  
  
 [Exemple de technologie d’informations réseau](http://go.microsoft.com/fwlink/?LinkID=179564)  
 Montre comment surveiller et afficher des informations réseau.  
  
 [Exemple de technologie de test ping à partir d’une application cliente](http://go.microsoft.com/fwlink/?LinkID=179565)  
 Montre une application cliente qui peut effectuer un test ping sur un hôte distant.  
  
 [Exemple de technologie WebClient](http://go.microsoft.com/fwlink/?LinkID=179566)  
 Montre comment effectuer des opérations courantes, telles que le chargement ou le téléchargement de fichiers ou de données.  
  
 [Exemple de flux sécurisés](http://go.microsoft.com/fwlink/?LinkID=179567)  
 Montre comment utiliser un flux sécurisé pour la communication entre un client et un serveur.  
  
 [Exemple de sockets IPv6](http://go.microsoft.com/fwlink/?LinkID=179568)  
 Montre comment utiliser des sockets lorsqu’IPv6 est activé.  
  
 [Exemple de technologie de l’explorateur FTP](http://go.microsoft.com/fwlink/?LinkID=179569)  
 Montre comment répertorier le contenu d’un serveur FTP.  
  
 [Exemple de technologie de performances de socket](http://go.microsoft.com/fwlink/?LinkID=179570)  
 Montre comment utiliser les améliorations apportées à la classe <xref:System.Net.Sockets.Socket> pour créer une application serveur qui utilise des E/S réseau asynchrones pour atteindre des performances maximales.  
  
 [Exemple de technologie PeerToPeer](http://go.microsoft.com/fwlink/?LinkID=179571)  
 Montre comment utiliser les nouvelles classes de l’espace de noms <xref:System.Net.PeerToPeer> pour inscrire et publier un nom de pair, puis le résoudre.
