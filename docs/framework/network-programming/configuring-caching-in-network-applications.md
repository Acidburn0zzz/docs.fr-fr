# Configuration de la mise en cache dans les applications r&#233;seau
Pour configurer la mise en cache, vous devez spécifier une stratégie de cache à l'application ou au niveau de <xref:System.Net.WebRequest> .  Les rubriques suivantes fournissent des exemples de code qui illustrent configurer des applications et des demandes d'utiliser la mise en cache.  
  
-   [Comment : Définir une stratégie de cache géolocalisée pour une application](../../../docs/framework/network-programming/how-to-set-a-location-based-cache-policy-for-an-application.md)  
  
-   [Comment : Définissez la stratégie de cache basée sur l'heure par défaut d'une application](../../../docs/framework/network-programming/how-to-set-the-default-time-based-cache-policy-for-an-application.md)  
  
-   [Comment : Personnalisez une stratégie de cache basée sur l'heure](../../../docs/framework/network-programming/how-to-customize-a-time-based-cache-policy.md)  
  
-   [Comment : Définissez la stratégie de cache pour une demande](../../../docs/framework/network-programming/how-to-set-cache-policy-for-a-request.md)  
  
 Vous pouvez également configurer la stratégie de cache à l'aide de l'application ou des fichiers de configuration machine.  Pour plus d'informations, consultez &#124; [\<requestCaching\>, élément \(paramètres réseau\)](../../../docs/framework/configure-apps/file-schema/network/requestcaching-element-network-settings.md).
