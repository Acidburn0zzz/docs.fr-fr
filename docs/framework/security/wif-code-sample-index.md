# Exemple d’index de code WIF
Voici des exemples de code pour Windows Identity Foundation 4.5 :  
  
-   [ClaimsAwareWebApp](http://go.microsoft.com/fwlink/?LinkID=248405) \-cet exemple illustre l'utilisation de l'externalisation de l'authentification de base \(pour le test local Service de jeton de sécurité à partir de l'identité et l'outil d'accès pour Visual Studio 11\) sur un ASP classique.NET d'application \(par opposition à un site web\).  
  
-   [ClaimsAwareWebService](http://go.microsoft.com/fwlink/?LinkID=248406) \-cet exemple illustre l'utilisation de l'externalisation de l'authentification sur un service WCF classique.  
  
-   [ClaimsAwareMvcApplication](http://go.microsoft.com/fwlink/?LinkID=248407) \- cet exemple montre comment intégrer WIF avec MVC, y compris la protection non permanente et code qui consacre l'authentification par formulaire redirige hors du contrôleur d'ouverture de session.  
  
-   [ClaimsAwareWebFarm](http://go.microsoft.com/fwlink/?LinkID=248408) \-cet exemple illustre un cache de session prêt de batterie de serveurs \(par opposition à un tokenreplycache\) afin que vous pouvez utiliser les sessions par référence plutôt que d'échanger des cookies big.  Il montre également un moyen plus simple de sécurisation des cookies dans une batterie de serveurs.  
  
-   [ClaimsAwareFormsAuthentication](http://go.microsoft.com/fwlink/?LinkID=248409) \-très simple, cet exemple montre que, dans.NET 4.5 pour obtenir des revendications dans vos entités indépendamment de la façon dont vous authentifiez vos utilisateurs.  
  
-   [ClaimsBasedAuthorization](http://go.microsoft.com/fwlink/?LinkID=248410)\- ce exemples illustre l'utilisation de votre classe de CLaimsAuthorizationManager et le ClaimsAuthorizationModule pour appliquer vos propres stratégies d'autorisation.  
  
-   [FederationMetadata](http://go.microsoft.com/fwlink/?LinkID=248411) – cet exemple illustre la génération dynamique \(sur un STS personnalisé\) et la consommation dynamique \(sur une application de la partie utilisatrice\) des documents de métadonnées.  
  
-   [CustomToken](http://go.microsoft.com/fwlink/?LinkID=248412) – cet exemple montre comment créer un type de jeton personnalisé Simple Web jeton \(SWT\).
