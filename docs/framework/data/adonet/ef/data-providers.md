# Fournisseurs de donn&#233;es Entity Framework
Cette section fournit des informations sur les fournisseurs de données qui prennent en charge [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
## Dans cette section  
 [Fournisseur EntityClient pour Entity Framework](../../../../../docs/framework/data/adonet/ef/entityclient-provider-for-the-entity-framework.md)  
 Décrit le fournisseur de données EntityClient.  Ce fournisseur transforme des requêtes sur un modèle de données en une arborescence de commandes canonique.  Ensuite, l'arborescence de commandes peut être consommée par un fournisseur de données [!INCLUDE[dnprdnshort](../../../../../includes/dnprdnshort-md.md)] pour [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
 [SqlClient pour Entity Framework](../../../../../docs/framework/data/adonet/ef/sqlclient-for-the-entity-framework.md)  
 Décrit le fournisseur de données [!INCLUDE[dnprdnshort](../../../../../includes/dnprdnshort-md.md)] qui prend en charge [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] à utiliser avec une base de données [!INCLUDE[ssNoVersion](../../../../../includes/ssnoversion-md.md)].
