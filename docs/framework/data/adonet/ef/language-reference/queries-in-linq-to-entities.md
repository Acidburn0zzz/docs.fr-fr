<span data-ttu-id="c822f-103">Une requête est une expression qui récupère des données d'une source de données.</span><span class="sxs-lookup"><span data-stu-id="c822f-103">A query is an expression that retrieves data from a data source.</span></span> <span data-ttu-id="c822f-104">En général, les requêtes sont exprimées dans un langage de requête spécialisé, tel que SQL pour les bases de données relationnelles et Xquery pour XML.</span><span class="sxs-lookup"><span data-stu-id="c822f-104">Queries are usually expressed in a specialized query language, such as SQL for relational databases and XQuery for XML.</span></span> <span data-ttu-id="c822f-105">Par conséquent, les développeurs ont dû apprendre un nouveau langage de requête pour chaque type de source de données ou format de données qu'ils interrogent.</span><span class="sxs-lookup"><span data-stu-id="c822f-105">Therefore, developers have had to learn a new query language for each type of data source or data format that they query.</span></span> <span data-ttu-id="c822f-106">LINQ (Language-Integrated Query) propose un modèle cohérent plus simple qui permet d'utiliser des données de types de sources et de formats divers.</span><span class="sxs-lookup"><span data-stu-id="c822f-106">Language-Integrated Query (LINQ) offers a simpler, consistent model for working with data across various kinds of data sources and formats.</span></span> <span data-ttu-id="c822f-107">Dans une requête LINQ, vous travaillez toujours avec des objets de programmation.</span><span class="sxs-lookup"><span data-stu-id="c822f-107">In a LINQ query, you always work with programming objects.</span></span>

 <span data-ttu-id="c822f-113">Dans la requête, vous spécifiez exactement les informations que vous voulez récupérer à partir de la source de données.</span><span class="sxs-lookup"><span data-stu-id="c822f-113">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="c822f-114">Une requête peut également spécifier la manière dont ces informations doivent être triées, regroupées et mises en forme avant d'être retournées.</span><span class="sxs-lookup"><span data-stu-id="c822f-114">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="c822f-115">Dans LINQ, une requête est stockée dans une variable.</span><span class="sxs-lookup"><span data-stu-id="c822f-115">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="c822f-116">Si la requête retourne une séquence de valeurs, la variable de requête doit elle-même être de type requêtable.</span><span class="sxs-lookup"><span data-stu-id="c822f-116">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="c822f-117">Cette variable de requête n'effectue aucune action et ne retourne aucune donnée ; elle stocke uniquement les informations de requête.</span><span class="sxs-lookup"><span data-stu-id="c822f-117">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="c822f-118">Une fois que vous avez créé une requête, vous devez l'exécuter pour extraire des données.</span><span class="sxs-lookup"><span data-stu-id="c822f-118">After you create a query you must execute that query to retrieve any data.</span></span>
