## <a name="group-based-aggregates"></a><span data-ttu-id="3fb02-182">Agrégats basés sur un groupe</span><span class="sxs-lookup"><span data-stu-id="3fb02-182">Group-based aggregates</span></span>

select avg(ol.Quantity) from LOB.OrderLines as ol
