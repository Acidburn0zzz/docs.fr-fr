# <a name="downloading-sample-databases"></a>Téléchargement d'exemples de base de données
Plusieurs exemples et procédures pas à pas dans la documentation de [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] utilisent l'exemple de base de données Northwind et SQL Server Express Edition. Vous pouvez télécharger ces produits gratuitement à partir du site de téléchargement Microsoft.  
  
## <a name="downloading-the-northwind-database"></a>Téléchargement de la base de données Northwind  
  
#### <a name="to-download-and-install-the-northwind-sample-database-for-sql-server"></a>Pour télécharger et installer l'exemple de base de données Northwind pour SQL Server  
  
1.  Démarrez Internet Explorer.  
  
2.  Accédez à la [Northwind et Pubs Sample Databases](http://go.microsoft.com/fwlink?linkid=64296) site Web.  
  
3.  Cliquez sur **Télécharger**.  
  
4.  Dans le **télécharger le fichier** boîte de dialogue, sélectionnez **enregistrer**.  
  
5.  Une fois le fichier téléchargé, double-cliquez sur le **Nwind.exe** pour installer la base de données.  
  
     Par défaut, la base de données est installé à *lecteur*: \SQL Server 2000 Sample Databases.  
  
## <a name="downloading-sql-server-express-edition"></a>Téléchargement de SQL Server Express Edition  
 SQL Server Express Edition est disponible gratuitement et vous pouvez le redistribuer avec les applications. Si vous utilisez [!INCLUDE[vsprvs](../../../../../../includes/vsprvs-md.md)], SQL Server Express Edition est inclus dans les éditions professionnelles et supérieures.  
  
#### <a name="to-download-and-install-sql-server-express-edition"></a>Pour télécharger et installer SQL Server Express Edition  
  
1.  Démarrez Internet Explorer.  
  
2.  Accédez à la [Microsoft Download Center](http://go.microsoft.com/fwlink?linkid=74602) site Web.  
  
3.  Dans la zone des mots clés, tapez **SQL Server Express**.  
  
4.  Cliquez sur **OK**.  
  
5.  Dans la page de résultats, cliquez sur le lien vers le **Microsoft SQL Server 2005 Express Edition** page de téléchargement.  
  
6.  Suivez les instructions d’installation indiquées sur le site web.  
  
## <a name="downloading-management-studio-express"></a>Téléchargement de Management Studio Express  
 Si vous souhaitez modifier une base de données que vous avez téléchargé, vous pouvez accéder à la base de données à partir de **l’Explorateur de serveurs** dans les [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)] l’environnement de développement intégré (IDE), ou l’utilisation de Microsoft SQL Server Management Studio Express () SSMSE).  
  
#### <a name="to-download-management-studio-express"></a>Pour télécharger Management Studio Express  
  
-   Suivez les instructions à le [site SSMSE](http://go.microsoft.com/fwlink/?LinkId=95933).
