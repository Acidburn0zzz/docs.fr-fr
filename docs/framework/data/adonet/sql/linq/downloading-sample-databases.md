# T&#233;l&#233;chargement d&#39;exemples de bases de donn&#233;es
Plusieurs exemples et procédures pas à pas dans la documentation de [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] utilisent l'exemple de base de données Northwind et SQL Server Express Edition.  Vous pouvez télécharger ces produits gratuitement à partir du site de téléchargement Microsoft.  
  
## Téléchargement de la base de données Northwind  
  
#### Pour télécharger et installer l'exemple de base de données Northwind pour SQL Server  
  
1.  Démarrez Internet Explorer.  
  
2.  Accédez au site web [Exemples de base de données Northwind et Pubs](http://go.microsoft.com/fwlink?linkid=64296).  
  
3.  Cliquez sur **Télécharger**.  
  
4.  Dans la boîte de dialogue **Téléchargement de fichier**, sélectionnez **Enregistrer**.  
  
5.  Une fois le fichier téléchargé, double\-cliquez sur le fichier **Nwind.exe** pour installer la base de données.  
  
     Par défaut, la base de données est installée dans le dossier *lecteur*:\\SQL Server 2000 Sample Databases.  
  
## Téléchargement de SQL Server Express Edition  
 SQL Server Express Edition est disponible gratuitement et vous pouvez le redistribuer avec les applications.  Si vous utilisez [!INCLUDE[vsprvs](../../../../../../includes/vsprvs-md.md)], SQL Server Express Edition est inclus dans les éditions professionnelles et supérieures.  
  
#### Pour télécharger et installer SQL Server Express Edition  
  
1.  Démarrez Internet Explorer.  
  
2.  Accédez au site web [Centre de téléchargement Microsoft](http://go.microsoft.com/fwlink?linkid=74602).  
  
3.  Dans la zone des mots clés, tapez SQL Server Express.  
  
4.  Cliquez sur **OK**.  
  
5.  Dans la page de résultats, cliquez sur le lien vers la page de téléchargement de **Microsoft SQL Server 2005 Express Edition**.  
  
6.  Suivez les instructions d'installation indiquées sur le site Web.  
  
## Téléchargement de Management Studio Express  
 Si vous souhaitez modifier une base de données que vous avez téléchargée, vous pouvez accéder à la base de données de l'**Explorateur de serveurs** dans l'environnement de développement intégré \(IDE\) [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)], ou utiliser Microsoft SQL Server Management Studio Express \(SSMSE\).  
  
#### Pour télécharger Management Studio Express  
  
-   Suivez les instructions fournies sur le [site SSMSE](http://go.microsoft.com/fwlink/?LinkId=95933).
