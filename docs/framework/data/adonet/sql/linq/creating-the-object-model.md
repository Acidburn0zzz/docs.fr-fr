# Cr&#233;ation du mod&#232;le objet
Vous pouvez créer votre modèle objet à partir d'une base de données existante et l'utiliser dans son état par défaut.  Vous pouvez également personnaliser de nombreux aspects du modèle ainsi que son comportement.  
  
 Si vous utilisez [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)], vous pouvez créer votre modèle objet à l'aide du [!INCLUDE[vs_ordesigner_long](../../../../../../includes/vs-ordesigner-long-md.md)].
