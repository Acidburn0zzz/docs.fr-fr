# D&#233;ploiement de .NET&#160;Framework
Cette section de la documentation .NET Framework fournit des informations pour les développeurs qui souhaitent installer le .NET Framework avec leurs applications, et administrateurs qui souhaitent déploiement du .NET Framework sur un réseau.  Il présente également l'activation et les erreurs de récupération associées au déploiement, et comment surveiller la progression de votre installation .NET Framework.  
  
## Dans cette section  
 [Guide de déploiement pour les développeurs](../../../docs/framework/deployment/deployment-guide-for-developers.md)  
 Explique comment les développeurs peuvent installer le .NET Framework sur les ordinateurs de leurs utilisateurs avec leurs applications.  
  
 [Guide de déploiement pour les administrateurs](../../../docs/framework/deployment/guide-for-administrators.md)  
 Explique comment un administrateur système peut déployer le .NET Framework et ses dépendances système dans un réseau grâce à System Center Configuration Manager \(SCCM\).  
  
 [Reducing System Restarts During .NET Framework 4.5 Installations](../../../docs/framework/deployment/reducing-system-restarts.md)  
 Décrit le Gestionnaire de redémarrage, qui empêché les redémarrages si possible, et explique comment les applications qui installent le .NET Framework peuvent tirer parti de ce dernier.  
  
 [Comment : obtenir la progression à partir du programme d'installation du .NET Framework 4.5](../../../docs/framework/deployment/how-to-get-progress-from-the-dotnet-installer.md)  
 Décrit comment lancer et suivre silencieusement le processus d'installation .NET Framework en affichant votre propre vue de la progression de l'installation.  
  
 [Erreurs d'initialisation de .NET Framework : gérer l'expérience utilisateur](../../../docs/framework/deployment/initialization-errors-managing-the-user-experience.md)  
 Illustre ce qui se produit lorsqu'une application .NET Framework exige une version CLR qui est incorrecte ou non installée sur l'ordinateur de l'utilisateur, comment résoudre ces erreurs, et comment contrôler le message d'erreur affiché à l'utilisateur.  
  
 [Comment : déboguer les problèmes d'activation du CLR](../../../docs/framework/deployment/how-to-debug-clr-activation-issues.md)  
 Explique comment afficher et déboguer des journaux de lancement du CLR pour résoudre les problèmes que vous pouvez rencontrer en obtenant votre application exécutée avec la version correcte du CLR.
