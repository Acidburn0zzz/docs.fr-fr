# <a name="installing-the-net-framework-on-windows"></a>Installation du .NET Framework sur Windows

Les documents suivants décrivent comment installer le .NET Framework sur chaque version de Windows.

## <a name="version-differences"></a>Différences entre les versions

Le .NET Framework est fourni avec Windows. Chaque version de Windows inclut une version du .NET Framework qui ne correspond peut-être pas à celle dont vous avez besoin. Si vous souhaitez utiliser une version différente du .NET Framework pour une version donnée de Windows, vous devez l’installer.

En général, les applications générées pour une ancienne version du .NET Framework peuvent être exécutées dans une version ultérieure. Par exemple, le .NET Framework 4.6.2 peut exécuter une application générée pour le .NET Framework 4.0. Il existe une exception à cette règle. Les applications générées pour le .NET Framework 1.0 à 3.5 s’exécutent sur le .NET Framework 3.5, mais pas sur le .NET Framework 4.0 ou ultérieur.
