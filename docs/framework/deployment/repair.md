# <a name="repair-the-net-framework"></a>Réparer le .NET Framework

Dans certains cas, votre installation du .NET Framework peut être endommagée et doit être réparée. Cela peut être le cas si votre application se bloque juste après avoir essayé de la démarrer ou si vous ne pouvez pas installer des versions plus récentes du .NET Framework.

Vous pouvez réparer votre installation du .NET Framework à l’aide de l’[outil de réparation du .NET Framework](http://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe). 

Si votre application ne démarre toujours pas après la réparation du .NET Framework, le problème peut être lié à l’application. Dans ce cas, vous devez contacter l’éditeur de l’application.
