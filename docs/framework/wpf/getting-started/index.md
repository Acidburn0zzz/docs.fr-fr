# Mise en route (WPF)
Windows Presentation Foundation \(WPF\) est une infrastructure d'interface utilisateur qui permet de créer des applications clientes riches et interactives.  La plateforme de développement WPF prend en charge un large éventail de fonctionnalités de développement d'applications, notamment un modèle d'application, des ressources, des contrôles, des graphiques, des dispositions, la liaison de données, les documents et la sécurité.  S'agissant d'un sous\-ensemble de .NET Framework, si vous avez déjà créé des applications avec .NET Framework en utilisant ASP.NET ou Windows Forms, l'environnement de programmation devrait vous être familier.  WPF utilise le langage XAML \(Extensible Application Markup Language\) pour fournir un modèle déclaratif à des fins de programmation d'applications.  Cette section contient des rubriques de présentation et d'aide à la prise en main de WPF.
