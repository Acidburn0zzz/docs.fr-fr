# ComboBox
Le contrôle <xref:System.Windows.Controls.ComboBox> présente aux utilisateurs une liste d'options.  La liste s'affiche ou disparaît lorsque le contrôle est développé ou réduit  Par défaut, la liste est réduite et affiche une seule option.  L'utilisateur cliquer sur un bouton pour consulter la liste complète des options.
