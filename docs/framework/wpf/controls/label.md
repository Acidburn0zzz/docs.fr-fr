# &#201;tiquette
Les contrôles <xref:System.Windows.Controls.Label> fournissent habituellement des informations dans le [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)].  Historiquement, un <xref:System.Windows.Controls.Label> ne contenait que du texte, mais comme le <xref:System.Windows.Controls.Label> fourni avec [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] est un <xref:System.Windows.Controls.ContentControl>, il peut contenir du texte ou un <xref:System.Windows.UIElement>.  
  
 Un <xref:System.Windows.Controls.Label> fournit une prise en charge à la fois fonctionnelle et visuelle pour les touches d'accès rapide.  Il est fréquemment utilisé pour activer les touches d'accès rapide aux contrôles tels qu'un <xref:System.Windows.Controls.TextBox>.  Pour assigner un <xref:System.Windows.Controls.Label> à un <xref:System.Windows.Controls.Control>, affectez la propriété <xref:System.Windows.Controls.Label.Target%2A?displayProperty=fullName> au contrôle qui doit obtenir le focus lorsque l'utilisateur appuie sur la touche d'accès rapide.  
  
 L'image suivante montre un "Thèmes" <xref:System.Windows.Controls.Label> qui cible un <xref:System.Windows.Controls.ComboBox>.  Lorsque l'utilisateur appuie sur ce bouton, le <xref:System.Windows.Controls.ComboBox> reçoit le focus.  Pour plus d'informations, consultez [How to: Set the Target Property of a Label](http://msdn.microsoft.com/fr-fr/b24c6977-ebcb-4855-a9bb-3fd4435af8f8).
