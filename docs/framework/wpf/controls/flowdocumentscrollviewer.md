# FlowDocumentScrollViewer
Le contrôle <xref:System.Windows.Controls.FlowDocumentScrollViewer> est utilisé pour consulter le contenu <xref:System.Windows.Documents.FlowDocument> dans un conteneur de défilement.  Contrairement au <xref:System.Windows.Controls.FlowDocumentPageViewer>, qui affiche le contenu par page.
