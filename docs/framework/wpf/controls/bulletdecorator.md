# BulletDecorator
<xref:System.Windows.Controls.Primitives.BulletDecorator> comporte deux propriétés de contenu : <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> et <xref:System.Windows.Controls.Decorator.Child%2A>.  La propriété <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> définit le <xref:System.Windows.UIElement> à utiliser comme puce.  La propriété <xref:System.Windows.Controls.Decorator.Child%2A> définit un <xref:System.Windows.UIElement> qui s'aligne visuellement avec la puce.
