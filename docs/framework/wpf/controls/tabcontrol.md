# TabControl
Les éléments <xref:System.Windows.Controls.TabControl> affichent le contenu dans des pages séparées auxquelles on accède en sélectionnant l'onglet approprié.  Chaque onglet contient un <xref:System.Windows.Controls.TabItem>.
