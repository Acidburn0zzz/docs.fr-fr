# PrintDialog
Le contrôle <xref:System.Windows.Controls.PrintDialog> est utilisé pour instancier une boîte de dialogue d'impression standard qui configure automatiquement <xref:System.Printing.PrintTicket> et <xref:System.Printing.PrintQueue> selon les entrées d'utilisateur.
