# FlowDocumentPageViewer
Le contrôle <xref:System.Windows.Controls.FlowDocumentPageViewer> est utilisé pour consulter le contenu <xref:System.Windows.Documents.FlowDocument> sur une base par page.  Contrastez avec le <xref:System.Windows.Controls.FlowDocumentScrollViewer>, qui présente le contenu <xref:System.Windows.Documents.FlowDocument> dans une visionneuse de défilement.
