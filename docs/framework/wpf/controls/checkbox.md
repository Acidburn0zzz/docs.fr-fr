# CheckBox
Vous pouvez utiliser une <xref:System.Windows.Controls.CheckBox> dans le [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] de votre application pour représenter les options qu'un utilisateur peut sélectionner ou désactiver.  Vous pouvez utiliser une case à cocher unique ou regrouper deux ou plusieurs cases à cocher.  
  
 Le graphique suivant affiche les différents états d'une <xref:System.Windows.Controls.CheckBox>.
