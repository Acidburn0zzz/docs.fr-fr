# Frame
Le contrôle <xref:System.Windows.Controls.Frame> prend en charge la navigation de contenu dans le contenu.  <xref:System.Windows.Controls.Frame> peut être hébergé par un élément racine comme <xref:System.Windows.Window>, <xref:System.Windows.Navigation.NavigationWindow>, <xref:System.Windows.Controls.Page>, <xref:System.Windows.Controls.UserControl>, <xref:System.Windows.Documents.FlowDocument>, ou en tant qu'îlot dans une arborescence de contenu qui appartient à un élément racine.
