# <a name="richtextbox"></a>RichTextBox
Le <xref:System.Windows.Controls.RichTextBox> élément définit un contrôle d’édition avec prise en charge intégrée des fonctionnalités telles que couper et coller, présentation de documents riche et sélection de contenu.
