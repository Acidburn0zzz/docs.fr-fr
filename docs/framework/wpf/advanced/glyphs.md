# Glyphes
Les glyphes représentent une description de bas niveau d'un caractère à tracer sur l'écran.  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] fournit un accès direct aux glyphes pour les clients qui souhaitent intercepter du texte et le rendre persistant après sa mise en forme.
