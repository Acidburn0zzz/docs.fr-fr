# &#201;l&#233;ments de base
Quatre classes clés \- <xref:System.Windows.UIElement>, <xref:System.Windows.ContentElement>, <xref:System.Windows.FrameworkElement> et <xref:System.Windows.FrameworkContentElement> \- implémentent une grande partie de la fonctionnalité des éléments courants disponibles dans la programmation WPF.  Ce Kit de développement logiciel fait référence à ces quatre classes comme des classes d'élément de base.
