# <a name="binding-markup-extension"></a><span data-ttu-id="5b599-102">Binding, extension de balisage</span><span class="sxs-lookup"><span data-stu-id="5b599-102">Binding Markup Extension</span></span>
<span data-ttu-id="5b599-103">Diffère d’une valeur de propriété une valeur liée aux données, création d’un objet d’expression et interprétant le contexte de données qui s’applique à l’élément et ses liaisons au moment de l’exécution.</span><span class="sxs-lookup"><span data-stu-id="5b599-103">Defers a property value to be a data-bound value, creating an intermediate expression object and interpreting the data context that applies to the element and its binding at run time.</span></span>  
  
## <a name="binding-expression-usage"></a><span data-ttu-id="5b599-104">Utilisation d’expressions de liaison</span><span class="sxs-lookup"><span data-stu-id="5b599-104">Binding Expression Usage</span></span>  
  
```  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```
