# Rubriques &quot;Comment&quot; relatives &#224; l&#39;impression
Les rubriques de cette section montrent comment utiliser l'impression et les fonctionnalités de gestion du système d'impression incluses avec [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] ainsi que le nouveau chemin d'impression [!INCLUDE[TLA#tla_xps](../../../../includes/tlasharptla-xps-md.md)].  
  
## Dans cette section  
 [Appeler une boîte de dialogue Imprimer](../../../../docs/framework/wpf/advanced/how-to-invoke-a-print-dialog.md)

 [Cloner une imprimante](../../../../docs/framework/wpf/advanced/how-to-clone-a-printer.md)

 Discussion étendue sur la façon dont les programmeurs de code managé peuvent étendre le schéma d'impression aux périphériques d'impression spécialisée ou aux imprimantes avec des fonctionnalités nouvelles ou inhabituelles.
