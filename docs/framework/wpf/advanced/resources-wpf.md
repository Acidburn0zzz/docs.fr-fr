# Ressources (WPF)
Une ressource est un objet qui peut être réutilisé dans différents endroits de votre application.  WPF prend en charge des types de ressources.  ces ressources sont principalement deux types de ressources : Ressources XAML et fichiers de données de ressources.  Les exemples de ressources XAML incluent les pinceaux et les styles.  Les fichiers de données de ressources sont des fichiers de données non exécutables dont une application a besoin.
