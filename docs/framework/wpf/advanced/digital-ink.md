# Encre num&#233;rique
Cette section étudie l'utilisation d'encre numérique dans [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].  Traditionnellement, l'encre numérique ne se trouvait que dans le Kit de développement logiciel de Tablet PC ; elle est désormais disponible dans Windows Presentation Foundation.  Cela signifie que vous pouvez désormais développer des applications complètes de Tablet PC à l'aide des capacités de Windows Presentation Foundation.
