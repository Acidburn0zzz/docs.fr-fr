# Documents
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] propose un jeu flexible de composants qui permettent aux développeurs de générer des applications dotées de fonctionnalités de document avancées et d'une expérience de lecture améliorée.
