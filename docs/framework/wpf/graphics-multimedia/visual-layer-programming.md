# Programmation de la couche visuelle
L'objet <xref:System.Windows.Media.Visual> est un objet [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] principal; dont le rôle est de fournir une prise en charge du rendu.  Les contrôles d'interface utilisateur, tels que <xref:System.Windows.Controls.Button> et <xref:System.Windows.Controls.TextBox>, dérivent de la classe <xref:System.Windows.Media.Visual> et l'utilisent pour rendre leurs données de rendu persistantes.
