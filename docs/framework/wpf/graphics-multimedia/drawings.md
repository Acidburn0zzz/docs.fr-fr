# Dessins
Les objets <xref:System.Windows.Media.Drawing> sont utilisés pour dessiner efficacement des formes, des images ou du texte.  Les dessins sont utilisés lorsque vous peignez avec un <xref:System.Windows.Media.DrawingBrush> ou que vous programmez avec les objets <xref:System.Windows.Media.Visual>.
