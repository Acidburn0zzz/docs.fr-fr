# Liaison de donn&#233;es (WPF)
La liaison de données Windows Presentation Foundation \(WPF\) offre un moyen simple et cohérent aux applications de présenter les données et d'interagir avec.  Les éléments peuvent être liés à des données émanant de diverses sources de données sous la forme d'objets Common Language Runtime \(CLR\) et de XML.
