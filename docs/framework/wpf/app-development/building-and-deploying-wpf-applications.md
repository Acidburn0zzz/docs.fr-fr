# G&#233;n&#233;ration et d&#233;ploiement d&#39;applications WPF
Le modèle de génération et de déploiement permet de générer et de déployer des applications localement et à distance, y compris les éléments suivants.  
  
-   MSBuild : le système de génération .NET situé dans l'espace de noms Microsoft.Build.Tasks.Windows.  
  
-   Ressources : utilisation de ressources d'interface graphique.  
  
-   Déploiement ClickOnce : le système de publication et de déploiement .NET.
