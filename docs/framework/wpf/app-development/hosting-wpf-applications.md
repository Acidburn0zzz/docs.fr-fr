# H&#233;bergement d&#39;applications WPF
Les applications de navigateur XAML WPF \(applications XBAP\) sont des applications clientes riches pouvant être déployées sur un serveur Web et être démarrées depuis un navigateur.  L'hôte WPF \(PresentationHost.exe\) est inscrit comme le shell et le gestionnaire MIME pour les fichiers XBAP et XAML.  Par conséquent, Internet Explorer sait qu'il doit démarrer l'hôte WPF lorsqu'une application XBAP est lancée.  Les utilisateurs de Firefox peuvent installer des compléments additionnels Firefox permettant à Firefox d'héberger également des applications XBAP.  Une application XBAP peut être hébergée par d'autres navigateurs ou applications autonomes à l'aide d'API d'hébergement de navigateur natives, fournies par WPF.
