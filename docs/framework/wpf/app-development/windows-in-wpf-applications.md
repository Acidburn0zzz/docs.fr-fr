# Fen&#234;tres dans les applications WPF
Les utilisateurs interagissent avec les applications à l'aide de fenêtres.  La fonction principale d'une fenêtre est d'héberger et d'afficher du contenu.  Le type de contenu qu'une fenêtre héberge dépend du type de données avec lesquelles une application fonctionne. Cela inclut les médias, les pages XAML \(eXtensible Application Markup Language\), les pages Web, les documents, les tables et les enregistrements de base de données et les informations système.
