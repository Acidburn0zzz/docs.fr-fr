# <a name="datamemberfieldeditor-class"></a>Classe de DataMemberFieldEditor

Fournit l’interface utilisateur de modifier les propriétés des objets liés aux données en répertoriant toutes les propriétés de la `DataSource` pour sélectionner la source de la valeur d’objet.  
  
## <a name="syntax"></a>Syntaxe
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> La `DataMemberFieldEditor` classe est interne et ne revient pas à être utilisée directement dans votre code.
> 
> Microsoft ne prend pas en charge l’utilisation de cette classe dans une application de production en toutes circonstances.
