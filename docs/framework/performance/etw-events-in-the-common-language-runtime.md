# ETW Events in the Common Language Runtime
Le Common Language Runtime \(CLR\) fournit des informations de diagnostic utiles pour le suivi d'événements pour Windows \(ETW\) par l'intermédiaire d'une grande variété d'événements de débogage et de profilage.  Les événements ETW du CLR tirent parti du système de suivi \(ETW\) Windows pour améliorer le profilage et le débogage déjà proposés par le Common Language Runtime.  
  
 Vous trouverez des informations supplémentaires sur ETW dans l'article [Améliorez le débogage et l'optimisation des performances avec ETW](http://go.microsoft.com/fwlink/?LinkID=161142) sur MSDN.  Des informations relatives à Xperf sont disponibles à la page du blog NTDebugging [Windows Performance Toolkit \- Xperf \(page éventuellement en anglais\)](http://go.microsoft.com/fwlink/?LinkID=161144).  
  
 Les outils supplémentaires CLR ETW sont fournis dans [Site Web CodePlex](http://go.microsoft.com/fwlink/?LinkID=111138) lorsqu'ils sont disponibles.  
  
 La version [!INCLUDE[net_v40_long](../../../includes/net-v40-long-md.md)] ou une version ultérieure est requise pour tous les rubriques décrits par événements.  Le système d'exploitation Windows Vista est le minimum client prise en charge, et Windows Server 2008 est le serveur minimum prise en charge.
