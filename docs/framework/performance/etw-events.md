# ETW Events in the .NET Framework
Le suivi d'événements pour Windows \(ETW\) est un système de suivi hautes performances, à faible charge de mémoire et évolutif fourni par les systèmes d'exploitation Windows.  Il complète les fonctions de profilage et de débogage fournies par le .NET Framework et peut être utilisé pour dépanner un certain nombre de situations.  Pour obtenir des informations générales sur le suivi ETW, consultez [Améliorer le débogage et le réglage des performances avec ETW](http://go.microsoft.com/fwlink/?LinkID=161142) dans MSDN Library.  
  
 Dans le .NET Framework, le suivi d'événements ETW est disponible pour le Common Language Runtime \(CLR\), la [bibliothèque parallèle de tâches](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md) et [Parallel LINQ \(PLINQ\)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md).
