# <a name="etw-events-in-the-net-framework"></a>Événements ETW dans le .NET Framework
Le suivi d’événements pour Windows (ETW) est un système de suivi hautes performances, à faible charge et scalable fourni par les systèmes d’exploitation Windows. Il complète la prise en charge du profilage et du débogage fournie par le .NET Framework et peut être utilisé pour dépanner un certain nombre de situations. Pour obtenir des informations d’ordre général sur ETW, consultez [Améliorez le débogage et l’optimisation des performances avec ETW](http://go.microsoft.com/fwlink/?LinkID=161142) dans MSDN Library.  
  
 Dans le .NET Framework, le suivi d’événements ETW est disponible pour le CLR (Common Language Runtime), la [bibliothèque parallèle de tâches](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md) et [PLINQ (Parallel LINQ)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md).
