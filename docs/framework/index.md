# <a name="net-framework-guide"></a>Guide du .NET framework

Le .NET Framework est une plateforme de développement permettant de générer des applications pour Windows, Windows Server et Microsoft Azure. Il se compose du common language runtime (CLR) et de la bibliothèque de classes .NET Framework, qui offre une vaste gamme de fonctionnalités et prend en charge de nombreux standards. 

Le .NET Framework propose plusieurs services, notamment la gestion de la mémoire, la cohérence des types, la sûreté de la mémoire, des fonctionnalités de sécurité, la mise en réseau et le déploiement d’applications. Il fournit des structures de données et des API faciles à utiliser qui permettent d’abstraire le système d’exploitation Windows de niveau inférieur. Vous pouvez utiliser divers langages de programmation avec le .NET Framework, notamment C#, F# et Visual Basic.

## <a name="deployment"></a>Déploiement

Le .NET Framework étant fourni avec Windows, vous pouvez exécuter des applications .NET Framework. Toutefois, vous aurez peut-être besoin d’une version du .NET Framework plus récente que celle fournie avec votre version de Windows. Pour savoir comment déployer le .NET Framework sur Windows, consultez [Déploiement du .NET Framework sur Windows](./deployment/windows/index.md).

Pour savoir comment réparer votre installation du .NET Framework si vous rencontrez des erreurs pendant l’installation, consultez [Réparer le .NET Framework](./deployment/repair.md).

Vous pouvez utiliser le .NET Framework avec Docker et des [conteneurs Windows](https://msdn.microsoft.com/virtualization/windowscontainers/about/about_overview). Pour apprendre à exécuter vos applications dans des conteneurs Docker, consultez [Déploiement d’applications .NET Framework avec Docker](./docker/index.md).
