# <a name="application-compatibility-in-the-net-framework"></a>Compatibilité des applications dans le .NET Framework
Les articles suivants décrivent les problèmes de compatibilité des applications dans les versions récentes du .NET Framework :  
  
-   Entre les versions 4 et 4.5 : [Compatibilité des applications dans la version 4.5](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-5.md)  
  
-   Entre les versions 4.5 et 4.5.1 : [Compatibilité des applications dans la version 4.5.1](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-5-1.md)  
  
-   Entre les versions 4.5.1 et 4.5.2 : [Compatibilité des applications dans la version 4.5.2](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-5-2.md)  
  
-   Entre les versions 4.5.2 et 4.6 : [Compatibilité des applications dans la version 4.6](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-6.md)  
  
-   Entre les versions 4.6 et le [!INCLUDE[net_v461](../../../includes/net-v461-md.md)] : [Compatibilité des applications dans la version 4.6.1](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-6-1.md)  
  
-   Entre les versions 4.6.1 et le [!INCLUDE[net_v462](../../../includes/net-v462-md.md)] : [Compatibilité des applications dans la version 4.6.2](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-6-2.md)  

- Entre les versions 4.6.2 et le .NET Framework 4.7 : [Compatibilité des applications dans .NET Framework 4.7](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-7.md)  

Nous avons globalement essayé de maintenir un très haut niveau de compatibilité descendante entre les versions du .NET Framework, pour que les applications qui ciblent une version spécifique s'exécutent sans avoir besoin de modifications sur les versions ultérieures. La plupart des problèmes de compatibilité des applications ont un impact faible et ne nécessitent pas de modifier la programmation de vos applications. Pour connaître ceux qui nécessitent une modification, reportez-vous à la colonne Impact dans les tableaux accessibles via ces liens.
