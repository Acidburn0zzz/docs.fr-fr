# <a name="guide-to-the-windows-workflow-documentation"></a>Guide de la documentation Windows Workflow
Cette rubrique contient des informations sur l'utilisation de la documentation de Windows Workflow Foundation. Les documents liés sont des points de départ recommandés regroupés en fonction d'intérêts et de niveaux d'expertise spécifiques.  
  
## <a name="new-to-windows-workflow-foundation-programming"></a>Nouveautés concernant la programmation Windows Workflow Foundation  
  
-   Si vous débutez en programmation avec Windows Workflow Foundation et que vous souhaitez simplement voir des exemples d’applications qui fonctionnent, consultez les rubriques répertoriées sous [exemples Windows Workflow](../../../docs/framework/windows-workflow-foundation/samples/index.md).  
  
-   Pour en savoir plus sur les différents modèles de contrôle de flux de Windows Workflow Foundation, consultez [concevoir le flux de travail](../../../docs/framework/windows-workflow-foundation/designing-workflows.md).  
  
-   Pour obtenir un didacticiel qui vous guide à travers les étapes de base de la création d’une application Windows Workflow Foundation, consultez [Getting Started Tutorial](../../../docs/framework/windows-workflow-foundation/getting-started-tutorial.md).  
  
-   Si vous êtes intéressé par les concepts de Windows Workflow Foundation, consultez les rubriques de la [vue d’ensemble conceptuelle de Windows Workflow](../../../docs/framework/windows-workflow-foundation/conceptual-overview.md) section.  
  
-   Pour obtenir la liste des termes utilisés dans la documentation de Windows Workflow Foundation, consultez [glossaire Windows Workflow Foundation pour .NET Framework 4.5](../../../docs/framework/windows-workflow-foundation/glossary.md).  
  
-   Pour obtenir la liste des nouveaux concepts et fonctionnalités, consultez [Nouveautés de Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/whats-new.md).  
  
## <a name="programming-in-depth"></a>Programmation approfondie  
  
-   Pour une discussion détaillée du modèle objet Windows Workflow Foundation, consultez [Architecture Windows Workflow](../../../docs/framework/windows-workflow-foundation/architecture.md).  
  
-   Si vous êtes prêt à commencer à développer une application, consultez [programmation de Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/programming.md).  
  
-   Si vous souhaitez étendre ou personnaliser Windows Workflow Foundation pour répondre à vos besoins, consultez [extension Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/extend.md).
