# Guide de la documentation Windows Workflow
Cette rubrique contient des informations sur l'utilisation de la documentation de Windows Workflow Foundation.Les documents liés sont des points de départ recommandés regroupés en fonction d'intérêts et de niveaux d'expertise spécifiques.  
  
## Nouveautés concernant la programmation Windows Workflow Foundation  
  
-   Si vous êtes novice dans la programmation à l'aide de Windows Workflow Foundation et que vous souhaitez simplement obtenir quelques exemples d'applications qui fonctionnent, consultez les rubriques répertoriées sous [Exemples Windows Workflow](../../../docs/framework/windows-workflow-foundation/samples/index.md).  
  
-   Pour en savoir plus sur les différents modèles de contrôle de flux Windows Workflow Foundation, consultez [Conception des flux de travaux](../../../docs/framework/windows-workflow-foundation//designing-workflows.md).  
  
-   Pour accéder à un didacticiel présentant les étapes de base de la création d'une application Windows Workflow Foundation, consultez [Didacticiel de mise en route](../../../docs/framework/windows-workflow-foundation//getting-started-tutorial.md).  
  
-   Si vous vous intéressez aux concepts sous\-jacents de Windows Workflow Foundation, consultez les rubriques de la section [Vue d'ensemble conceptuelle de Windows Workflow](../../../docs/framework/windows-workflow-foundation//conceptual-overview.md).  
  
-   Pour obtenir une liste des termes utilisés dans la documentation de Windows Workflow Foundation, consultez [Glossaire Windows Workflow Foundation pour .NET Framework 4.5](../../../docs/framework/windows-workflow-foundation//glossary.md).  
  
-   Pour obtenir une liste des nouveaux concepts et des nouvelles fonctionnalités, consultez [Nouveautés dans Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation//whats-new.md).  
  
## Programmation approfondie  
  
-   Pour obtenir une présentation détaillée du modèle objet Windows Workflow Foundation, consultez [Architecture Windows Workflow](../../../docs/framework/windows-workflow-foundation//architecture.md).  
  
-   Si vous avez le niveau requis pour développer une application, consultez [Programmation Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation//programming.md).  
  
-   Si vous souhaitez étendre ou adapter Windows Workflow Foundation à vos besoins spécifiques, consultez [Extension de Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation//extend.md).
