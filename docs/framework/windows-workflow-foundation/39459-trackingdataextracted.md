# 39459 - TrackingDataExtracted
## Propriétés  
  
|||  
|-|-|  
|ID|39459|  
|Mots clés|WFRuntime|  
|Niveau|Verbose|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Indique que les données de suivi ont été extraites dans une activité.  
  
## Message  
 Données de suivi %1 extraites dans l'activité %2.  
  
## Détails  
  
|Nom d'élément de données|Type d'élément de données|Description|  
|------------------------------|-------------------------------|-----------------|  
|Données|xs:string|Nom des données extraites.|  
|Activité|xs:string|Nom de l'activité.|  
|AppDomain|xs:string|Chaîne retournée par AppDomain.CurrentDomain.FriendlyName.|
