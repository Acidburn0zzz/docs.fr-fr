# Cha&#238;ne de connexion et nom de la cha&#238;ne de connexion
La propriété **Connection String** spécifie la chaîne de connexion que le magasin d'instances de workflow SQL doit utiliser pour se connecter à une base de données de persistance.Ce paramètre est facultatif.La propriété **Connection String Name** spécifie le nom de la chaîne de connexion nommée que le magasin d'instances de workflow SQL doit utiliser pour se connecter à la base de données de persistance.Ce paramètre est facultatif.Vous devez spécifier une valeur pour les propriétés Connection String Name ou Connection String si vous ne souhaitez pas que le magasin d'instances de workflow SQL utilise la chaîne de connexion nommée par défaut **DefaultSqlWorkflowInstanceStoreConnectionString**.
