# 2026 - CompileVbExpressionStart
## Propriétés  
  
|||  
|-|-|  
|ID|2026|  
|Mots clés|WFRuntime|  
|Niveau|Verbose|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Indique le début d'une compilation d'expression VB.

 Compilation de l'expression Visual Basic « %1 »
