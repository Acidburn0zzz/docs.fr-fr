|||  
|-|-|  
|ID|2025|  
|Mots clés|WFRuntime|  
|Niveau|Verbose|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description

 InternalCacheMetadata s'est arrêté sur l'activité « %1 ».
