# 1147 - FlowchartSwitchDefault
## Propriétés  
  
|||  
|-|-|  
|ID|1147|  
|Mots clés|WFActivities|  
|Niveau|Information|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Débogage|  
  
## Description  
 Indique que le cas par défaut a été sélectionné dans un commutateur Flowchart.  
  
## Message  
 Flowchart '%1'\/FlowSwitch \- cas par défaut a été sélectionné.  
  
## Détails  
  
|Nom d'élément de données|Type d'élément de données|Description|  
|------------------------------|-------------------------------|-----------------|  
|Organigramme|xs:string|Nom complet de l'organigramme.|  
|AppDomain|xs:string|Chaîne retournée par AppDomain.CurrentDomain.FriendlyName.|
