|||  
|-|-|  
|ID|2028|  
|Mots clés|WFRuntime|  
|Niveau|Verbose|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|

 CacheRootMetadata s'est arrêté sur l'activité %1.
