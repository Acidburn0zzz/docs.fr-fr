# <a name="441--stopsignpostevent1"></a>441- StopSignpostEvent1
## <a name="properties"></a>Propriétés  
  
|||  
|-|-|  
|ID|441|  
|Mots clés|Résolution des problèmes|  
|Niveau|Information|  
|Canal|Microsoft-Windows-Application Server-Applications/Analyse|  
  
## <a name="description"></a>Description  
 Dans le suivi des activités, indique qu'un message a traversé une limite d'activité lors d'une opération d'envoi ou de réception.  
  
## <a name="message"></a>Message  
 Limite d'activité.

|AppDomain|`xs:string`|Chaîne retournée par AppDomain.CurrentDomain.FriendlyName.|
