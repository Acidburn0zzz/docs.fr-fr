# Biblioth&#232;que d&#39;activit&#233;s int&#233;gr&#233;e .NET Framework&#160;4.5
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] inclut une nouvelle bibliothèque d'activités avec des fonctionnalités étendues.Les types d'activités sont les suivants :  
  
-   [Flux de contrôle](../../../docs/framework/windows-workflow-foundation//control-flow-activities-in-wf.md)  
  
-   [Organigramme](../../../docs/framework/windows-workflow-foundation//flowchart-activities-in-wf.md)  
  
-   [Ordinateur d'état](../../../docs/framework/windows-workflow-foundation//state-machine-activities-in-wf.md)  
  
-   [Activités de messagerie](../../../docs/framework/wcf/feature-details/messaging-activities.md)  
  
-   [Exécution](../../../docs/framework/windows-workflow-foundation//runtime-activities-in-wf.md)  
  
-   [Primitives](../../../docs/framework/windows-workflow-foundation//primitives-activities-in-wf.md)  
  
-   [Transaction](../../../docs/framework/windows-workflow-foundation//transaction-activities-in-wf.md)  
  
-   [Collection](../../../docs/framework/windows-workflow-foundation//collection-activities-in-wf.md)  
  
-   [Gestion des erreurs](../../../docs/framework/windows-workflow-foundation//error-handling-activities-in-wf.md)  
  
-   [Migration](../../../docs/framework/windows-workflow-foundation//migration-activity-in-wf.md)  
  
 Les activités de la bibliothèque d'activités intégrée peuvent être composées de workflows, ou être utilisées avec des activités personnalisées.Les activités de la bibliothèque d'activités intégrée sont scellées ; elles ne sont pas destinées à être utilisées pour créer de nouvelles fonctionnalités via l'héritage.
