# <a name="validation"></a>Validation
Cette section contient des exemples qui illustrent la validation d’activité.  
  
## <a name="in-this-section"></a>Dans cette section  
 [Validation d’activité externe](../../../../docs/framework/windows-workflow-foundation/samples/external-activity-validation.md)  
 Cet exemple montre comment ajouter une logique de validation à une activité intégrée dont vous n’êtes pas l’auteur.  
  
 [Validation de base](../../../../docs/framework/windows-workflow-foundation/samples/basic-validation.md)  
 Illustre une activité, `CreateProduct`, qui valide que son argument `Cost` est inférieur ou égal à son argument `Price`.  
  
 [OverloadGroups](../../../../docs/framework/windows-workflow-foundation/samples/overloadgroups.md)  
 Illustre une activité, `CreateLocation`, qui a des arguments requis et des arguments facultatifs, et qui permet à l'utilisateur de fournir un jeu d'arguments sélectionné parmi deux jeux d'arguments différents.  
  
 [Validation des relations d’activité](../../../../docs/framework/windows-workflow-foundation/samples/activity-relationships-validation.md)  
 Montre comment valider des relations entre des activités.
