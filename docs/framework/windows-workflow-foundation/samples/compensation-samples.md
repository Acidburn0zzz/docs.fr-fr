# <a name="compensation-samples"></a>Compensation (exemples)
Cette section contient des exemples liés à la compensation dans Windows Workflow Foundation (WF).  
  
## <a name="in-this-section"></a>Dans cette section  
 [Activité compensable](../../../../docs/framework/windows-workflow-foundation/samples/compensable-activity-sample.md)  
 Montre comment utiliser l'activité <xref:System.Activities.Statements.CompensableActivity> pour définir le travail à faire pour une action donnée pendant une exécution normale et le travail qui doit être fait pour compenser cette action, si nécessaire ultérieurement.  
  
 [Compensation personnalisée](../../../../docs/framework/windows-workflow-foundation/samples/custom-compensation-sample.md)  
 Montre comment utiliser <xref:System.Activities.Statements.CompensableActivity> et son gestionnaire de compensation pour définir une logique de compensation personnalisée.  
  
 [Confirmation](../../../../docs/framework/windows-workflow-foundation/samples/confirmation.md)  
 Illustre quatre scénarios courants se rapportant à l'utilisation de <xref:System.Activities.Statements.CompensableActivity> et de la confirmation.  
  
 [Gestionnaire d’annulation sur une activité compensable](../../../../docs/framework/windows-workflow-foundation/samples/cancellation-handler-on-compensable-activity.md)  
 Illustre l'utilisation d'un gestionnaire d'annulation sur un <xref:System.Activities.Statements.CompensableActivity>.
