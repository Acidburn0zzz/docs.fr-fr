# <a name="extending-windows-workflow-foundation"></a>Extension de Windows Workflow Foundation
La section suivante décrit comment étendre Windows Workflow Foundation (WF) avec des activités personnalisées et concepteurs dans les environnements réhébergés en dehors de Visual Studio 2010.

## <a name="in-this-section"></a>Dans cette section
 [Personnalisation de l’expérience de conception de Workflow](../../../docs/framework/windows-workflow-foundation/customizing-the-workflow-design-experience.md) indique comment les scénarios de conception d’activités personnalisées et au réhébergement du [!INCLUDE[wfd1](../../../includes/wfd1-md.md)] ont été considérablement simplifiés dans [!INCLUDE[netfx40_short](../../../includes/netfx40-short-md.md)]. Développement et le déploiement sont désormais plus facile et plus flexible, car le nouveau modèle de programmation concepteur activité repose sur Windows Presentation Foundation (WPF).
