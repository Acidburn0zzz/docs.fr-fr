# UI Automation Specification and Community Promise
UI Automation fournit l'accès par programme à la plupart des éléments d'interface utilisateur \(UI\) du bureau, permettant aux produits de technologie d'assistance tels que les lecteurs d'écran de fournir aux utilisateurs finaux des informations sur l'interface utilisateur et de manipuler l'interface utilisateur par d'autres moyens que l'entrée standard.
