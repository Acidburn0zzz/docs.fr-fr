# <a name="ui-automation-specification-and-community-promise"></a>Spécification UI Automation et promesse de la communauté
UI Automation fournit un accès par programme à la plupart des éléments d’interface (UI) utilisateur sur le bureau, l’activation de produits de technologie d’assistance tels que les lecteurs d’écran pour fournir des informations sur l’interface utilisateur pour les utilisateurs finaux et de manipuler l’interface utilisateur par signifie que l’entrée standard .
