# <a name="endregion-c-reference"></a>#endregion (référence C#)
`#endregion` marque la fin d’un bloc [#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md). Exemple :  
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
