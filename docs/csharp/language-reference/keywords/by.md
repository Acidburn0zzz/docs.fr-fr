# <a name="by-c-reference"></a>by (Référence C#)

Le mot clé contextuel `by` est utilisé dans la clause `group` dans une expression de requête pour spécifier la façon dont les éléments retournés doivent être groupés. Pour plus d’informations, consultez [group, clause](../../../csharp/language-reference/keywords/group-clause.md).
