# <a name="try-catch-finally-c-reference"></a>try-catch-finally (référence C#)
`catch` et `finally` sont souvent utilisés ensemble pour obtenir et utiliser des ressources dans un bloc `try`, pour traiter des circonstances exceptionnelles dans un bloc `catch` et pour libérer les ressources dans le bloc `finally`.  
  
 Pour plus d’informations et des exemples sur la génération répétée d’exceptions, consultez [try-catch](../../../csharp/language-reference/keywords/try-catch.md) et [Génération d’exceptions](../../../standard/exceptions/index.md). Pour plus d’informations sur le bloc `finally`, consultez [try-finally](../../../csharp/language-reference/keywords/try-finally.md).
