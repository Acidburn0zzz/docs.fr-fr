# <a name="new-c-reference"></a>new (référence C#)
En C#, le mot clé `new` peut servir d’opérateur, de modificateur ou de contrainte.  
  
 [new, opérateur](../../../csharp/language-reference/keywords/new-operator.md)  
 Cet opérateur s’utilise pour créer des objets et appeler des constructeurs.  
  
 [new, modificateur](../../../csharp/language-reference/keywords/new-modifier.md)  
 Permet de masquer un membre hérité d’un membre de classe de base.  
  
 [new, contrainte](../../../csharp/language-reference/keywords/new-constraint.md)  
 Permet de restreindre les types susceptibles d’être utilisés comme arguments pour un paramètre de type dans une déclaration générique.
