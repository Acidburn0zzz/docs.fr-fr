# <a name="operator-keywords-c-reference"></a>Mots clés des opérateurs (référence C#)

Permettent d’effectuer diverses actions comme créer des objets, vérifier le type d’un objet au moment de l’exécution, obtenir la taille d’un type, etc. Cette section présente les mots clés suivants :

- [as](as.md) : convertit un objet en type compatible.

- [await](await.md) : suspend une méthode [async](async.md) jusqu’à ce qu’une tâche attendue soit terminée.

- [is](is.md) : vérifie le type de runtime d’un objet, ou (avec C# 7.0) teste une expression par rapport à un modèle.

- [new](new.md)

  - [Opérateur new](new-operator.md) : crée des objets.

  - [Modificateur new](new-modifier.md) : masque un membre hérité.

  - [Contrainte new](new-constraint.md) : qualifie un paramètre de type.

- [nameof](nameof.md) : obtient le nom de chaîne simple (non qualifié) d’une variable, d’un type ou d’un membre.

- [sizeof](sizeof.md) : obtient la taille d'un type non managé.  

- [typeof](typeof.md) : obtient l’objet <xref:System.Type?displayProperty=nameWithType> pour un type.  

- [true](true.md)  

  - [true Operator](true-false-operators.md) : retourne la valeur [bool](bool.md) `true` pour indiquer que l’opérande est true.

  - [true Literal](true-literal.md) : représente la valeur [bool](bool.md) `true`.

- [false](false.md)  

  - [false Operator](true-false-operators.md) : retourne la valeur [bool](bool.md) `true` pour indiquer que l’opérande est false.

  - [false Literal](false-literal.md) : représente la valeur [bool](bool.md) `false`.

- [stackalloc](stackalloc.md) : alloue un bloc de mémoire dans la pile.  

Les mots clés suivants, qui peuvent être utilisés comme opérateurs et instructions, sont décrits dans la section [Instructions](statement-keywords.md) :

- [checked](checked.md) : indique un contexte vérifié.  

- [unchecked](unchecked.md) : indique un contexte non vérifié.
