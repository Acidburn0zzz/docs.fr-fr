# <a name="floating-point-types-table-c-reference"></a>Tableau des types virgule flottante (référence C#)

Le tableau suivant indique la plage de valeurs approximative et la précision pour les types virgule flottante.  
  
|Type|Plage approximative|Précision|  
|----------|-----------------------|---------------|  
|[float](float.md)|±1,5 x 10<sup>−45</sup> à ±3,4 x 10<sup>38</sup>|~6-9 chiffres|  
|[double](double.md)|De ±5,0 × 10<sup>−324</sup> à ±1,7 × 10<sup>308</sup>|~15-17 chiffres|  
|[decimal](decimal.md)|±1,0 x 10<sup>-28</sup> to ±7,9228 x 10<sup>28</sup>|28 à 29 chiffres|
