```csharp

```

<span data-ttu-id="d99c6-126">Utilisez l’opérateur [typeof](typeof.md) pour obtenir l’instance de <xref:System.Type?displayProperty=nameWithType>, qui représente le type spécifié :</span><span class="sxs-lookup"><span data-stu-id="d99c6-126">Use the [typeof](typeof.md) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
// System.Double
```
