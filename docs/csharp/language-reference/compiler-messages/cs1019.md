# <a name="compiler-error-cs1019"></a><span data-ttu-id="acfc3-102">Erreur du compilateur CS1019</span><span class="sxs-lookup"><span data-stu-id="acfc3-102">Compiler Error CS1019</span></span>
<span data-ttu-id="acfc3-103">Opérateur unaire surchargeable attendu</span><span class="sxs-lookup"><span data-stu-id="acfc3-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="acfc3-104">Un élément semblable à un opérateur unaire surchargé a été déclaré, mais l’opérateur est manquant ou ne figure pas à l’emplacement approprié dans la signature.</span><span class="sxs-lookup"><span data-stu-id="acfc3-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="acfc3-105">Un *opérateur unaire* s’applique à un opérande unique.</span><span class="sxs-lookup"><span data-stu-id="acfc3-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="acfc3-106">Par exemple, `++` est un opérateur unaire.</span><span class="sxs-lookup"><span data-stu-id="acfc3-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="acfc3-107">Vous pouvez surcharger certains opérateurs unaires en utilisant le mot clé `operator` et en spécifiant un paramètre unique du type auquel l’opérateur s’applique.</span><span class="sxs-lookup"><span data-stu-id="acfc3-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="acfc3-108">Par exemple, si vous souhaitez surcharger l’opérateur `++` pour une classe définie par l’utilisateur `Temperature` de façon à pouvoir écrire `Temperature++`, vous pouvez l’indiquer de la façon suivante :</span><span class="sxs-lookup"><span data-stu-id="acfc3-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="acfc3-109">Quand vous recevez cette erreur, vous avez déclaré un élément semblable à un opérateur unaire surchargé, sauf que l’opérateur lui-même est manquant ou ne figure pas à l’emplacement approprié dans la signature.</span><span class="sxs-lookup"><span data-stu-id="acfc3-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="acfc3-110">Si vous supprimez `++` de la signature dans l’exemple précédent, vous générez l’erreur CS1019.</span><span class="sxs-lookup"><span data-stu-id="acfc3-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="acfc3-111">Le code suivant génère l’erreur CS1019 :</span><span class="sxs-lookup"><span data-stu-id="acfc3-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)     
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;   
   }  
  
   public static void Main()  
   {  
   }  
}  
```
