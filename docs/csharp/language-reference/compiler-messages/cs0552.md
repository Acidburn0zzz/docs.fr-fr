# <a name="compiler-error-cs0552"></a><span data-ttu-id="9614a-102">Erreur du compilateur CS0552</span><span class="sxs-lookup"><span data-stu-id="9614a-102">Compiler Error CS0552</span></span>
<span data-ttu-id="9614a-103">'conversion routine' : conversion définie par l’utilisateur vers/depuis l’interface</span><span class="sxs-lookup"><span data-stu-id="9614a-103">'conversion routine' : user defined conversion to/from interface</span></span>  
  
 <span data-ttu-id="9614a-104">Vous ne pouvez pas créer une conversion définie par l’utilisateur vers ou depuis une interface.</span><span class="sxs-lookup"><span data-stu-id="9614a-104">You cannot create a user-defined conversion to or from an interface.</span></span> <span data-ttu-id="9614a-105">Si vous avez besoin de la routine de conversion, corrigez cette erreur en transformant l’interface en classe ou dérivez une classe de l’interface.</span><span class="sxs-lookup"><span data-stu-id="9614a-105">If you need the conversion routine, resolve this error by making the interface a class or derive a class from the interface.</span></span>  
  
 <span data-ttu-id="9614a-106">L’exemple suivant génère l’erreur CS0552 :</span><span class="sxs-lookup"><span data-stu-id="9614a-106">The following sample generates CS0552:</span></span>  
  
```csharp  
// CS0552.cs  
public interface ii  
{  
}  
  
public class a  
{  
   // delete the routine to resolve CS0552  
   public static implicit operator ii(a aa) // CS0552  
   {  
      return new ii();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
