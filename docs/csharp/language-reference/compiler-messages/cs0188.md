# <a name="compiler-error-cs0188"></a><span data-ttu-id="83f17-102">Erreur du compilateur CS0188</span><span class="sxs-lookup"><span data-stu-id="83f17-102">Compiler Error CS0188</span></span>
<span data-ttu-id="83f17-103">L’objet 'this' ne peut pas être utilisé avant que tous ses champs ne soient assignés</span><span class="sxs-lookup"><span data-stu-id="83f17-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="83f17-104">Tous les champs d’un `struct` doivent être assignés par un constructeur avant que le constructeur ne puisse appeler une méthode dans le `struct`.</span><span class="sxs-lookup"><span data-stu-id="83f17-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="83f17-105">Si cette erreur se produit quand vous tentez d’initialiser une propriété dans un constructeur de struct, la solution est de modifier le paramètre de constructeur pour spécifier le champ de stockage au lieu de la propriété elle-même.</span><span class="sxs-lookup"><span data-stu-id="83f17-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="83f17-106">Les propriétés implémentées automatiquement doivent être évitées dans les structs parce qu’elles n’ont pas de champ de stockage et par conséquent ne peuvent pas être initialisées à partir du constructeur.</span><span class="sxs-lookup"><span data-stu-id="83f17-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>  
  
 <span data-ttu-id="83f17-107">Pour plus d’informations, consultez [Utilisation de structs](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span><span class="sxs-lookup"><span data-stu-id="83f17-107">For more information, see [Using Structs](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="83f17-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="83f17-108">Example</span></span>  
 <span data-ttu-id="83f17-109">L’exemple suivant génère l’erreur CS0188 :</span><span class="sxs-lookup"><span data-stu-id="83f17-109">The following sample generates CS0188:</span></span>  
  
```csharp  
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
