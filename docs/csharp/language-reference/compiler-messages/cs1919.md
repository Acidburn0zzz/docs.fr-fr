## <a name="example"></a><span data-ttu-id="4cd98-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="4cd98-110">Example</span></span>

 <span data-ttu-id="4cd98-111">L’exemple suivant génère l’erreur CS1919, car un type de pointeur est unsafe :</span><span class="sxs-lookup"><span data-stu-id="4cd98-111">The following example generates CS1919 because a pointer type is unsafe:</span></span>

```csharp

unsafe public class C  
{  
    public static int Main()  
    {  
        var col1 = new int* { }; // CS1919  
        var col2 = new char* { }; // CS1919  
        return 1;  
    }  
}  
```
