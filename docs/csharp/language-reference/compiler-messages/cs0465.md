<span data-ttu-id="0d90a-103">L’introduction d’une méthode 'Finalize' peut interférer avec un appel destructeur.</span><span class="sxs-lookup"><span data-stu-id="0d90a-103">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="0d90a-104">Souhaitiez-vous déclarer un destructeur ?</span><span class="sxs-lookup"><span data-stu-id="0d90a-104">Did you intend to declare a destructor?</span></span>

 <span data-ttu-id="0d90a-106">Si cette classe est utilisée comme classe de base et si la classe dérivée définit un destructeur, le destructeur se substitue à la méthode `Finalize` de la classe de base, pas à <xref:System.Object.Finalize%2A>.</span><span class="sxs-lookup"><span data-stu-id="0d90a-106">If such a class is used as a base class and if the deriving class defines a destructor, the destructor will override the base class `Finalize` method, not <xref:System.Object.Finalize%2A>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0d90a-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="0d90a-107">Example</span></span>  
 <span data-ttu-id="0d90a-108">L’exemple suivant génère l’avertissement CS0465.</span><span class="sxs-lookup"><span data-stu-id="0d90a-108">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```
