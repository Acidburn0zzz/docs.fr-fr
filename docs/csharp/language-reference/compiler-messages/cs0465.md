## <a name="example"></a><span data-ttu-id="8f806-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="8f806-107">Example</span></span>  
 <span data-ttu-id="8f806-108">L’exemple suivant génère l’avertissement CS0465.</span><span class="sxs-lookup"><span data-stu-id="8f806-108">The following sample generates CS0465.</span></span>

```csharp  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}

{  
   ~B() {}  
}  
```
