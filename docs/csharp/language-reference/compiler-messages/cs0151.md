## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="142c1-106">Exemple de conversion ambiguë</span><span class="sxs-lookup"><span data-stu-id="142c1-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="142c1-107">Cette erreur peut se produire quand il n’existe aucune conversion ou si les conversions implicites disponibles ont comme résultat une situation ambiguë.</span><span class="sxs-lookup"><span data-stu-id="142c1-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="142c1-108">L’exemple suivant génère l’erreur CS0151 :</span><span class="sxs-lookup"><span data-stu-id="142c1-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="142c1-109">Exemple de méthode void</span><span class="sxs-lookup"><span data-stu-id="142c1-109">Example of void method</span></span>

<span data-ttu-id="142c1-110">Un appel de méthode [void](../../../csharp/language-reference/keywords/void.md) dans une expression de correspondance [switch](../keywords/switch.md) génère l’erreur CS0151.</span><span class="sxs-lookup"><span data-stu-id="142c1-110">A [void](../../../csharp/language-reference/keywords/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="142c1-111">Vous pouvez la résoudre en appelant une méthode qui retourne plutôt un type intégral comme [int](../../../csharp/language-reference/keywords/int.md) ou [long](../../../csharp/language-reference/keywords/long.md).</span><span class="sxs-lookup"><span data-stu-id="142c1-111">You can fix the error by calling a method that returns an integral type such as [int](../../../csharp/language-reference/keywords/int.md) or [long](../../../csharp/language-reference/keywords/long.md) instead.</span></span>

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
