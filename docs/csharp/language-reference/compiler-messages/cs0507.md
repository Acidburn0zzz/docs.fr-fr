# <a name="compiler-error-cs0507"></a><span data-ttu-id="14325-102">Erreur du compilateur CS0507</span><span class="sxs-lookup"><span data-stu-id="14325-102">Compiler Error CS0507</span></span>
<span data-ttu-id="14325-103">'function1' : impossible de changer les modificateurs d’accès en cas de substitution du membre hérité 'access' 'function2'</span><span class="sxs-lookup"><span data-stu-id="14325-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="14325-104">Une tentative de modification de la spécification de l’accès dans une substitution de méthode a été effectuée.</span><span class="sxs-lookup"><span data-stu-id="14325-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="14325-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="14325-105">Example</span></span>  
 <span data-ttu-id="14325-106">L’exemple suivant génère l’erreur CS0507.</span><span class="sxs-lookup"><span data-stu-id="14325-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="14325-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="14325-107">Example</span></span>  
 <span data-ttu-id="14325-108">L’erreur CS0507 peut également se produire si une classe essaie de substituer une méthode marquée comme `protected internal` définie dans les métadonnées référencées.</span><span class="sxs-lookup"><span data-stu-id="14325-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="14325-109">Dans cette situation, la méthode de substitution doit être marquée comme `protected`.</span><span class="sxs-lookup"><span data-stu-id="14325-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs

abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="14325-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="14325-110">Example</span></span>  
 <span data-ttu-id="14325-111">L’exemple suivant génère l’erreur CS0507.</span><span class="sxs-lookup"><span data-stu-id="14325-111">The following sample generates CS0507.</span></span>
