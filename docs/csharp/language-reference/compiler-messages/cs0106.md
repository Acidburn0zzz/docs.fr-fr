## <a name="example"></a><span data-ttu-id="55873-115">Exemple</span><span class="sxs-lookup"><span data-stu-id="55873-115">Example</span></span>  
 <span data-ttu-id="55873-116">L’exemple suivant génère l’erreur CS0106.</span><span class="sxs-lookup"><span data-stu-id="55873-116">The following sample generates CS0106.</span></span>  
  
```csharp

{  
   interface I  
   {  
      void m();  
      static public void f();   // CS0106

   public class MyClass  
   {  
      public void I.m() {}   // CS0106  
      public static void Main() {}  
   }  
}
```
