# Compiler Error CS0001
Erreur interne du compilateur  
  
 Essayez de déterminer si l'erreur du compilateur est due à l'impossibilité d'analyser une syntaxe inattendue.  Si l'erreur persiste, veuillez prendre contact avec Microsoft.
