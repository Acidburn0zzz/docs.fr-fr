# <a name="compiler-error-cs0001"></a>Erreur du compilateur CS0001
Erreur interne du compilateur  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si l’erreur persiste, contactez Microsoft.
