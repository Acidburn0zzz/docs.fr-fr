# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="96b83-102">Avertissement du compilateur (niveau 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="96b83-102">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="96b83-103">La méthode surchargée 'méthode' qui diffère uniquement par les types de tableau sans nom n'est pas conforme CLS</span><span class="sxs-lookup"><span data-stu-id="96b83-103">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="96b83-104">Cette erreur se produit lorsqu'une méthode surchargée accepte un tableau en escalier ; la seule différence entre les signatures de méthodes réside dans le type d'élément du tableau.</span><span class="sxs-lookup"><span data-stu-id="96b83-104">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="96b83-105">Pour éviter cette erreur, utilisez un tableau rectangulaire plutôt qu'un tableau en escalier; ainsi qu'un paramètre supplémentaire pour supprimer l'ambiguïté de l'appel de fonction ; renommez une ou plusieurs méthodes surchargées ; ou encore, si la Conformité CLS n'est pas exigée, désactivez l'attribut <xref:System.CLSCompliantAttribute>.</span><span class="sxs-lookup"><span data-stu-id="96b83-105">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="96b83-106">Pour plus d’informations sur la conformité CLS, consultez [Indépendance du langage et composants indépendants du langage](https://msdn.microsoft.com/library/12a7a7h3).</span><span class="sxs-lookup"><span data-stu-id="96b83-106">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](https://msdn.microsoft.com/library/12a7a7h3).</span></span>  
  
## <a name="example"></a><span data-ttu-id="96b83-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="96b83-107">Example</span></span>  
 <span data-ttu-id="96b83-108">L'exemple suivant génère l'erreur CS3007 :</span><span class="sxs-lookup"><span data-stu-id="96b83-108">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or   
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
