# <a name="compiler-error-cs1674"></a><span data-ttu-id="f47de-102">Erreur du compilateur CS1674</span><span class="sxs-lookup"><span data-stu-id="f47de-102">Compiler Error CS1674</span></span>
<span data-ttu-id="f47de-103">'T' : le type utilisé dans une instruction using doit être convertible implicitement en 'System.IDisposable'</span><span class="sxs-lookup"><span data-stu-id="f47de-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="f47de-104">L’[instruction using](../../../csharp/language-reference/keywords/using-statement.md) doit être utilisée pour garantir la suppression d’un objet à la fin du bloc `using` ; par conséquent, seuls les types pouvant être supprimés peuvent être utilisés dans ce type d’instruction.</span><span class="sxs-lookup"><span data-stu-id="f47de-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="f47de-105">Par exemple, les types valeur ne peuvent pas être supprimés, et les paramètres de type qui ne sont pas contraints à être des classes ne peuvent pas être considérés comme pouvant être supprimés.</span><span class="sxs-lookup"><span data-stu-id="f47de-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f47de-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="f47de-106">Example</span></span>  
 <span data-ttu-id="f47de-107">L’exemple suivant génère l’erreur CS1674.</span><span class="sxs-lookup"><span data-stu-id="f47de-107">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674.cs  
class C  
{  
   public static void Main()  
   {  
      int a = 0;  
      a++;  
  
      using (a) {}   // CS1674  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="f47de-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="f47de-108">Example</span></span>  
 <span data-ttu-id="f47de-109">L’exemple suivant génère l’erreur CS1674.</span><span class="sxs-lookup"><span data-stu-id="f47de-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }

class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
