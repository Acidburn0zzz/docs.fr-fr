# <a name="compiler-error-cs0311"></a><span data-ttu-id="9c176-102">Erreur du compilateur CS0311</span><span class="sxs-lookup"><span data-stu-id="9c176-102">Compiler Error CS0311</span></span>
<span data-ttu-id="9c176-103">Impossible d’utiliser le type 'type1' comme paramètre de type 'T' dans le type ou la méthode générique '\<nom>'.</span><span class="sxs-lookup"><span data-stu-id="9c176-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="9c176-104">Il n’existe aucune conversion de référence implicite de 'type1' vers 'type2'.</span><span class="sxs-lookup"><span data-stu-id="9c176-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="9c176-105">Quand une contrainte est appliquée à un paramètre de type générique, il doit exister une conversion de référence ou d’identité implicite de l’argument concret en type de la contrainte.</span><span class="sxs-lookup"><span data-stu-id="9c176-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9c176-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="9c176-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="9c176-107">Changez l’argument que vous utilisez pour créer la classe.</span><span class="sxs-lookup"><span data-stu-id="9c176-107">Change the argument you are using to create the class.</span></span>  
  
2.  <span data-ttu-id="9c176-108">Si vous êtes propriétaire de la classe, vous pouvez supprimer la contrainte ou faire en sorte qu’il y ait une conversion d’identité ou de référence implicite.</span><span class="sxs-lookup"><span data-stu-id="9c176-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="9c176-109">Par exemple, vous pouvez faire en sorte que le deuxième type hérite du premier.</span><span class="sxs-lookup"><span data-stu-id="9c176-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9c176-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="9c176-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B{}  
class C{}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="9c176-111">Si cette erreur se produit quand vous tentez d’utiliser un argument de type valeur, notez qu’une conversion numérique implicite, par exemple de `short` en `int`, ne satisfait pas à un paramètre de type générique.</span><span class="sxs-lookup"><span data-stu-id="9c176-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
