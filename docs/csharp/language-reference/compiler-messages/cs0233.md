## <a name="example"></a><span data-ttu-id="44b17-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="44b17-107">Example</span></span>  
 <span data-ttu-id="44b17-108">L’exemple suivant génère l’erreur CS0233 :</span><span class="sxs-lookup"><span data-stu-id="44b17-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
