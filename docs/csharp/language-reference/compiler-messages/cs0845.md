# <a name="compiler-error-cs0845"></a><span data-ttu-id="3c783-102">Erreur du compilateur CS0845</span><span class="sxs-lookup"><span data-stu-id="3c783-102">Compiler Error CS0845</span></span>
<span data-ttu-id="3c783-103">Une arborescence d’expressions lambda ne peut pas contenir un opérateur de fusion avec une partie gauche de littéral null.</span><span class="sxs-lookup"><span data-stu-id="3c783-103">An expression tree lambda may not contain a coalescing operator with a null literal left-hand side.</span></span>  
  
 <span data-ttu-id="3c783-104">Null n’ayant pas de type propre, l’opérateur de fusion null ne peut pas l’utiliser.</span><span class="sxs-lookup"><span data-stu-id="3c783-104">Because null by itself does not have a type, the null coalescing operator cannot operate on it.</span></span>

1.  <span data-ttu-id="3c783-106">Effectuez un cast du littéral null en un objet.</span><span class="sxs-lookup"><span data-stu-id="3c783-106">Cast the null literal to an object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3c783-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="3c783-107">Example</span></span>  
 <span data-ttu-id="3c783-108">Le code suivant génère l’erreur CS0845 :</span><span class="sxs-lookup"><span data-stu-id="3c783-108">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
