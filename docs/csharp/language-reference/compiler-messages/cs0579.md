# <a name="compiler-error-cs0579"></a><span data-ttu-id="8dea5-102">Erreur du compilateur CS0579</span><span class="sxs-lookup"><span data-stu-id="8dea5-102">Compiler Error CS0579</span></span>
<span data-ttu-id="8dea5-103">L’attribut 'attribute' est dupliqué</span><span class="sxs-lookup"><span data-stu-id="8dea5-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="8dea5-104">Il n’est pas possible de spécifier plusieurs fois le même attribut, sauf si l’attribut spécifie **AllowMultiple=true** dans [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span><span class="sxs-lookup"><span data-stu-id="8dea5-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="8dea5-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="8dea5-105">Example</span></span>  
 <span data-ttu-id="8dea5-106">L’exemple suivant génère l’erreur CS0579.</span><span class="sxs-lookup"><span data-stu-id="8dea5-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
