# <a name="compiler-warning-level-1-cs1616"></a><span data-ttu-id="9399a-102">Avertissement du compilateur (niveau 1) CS1616</span><span class="sxs-lookup"><span data-stu-id="9399a-102">Compiler Warning (level 1) CS1616</span></span>
<span data-ttu-id="9399a-103">L’option 'option' se substitue à l’attribut 'attribute' spécifié dans un fichier source ou un module ajouté</span><span class="sxs-lookup"><span data-stu-id="9399a-103">Option 'option' overrides attribute 'attribute' given in a source file or added module</span></span>  
  
 <span data-ttu-id="9399a-104">Cet avertissement se produit si les attributs d’assembly <xref:System.Reflection.AssemblyKeyFileAttribute> ou <xref:System.Reflection.AssemblyKeyNameAttribute> détectés dans la source sont en conflit avec l’option de ligne de commande [/keyfile](../../../csharp/language-reference/compiler-options/keyfile-compiler-option.md) ou [/keycontainer](../../../csharp/language-reference/compiler-options/keycontainer-compiler-option.md), ou bien avec le nom du fichier de clé ou le conteneur de clé spécifié dans les propriétés du projet.</span><span class="sxs-lookup"><span data-stu-id="9399a-104">This warning occurs if the assembly attributes <xref:System.Reflection.AssemblyKeyFileAttribute> or <xref:System.Reflection.AssemblyKeyNameAttribute> found in source conflict with the [/keyfile](../../../csharp/language-reference/compiler-options/keyfile-compiler-option.md) or [/keycontainer](../../../csharp/language-reference/compiler-options/keycontainer-compiler-option.md) command line option or key file name or key container specified in the Project Properties.</span></span>  
  
 <span data-ttu-id="9399a-105">Pour l’exemple ci-dessous, supposons que vous ayez un fichier de clé nommé `cs1616.snk`.</span><span class="sxs-lookup"><span data-stu-id="9399a-105">For the example below, assume you have a key file named `cs1616.snk`.</span></span> <span data-ttu-id="9399a-106">Générez ce fichier avec la ligne de commande :</span><span class="sxs-lookup"><span data-stu-id="9399a-106">Generate this file with the command line:</span></span>

```console  
sn -k CS1616.snk  
```  
  
 <span data-ttu-id="9399a-107">L’exemple suivant génère l’avertissement CS1616 :</span><span class="sxs-lookup"><span data-stu-id="9399a-107">The following sample generates CS1616:</span></span>  
  
```csharp

// compile with: /keyfile:cs1616.snk

  }

```
