<span data-ttu-id="90607-103">L’option 'option' se substitue à l’attribut 'attribute' spécifié dans un fichier source ou un module ajouté</span><span class="sxs-lookup"><span data-stu-id="90607-103">Option 'option' overrides attribute 'attribute' given in a source file or added module</span></span>

 <span data-ttu-id="90607-105">Pour l’exemple ci-dessous, supposons que vous ayez un fichier de clé nommé `cs1616.snk`.</span><span class="sxs-lookup"><span data-stu-id="90607-105">For the example below, assume you have a key file named `cs1616.snk`.</span></span> <span data-ttu-id="90607-106">Générez ce fichier avec la ligne de commande :</span><span class="sxs-lookup"><span data-stu-id="90607-106">Generate this file with the command line:</span></span>
