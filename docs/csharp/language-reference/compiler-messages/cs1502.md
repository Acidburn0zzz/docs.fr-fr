# <a name="compiler-error-cs1502"></a><span data-ttu-id="b49f9-102">Erreur du compilateur CS1502</span><span class="sxs-lookup"><span data-stu-id="b49f9-102">Compiler Error CS1502</span></span>
<span data-ttu-id="b49f9-103">La méthode surchargée correspondant le mieux à 'name' possède des arguments non valides</span><span class="sxs-lookup"><span data-stu-id="b49f9-103">The best overloaded method match for 'name' has some invalid arguments</span></span>  
  
 <span data-ttu-id="b49f9-104">Cette erreur se produit quand les types d’arguments passés à la méthode ne correspondent pas aux types de paramètres de cette méthode.</span><span class="sxs-lookup"><span data-stu-id="b49f9-104">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="b49f9-105">Si la méthode appelée est surchargée, aucune des versions surchargées n’a de signature correspondant aux types d’arguments passés.</span><span class="sxs-lookup"><span data-stu-id="b49f9-105">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="b49f9-106">Pour résoudre ce problème, effectuez l’une des opérations suivantes :</span><span class="sxs-lookup"><span data-stu-id="b49f9-106">To resolve this problem, do one of the following:</span></span>  
  
-   <span data-ttu-id="b49f9-107">Vérifiez soigneusement les types des arguments passés.</span><span class="sxs-lookup"><span data-stu-id="b49f9-107">Double-check the types of the arguments being passed.</span></span> <span data-ttu-id="b49f9-108">Vérifiez qu’ils correspondent aux arguments de la méthode appelée.</span><span class="sxs-lookup"><span data-stu-id="b49f9-108">Make sure that they match the arguments of the method being called.</span></span>  
  
-   <span data-ttu-id="b49f9-109">Le cas échéant, convertissez tous les paramètres sans correspondance à l’aide de la classe <xref:System.Convert>.</span><span class="sxs-lookup"><span data-stu-id="b49f9-109">If appropriate, convert any mismatched parameters using the <xref:System.Convert> class.</span></span>  
  
-   <span data-ttu-id="b49f9-110">Le cas échéant,effectuez un cast de tout paramètre non concordant pour qu’il corresponde au type que la méthode attend.</span><span class="sxs-lookup"><span data-stu-id="b49f9-110">If appropriate, cast any mismatched parameters to match the type that the method is expecting.</span></span>  
  
-   <span data-ttu-id="b49f9-111">Le cas échéant, définissez une autre version surchargée de la méthode pour qu’elle corresponde aux types de paramètres envoyés.</span><span class="sxs-lookup"><span data-stu-id="b49f9-111">If appropriate, define another overloaded version of the method to match the parameter types that are being sent.</span></span>  
  
 <span data-ttu-id="b49f9-112">L’exemple suivant génère l’erreur CS1502 :</span><span class="sxs-lookup"><span data-stu-id="b49f9-112">The following sample generates CS1502:</span></span>  
  
```csharp

{  
   public class a  
   {  
      public a(char i)  
      // try the following constructor instead  
      // public a(int i)  
      {  
      }  
  
      public static void Main()  
      {  
         a aa = new a(2222);   // CS1502  
      }  
   }  
}  
```
