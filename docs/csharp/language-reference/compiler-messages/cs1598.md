# <a name="compiler-warning-level-1-cs1598"></a>Avertissement du compilateur (niveau 1) CS1598
L’analyseur XML n’a pas pu être chargé pour la raison suivante : 'reason'. Le fichier de documentation XML 'file' ne sera pas généré.  
  
 L’option [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) a été spécifiée, mais le compilateur n’a pas pu trouver et charger msxml3.dll. Vérifiez que le fichier msxml3.dll est installé et inscrit.
