 <span data-ttu-id="d5107-104">L’attribut [Conditional](http://msdn.microsoft.com/en-us/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) ne peut pas être utilisé sur l’implémentation d’une interface.</span><span class="sxs-lookup"><span data-stu-id="d5107-104">The [Conditional](http://msdn.microsoft.com/en-us/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="d5107-105">L’exemple suivant génère l’erreur CS0629 :</span><span class="sxs-lookup"><span data-stu-id="d5107-105">The following sample generates CS0629:</span></span>  
  
```  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
