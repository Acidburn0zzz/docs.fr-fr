# <a name="compiler-error-cs0538"></a><span data-ttu-id="cabe4-102">Erreur du compilateur CS0538</span><span class="sxs-lookup"><span data-stu-id="cabe4-102">Compiler Error CS0538</span></span>
<span data-ttu-id="cabe4-103">'nom' dans une déclaration d’interface explicite n’est pas une interface</span><span class="sxs-lookup"><span data-stu-id="cabe4-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="cabe4-104">Une tentative a été effectuée pour déclarer explicitement une [interface](../../csharp/language-reference/keywords/interface.md), mais aucune interface n’a été spécifiée.</span><span class="sxs-lookup"><span data-stu-id="cabe4-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="cabe4-105">L’exemple suivant génère l’erreur CS0538 :</span><span class="sxs-lookup"><span data-stu-id="cabe4-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
