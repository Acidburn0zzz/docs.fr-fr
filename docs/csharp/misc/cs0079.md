# <a name="compiler-error-cs0079"></a><span data-ttu-id="26144-102">Erreur du compilateur CS0079</span><span class="sxs-lookup"><span data-stu-id="26144-102">Compiler Error CS0079</span></span>
<span data-ttu-id="26144-103">L’événement 'event' peut apparaître uniquement à gauche de += ou -=</span><span class="sxs-lookup"><span data-stu-id="26144-103">The event 'event' can only appear on the left hand side of += or -=</span></span>  
  
 <span data-ttu-id="26144-104">Un [événement](../../csharp/language-reference/keywords/event.md) a été appelé de façon incorrecte.</span><span class="sxs-lookup"><span data-stu-id="26144-104">An [event](../../csharp/language-reference/keywords/event.md) was called incorrectly.</span></span> <span data-ttu-id="26144-105">Pour plus d’informations, consultez [événements](../../csharp/programming-guide/events/index.md) et [délégués](../../csharp/programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="26144-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="26144-106">L’exemple suivant génère l’erreur CS0079 :</span><span class="sxs-lookup"><span data-stu-id="26144-106">The following sample generates CS0079:</span></span>  
  
```csharp  
// CS0079.cs  
using System;  
  
public delegate void MyEventHandler();  
  
public class Class1  
{  
   private MyEventHandler _e;  
  
   public event MyEventHandler Pow  
   {  
      add  
      {  
         _e += value;  
         Console.WriteLine("in add accessor");  
      }  
      remove  
      {  
         _e -= value;  
         Console.WriteLine("in remove accessor");  
      }  
   }  
  
   public void Handler()  
   {  
   }  
  
   public void Fire()  
   {  
      if (_e != null)  
      {  
         Pow();   // CS0079  
         // try the following line instead  
         // _e();  
      }  
   }  
  
   public static void Main()  
   {  
      Class1 p = new Class1();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p._e -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
      p.Pow -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
   }  
}  
```
