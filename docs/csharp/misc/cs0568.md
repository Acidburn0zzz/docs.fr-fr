 <span data-ttu-id="86fd7-104">Chaque [struct](../../csharp/language-reference/keywords/struct.md) a déjà un constructeur par défaut qui initialise l’objet à zéro.</span><span class="sxs-lookup"><span data-stu-id="86fd7-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="86fd7-105">Les constructeurs que vous pouvez créer pour un struct doivent donc accepter un ou plusieurs paramètres.</span><span class="sxs-lookup"><span data-stu-id="86fd7-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>

```  
   // }

{  
   public static void Main()  
   {  
   }  
}  
```
