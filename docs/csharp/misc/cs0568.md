# <a name="compiler-error-cs0568"></a><span data-ttu-id="8e84f-102">Erreur du compilateur CS0568</span><span class="sxs-lookup"><span data-stu-id="8e84f-102">Compiler Error CS0568</span></span>
<span data-ttu-id="8e84f-103">Les structures ne peuvent pas contenir de constructeurs exempts de paramètres explicites</span><span class="sxs-lookup"><span data-stu-id="8e84f-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="8e84f-104">Chaque [struct](../../csharp/language-reference/keywords/struct.md) a déjà un constructeur par défaut qui initialise l’objet à zéro.</span><span class="sxs-lookup"><span data-stu-id="8e84f-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="8e84f-105">Les constructeurs que vous pouvez créer pour un struct doivent donc accepter un ou plusieurs paramètres.</span><span class="sxs-lookup"><span data-stu-id="8e84f-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="8e84f-106">L’exemple suivant génère l’erreur CS0568 :</span><span class="sxs-lookup"><span data-stu-id="8e84f-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
