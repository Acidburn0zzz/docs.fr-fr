# Avertissement du compilateur (niveau&#160;1) CS1592
Code XML incorrect dans le fichier de commentaires inclus \-\- 'reason'  
  
 Un problème, signalé par ***raison***, a été trouvé dans le fichier spécifié par la balise [\<include\>](../../csharp/programming-guide/xmldoc/include.md).
