## <a name="example"></a><span data-ttu-id="1e45b-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="1e45b-107">Example</span></span>  
 <span data-ttu-id="1e45b-108">L’exemple suivant génère l’erreur CS0754 :</span><span class="sxs-lookup"><span data-stu-id="1e45b-108">The following code generates CS0754:</span></span>

    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF

        partial void IF.Part(); //CS0754

        {  
            return 1;  
        }  
    }
