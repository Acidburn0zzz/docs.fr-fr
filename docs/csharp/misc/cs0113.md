# <a name="compiler-error-cs0113"></a><span data-ttu-id="b4a36-102">Erreur du compilateur CS0113</span><span class="sxs-lookup"><span data-stu-id="b4a36-102">Compiler Error CS0113</span></span>
<span data-ttu-id="b4a36-103">Un membre 'fonction' marqué comme override ne peut pas être marqué comme new ni virtual</span><span class="sxs-lookup"><span data-stu-id="b4a36-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="b4a36-104">Marquer une méthode avec les mots clés [new](../../csharp/language-reference/keywords/new.md) et [override](../../csharp/language-reference/keywords/override.md) sont des opérations qui s’excluent mutuellement.</span><span class="sxs-lookup"><span data-stu-id="b4a36-104">It is mutually exclusive to mark a method with the [new](../../csharp/language-reference/keywords/new.md) and [override](../../csharp/language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="b4a36-105">L’exemple suivant génère l’erreur CS0113 :</span><span class="sxs-lookup"><span data-stu-id="b4a36-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
