# <a name="compiler-error-cs0555"></a><span data-ttu-id="11571-102">Erreur du compilateur CS0555</span><span class="sxs-lookup"><span data-stu-id="11571-102">Compiler Error CS0555</span></span>

<span data-ttu-id="11571-103">L'opérateur défini par l'utilisateur ne peut pas prendre un objet du type englobant et le convertir en un objet du type englobant</span><span class="sxs-lookup"><span data-stu-id="11571-103">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>

<span data-ttu-id="11571-104">Les conversions définies par l’utilisateur dans des valeurs de la classe englobante ne sont pas autorisées. Ce type d’opérateur n’est pas nécessaire.</span><span class="sxs-lookup"><span data-stu-id="11571-104">User-defined conversions to values of the enclosing class are not allowed; you do not need such an operator.</span></span>

<span data-ttu-id="11571-105">L’exemple suivant génère l’erreur CS0555 :</span><span class="sxs-lookup"><span data-stu-id="11571-105">The following sample generates CS0555:</span></span>

```csharp
// CS0555.cs
public class MyClass
{
   // delete the following operator to resolve this CS0555
   public static implicit operator MyClass(MyClass aa)   // CS0555
   {
      return new MyClass();
   }

   public static void Main()
   {
   }
}
```
