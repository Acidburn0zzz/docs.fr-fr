# <a name="compiler-error-cs0448"></a><span data-ttu-id="904cb-102">Erreur du compilateur CS0448</span><span class="sxs-lookup"><span data-stu-id="904cb-102">Compiler Error CS0448</span></span>
<span data-ttu-id="904cb-103">Le type de retour pour l'opérateur ++ ou -- doit être le type conteneur ou dérivé du type conteneur</span><span class="sxs-lookup"><span data-stu-id="904cb-103">The return type for ++ or -- operator must be the containing type or derived from the containing type</span></span>  
  
 <span data-ttu-id="904cb-104">Lorsque vous remplacez les opérateurs `++` ou `--` , ils doivent retourner le même type que le type conteneur, ou retourner un type dérivé du type conteneur.</span><span class="sxs-lookup"><span data-stu-id="904cb-104">When you override the `++` or `--` operators, they must return the same type as the containing type, or return a type that is derived from the containing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="904cb-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="904cb-105">Example</span></span>  
 <span data-ttu-id="904cb-106">L’exemple suivant génère l’erreur CS0448.</span><span class="sxs-lookup"><span data-stu-id="904cb-106">The following sample generates CS0448.</span></span>  
  
```  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```

 <span data-ttu-id="904cb-108">L’exemple suivant génère l’erreur CS0448.</span><span class="sxs-lookup"><span data-stu-id="904cb-108">The following sample generates CS0448.</span></span>  
  
```  
// CS0448_b.cs

{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448

public struct T

   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
