<span data-ttu-id="017e4-103">Bloc switch vide</span><span class="sxs-lookup"><span data-stu-id="017e4-103">Empty switch block</span></span>  
  
 <span data-ttu-id="017e4-104">Le compilateur a détecté un bloc [switch](../../csharp/language-reference/keywords/switch.md) sans aucune instruction **case** ou **default** .</span><span class="sxs-lookup"><span data-stu-id="017e4-104">The compiler detected a [switch](../../csharp/language-reference/keywords/switch.md) block with no **case** or **default** statement.</span></span> <span data-ttu-id="017e4-105">Or, un bloc `switch` doit avoir une ou plusieurs instructions **case** ou **default** .</span><span class="sxs-lookup"><span data-stu-id="017e4-105">A `switch` block must have one or more **case** or **default** statements.</span></span>  
  
 <span data-ttu-id="017e4-106">L’exemple suivant génère l’erreur CS1522 :</span><span class="sxs-lookup"><span data-stu-id="017e4-106">The following sample generates CS1522:</span></span>  
  
```csharp  
// CS1522.cs  
// compile with: /W:1

class x  
{  
   public static void Main()  
   {  
      int i = 6;  
  
      switch(i)   // CS1522  
      {  
         // add something to the switch block, for example:  
         /*  
         case (5):  
            Console.WriteLine("5");  
            return;  
         default:  
            Console.WriteLine("not 5");  
            return;  
         */  
      }  
   }  
}  
```
