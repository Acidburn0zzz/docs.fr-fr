 <span data-ttu-id="5bd61-104">Quand vous utilisez `goto case`, une conversion implicite de la valeur du goto case vers le type du commutateur doit avoir lieu.</span><span class="sxs-lookup"><span data-stu-id="5bd61-104">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5bd61-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="5bd61-105">Example</span></span>  
 <span data-ttu-id="5bd61-106">L’exemple suivant génère l’avertissement CS0469.</span><span class="sxs-lookup"><span data-stu-id="5bd61-106">The following sample generates CS0469.</span></span>  
  
```  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:   
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
