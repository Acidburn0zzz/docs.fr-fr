# <a name="compiler-error-cs0835"></a><span data-ttu-id="49097-102">Erreur du compilateur CS0835</span><span class="sxs-lookup"><span data-stu-id="49097-102">Compiler Error CS0835</span></span>
<span data-ttu-id="49097-103">Impossible de convertir une expression lambda en une arborescence d’expression dont l’argument de type 'type' n’est pas un type délégué.</span><span class="sxs-lookup"><span data-stu-id="49097-103">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="49097-104">Si une expression lambda est convertie en une arborescence d’expression, elle doit disposer d’un type délégué pour son argument.</span><span class="sxs-lookup"><span data-stu-id="49097-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="49097-105">De plus, l’expression lambda doit pouvoir être convertie en type délégué.</span><span class="sxs-lookup"><span data-stu-id="49097-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="49097-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="49097-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="49097-107">Remplacez le paramètre de type `int` par un type délégué, par exemple `Func<int,int>`.</span><span class="sxs-lookup"><span data-stu-id="49097-107">Change the type parameter from `int` to a delegate type, for example `Func<int,int>`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="49097-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="49097-108">Example</span></span>  
 <span data-ttu-id="49097-109">L’exemple suivant génère l’erreur CS0835 :</span><span class="sxs-lookup"><span data-stu-id="49097-109">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
