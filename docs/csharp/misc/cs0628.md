<span data-ttu-id="eab32-103">'membre' : nouveau membre protected déclaré dans une classe sealed</span><span class="sxs-lookup"><span data-stu-id="eab32-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="eab32-104">Une classe [sealed](../../csharp/language-reference/keywords/sealed.md) ne peut pas introduire un membre [protected](../../csharp/language-reference/keywords/protected.md) , car aucune autre classe ne pourra hériter de la classe `sealed` et utiliser le membre `protected` .</span><span class="sxs-lookup"><span data-stu-id="eab32-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="eab32-105">L’exemple suivant génère l’erreur CS0628 :</span><span class="sxs-lookup"><span data-stu-id="eab32-105">The following sample generates CS0628:</span></span>  
  
```csharp

{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
