# <a name="compiler-error-cs1623"></a><span data-ttu-id="8324c-102">Erreur du compilateur CS1623</span><span class="sxs-lookup"><span data-stu-id="8324c-102">Compiler Error CS1623</span></span>
<span data-ttu-id="8324c-103">Les itérateurs ne peuvent pas avoir de paramètres out ou ref</span><span class="sxs-lookup"><span data-stu-id="8324c-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="8324c-104">Cette erreur se produit si une méthode iterator prend un `in`, `ref`, ou `out` paramètre.</span><span class="sxs-lookup"><span data-stu-id="8324c-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="8324c-105">Pour éviter cette erreur, supprimez le `in`, `ref`, ou `out` (mot clé) à partir de la signature de méthode.</span><span class="sxs-lookup"><span data-stu-id="8324c-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8324c-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="8324c-106">Example</span></span>  
 <span data-ttu-id="8324c-107">L’exemple suivant génère l’erreur CS1623 :</span><span class="sxs-lookup"><span data-stu-id="8324c-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
