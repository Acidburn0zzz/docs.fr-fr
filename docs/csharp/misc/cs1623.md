# <a name="compiler-error-cs1623"></a><span data-ttu-id="dbcdc-102">Erreur du compilateur CS1623</span><span class="sxs-lookup"><span data-stu-id="dbcdc-102">Compiler Error CS1623</span></span>

 <span data-ttu-id="dbcdc-104">Cette erreur se produit si une méthode Iterator prend un paramètre `ref` ou `out` .</span><span class="sxs-lookup"><span data-stu-id="dbcdc-104">This error occurs if an iterator method takes a `ref` or `out` parameter.</span></span> <span data-ttu-id="dbcdc-105">Pour éviter cette erreur, supprimez le mot clé `ref` ou `out` de la signature de la méthode.</span><span class="sxs-lookup"><span data-stu-id="dbcdc-105">To avoid this error, remove the `ref` or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dbcdc-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="dbcdc-106">Example</span></span>  
 <span data-ttu-id="dbcdc-107">L’exemple suivant génère l’erreur CS1623 :</span><span class="sxs-lookup"><span data-stu-id="dbcdc-107">The following sample generates CS1623:</span></span>  
  
```  
// CS1623.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {  
        yield return i;  
    }  
  
    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {  
        f = 0.0F;  
        yield return f;  
    }  
}  
```
