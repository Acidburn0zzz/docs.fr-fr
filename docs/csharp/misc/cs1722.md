## <a name="example"></a><span data-ttu-id="714d4-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="714d4-105">Example</span></span>  
 <span data-ttu-id="714d4-106">L’exemple suivant génère l’erreur CS1722.</span><span class="sxs-lookup"><span data-stu-id="714d4-106">The following sample generates CS1722.</span></span>
