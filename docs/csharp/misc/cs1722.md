## <a name="example"></a><span data-ttu-id="5edd5-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="5edd5-105">Example</span></span>  
 <span data-ttu-id="5edd5-106">L’exemple suivant génère l’erreur CS1722.</span><span class="sxs-lookup"><span data-stu-id="5edd5-106">The following sample generates CS1722.</span></span>  
  
```csharp

public class A {}  
interface I {}  
  
public class MyClass : I, A {}   // CS1722
```
