# <a name="compiler-error-cs0155"></a><span data-ttu-id="02dcf-102">Erreur du compilateur CS0155</span><span class="sxs-lookup"><span data-stu-id="02dcf-102">Compiler Error CS0155</span></span>
<span data-ttu-id="02dcf-103">Le type intercepté ou levé doit être dérivé de System.Exception</span><span class="sxs-lookup"><span data-stu-id="02dcf-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="02dcf-104">Une tentative a été effectuée pour passer un type de données qui ne dérive pas de **System.Exception** dans un [catch](../../csharp/language-reference/keywords/try-catch.md) bloc.</span><span class="sxs-lookup"><span data-stu-id="02dcf-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="02dcf-105">Seuls les types de données qui dérivent de **System.Exception** peuvent être passés à un bloc **catch** .</span><span class="sxs-lookup"><span data-stu-id="02dcf-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="02dcf-106">Pour plus d’informations, consultez [instructions de gestion des exceptions](../../csharp/language-reference/keywords/exception-handling-statements.md) et [les Exceptions et la gestion des exceptions](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="02dcf-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="02dcf-107">L’exemple suivant génère l’erreur CS0155 :</span><span class="sxs-lookup"><span data-stu-id="02dcf-107">The following sample generates CS0155:</span></span>  
  
```  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
