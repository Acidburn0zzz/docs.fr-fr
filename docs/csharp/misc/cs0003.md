# <a name="compiler-error-cs0003"></a>Erreur du compilateur CS0003
Mémoire insuffisante  
  
 Le compilateur n’a pas pu allouer suffisamment de mémoire virtuelle pour terminer la compilation. Fermez toutes les applications dont vous n’avez pas besoin, puis recompilez.  
  
 Vous pouvez également augmenter la taille du fichier d’échange et vous assurer que vous disposez d’espace disque disponible.  
  
 Cette erreur peut également apparaître à la suite d’avoir versions incompatibles de le [!INCLUDE[winsdklong](../../csharp/language-reference/compiler-options/includes/winsdklong_md.md)] et le compilateur c#, ou un ou plusieurs fichiers endommagés qui prennent en charge le compilateur c#, réinstallez Visual Studio.
