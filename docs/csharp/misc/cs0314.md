## <a name="example"></a><span data-ttu-id="6c583-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="6c583-108">Example</span></span>  
 <span data-ttu-id="6c583-109">Le code suivant génère l’erreur CS0314 :</span><span class="sxs-lookup"><span data-stu-id="6c583-109">The following code generates CS0314:</span></span>  
  
```  
// cs0314.cs  
// Compile with: /target:library

public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.

{ }  
```
