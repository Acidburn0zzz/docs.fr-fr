## <a name="example"></a><span data-ttu-id="6ec3a-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="6ec3a-107">Example</span></span>

```csharp

{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411

    }

```
