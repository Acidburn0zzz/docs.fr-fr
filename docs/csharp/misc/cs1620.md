# <a name="compiler-error-cs1620"></a><span data-ttu-id="858ba-102">Erreur du compilateur CS1620</span><span class="sxs-lookup"><span data-stu-id="858ba-102">Compiler Error CS1620</span></span>
<span data-ttu-id="858ba-103">L’argument 'nombre' doit être passé avec le mot clé 'mot_clé'</span><span class="sxs-lookup"><span data-stu-id="858ba-103">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="858ba-104">Cette erreur se produit si vous passez un argument à une fonction qui accepte un [ref](../../csharp/language-reference/keywords/ref.md) ou [hors](../../csharp/language-reference/keywords/out-parameter-modifier.md) paramètre et que vous n’incluez pas le `ref` ou `out` au point d’appel, ou que vous incluez le mot clé incorrect.</span><span class="sxs-lookup"><span data-stu-id="858ba-104">This error occurs if you are passing an argument to a function that takes a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="858ba-105">Le texte d’erreur indique le mot clé approprié à utiliser et l’argument qui a provoqué l’échec.</span><span class="sxs-lookup"><span data-stu-id="858ba-105">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="858ba-106">L’exemple suivant génère l’avertissement CS1620 :</span><span class="sxs-lookup"><span data-stu-id="858ba-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
