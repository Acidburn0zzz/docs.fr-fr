# <a name="compiler-error-cs1650"></a><span data-ttu-id="65e1c-102">Erreur du compilateur CS1650</span><span class="sxs-lookup"><span data-stu-id="65e1c-102">Compiler Error CS1650</span></span>
<span data-ttu-id="65e1c-103">Impossible d’assigner les champs du champ readonly statique 'identifier' (sauf s’ils appartiennent à un constructeur statique ou un initialiseur de variable)</span><span class="sxs-lookup"><span data-stu-id="65e1c-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="65e1c-104">Cette erreur se produit quand vous tentez de modifier un membre d’un champ readonly et static qui n’autorise pas les modifications.</span><span class="sxs-lookup"><span data-stu-id="65e1c-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="65e1c-105">Pour résoudre cette erreur, limitez les assignations aux champs readonly au constructeur ou à l’initialiseur de variable, ou supprimez le mot clé `readonly` de la déclaration du champ.</span><span class="sxs-lookup"><span data-stu-id="65e1c-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
