 <span data-ttu-id="45bbd-104">Cette erreur se produit quand une variable est passée en tant qu’argument [ref](../../csharp/language-reference/keywords/ref.md) ou [out](../../csharp/language-reference/keywords/out.md) dans un contexte dans lequel cette variable est en lecture seule.</span><span class="sxs-lookup"><span data-stu-id="45bbd-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="45bbd-105">Les contextes en lecture seule incluent les variables d’itération [foreach](../../csharp/language-reference/keywords/foreach-in.md) , les variables [using](../../csharp/language-reference/keywords/using-statement.md) et les variables `fixed` .</span><span class="sxs-lookup"><span data-stu-id="45bbd-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="45bbd-106">Pour résoudre cette erreur, n’appelez pas les fonctions qui acceptent la variable `foreach`, `using` ou `fixed` en tant que paramètre `ref` ou `out` dans des blocs `using` , des instructions `foreach` et des instructions `fixed` .</span><span class="sxs-lookup"><span data-stu-id="45bbd-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="45bbd-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="45bbd-107">Example</span></span>  
 <span data-ttu-id="45bbd-108">L’exemple suivant génère l’erreur CS1657 :</span><span class="sxs-lookup"><span data-stu-id="45bbd-108">The following sample generates CS1657:</span></span>  
  
```  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
