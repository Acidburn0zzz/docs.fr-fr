 <span data-ttu-id="c0c5a-104">Lorsque vous déclarez une [propriété](../../csharp/programming-guide/classes-and-structs/using-properties.md), vous devez également déclarer ses méthodes d’accesseur.</span><span class="sxs-lookup"><span data-stu-id="c0c5a-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="c0c5a-105">Toutefois, une propriété ne peut pas avoir plusieurs méthodes d’accesseur `get` ni plusieurs méthodes d’accesseur `set` .</span><span class="sxs-lookup"><span data-stu-id="c0c5a-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c0c5a-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="c0c5a-106">Example</span></span>

```csharp  
        get

            return 0;

    }

}  
```
