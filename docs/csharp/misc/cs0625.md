# <a name="compiler-error-cs0625"></a><span data-ttu-id="1c37a-102">Erreur du compilateur CS0625</span><span class="sxs-lookup"><span data-stu-id="1c37a-102">Compiler Error CS0625</span></span>
<span data-ttu-id="1c37a-103">'field' : types de champ d’instance marqués avec StructLayout (LayoutKind.Explicit) doivent posséder un attribut FieldOffset.</span><span class="sxs-lookup"><span data-stu-id="1c37a-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="1c37a-104">Quand un struct est marqué avec un attribut **StructLayout** explicite, tous les champs du struct doivent avoir l’attribut [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) .</span><span class="sxs-lookup"><span data-stu-id="1c37a-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="1c37a-105">Pour plus d’informations, consultez [StructLayoutAttribute, classe](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="1c37a-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span> 

<span data-ttu-id="1c37a-106">L’exemple suivant génère l’erreur CS0625 :</span><span class="sxs-lookup"><span data-stu-id="1c37a-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
