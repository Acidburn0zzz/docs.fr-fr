# <a name="compiler-error-cs0077"></a><span data-ttu-id="c8c7b-102">Erreur du compilateur CS0077</span><span class="sxs-lookup"><span data-stu-id="c8c7b-102">Compiler Error CS0077</span></span>
<span data-ttu-id="c8c7b-103">L’opérateur as doit être utilisé avec un type référence ou un type Nullable ('int' est un type valeur qui n’autorise pas les valeurs null).</span><span class="sxs-lookup"><span data-stu-id="c8c7b-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="c8c7b-104">Un [type valeur](../../csharp/language-reference/keywords/as.md) a été passé à l’opérateur [as](../../csharp/language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="c8c7b-104">The [as](../../csharp/language-reference/keywords/as.md) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="c8c7b-105">Comme `as` peut retourner [null](../../csharp/language-reference/keywords/null.md), seuls des [types référence](../../csharp/language-reference/keywords/reference-types.md) ou un type Nullable peuvent lui être passés.</span><span class="sxs-lookup"><span data-stu-id="c8c7b-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="c8c7b-106">Pour plus d’informations sur les types nullable, consultez [Types Nullable](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="c8c7b-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="c8c7b-107">L’exemple suivant génère l’erreur CS0077 :</span><span class="sxs-lookup"><span data-stu-id="c8c7b-107">The following sample generates CS0077:</span></span>  
  
```  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
