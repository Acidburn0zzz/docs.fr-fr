<span data-ttu-id="1e614-103">Impossible d’utiliser au niveau de ref ou out ' paramètre ' à l’intérieur d’une méthode anonyme, une expression lambda ou une expression de requête</span><span class="sxs-lookup"><span data-stu-id="1e614-103">Cannot use in ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="1e614-104">Cette erreur se produit si vous utilisez un `in`, `ref`, ou `out` paramètre à l’intérieur d’un bloc de méthode anonyme.</span><span class="sxs-lookup"><span data-stu-id="1e614-104">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="1e614-105">Pour éviter cette erreur, utilisez une variable locale ou une autre construction.</span><span class="sxs-lookup"><span data-stu-id="1e614-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="1e614-106">L’exemple suivant génère l’erreur CS1628 :</span><span class="sxs-lookup"><span data-stu-id="1e614-106">The following sample generates CS1628:</span></span>

```csharp  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
