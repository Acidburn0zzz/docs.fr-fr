# <a name="compiler-error-cs1621"></a><span data-ttu-id="bd5a6-102">Erreur du compilateur CS1621</span><span class="sxs-lookup"><span data-stu-id="bd5a6-102">Compiler Error CS1621</span></span>
<span data-ttu-id="bd5a6-103">L'instruction yield ne peut pas être utilisée dans une méthode anonyme ou une expression lambda</span><span class="sxs-lookup"><span data-stu-id="bd5a6-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="bd5a6-104">L’instruction yield ne peut pas figurer dans un bloc de méthode anonyme dans un itérateur.</span><span class="sxs-lookup"><span data-stu-id="bd5a6-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bd5a6-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="bd5a6-105">Example</span></span>  
 <span data-ttu-id="bd5a6-106">L’exemple suivant génère l’erreur CS1621 :</span><span class="sxs-lookup"><span data-stu-id="bd5a6-106">The following sample generates CS1621:</span></span>  
  
```  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
