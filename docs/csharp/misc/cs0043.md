# <a name="compiler-error-cs0043"></a>Erreur du compilateur CS0043
Le format du fichier PDB 'fichier' est incorrect ou obsolète. Supprimez-le et relancez une génération.  
  
 Supprimez le fichier .pdb pour cette compilation et recompilez.
