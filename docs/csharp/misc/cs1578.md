 <span data-ttu-id="0cd46-104">Après une directive [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) , seul un nom de fichier (entre guillemets doubles) ou un commentaire sur une ligne est autorisé.</span><span class="sxs-lookup"><span data-stu-id="0cd46-104">After a [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="0cd46-105">L’exemple suivant génère l’erreur CS1578 :</span><span class="sxs-lookup"><span data-stu-id="0cd46-105">The following sample generates CS1578:</span></span>

```csharp  
   {  
      #line 101 abc.cs   // CS1578

   }  
}  
```
