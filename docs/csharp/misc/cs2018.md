# Erreur du compilateur CS2018
Fichier de messages 'cscmsgs.dll' introuvable  
  
 Le fichier .dll qui contient les messages d’erreurs et d’avertissement du compilateur est introuvable. Ce fichier doit être présent dans le même répertoire que les autres fichiers de prise en charge du compilateur.
