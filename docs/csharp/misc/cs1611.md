# <a name="compiler-error-cs1611"></a><span data-ttu-id="52799-102">Erreur du compilateur CS1611</span><span class="sxs-lookup"><span data-stu-id="52799-102">Compiler Error CS1611</span></span>
<span data-ttu-id="52799-103">Le paramètres params ne peut pas être déclaré comme ref ou out</span><span class="sxs-lookup"><span data-stu-id="52799-103">The params parameter cannot be declared as ref or out</span></span>  
  
 <span data-ttu-id="52799-104">Vous ne pouvez pas utiliser les mots clés [ref](../../csharp/language-reference/keywords/ref.md) ou [out](../../csharp/language-reference/keywords/out.md) avec le mot clé [params](../../csharp/language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="52799-104">The keywords [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) cannot be used with the [params](../../csharp/language-reference/keywords/params.md) keyword.</span></span>  
  
 <span data-ttu-id="52799-105">L’exemple suivant génère l’erreur CS1611 :</span><span class="sxs-lookup"><span data-stu-id="52799-105">The following sample generates CS1611:</span></span>  
  
```  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
