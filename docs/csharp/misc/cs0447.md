# <a name="compiler-error-cs0447"></a><span data-ttu-id="ff573-102">Erreur du compilateur CS0447</span><span class="sxs-lookup"><span data-stu-id="ff573-102">Compiler Error CS0447</span></span>
<span data-ttu-id="ff573-103">Les attributs ne peuvent pas être utilisés sur des arguments de type, uniquement sur des paramètres de type</span><span class="sxs-lookup"><span data-stu-id="ff573-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="ff573-104">Cette erreur se produit quand vous appliquez un attribut à un argument de type qui se trouve dans une instruction d’appel.</span><span class="sxs-lookup"><span data-stu-id="ff573-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="ff573-105">Vous pouvez appliquer un attribut à un paramètre de type dans une instruction de déclaration de classe ou de méthode, telle que la suivante :</span><span class="sxs-lookup"><span data-stu-id="ff573-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

 <span data-ttu-id="ff573-106">La ligne de code suivante génère cette erreur.</span><span class="sxs-lookup"><span data-stu-id="ff573-106">The following line of code will generate this error.</span></span> <span data-ttu-id="ff573-107">Il est supposé que la classe `C`, qui est définie dans la ligne de code précédente, possède une méthode statique nommée `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="ff573-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

