## <a name="example"></a><span data-ttu-id="bce67-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="bce67-106">Example</span></span>  
 <span data-ttu-id="bce67-107">L’exemple suivant génère l’erreur CS1642.</span><span class="sxs-lookup"><span data-stu-id="bce67-107">The following sample generates CS1642.</span></span>  
  
```  
// CS1642.cs

unsafe class C  
{  
   fixed int a[10];   // CS1642  
}

{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
