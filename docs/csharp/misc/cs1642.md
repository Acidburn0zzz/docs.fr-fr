## <a name="example"></a><span data-ttu-id="82cb4-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="82cb4-106">Example</span></span>  
 <span data-ttu-id="82cb4-107">L’exemple suivant génère l’erreur CS1642.</span><span class="sxs-lookup"><span data-stu-id="82cb4-107">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
