 <span data-ttu-id="6e7a6-104">Une variable [readonly](../../csharp/language-reference/keywords/readonly.md) doit avoir la même utilisation de [static](../../csharp/language-reference/keywords/static.md) que le constructeur dans lequel vous voulez l’initialiser.</span><span class="sxs-lookup"><span data-stu-id="6e7a6-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="6e7a6-105">Pour plus d’informations, consultez [Constructeurs statiques](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="6e7a6-105">For more information, see [Static Constructors](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span></span>

   public static readonly int TestInt = 6;
