# <a name="compiler-error-cs1715"></a><span data-ttu-id="9565a-102">Erreur du compilateur CS1715</span><span class="sxs-lookup"><span data-stu-id="9565a-102">Compiler Error CS1715</span></span>
<span data-ttu-id="9565a-103">'Type1': le type doit être 'Type2' pour correspondre au membre substitué 'nom_membre'</span><span class="sxs-lookup"><span data-stu-id="9565a-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="9565a-104">Cette erreur est identique à [Compiler Error CS0508](../../csharp/misc/cs0508.md), sauf que CS0508 s’applique maintenant uniquement aux méthodes qui ont des types de retour, alors que l’erreur CS1715 s’applique aux propriétés et aux indexeurs qui ont uniquement des « types » plutôt que des « types de retour ».</span><span class="sxs-lookup"><span data-stu-id="9565a-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9565a-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="9565a-105">Example</span></span>  
 <span data-ttu-id="9565a-106">Le code suivant génère l’erreur CS1715.</span><span class="sxs-lookup"><span data-stu-id="9565a-106">The following code generates CS1715.</span></span>  
  
```  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
