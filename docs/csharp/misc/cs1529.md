 <span data-ttu-id="1da76-106">L’exemple suivant génère l’erreur CS1529 :</span><span class="sxs-lookup"><span data-stu-id="1da76-106">The following sample generates CS1529:</span></span>  
  
```csharp

{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass

}

```
