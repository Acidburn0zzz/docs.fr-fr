 <span data-ttu-id="91aa8-106">L’exemple suivant génère l’erreur CS1529 :</span><span class="sxs-lookup"><span data-stu-id="91aa8-106">The following sample generates CS1529:</span></span>

```  
    {  
        using Microsoft;

```
