# Erreur du compilateur CS1038
directive \#endregion attendue  
  
 Une directive [\#region](../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) n’avait pas de directive [\#endregion](../../csharp/language-reference/preprocessor-directives/preprocessor-endregion.md) correspondante.  
  
 L’exemple suivant génère l’erreur CS1038 :  
  
```  
// CS1038.cs #region testing public class clx { public static void Main() { } } // CS1038 // uncomment the next line to resolve // #endregion  
```
