# <a name="compiler-error-cs1601"></a><span data-ttu-id="31e13-102">Erreur du compilateur CS1601</span><span class="sxs-lookup"><span data-stu-id="31e13-102">Compiler Error CS1601</span></span>
<span data-ttu-id="31e13-103">Le paramètre de la méthode ou du délégué ne peut pas être de type 'type'</span><span class="sxs-lookup"><span data-stu-id="31e13-103">Method or delegate parameter cannot be of type 'type'</span></span>  
  
 <span data-ttu-id="31e13-104">Certains types de la bibliothèque de classes .NET Framework, par exemple <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> et <xref:System.ArgIterator> , ne peuvent pas être utilisés comme paramètres [ref](../../csharp/language-reference/keywords/ref.md) ou [out](../../csharp/language-reference/keywords/out.md) , car ils peuvent potentiellement être utilisés pour des opérations non sécurisées.</span><span class="sxs-lookup"><span data-stu-id="31e13-104">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) parameters because they could potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="31e13-105">L’exemple suivant génère l’erreur CS1601 :</span><span class="sxs-lookup"><span data-stu-id="31e13-105">The following sample generates CS1601:</span></span>  
  
```  
// CS1601.cs  
using System;  
  
class MyClass  
{  
   public void Test1 (ref TypedReference t)   // CS1601  
   {  
   }  
  
   public void Test2 (out ArgIterator t)   // CS1601  
   {  
   }  
}  
```
