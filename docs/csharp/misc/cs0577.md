<span data-ttu-id="bfbe3-103">L’attribut Conditional n’est pas valide sur 'function', car il s’agit d’un constructeur, d’un destructeur, d’un opérateur ou d’une implémentation d’interface explicite</span><span class="sxs-lookup"><span data-stu-id="bfbe3-103">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>

```csharp  
// CS0577.cs  
// compile with: /target:library

{  
   void m();  
}

   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
