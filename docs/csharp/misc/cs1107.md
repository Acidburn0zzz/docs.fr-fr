# <a name="compiler-error-cs1107"></a><span data-ttu-id="b950f-102">Erreur du compilateur CS1107</span><span class="sxs-lookup"><span data-stu-id="b950f-102">Compiler Error CS1107</span></span>
<span data-ttu-id="b950f-103">Un paramètre ne peut posséder qu’un modificateur 'modifier name'.</span><span class="sxs-lookup"><span data-stu-id="b950f-103">A parameter can only have one 'modifier name' modifier.</span></span>  
  
 <span data-ttu-id="b950f-104">Il n’est pas de modificateurs de paramètre comme `this`, `ref`, `in`, et `out` apparaître plusieurs fois dans une définition de paramètre.</span><span class="sxs-lookup"><span data-stu-id="b950f-104">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b950f-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="b950f-105">Example</span></span>  
 <span data-ttu-id="b950f-106">L’exemple suivant génère l’erreur CS1107 :</span><span class="sxs-lookup"><span data-stu-id="b950f-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{ 
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
