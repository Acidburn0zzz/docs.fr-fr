## <a name="example"></a><span data-ttu-id="78716-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="78716-110">Example</span></span>  
 <span data-ttu-id="78716-111">Le code suivant génère l’erreur CS0736, car `Program.testMethod` est déclaré statique :</span><span class="sxs-lookup"><span data-stu-id="78716-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>

```  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
