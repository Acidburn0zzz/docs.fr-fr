## <a name="example"></a><span data-ttu-id="a849e-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="a849e-109">Example</span></span>  
 <span data-ttu-id="a849e-110">Le code suivant génère l’erreur CS1914, car l’initialiseur essaie d’initialiser le champ `TestClass.Number` , qui est de type `static`:</span><span class="sxs-lookup"><span data-stu-id="a849e-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }      
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
