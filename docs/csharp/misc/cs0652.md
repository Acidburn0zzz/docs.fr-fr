# <a name="compiler-warning-level-2-cs0652"></a><span data-ttu-id="025b0-102">Avertissement du compilateur (niveau 2) CS0652</span><span class="sxs-lookup"><span data-stu-id="025b0-102">Compiler Warning (level 2) CS0652</span></span>
<span data-ttu-id="025b0-103">La comparaison à la constante intégrale est inutile, car la constante se situe hors de la plage du type 'type'</span><span class="sxs-lookup"><span data-stu-id="025b0-103">Comparison to integral constant is useless; the constant is outside the range of type 'type'</span></span>  
  
 <span data-ttu-id="025b0-104">Le compilateur a détecté une comparaison entre une constante et une variable pour laquelle la constante est hors de la plage de la variable.</span><span class="sxs-lookup"><span data-stu-id="025b0-104">The compiler detected a comparison between a constant and a variable where the constant is out of the range of the variable.</span></span>  
  
 <span data-ttu-id="025b0-105">L’exemple suivant génère l’avertissement CS0652 :</span><span class="sxs-lookup"><span data-stu-id="025b0-105">The following sample generates CS0652:</span></span>  
  
```csharp  
// CS0652.cs  
// compile with: /W:2  
public class Class1  
{  
   private static byte i = 0;  
   public static void Main()  
   {  
      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte  
         i = 0;  
   }  
}  
```
