# <a name="compiler-error-cs0569"></a>Erreur du compilateur CS0569
'method2' : impossible de substituer 'method1', car il n’est pas pris en charge par le langage  
  
 Cette erreur se produit quand vous dérivez d’une classe de base qui a été écrite dans un autre langage et que le compilateur ne prend pas en charge la méthode que vous essayez de substituer.
