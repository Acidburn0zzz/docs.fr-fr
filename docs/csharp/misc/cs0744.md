# <a name="compiler-error-cs0744"></a><span data-ttu-id="94d05-102">Erreur du compilateur CS0744</span><span class="sxs-lookup"><span data-stu-id="94d05-102">Compiler Error CS0744</span></span>
<span data-ttu-id="94d05-103">Mot clé contextuel 'equals' attendu</span><span class="sxs-lookup"><span data-stu-id="94d05-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="94d05-104">Le modèle pour une clause `join` est `join`...`in`...`on`...`equals`, comme illustré dans cet exemple :</span><span class="sxs-lookup"><span data-stu-id="94d05-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

 <span data-ttu-id="94d05-108">L’exemple suivant génère l’erreur CS0744 :</span><span class="sxs-lookup"><span data-stu-id="94d05-108">The following code generates CS0744:</span></span>

{  
    public static int Main()
