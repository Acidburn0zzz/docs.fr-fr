# <a name="compiler-error-cs1606"></a>Erreur du compilateur CS1606
Échec de la signature de l’assembly ; la sortie n’est peut-être pas signée -- raison  
  
 L’assembly a été créé, mais quand le compilateur a essayé de finir de le signer, un échec s’est produit.
