# <a name="compiler-error-cs0517"></a>Erreur du compilateur CS0517
'class' ne possède pas de classe de base et ne peut pas appeler un constructeur de base  
  
 L’erreur CS0517 peut se produire uniquement quand le Common Language Runtime du .NET Framework compile le code source pour la classe d’objets, qui est la seule classe qui ne dispose d’aucune classe de base.
