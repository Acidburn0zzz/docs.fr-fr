## <a name="example"></a><span data-ttu-id="e40ff-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="e40ff-106">Example</span></span>  
 <span data-ttu-id="e40ff-107">L’exemple suivant génère l’erreur CS0136 :</span><span class="sxs-lookup"><span data-stu-id="e40ff-107">The following sample generates CS0136:</span></span>  
  
```csharp
// CS0136.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }  
}  
```
