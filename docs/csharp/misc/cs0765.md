# <a name="compiler-error-cs0765"></a><span data-ttu-id="a287e-102">Erreur du compilateur CS0765</span><span class="sxs-lookup"><span data-stu-id="a287e-102">Compiler Error CS0765</span></span>
<span data-ttu-id="a287e-103">Les méthodes partielles avec uniquement une déclaration de définition ou des méthodes conditionnelles supprimées ne peuvent pas être utilisées dans des arborescences d'expressions</span><span class="sxs-lookup"><span data-stu-id="a287e-103">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="a287e-104">Bien qu’un appel à une méthode partielle supprimée soit une expression, ce n’est pas une expression acceptable dans une arborescence d’expression.</span><span class="sxs-lookup"><span data-stu-id="a287e-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a287e-105">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="a287e-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="a287e-106">Ajoutez une déclaration d’implémentation pour la méthode partielle ou supprimez le code qui entraîne l’exclusion de la méthode conditionnelle de la compilation.</span><span class="sxs-lookup"><span data-stu-id="a287e-106">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a287e-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="a287e-107">Example</span></span>  
 <span data-ttu-id="a287e-108">Le code suivant génère CS0765 à deux emplacements :</span><span class="sxs-lookup"><span data-stu-id="a287e-108">The following code generates CS0765 in two locations:</span></span>  
  
```  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
