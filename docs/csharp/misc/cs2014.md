# Avertissement du compilateur (niveau&#160;1) CS2014
L’option de compilateur 'ancienne option' est obsolète. Utilisez plutôt 'nouvelle option'  
  
 La forme de l’option de compilateur est déconseillée. Pour plus d’informations, consultez [C\# Compiler Options](../../csharp/language-reference/compiler-options/index.md).
