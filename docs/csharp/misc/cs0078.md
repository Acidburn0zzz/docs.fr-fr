# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="a74fb-102">Avertissement du compilateur (niveau 4) CS0078</span><span class="sxs-lookup"><span data-stu-id="a74fb-102">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="a74fb-103">Le suffixe 'l' risque d'être facilement confondu avec le chiffre '1' -- utilisez plutôt 'L'</span><span class="sxs-lookup"><span data-stu-id="a74fb-103">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="a74fb-104">Le compilateur vous avertit lorsqu’il détecte un cast en long à l’aide d’un l minuscule au lieu d’un L majuscule.</span><span class="sxs-lookup"><span data-stu-id="a74fb-104">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="a74fb-105">L’exemple suivant génère l’avertissement CS0078 :</span><span class="sxs-lookup"><span data-stu-id="a74fb-105">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
