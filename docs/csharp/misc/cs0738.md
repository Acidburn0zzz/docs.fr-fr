## <a name="example"></a><span data-ttu-id="4d888-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="4d888-108">Example</span></span>  
 <span data-ttu-id="4d888-109">Le code suivant génère CS0738, car la méthode de classe retourne `void` et le membre d’interface du même nom retourne `int`:</span><span class="sxs-lookup"><span data-stu-id="4d888-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

{  
    int TestMethod();
