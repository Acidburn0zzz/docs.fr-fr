 <span data-ttu-id="acec6-104">Les mots clés [is](../../csharp/language-reference/keywords/is.md) et [as](../../csharp/language-reference/keywords/as.md) ne sont pas valides pour une utilisation sur les types de pointeurs.</span><span class="sxs-lookup"><span data-stu-id="acec6-104">The [is](../../csharp/language-reference/keywords/is.md) and [as](../../csharp/language-reference/keywords/as.md) keywords are not valid for use on pointer types.</span></span> <span data-ttu-id="acec6-105">Pour plus d’informations, consultez [Pointeurs et code unsafe](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="acec6-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="acec6-106">L’exemple suivant génère l’erreur CS0244 :</span><span class="sxs-lookup"><span data-stu-id="acec6-106">The following sample generates CS0244:</span></span>  
  
```  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
