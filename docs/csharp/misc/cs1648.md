# <a name="compiler-error-cs1648"></a><span data-ttu-id="4e9e3-102">Erreur du compilateur CS1648</span><span class="sxs-lookup"><span data-stu-id="4e9e3-102">Compiler Error CS1648</span></span>
<span data-ttu-id="4e9e3-103">Les membres d’un champ readonly 'identifier' ne peuvent pas être modifiés (sauf s’ils appartiennent à un constructeur ou un initialiseur de variable)</span><span class="sxs-lookup"><span data-stu-id="4e9e3-103">Members of readonly field 'identifier' cannot be modified (except in a constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="4e9e3-104">Cette erreur se produit quand vous tentez de modifier un membre de champ readonly qui ne peut pas être modifié.</span><span class="sxs-lookup"><span data-stu-id="4e9e3-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="4e9e3-105">Pour résoudre cette erreur, limitez les assignations aux champs readonly au constructeur ou à l’initialiseur de variable, ou supprimez le mot clé readonly de la déclaration du champ.</span><span class="sxs-lookup"><span data-stu-id="4e9e3-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>  
  
 <span data-ttu-id="4e9e3-106">L’exemple suivant génère l’erreur CS1648 :</span><span class="sxs-lookup"><span data-stu-id="4e9e3-106">The following sample generates CS1648:</span></span>  
  
```csharp  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
