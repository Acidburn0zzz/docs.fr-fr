# <a name="compiler-error-cs1676"></a><span data-ttu-id="f310f-102">Erreur du compilateur CS1676</span><span class="sxs-lookup"><span data-stu-id="f310f-102">Compiler Error CS1676</span></span>
<span data-ttu-id="f310f-103">Le paramètre 'number' doit être déclaré avec le mot clé 'keyword'</span><span class="sxs-lookup"><span data-stu-id="f310f-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="f310f-104">Cette erreur se produit quand le modificateur de type de paramètre dans une méthode anonyme est différent de celui utilisé dans la déclaration du délégué vers laquelle vous effectuez un cast de la méthode.</span><span class="sxs-lookup"><span data-stu-id="f310f-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="f310f-105">L’exemple suivant génère l’erreur CS1676 :</span><span class="sxs-lookup"><span data-stu-id="f310f-105">The following sample generates CS1676:</span></span>  
  
```  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
