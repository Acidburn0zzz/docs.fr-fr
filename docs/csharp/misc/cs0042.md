# Erreur du compilateur CS0042
Erreur inattendue lors de la création du fichier des informations de débogage 'fichier' — 'raison'  
  
 Le compilateur n’a pas pu créer les informations de débogage ; *raison* contient des informations supplémentaires sur la condition d’erreur.
