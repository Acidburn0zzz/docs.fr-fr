<span data-ttu-id="3ecea-103">'class' : une classe avec l’attribut ComImport ne peut pas spécifier une classe de base</span><span class="sxs-lookup"><span data-stu-id="3ecea-103">'class': a class with the ComImport attribute cannot specify a base class</span></span>  
  
 <span data-ttu-id="3ecea-104">Spécifier l’attribut <xref:System.Runtime.InteropServices.ComImportAttribute> implique que l’implémentation de la classe soit importée à partir d’un module COM.</span><span class="sxs-lookup"><span data-stu-id="3ecea-104">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="3ecea-105">Les méthodes ou champs supplémentaires hérités de la classe de base ne peuvent pas être ajoutés à l’implémentation définie dans le module COM.</span><span class="sxs-lookup"><span data-stu-id="3ecea-105">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>

