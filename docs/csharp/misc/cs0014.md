# <a name="compiler-error-cs0014"></a>Erreur du compilateur CS0014
Le fichier requis 'fichier' est introuvable  
  
 Un fichier requis par le compilateur est introuvable sur le système. Vérifiez que le chemin est correct. Si le fichier est un [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] système de fichiers, puis vous devrez peut-être réparer votre installation ou un autre supprimer et réinstaller [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] complètement.
