# Erreur du compilateur CS0014
Le fichier requis 'fichier' est introuvable  
  
 Un fichier requis par le compilateur est introuvable sur le système. Vérifiez que le chemin est correct. S’il s’agit d’un fichier système [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)], vous devrez peut\-être réparer votre installation ou supprimer et réinstaller complètement [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].
