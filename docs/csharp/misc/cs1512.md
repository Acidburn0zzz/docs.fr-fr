 <span data-ttu-id="2a03d-104">Le mot clé [base](../../csharp/language-reference/keywords/base.md) a été utilisé en dehors d’une méthode, d’une propriété ou d’un constructeur.</span><span class="sxs-lookup"><span data-stu-id="2a03d-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="2a03d-105">L’exemple suivant génère l’erreur CS1512 :</span><span class="sxs-lookup"><span data-stu-id="2a03d-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
