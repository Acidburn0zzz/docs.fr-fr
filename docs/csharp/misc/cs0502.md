# <a name="compiler-error-cs0502"></a><span data-ttu-id="c797d-102">Erreur du compilateur CS0502</span><span class="sxs-lookup"><span data-stu-id="c797d-102">Compiler Error CS0502</span></span>

 <span data-ttu-id="c797d-104">Un membre de classe ne peut pas être à la fois [abstract](../../csharp/language-reference/keywords/abstract.md) et [sealed](../../csharp/language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="c797d-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="c797d-105">L’exemple suivant génère l’erreur CS0502 :</span><span class="sxs-lookup"><span data-stu-id="c797d-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
