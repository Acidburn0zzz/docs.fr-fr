# <a name="compiler-error-cs1556"></a>Erreur du compilateur CS1556
La valeur 'construct' spécifiée pour la méthode Main doit être une classe ou une structure valide  
  
 Un identificateur qui n’était pas un nom de classe a été passé à l’option de compilateur [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) .
