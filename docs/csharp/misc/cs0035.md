<span data-ttu-id="6410a-103">L’opérateur 'opérateur' est ambigu pour un opérande de type 'type'</span><span class="sxs-lookup"><span data-stu-id="6410a-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="6410a-104">Plusieurs conversions sont disponibles et le compilateur ne sait pas laquelle choisir avant d’appliquer l’opérateur.</span><span class="sxs-lookup"><span data-stu-id="6410a-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="6410a-105">Pour plus d’informations, consultez [les Conversions définies par utilisateur basés sur des modèles](https://docs.microsoft.com/cpp/misc/templated-user-defined-conversions) et [les opérateurs de Conversion](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="6410a-105">For more information, see [Templated User Defined Conversions](https://docs.microsoft.com/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="6410a-106">L’exemple suivant génère l’erreur CS0035 :</span><span class="sxs-lookup"><span data-stu-id="6410a-106">The following sample generates CS0035:</span></span>  
  
```  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
