<span data-ttu-id="85d8e-103">Impossible de marquer 'type' comme conforme CLS car il est membre du type non conforme CLS 'type'</span><span class="sxs-lookup"><span data-stu-id="85d8e-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>  
  
 <span data-ttu-id="85d8e-104">Cet avertissement se produit si une classe imbriquée avec l’attribut CLSCompliant défini sur `true` est déclarée comme membre d’une classe déclarée avec l’attribut CLSCompliant défini sur `false`.</span><span class="sxs-lookup"><span data-stu-id="85d8e-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="85d8e-105">Cela n’est pas autorisé car une classe imbriquée ne peut pas être conforme CLS si elle est membre d’une classe externe qui n’est pas conforme CLS.</span><span class="sxs-lookup"><span data-stu-id="85d8e-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="85d8e-106">Pour remédier à cet avertissement, supprimez l’attribut CLSCompliant de la classe imbriquée ou remplacez `true` par `false`.</span><span class="sxs-lookup"><span data-stu-id="85d8e-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="85d8e-107">Pour plus d’informations sur la conformité CLS, consultez [Écriture d’un code conforme CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) et [Indépendance du langage et composants indépendants du langage](../../../docs/standard/language-independence-and-language-independent-components.md).</span><span class="sxs-lookup"><span data-stu-id="85d8e-107">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) and [Language Independence and Language-Independent Components](../../../docs/standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="85d8e-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="85d8e-108">Example</span></span>  
 <span data-ttu-id="85d8e-109">L’exemple suivant génère l’avertissement CS3018.</span><span class="sxs-lookup"><span data-stu-id="85d8e-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
