## <a name="example"></a><span data-ttu-id="a90f4-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="a90f4-107">Example</span></span>

```  
public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010

{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
