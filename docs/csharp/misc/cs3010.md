## <a name="example"></a><span data-ttu-id="772ae-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="772ae-107">Example</span></span>  
 <span data-ttu-id="772ae-108">L’exemple suivant génère l’erreur CS3010 :</span><span class="sxs-lookup"><span data-stu-id="772ae-108">The following example generates CS3010:</span></span>  
  
```csharp

public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010

{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
