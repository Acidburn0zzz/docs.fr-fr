 <span data-ttu-id="c0be1-105">Le compilateur c# n’autorise pas l’utilisation de <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; utilisez [params](../../csharp/language-reference/keywords/params.md) à la place.</span><span class="sxs-lookup"><span data-stu-id="c0be1-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="c0be1-106">L’exemple suivant génère l’erreur CS0674 :</span><span class="sxs-lookup"><span data-stu-id="c0be1-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
