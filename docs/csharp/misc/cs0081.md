# <a name="compiler-error-cs0081"></a><span data-ttu-id="fe4e7-102">Erreur du compilateur CS0081</span><span class="sxs-lookup"><span data-stu-id="fe4e7-102">Compiler Error CS0081</span></span>
<span data-ttu-id="fe4e7-103">La déclaration du paramètre de type doit être un identificateur et non un type</span><span class="sxs-lookup"><span data-stu-id="fe4e7-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="fe4e7-104">Quand vous déclarez une méthode ou un type générique, spécifiez le paramètre de type comme identificateur, par exemple « T » ou « inputType ».</span><span class="sxs-lookup"><span data-stu-id="fe4e7-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="fe4e7-105">Quand le code client appelle la méthode, il fournit le type, qui remplace chaque occurrence de l’identificateur dans le corps de la méthode ou de la classe.</span><span class="sxs-lookup"><span data-stu-id="fe4e7-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="fe4e7-106">Pour plus d’informations, consultez [des paramètres de Type générique](../../csharp/programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="fe4e7-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
