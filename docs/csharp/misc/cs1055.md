<span data-ttu-id="6e87e-103">Un accesseur add ou remove est attendu</span><span class="sxs-lookup"><span data-stu-id="6e87e-103">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="6e87e-104">Si votre [événement](../../csharp/language-reference/keywords/event.md) n'est pas déclaré comme un champ, il doit définir les deux fonctions d'accesseur **add** et **remove** .</span><span class="sxs-lookup"><span data-stu-id="6e87e-104">If your [event](../../csharp/language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="6e87e-105">L’exemple suivant génère l’erreur CS1055 :</span><span class="sxs-lookup"><span data-stu-id="6e87e-105">The following sample generates CS1055:</span></span>  
  
```  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
