# <a name="compiler-warning-level-3-cs0660"></a><span data-ttu-id="f17f6-102">Avertissement du compilateur (niveau 3) CS0660</span><span class="sxs-lookup"><span data-stu-id="f17f6-102">Compiler Warning (level 3) CS0660</span></span>
<span data-ttu-id="f17f6-103">'class' définit l’opérateur == ou l’opérateur != mais ne se substitue pas à Object.Equals(object o)</span><span class="sxs-lookup"><span data-stu-id="f17f6-103">'class' defines operator == or operator != but does not override Object.Equals(object o)</span></span>  
  
 <span data-ttu-id="f17f6-104">Le compilateur a détecté l’opérateur d’égalité ou d’inégalité défini par l’utilisateur, mais n’a détecté aucune substitution pour la fonction **Equals** .</span><span class="sxs-lookup"><span data-stu-id="f17f6-104">The compiler detected the user-defined equality or inequality operator, but no override for the **Equals** function.</span></span> <span data-ttu-id="f17f6-105">Un opérateur d’égalité ou d’inégalité défini par l’utilisateur sous-entend que vous souhaitez également substituer la fonction **Equals** .</span><span class="sxs-lookup"><span data-stu-id="f17f6-105">A user-defined equality or inequality operator implies that you also want to override the **Equals** function.</span></span> <span data-ttu-id="f17f6-106">Pour plus d’informations, consultez [NIB - indications concernant la substitution de Equals() et de l’opérateur == (Guide de programmation c#)](http://msdn.microsoft.com/library/7e4c24c5-7693-4c45-88fb-ba5204fbcb20).</span><span class="sxs-lookup"><span data-stu-id="f17f6-106">For more information, see [NIB - Guidelines for Overriding Equals() and Operator == (C# Programming Guide)](http://msdn.microsoft.com/library/7e4c24c5-7693-4c45-88fb-ba5204fbcb20).</span></span>  
  
 <span data-ttu-id="f17f6-107">L’exemple suivant génère l’avertissement CS0660 :</span><span class="sxs-lookup"><span data-stu-id="f17f6-107">The following sample generates CS0660:</span></span>  
  
```csharp  
// CS0660.cs  
// compile with: /W:3 /warnaserror  
class Test   // CS0660  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   // uncomment the Equals function to resolve  
   // public override bool Equals(object o)  
   // {  
   //    return true;  
   // }  
  
   public override int GetHashCode()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
