 <span data-ttu-id="db5a8-108">La séquence suivante de compilations provoque une erreur CS0012 :</span><span class="sxs-lookup"><span data-stu-id="db5a8-108">The following sequence of compilations will result in CS0012:</span></span>  
  
```csharp  
// cs0012a.cs  
// compile with: /target:library  
public class A {}  
```  
  
 <span data-ttu-id="db5a8-109">Puis :</span><span class="sxs-lookup"><span data-stu-id="db5a8-109">Then:</span></span>  
  
```csharp  
// cs0012b.cs  
// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="db5a8-110">Puis :</span><span class="sxs-lookup"><span data-stu-id="db5a8-110">Then:</span></span>  
  
```csharp

// compile with: /reference:cs0012b.dll

{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```  
  
 <span data-ttu-id="db5a8-111">Vous pouvez résoudre cette erreur CS0012 en compilant avec `/reference:cs0012b.dll;cs0012a.dll`, ou dans Visual Studio à l’aide de la [boîte de dialogue Ajouter une référence](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) pour ajouter une référence à cs0012a.dll en plus de cs0012b.dll.</span><span class="sxs-lookup"><span data-stu-id="db5a8-111">You could resolve this CS0012 by compiling with `/reference:cs0012b.dll;cs0012a.dll`, or in Visual Studio by using the [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) to add a reference to cs0012a.dll in addition to cs0012b.dll.</span></span>
