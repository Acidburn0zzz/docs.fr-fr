  
 <span data-ttu-id="9c0ea-109">Puis :</span><span class="sxs-lookup"><span data-stu-id="9c0ea-109">Then:</span></span>

```csharp  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="9c0ea-110">Puis :</span><span class="sxs-lookup"><span data-stu-id="9c0ea-110">Then:</span></span>

```csharp  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
