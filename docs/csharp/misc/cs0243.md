 <span data-ttu-id="633f6-104">Le [conditionnelle](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribut n’est pas autorisé sur une méthode qui est marquée avec la [remplacer](../../csharp/language-reference/keywords/override.md) (mot clé).</span><span class="sxs-lookup"><span data-stu-id="633f6-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="633f6-105">Pour plus d’informations, consultez [connaître lors de la remplacer de l’utilisation et de nouveaux mots clés](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="633f6-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="633f6-106">Le compilateur ne lie jamais à des méthodes override ; il lie uniquement à la méthode de base et le Common Language Runtime appelle la substitution comme il convient.</span><span class="sxs-lookup"><span data-stu-id="633f6-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="633f6-107">L’exemple suivant génère l’erreur CS0243 :</span><span class="sxs-lookup"><span data-stu-id="633f6-107">The following sample generates CS0243:</span></span>  
  
```  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
