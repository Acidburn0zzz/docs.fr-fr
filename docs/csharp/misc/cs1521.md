# <a name="compiler-error-cs1521"></a>Erreur du compilateur CS1521
Type de base non valide  
  
 Une spécification de classe de [base](../../csharp/language-reference/keywords/base.md) était incorrecte.  
  
 L’exemple suivant génère l’erreur CS1521 :  
  
```csharp  
// CS1521.cs  
class CMyClass  
{  
   public static void Main()  
   {  
   }  
}  
  
class CMyClass1 : CMyClass     // OK  
{  
}  
  
class CMyClass2 : CMyClass[]   // CS1521  
{  
}  
  
class CMyClass3 : CMyClass*    // CS1521  
{  
}  
```
