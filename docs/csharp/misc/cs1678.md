# <a name="compiler-error-cs1678"></a><span data-ttu-id="8214d-102">Erreur du compilateur CS1678</span><span class="sxs-lookup"><span data-stu-id="8214d-102">Compiler Error CS1678</span></span>
<span data-ttu-id="8214d-103">Le paramètre 'number' est déclaré comme type 'type1' mais doit être 'type2'</span><span class="sxs-lookup"><span data-stu-id="8214d-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="8214d-104">Cette erreur se produit quand le type de paramètre dans une méthode anonyme est différent de la déclaration du délégué vers laquelle vous effectuez un cast de la méthode.</span><span class="sxs-lookup"><span data-stu-id="8214d-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="8214d-105">L’exemple suivant génère l’erreur CS1678 :</span><span class="sxs-lookup"><span data-stu-id="8214d-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
