## <a name="example"></a><span data-ttu-id="4a0f8-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="4a0f8-108">Example</span></span>  
 <span data-ttu-id="4a0f8-109">L’exemple suivant génère l’erreur CS1102 :</span><span class="sxs-lookup"><span data-stu-id="4a0f8-109">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
