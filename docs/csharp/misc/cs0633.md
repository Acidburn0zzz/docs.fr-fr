## <a name="example"></a><span data-ttu-id="00bbe-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="00bbe-106">Example</span></span>  
 <span data-ttu-id="00bbe-107">Cet exemple montre l’erreur CS0633 avec l’utilisation de <xref:System.Diagnostics.ConditionalAttribute>.</span><span class="sxs-lookup"><span data-stu-id="00bbe-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="00bbe-108">L’exemple suivant génère l’erreur CS0633 :</span><span class="sxs-lookup"><span data-stu-id="00bbe-108">The following sample generates CS0633.</span></span>  
  
```

using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="00bbe-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="00bbe-109">Example</span></span>  
 <span data-ttu-id="00bbe-110">Cet exemple montre l’erreur CS0633 avec l’utilisation de <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span><span class="sxs-lookup"><span data-stu-id="00bbe-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
