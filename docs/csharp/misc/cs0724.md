## <a name="example"></a><span data-ttu-id="011b9-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="011b9-105">Example</span></span>  
 <span data-ttu-id="011b9-106">L’exemple suivant génère l’erreur CS0724.</span><span class="sxs-lookup"><span data-stu-id="011b9-106">The following example generates CS0724.</span></span>  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
