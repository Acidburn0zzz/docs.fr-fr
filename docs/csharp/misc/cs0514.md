## <a name="example"></a><span data-ttu-id="2756c-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="2756c-107">Example</span></span>  
 <span data-ttu-id="2756c-108">L’exemple suivant génère l’erreur CS0514 :</span><span class="sxs-lookup"><span data-stu-id="2756c-108">The following example generates CS0514:</span></span>

```csharp  
    {  
    }

{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
