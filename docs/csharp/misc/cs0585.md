# <a name="compiler-error-cs0585"></a>Erreur du compilateur CS0585
Erreur interne du compilateur : étape 'étape'  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si tel n’est pas le cas, essayez de [nous contacter](/visualstudio/ide/talk-to-us).
