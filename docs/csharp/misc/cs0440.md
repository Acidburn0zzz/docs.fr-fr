## <a name="example"></a><span data-ttu-id="92169-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="92169-105">Example</span></span>  
 <span data-ttu-id="92169-106">L’exemple suivant génère l’avertissement CS0440 :</span><span class="sxs-lookup"><span data-stu-id="92169-106">The following example generates CS0440:</span></span>  
  
```  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
