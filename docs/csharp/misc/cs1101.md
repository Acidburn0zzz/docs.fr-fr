# <a name="compiler-error-cs1101"></a><span data-ttu-id="03f36-102">Erreur du compilateur CS1101</span><span class="sxs-lookup"><span data-stu-id="03f36-102">Compiler Error CS1101</span></span>
<span data-ttu-id="03f36-103">Le modificateur de paramètre 'ref' ne peut pas être utilisé avec 'this'.</span><span class="sxs-lookup"><span data-stu-id="03f36-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="03f36-104">Lorsque le mot-clé `this` modifie le premier paramètre d’une méthode statique, il signale au compilateur que la méthode est une méthode d’extension.</span><span class="sxs-lookup"><span data-stu-id="03f36-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="03f36-105">Aucun autre modificateur n’est nécessaire ni autorisé sur le premier paramètre d’une méthode d’extension.</span><span class="sxs-lookup"><span data-stu-id="03f36-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="03f36-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="03f36-106">Example</span></span>  
 <span data-ttu-id="03f36-107">L’exemple suivant génère l’erreur CS1101 :</span><span class="sxs-lookup"><span data-stu-id="03f36-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
