# <a name="compiler-error-cs0663"></a><span data-ttu-id="7319f-102">Erreur du compilateur CS0663</span><span class="sxs-lookup"><span data-stu-id="7319f-102">Compiler Error CS0663</span></span>
<span data-ttu-id="7319f-103">Impossible de définir des méthodes surchargées qui se différencient uniquement au niveau des paramètres ref et out</span><span class="sxs-lookup"><span data-stu-id="7319f-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="7319f-104">Les méthodes qui diffèrent uniquement sur leur utilisation de [dans](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) et [hors](../../csharp/language-reference/keywords/out-parameter-modifier.md) sur un paramètre ne sont pas autorisés.</span><span class="sxs-lookup"><span data-stu-id="7319f-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="7319f-105">L’exemple suivant génère l’erreur CS0663 :</span><span class="sxs-lookup"><span data-stu-id="7319f-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
