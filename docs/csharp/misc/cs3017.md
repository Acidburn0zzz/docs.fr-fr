## <a name="example"></a><span data-ttu-id="fca40-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="fca40-108">Example</span></span>  
 <span data-ttu-id="fca40-109">L’exemple suivant génère l’avertissement CS3017 :</span><span class="sxs-lookup"><span data-stu-id="fca40-109">The following example generates CS3017:</span></span>  
  
```

// compile with: /target:module

[module: CLSCompliant(true)]  
[assembly: CLSCompliant(false)]  // CS3017  
// Try this line instead:  
// [assembly: CLSCompliant(true)]

{  
    static void Main() {}  
}  
```
