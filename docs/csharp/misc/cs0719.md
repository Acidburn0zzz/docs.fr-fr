# <a name="compiler-error-cs0719"></a><span data-ttu-id="0f4ef-102">Erreur du compilateur CS0719</span><span class="sxs-lookup"><span data-stu-id="0f4ef-102">Compiler Error CS0719</span></span>
<span data-ttu-id="0f4ef-103">'type' : les éléments de tableau ne peuvent pas être de type static</span><span class="sxs-lookup"><span data-stu-id="0f4ef-103">'type': array elements cannot be of static type</span></span>  
  
 <span data-ttu-id="0f4ef-104">Un tableau de type static n’a pas de sens dans la mesure où les éléments du tableau sont des instances et qu’il n’est pas possible de créer des instances de types static.</span><span class="sxs-lookup"><span data-stu-id="0f4ef-104">An array of static type does not make sense since array elements are instances, but it is not possible to create instances of static types.</span></span>  
  
 <span data-ttu-id="0f4ef-105">L’exemple suivant génère l’erreur CS0719 :</span><span class="sxs-lookup"><span data-stu-id="0f4ef-105">The following sample generates CS0719:</span></span>  
  
```csharp  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
