 <span data-ttu-id="6d4e8-104">Lors d’une tentative de référencement d’une forme surchargée d’une méthode, le compilateur a détecté une erreur de syntaxe.</span><span class="sxs-lookup"><span data-stu-id="6d4e8-104">When attempting to reference an overload form of a method, the compiler detected a syntax error.</span></span> <span data-ttu-id="6d4e8-105">Cela indique généralement que le nom du paramètre, et non le type, a été spécifié.</span><span class="sxs-lookup"><span data-stu-id="6d4e8-105">Typically, this indicates that the parameter name, and not the type, was specified.</span></span> <span data-ttu-id="6d4e8-106">Une ligne incorrecte apparaît dans le fichier XML généré.</span><span class="sxs-lookup"><span data-stu-id="6d4e8-106">A malformed line will appear in the generated XML file.</span></span>  
  
 <span data-ttu-id="6d4e8-107">L’exemple suivant génère l’avertissement CS1580 :</span><span class="sxs-lookup"><span data-stu-id="6d4e8-107">The following sample generates CS1580:</span></span>  
  
```csharp

/// <seealso cref="Test(i)"/>   // CS1580  
// try the following line instead  
// /// <seealso cref="Test(int)"/>

{  
   /// <summary>help text</summary>

   {  
   }  
  
   /// <summary>help text</summary>  
   public void Test(int i)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public void Test(char i)  
   {  
   }  
}  
```
