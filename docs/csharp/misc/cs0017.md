 <span data-ttu-id="7c8db-107">L’exemple suivant génère l’erreur CS0017 :</span><span class="sxs-lookup"><span data-stu-id="7c8db-107">The following sample generates CS0017:</span></span>

```  
{  
   static public void Main()  
   {  
   }

   public static void Main()   // CS0017, delete one Main or use /main

   }

```
