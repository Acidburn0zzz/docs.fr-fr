<span data-ttu-id="c36aa-103">Plusieurs points d’entrée sont définis dans le programme 'nom_fichier_sortie'.</span><span class="sxs-lookup"><span data-stu-id="c36aa-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="c36aa-104">Compilez avec l’option /main pour spécifier le type qui contient le point d’entrée.</span><span class="sxs-lookup"><span data-stu-id="c36aa-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="c36aa-105">Un programme peut posséder une seule [Main](../../csharp/programming-guide/main-and-command-args/index.md) méthode.</span><span class="sxs-lookup"><span data-stu-id="c36aa-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="c36aa-106">Pour résoudre cette erreur, vous pouvez supprimer toutes les méthodes Main de votre code, sauf un, ou vous pouvez utiliser la [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) option du compilateur pour spécifier la méthode Main que vous souhaitez utiliser.</span><span class="sxs-lookup"><span data-stu-id="c36aa-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="c36aa-107">L’exemple suivant génère l’erreur CS0017 :</span><span class="sxs-lookup"><span data-stu-id="c36aa-107">The following sample generates CS0017:</span></span>  
  
```

{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
