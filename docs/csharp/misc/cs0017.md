<span data-ttu-id="2a53d-103">Plusieurs points d’entrée sont définis dans le programme 'nom_fichier_sortie'.</span><span class="sxs-lookup"><span data-stu-id="2a53d-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="2a53d-104">Compilez avec l’option /main pour spécifier le type qui contient le point d’entrée.</span><span class="sxs-lookup"><span data-stu-id="2a53d-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="2a53d-105">Un programme ne peut avoir qu’une seule méthode [Main](../../csharp/programming-guide/main-and-command-args/index.md) .</span><span class="sxs-lookup"><span data-stu-id="2a53d-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="2a53d-106">Pour résoudre cette erreur, vous pouvez supprimer toutes les méthodes Main de votre code sauf une, ou vous pouvez utiliser l’option du compilateur [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) pour spécifier la méthode Main à utiliser.</span><span class="sxs-lookup"><span data-stu-id="2a53d-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="2a53d-107">L’exemple suivant génère l’erreur CS0017 :</span><span class="sxs-lookup"><span data-stu-id="2a53d-107">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe

{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
