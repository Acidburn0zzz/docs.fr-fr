# <a name="compiler-error-cs1627"></a><span data-ttu-id="731d0-102">Erreur du compilateur CS1627</span><span class="sxs-lookup"><span data-stu-id="731d0-102">Compiler Error CS1627</span></span>
<span data-ttu-id="731d0-103">Expression attendue après yield return</span><span class="sxs-lookup"><span data-stu-id="731d0-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="731d0-104">Cette erreur se produit si `yield` est utilisé sans expression.</span><span class="sxs-lookup"><span data-stu-id="731d0-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="731d0-105">Pour éviter cette erreur, insérez l’expression appropriée dans l’instruction.</span><span class="sxs-lookup"><span data-stu-id="731d0-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="731d0-106">L’exemple suivant génère l’erreur CS1627 :</span><span class="sxs-lookup"><span data-stu-id="731d0-106">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
