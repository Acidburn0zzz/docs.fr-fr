# <a name="compiler-error-cs2001"></a>Erreur du compilateur CS2001
Le fichier source 'fichier' est introuvable  
  
 Un nom de fichier source a été passé au compilateur, mais il est introuvable. Vérifiez l’orthographe du nom de fichier et l’emplacement du fichier.
