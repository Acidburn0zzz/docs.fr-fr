## <a name="example"></a><span data-ttu-id="cc16b-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="cc16b-108">Example</span></span>  
 <span data-ttu-id="cc16b-109">L’exemple suivant génère l’erreur CS1666.</span><span class="sxs-lookup"><span data-stu-id="cc16b-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);   
   }   
}  
```
