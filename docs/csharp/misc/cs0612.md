# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="1dcc6-102">Avertissement du compilateur (niveau 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="1dcc6-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="1dcc6-103">'membre' est obsolète.</span><span class="sxs-lookup"><span data-stu-id="1dcc6-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="1dcc6-104">Le Concepteur de classes a marqué un membre avec le [obsolète](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribut.</span><span class="sxs-lookup"><span data-stu-id="1dcc6-104">The class designer marked a member with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute.</span></span> <span data-ttu-id="1dcc6-105">Cela signifie que le membre ne sera peut-être pas pris en charge dans une version ultérieure de la classe.</span><span class="sxs-lookup"><span data-stu-id="1dcc6-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="1dcc6-106">L’exemple suivant montre comment l’accès à un membre obsolète génère l’avertissement CS0612 :</span><span class="sxs-lookup"><span data-stu-id="1dcc6-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
