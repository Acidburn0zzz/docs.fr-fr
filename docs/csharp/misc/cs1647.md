# Erreur du compilateur CS1647
Une expression est trop longue ou complexe pour compiler près de 'code'  
  
 Un dépassement de capacité de la pile s’est produit dans le compilateur qui traite votre code. Pour résoudre cette erreur, simplifiez votre code. Si votre code est valide, contactez le [support technique](http://msdn.microsoft.com/fr-fr/77e75b8b-817d-45bf-9c38-458930d873b4).
