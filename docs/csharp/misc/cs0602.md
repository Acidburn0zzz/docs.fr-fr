# <a name="compiler-warning-level-1-cs0602"></a>Avertissement du compilateur (niveau 1) CS0602
La fonctionnalité 'ancienne_fonctionnalité' est déconseillée. Utilisez plutôt 'nouvelle_fonctionnalité'  
  
 Une fonctionnalité de langage utilisée dans votre code (*ancienne_fonctionnalité*) est toujours prise en charge, mais cette prise en charge risque d’être retirée dans une version ultérieure. Utilisez plutôt la syntaxe recommandée (*nouvelle_fonctionnalité*).
