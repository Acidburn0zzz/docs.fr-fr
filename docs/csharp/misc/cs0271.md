<span data-ttu-id="b925a-103">La propriété ou l’indexeur 'property/indexer' ne peut pas être utilisé dans ce contexte, car l’accesseur get n’est pas accessible</span><span class="sxs-lookup"><span data-stu-id="b925a-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="b925a-104">Cette erreur se produit quand vous essayez d’accéder à un accesseur `get` inaccessible.</span><span class="sxs-lookup"><span data-stu-id="b925a-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="b925a-105">Pour résoudre cette erreur, augmentez l’accessibilité de l’accesseur ou modifiez l’emplacement d’appel.</span><span class="sxs-lookup"><span data-stu-id="b925a-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="b925a-106">Pour plus d’informations, consultez [accessibilité des accesseurs](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) et [propriétés](../../csharp/programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="b925a-106">For more information, see [Accessor Accessibility](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="b925a-107">L’exemple suivant génère l’erreur CS0271 :</span><span class="sxs-lookup"><span data-stu-id="b925a-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
