# Erreur du compilateur CS0005
L’option du compilateur 'option\_compilateur' doit être suivie d’un argument  
  
 Certaines options du compilateur nécessitent des paramètres. Si vous ne passez pas les arguments dont a besoin l’option du compilateur, l’erreur CS0005 est générée.  
  
 Pour plus d’informations, consultez [Options du compilateur C\#](../../csharp/language-reference/compiler-options/index.md).
