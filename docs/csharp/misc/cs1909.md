## <a name="example"></a><span data-ttu-id="0a311-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="0a311-105">Example</span></span>  
 <span data-ttu-id="0a311-106">L’exemple suivant génère l’erreur CS1909 :</span><span class="sxs-lookup"><span data-stu-id="0a311-106">The following sample generates CS1909.</span></span>  
  
```  
// CS1909.cs  
// compile with: /target:library

public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
