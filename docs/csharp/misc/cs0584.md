# Erreur du compilateur CS0584
Erreur interne du compilateur : étape 'étape' symbole 'symbole'  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si ce n’est pas le cas, essayez [Nous contacter](/visual-studio/ide/talk-to-us).
