# <a name="compiler-error-cs0584"></a>Erreur du compilateur CS0584
Erreur interne du compilateur : étape 'étape' symbole 'symbole'  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si tel n’est pas le cas, essayez de [nous contacter](https://docs.microsoft.com/visualstudio/ide/talk-to-us).
