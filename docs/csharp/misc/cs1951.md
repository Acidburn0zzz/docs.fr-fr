# <a name="compiler-error-cs1951"></a><span data-ttu-id="8e110-102">Erreur du compilateur CS1951</span><span class="sxs-lookup"><span data-stu-id="8e110-102">Compiler Error CS1951</span></span>
<span data-ttu-id="8e110-103">Une arborescence d’expression lambda ne peut pas contenir une, entrée/sortie, ou le paramètre ref.</span><span class="sxs-lookup"><span data-stu-id="8e110-103">An expression tree lambda may not contain an in, out, or ref parameter.</span></span>  
  
 <span data-ttu-id="8e110-104">Une arborescence d’expression représente simplement des expressions en tant que structures de données.</span><span class="sxs-lookup"><span data-stu-id="8e110-104">An expression tree just represents expressions as data structures.</span></span> <span data-ttu-id="8e110-105">Il n’existe aucun moyen de représenter des emplacements de mémoire spécifiques, comme cela est nécessaire quand vous passez un paramètre par référence.</span><span class="sxs-lookup"><span data-stu-id="8e110-105">There is no way to represent specific memory locations as is required when you pass a parameter by reference.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8e110-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="8e110-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="8e110-107">La seule option consiste à supprimer le modificateur dans la définition du délégué et passez le paramètre par valeur.</span><span class="sxs-lookup"><span data-stu-id="8e110-107">The only option is to remove the modifier in the delegate definition and pass the parameter by value.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8e110-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="8e110-108">Example</span></span>  
 <span data-ttu-id="8e110-109">L’exemple suivant génère l’erreur CS1951 :</span><span class="sxs-lookup"><span data-stu-id="8e110-109">The following example generates CS1951:</span></span>
