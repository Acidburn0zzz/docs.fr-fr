## <a name="to-correct-this-error"></a><span data-ttu-id="3f457-105">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="3f457-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="3f457-106">Créez et initialisez le tableau avant de créer l’arborescence d’expression.</span><span class="sxs-lookup"><span data-stu-id="3f457-106">Create and initialize the array before creating the expression tree.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3f457-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="3f457-107">Example</span></span>  
 <span data-ttu-id="3f457-108">L’exemple suivant génère l’erreur CS0838 :</span><span class="sxs-lookup"><span data-stu-id="3f457-108">The following example generates CS0838:</span></span>  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;   
  
            return 1;  
        }  
    }  
}  
```
