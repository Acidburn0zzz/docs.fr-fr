 <span data-ttu-id="f9d0f-104">Cette erreur se produit si vous utilisez un attribut IndexerName sur un indexeur correspondant à une implémentation explicite d’une interface.</span><span class="sxs-lookup"><span data-stu-id="f9d0f-104">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="f9d0f-105">Cette erreur peut être évitée grâce à la suppression du nom d’interface de la déclaration de l’indexeur, si possible.</span><span class="sxs-lookup"><span data-stu-id="f9d0f-105">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="f9d0f-106">Pour plus d’informations, consultez [IndexerNameAttribute, classe](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="f9d0f-106">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="f9d0f-107">L’exemple suivant génère l’erreur CS0415 :</span><span class="sxs-lookup"><span data-stu-id="f9d0f-107">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
