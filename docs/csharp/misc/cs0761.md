 <span data-ttu-id="250f5-104">Si une méthode partielle a une implémentation, la contrainte de type générique doit être identique à la contrainte définie sur la signature de méthode.</span><span class="sxs-lookup"><span data-stu-id="250f5-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
