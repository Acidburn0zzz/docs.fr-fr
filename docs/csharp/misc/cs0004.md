# <a name="compiler-error-cs0004"></a>Erreur du compilateur CS0004
Avertissement considéré comme une erreur  
  
 Vous avez effectué la compilation avec l’option du compilateur `/warnaserror` , ce qui force le compilateur à générer les avertissements en tant qu’erreurs.
