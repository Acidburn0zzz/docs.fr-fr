## <a name="example"></a><span data-ttu-id="de358-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="de358-106">Example</span></span>  
 <span data-ttu-id="de358-107">L’exemple suivant génère l’avertissement CS3015.</span><span class="sxs-lookup"><span data-stu-id="de358-107">The following sample generates C3015.</span></span>
