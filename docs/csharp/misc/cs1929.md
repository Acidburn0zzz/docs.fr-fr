## <a name="example"></a><span data-ttu-id="ac74b-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="ac74b-108">Example</span></span>  
 <span data-ttu-id="ac74b-109">Le code suivant génère les erreurs CS1928 et CS1929 :</span><span class="sxs-lookup"><span data-stu-id="ac74b-109">The following code generates CS1928 and CS1929:</span></span>  
  
```  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
