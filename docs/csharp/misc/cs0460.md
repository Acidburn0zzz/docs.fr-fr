# <a name="compiler-error-cs0460"></a><span data-ttu-id="00412-102">Erreur du compilateur CS0460</span><span class="sxs-lookup"><span data-stu-id="00412-102">Compiler Error CS0460</span></span>

## <a name="example"></a><span data-ttu-id="00412-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="00412-106">Example</span></span>  
 <span data-ttu-id="00412-107">L’exemple suivant génère l’erreur CS0460.</span><span class="sxs-lookup"><span data-stu-id="00412-107">The following sample generates CS0460.</span></span>  
  
```  
// CS0460.cs  
// compile with: /target:library  
class BaseClass   
{  
   BaseClass() { }  
}  
  
interface I  
{  
   void F1<T>() where T : BaseClass;  
   void F2<T>() where T : struct;  
   void F3<T>() where T : BaseClass;  
}  
  
class ExpImpl : I  
{  
   void I.F1<T>() where T : BaseClass {}   // CS0460  
   void I.F2<T>() where T : class {}  // CS0460  
}  
```
