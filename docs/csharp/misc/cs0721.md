# <a name="compiler-error-cs0721"></a><span data-ttu-id="ba229-102">Erreur du compilateur CS0721</span><span class="sxs-lookup"><span data-stu-id="ba229-102">Compiler Error CS0721</span></span>
<span data-ttu-id="ba229-103">'type' : les types static ne peuvent pas être utilisés en tant que paramètres</span><span class="sxs-lookup"><span data-stu-id="ba229-103">'type': static types cannot be used as parameters</span></span>  
  
 <span data-ttu-id="ba229-104">Un type static ne peut pas être utilisé comme un paramètre.</span><span class="sxs-lookup"><span data-stu-id="ba229-104">A static type is not meaningful as a parameter.</span></span> <span data-ttu-id="ba229-105">Puisqu’aucune instance de type static ne peut être créée, aucune instance ne pourra être passée en tant que paramètre.</span><span class="sxs-lookup"><span data-stu-id="ba229-105">Since no instances of static types may be created, no instance could ever be passed as a parameter.</span></span>  
  
 <span data-ttu-id="ba229-106">L’exemple suivant génère l’erreur CS0721 :</span><span class="sxs-lookup"><span data-stu-id="ba229-106">The following sample generates CS0721:</span></span>  
  
```csharp  
// CS0721.cs

{  
}

{  
   public void F(SC sc)  // CS0721  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
