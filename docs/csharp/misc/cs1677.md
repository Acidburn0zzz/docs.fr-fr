## <a name="example"></a><span data-ttu-id="f5d1b-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="f5d1b-105">Example</span></span>  
 <span data-ttu-id="f5d1b-106">L’exemple suivant génère l’erreur CS1677 :</span><span class="sxs-lookup"><span data-stu-id="f5d1b-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs

class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
