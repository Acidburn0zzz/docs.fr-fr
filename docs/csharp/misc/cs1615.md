# <a name="compiler-error-cs1615"></a><span data-ttu-id="e78d9-102">Erreur du compilateur CS1615</span><span class="sxs-lookup"><span data-stu-id="e78d9-102">Compiler Error CS1615</span></span>
<span data-ttu-id="e78d9-103">L’argument 'nombre' doit être passé avec le mot clé 'mot_clé'</span><span class="sxs-lookup"><span data-stu-id="e78d9-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="e78d9-104">L’un des mots clés `ref` ou **out** a été utilisé alors que la fonction ne prenait pas de paramètre `ref` ou **out** pour cet argument.</span><span class="sxs-lookup"><span data-stu-id="e78d9-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="e78d9-105">Pour résoudre cette erreur, supprimez le mot clé incorrect et utilisez le mot clé approprié qui correspond à la déclaration de la fonction, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="e78d9-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="e78d9-106">L’exemple suivant génère l’erreur CS1615 :</span><span class="sxs-lookup"><span data-stu-id="e78d9-106">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
