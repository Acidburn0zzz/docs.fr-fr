# <a name="compiler-error-cs0112"></a><span data-ttu-id="6c362-102">Erreur du compilateur CS0112</span><span class="sxs-lookup"><span data-stu-id="6c362-102">Compiler Error CS0112</span></span>
<span data-ttu-id="6c362-103">Un membre static 'function' ne peut pas être marqué comme override, virtual ou abstract</span><span class="sxs-lookup"><span data-stu-id="6c362-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="6c362-104">Toute déclaration de méthode qui utilise le mot clé `override`, **virtual**ou **abstract** ne peut pas aussi utiliser le mot clé **static** .</span><span class="sxs-lookup"><span data-stu-id="6c362-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="6c362-105">Pour plus d’informations, consultez [Méthodes](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="6c362-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="6c362-106">L’exemple suivant génère l’erreur CS0112 :</span><span class="sxs-lookup"><span data-stu-id="6c362-106">The following sample generates CS0112:</span></span>  
  
```  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
