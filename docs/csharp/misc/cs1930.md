## <a name="example"></a><span data-ttu-id="0abf0-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="0abf0-108">Example</span></span>  
 <span data-ttu-id="0abf0-109">L’exemple suivant génère l’erreur CS1930, car l’identificateur `num` est utilisé pour la variable de portée dans la clause `from` et pour la variable de portée ajoutée par la clause `let` .</span><span class="sxs-lookup"><span data-stu-id="0abf0-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```
