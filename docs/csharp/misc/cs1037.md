# Erreur du compilateur CS1037
Opérateur surchargeable attendu  
  
 Lors de la spécification d’un commentaire avec [\/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md), le compilateur a rencontré un lien non valide.
