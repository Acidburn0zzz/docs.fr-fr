## <a name="example"></a><span data-ttu-id="f398c-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="f398c-107">Example</span></span>  
 <span data-ttu-id="f398c-108">L’exemple suivant génère l’erreur CS1706.</span><span class="sxs-lookup"><span data-stu-id="f398c-108">The following example generates CS1706.</span></span>

```  
delegate void MyDelegate();  
class MyAttribute : Attribute  
{  
    public MyAttribute(MyDelegate d) { }  
}  
  
// Anonymous Method in Attribute declaration is not allowed.  
[MyAttribute(delegate{/* anonymous Method in Attribute declaration */})]  // CS1706  
class Program  
{  
}  
```
