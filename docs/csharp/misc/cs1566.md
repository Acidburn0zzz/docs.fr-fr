# <a name="compiler-error-cs1566"></a>Erreur du compilateur CS1566
Erreur lors de la lecture du fichier de ressources 'fichier' — 'raison'  
  
 Le compilateur a rencontré une erreur avec le nom de fichier passé à l’option de compilateur [/resource](../../csharp/language-reference/compiler-options/resource-compiler-option.md) .
