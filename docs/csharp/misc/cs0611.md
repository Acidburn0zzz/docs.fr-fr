# <a name="compiler-error-cs0611"></a><span data-ttu-id="71bd2-102">Erreur du compilateur CS0611</span><span class="sxs-lookup"><span data-stu-id="71bd2-102">Compiler Error CS0611</span></span>

 <span data-ttu-id="71bd2-104">Certains types ne peuvent pas être appliqués à un tableau.</span><span class="sxs-lookup"><span data-stu-id="71bd2-104">There are some types that cannot be used as the type of an array.</span></span> <span data-ttu-id="71bd2-105">Ces types incluent **System.TypedReference** et **System.ArgIterator**.</span><span class="sxs-lookup"><span data-stu-id="71bd2-105">These types include **System.TypedReference** and **System.ArgIterator**.</span></span>  
  
 <span data-ttu-id="71bd2-106">L’exemple suivant génère l’erreur CS0611 à la suite de l’utilisation de **System.TypedReference** en tant qu’élément de tableau :</span><span class="sxs-lookup"><span data-stu-id="71bd2-106">The following sample generates CS0611 as a result of using **System.TypedReference** as an array element:</span></span>  
  
```csharp

{  
   public static void Main()  
   {  
      System.TypedReference[] ao = new System.TypedReference [10];   // CS0611  
      // try the following line instead  
      // int[] ao = new int[10];  
   }  
}  
```
