 <span data-ttu-id="22a9e-104">L’instruction yield n’est pas autorisée à partir du corps d’une clause catch.</span><span class="sxs-lookup"><span data-stu-id="22a9e-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="22a9e-105">Pour éviter cette erreur, placez l’instruction yield en dehors du corps de la clause catch.</span><span class="sxs-lookup"><span data-stu-id="22a9e-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>

```  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
