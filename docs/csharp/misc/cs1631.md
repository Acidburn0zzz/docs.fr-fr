# <a name="compiler-error-cs1631"></a><span data-ttu-id="78de5-102">Erreur du compilateur CS1631</span><span class="sxs-lookup"><span data-stu-id="78de5-102">Compiler Error CS1631</span></span>
<span data-ttu-id="78de5-103">Impossible de générer une valeur dans le corps d'une clause catch</span><span class="sxs-lookup"><span data-stu-id="78de5-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="78de5-104">L’instruction yield n’est pas autorisée à partir du corps d’une clause catch.</span><span class="sxs-lookup"><span data-stu-id="78de5-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="78de5-105">Pour éviter cette erreur, placez l’instruction yield en dehors du corps de la clause catch.</span><span class="sxs-lookup"><span data-stu-id="78de5-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="78de5-106">L’exemple suivant génère l’erreur CS1631 :</span><span class="sxs-lookup"><span data-stu-id="78de5-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
