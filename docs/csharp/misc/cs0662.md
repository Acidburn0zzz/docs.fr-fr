# <a name="compiler-error-cs0662"></a><span data-ttu-id="e8190-102">Erreur du compilateur CS0662</span><span class="sxs-lookup"><span data-stu-id="e8190-102">Compiler Error CS0662</span></span>

<span data-ttu-id="e8190-103">'method' ne peut pas spécifier uniquement un attribut Out sur un paramètre ref.</span><span class="sxs-lookup"><span data-stu-id="e8190-103">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="e8190-104">Utilisez les deux attributs In et Out ou aucun des deux.</span><span class="sxs-lookup"><span data-stu-id="e8190-104">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="e8190-105">Une méthode d’interface a un paramètre qui utilise [ref](../../csharp/language-reference/keywords/ref.md) avec uniquement l’attribut [Out](xref:System.Runtime.InteropServices.OutAttribute) .</span><span class="sxs-lookup"><span data-stu-id="e8190-105">An interface method has a parameter that uses [ref](../../csharp/language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="e8190-106">Un paramètre `ref` qui utilise l’attribut **Out** doit également utiliser l’attribut [In](xref:System.Runtime.InteropServices.InAttribute) .</span><span class="sxs-lookup"><span data-stu-id="e8190-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="e8190-107">L’exemple suivant génère l’erreur CS0662 :</span><span class="sxs-lookup"><span data-stu-id="e8190-107">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
