# <a name="compiler-error-cs0743"></a><span data-ttu-id="11b30-102">Erreur du compilateur CS0743</span><span class="sxs-lookup"><span data-stu-id="11b30-102">Compiler Error CS0743</span></span>
<span data-ttu-id="11b30-103">Mot clé contextuel 'on' attendu</span><span class="sxs-lookup"><span data-stu-id="11b30-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="11b30-104">Le modèle pour une clause `join` est `join`...`in`...`on`...`equals`, comme illustré dans cet exemple :</span><span class="sxs-lookup"><span data-stu-id="11b30-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y

## <a name="example"></a><span data-ttu-id="11b30-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="11b30-107">Example</span></span>  
 <span data-ttu-id="11b30-108">Le code suivant génère l’erreur CS0743 :</span><span class="sxs-lookup"><span data-stu-id="11b30-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
