## <a name="example"></a><span data-ttu-id="295ea-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="295ea-105">Example</span></span>  
 <span data-ttu-id="295ea-106">L’exemple suivant génère l’erreur CS0135 :</span><span class="sxs-lookup"><span data-stu-id="295ea-106">The following sample generates CS0135:</span></span>  
  
```csharp
// CS0135.cs  
public class MyClass2  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
      {  
         int i = 4;  
         i++;  
      }  
      i = 0;   // CS0135  
   }  
}  
```
