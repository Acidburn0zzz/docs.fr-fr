# <a name="compiler-warning-level-3-cs0642"></a><span data-ttu-id="967a7-102">Avertissement du compilateur (niveau 3) CS0642</span><span class="sxs-lookup"><span data-stu-id="967a7-102">Compiler Warning (level 3) CS0642</span></span>
<span data-ttu-id="967a7-103">Possibilité d'instruction vide erronée</span><span class="sxs-lookup"><span data-stu-id="967a7-103">Possible mistaken empty statement</span></span>  
  
 <span data-ttu-id="967a7-104">Si un point-virgule se trouve après une instruction conditionnelle, votre code peut ne pas s’exécuter comme attendu.</span><span class="sxs-lookup"><span data-stu-id="967a7-104">A semicolon after a conditional statement may cause your code to execute differently than intended.</span></span>  
  
 <span data-ttu-id="967a7-105">Vous pouvez utiliser **/nowarn** option du compilateur ou `#pragmas warning` pour désactiver cet avertissement ; consultez [/nowarn (Options du compilateur c#)](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) ou [#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) pour plus d’informations.</span><span class="sxs-lookup"><span data-stu-id="967a7-105">You can use **/nowarn** compiler option or `#pragmas warning` to disable this warning; see [/nowarn (C# Compiler Options)](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) or [#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) for more information.</span></span>  
  
 <span data-ttu-id="967a7-106">L’exemple suivant génère l’erreur CS0642 :</span><span class="sxs-lookup"><span data-stu-id="967a7-106">The following sample generates CS0642:</span></span>  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
