# <a name="compiler-error-cs1559"></a>Erreur du compilateur CS1559
Impossible d’utiliser 'object' pour la méthode Main, car il est importé  
  
 Une classe non valide a été spécifiée pour le [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) option du compilateur ; la classe ne peut pas être utilisée comme un emplacement pour le [Main](../../csharp/programming-guide/main-and-command-args/index.md) (méthode).
