 <span data-ttu-id="0f338-106">L’exemple suivant génère l’erreur CS1641 :</span><span class="sxs-lookup"><span data-stu-id="0f338-106">The following sample generates CS1641:</span></span>  
  
```csharp

   fixed int [] a;  // CS1641

}
```
