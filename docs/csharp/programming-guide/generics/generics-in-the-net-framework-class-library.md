# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>Génériques dans la bibliothèque de classes .NET Framework (Guide de programmation C#)
La version 2.0 de la bibliothèque de classes .NET Framework fournit un nouvel espace de noms, <xref:System.Collections.Generic>, qui inclut plusieurs classes de collection générique prêtes à l’emploi, ainsi que l’interface qui leur est associée. D’autres espaces de noms, comme <xref:System>, fournissent également de nouvelles interfaces génériques telles que <xref:System.IComparable%601>. Ces classes et ces interfaces sont de type sécurisé et sont plus efficaces que les classes de collection non générique fournies dans les versions antérieures du .NET Framework. Avant de concevoir et d’implémenter vos propres classes de collection personnalisées, envisagez d’utiliser ou de dériver une classe d’une des classes fournies dans la bibliothèque de classes .NET Framework.
