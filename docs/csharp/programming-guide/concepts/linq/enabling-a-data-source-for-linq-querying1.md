-   <span data-ttu-id="0bd83-111">Création d’un fournisseur pour votre source de données qui exploite une technologie [!INCLUDE[vbteclinq](~/includes/vbteclinq-md.md)] existante.</span><span class="sxs-lookup"><span data-stu-id="0bd83-111">Creating a provider for your data source that takes advantage of an existing [!INCLUDE[vbteclinq](~/includes/vbteclinq-md.md)] technology.</span></span> <span data-ttu-id="0bd83-112">Ce type de fournisseur permettrait non seulement l'interrogation, mais également l'insertion, la mise à jour et la suppression d'opérations et le mappage pour des types définis par l'utilisateur.</span><span class="sxs-lookup"><span data-stu-id="0bd83-112">Such a provider would enable not only querying, but also insert, update, and delete operations and mapping for user-defined types.</span></span>
